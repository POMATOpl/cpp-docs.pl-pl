# <a name="platformenum-class"></a>Platform:: enum, Klasa

Klasa wartości, która reprezentuje zestaw nazwanych stałych.

## <a name="syntax"></a>Składnia

```cpp
public class Enum
```

### <a name="members"></a>Elementy członkowskie

Klasa enum dziedziczy metody Equals (), GetHashCode () i ToString () z [klasy platform:: Object](../cppcx/platform-object-class.md).

### <a name="remarks"></a>Uwagi

Użyj słowa kluczowego [public class enum](../extensions/enum-class-cpp-component-extensions.md) do tworzenia wyliczeń. Nie używaj jawnie typu platform:: enum. Aby uzyskać więcej informacji, zobacz [wyliczenia](../cppcx/enums-c-cx.md).

### <a name="requirements"></a>Wymagania

**Minimalny obsługiwany klient:** System Windows 8

**Minimalny obsługiwany serwer:** System Windows Server 2012

**Przestrzeń nazw:** Platformach

**Metadane:** obiekt platform. winmd
