# <a name="platformenum-class"></a>Platform::Enum, klasa

Klasa wartość, która reprezentuje zestaw nazwanych stałych.

## <a name="syntax"></a>Składnia

```cpp
public class Enum
```

### <a name="members"></a>Elementy członkowskie

Klasa wyliczeniowa dziedziczy metodę Equals, element GetHashCode() i ToString() metody z [Platform::Object, klasa](../cppcx/platform-object-class.md).

### <a name="remarks"></a>Uwagi

Użyj [klasy publicznym typie wyliczeniowym](../extensions/enum-class-cpp-component-extensions.md) — słowo kluczowe do tworzenia wyliczenia. Nie należy używać typu Platform::Enum jawnie. Aby uzyskać więcej informacji, zobacz [wyliczenia](../cppcx/enums-c-cx.md).

### <a name="requirements"></a>Wymagania

**Minimalna obsługiwana klienta:** Windows 8

**Minimalna obsługiwana serwera:** Windows Server 2012

**Namespace:** Platforma

**Metadane:** platform.winmd
