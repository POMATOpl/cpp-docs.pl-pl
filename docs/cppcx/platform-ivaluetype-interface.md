# <a name="platformivaluetype-interface"></a>Platform:: IValueType, interfejs

`Platform::IValueType` jest interfejsem infrastruktury, który jest implementowany przez klasy wartości i struktury wartości. Nie można jawnie używać w kodzie.

## <a name="syntax"></a>Składnia

```cpp
interface class IValueType
```
