# <a name="interoperating-with-other-languages-ccx"></a>Współdziałanie z innymi językami (C++/CX)

W tej części dokumentacji opisano, jak używać języka C++/CX do tworzenia składników środowisko wykonawcze systemu Windows, które mogą być używane przez inne języki programowania i biblioteki. Można również tworzyć składniki, których nie można używać we wszystkich językach. W tej sekcji opisano różne aspekty, które należy wziąć pod uwagę, gdy aplikacja C++/CX współpracuje ze składnikami, które są zapisywane przy użyciu języka JavaScript, .NET Framework języku zarządzanym lub biblioteki szablonów środowisko wykonawcze systemu Windows C++.
