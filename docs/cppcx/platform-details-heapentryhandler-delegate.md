# <a name="platformdetailsheapentryhandler-delegate"></a>Delegat platform::details::HeapEntryHandler
Ten delegat jest przeznaczony tylko do użytku wewnętrznego i nie mają być używane do tworzenia aplikacji.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
  
delegate HeapEntryHandler;  
```  
  
### <a name="remarks"></a>Uwagi  
 Ta klasa jest dostępna wyłącznie dla kompletności, ponieważ mogą być kontrolowane z przeglądarkami.
