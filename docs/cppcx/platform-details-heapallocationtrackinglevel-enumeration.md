# <a name="platformdetailsheapallocationtrackinglevel-enumeration"></a>Platform::details::HeapAllocationTrackingLevel — wyliczenie
To wyliczenie jest przeznaczony tylko do użytku wewnętrznego i nie mają być używane do tworzenia aplikacji.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
  
enumm class HeapAllocationTrackingLevel;  
```  
  
### <a name="remarks"></a>Uwagi  
 To wyliczenie jest dostępne wyłącznie w celu kompletności, ponieważ mogą być kontrolowane z przeglądarkami.
