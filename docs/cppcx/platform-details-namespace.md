# <a name="platformdetails-namespace"></a>Namespace platform::details
Ta przestrzeń nazw jest przeznaczony tylko do użytku wewnętrznego i nie mają być używane do tworzenia aplikacji.  
  
## <a name="syntax"></a>Składnia  
  
```cpp

   namespace Details {  
}}  
```
