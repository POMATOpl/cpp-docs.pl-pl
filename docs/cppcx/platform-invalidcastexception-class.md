# <a name="platforminvalidcastexception-class"></a>Platform::InvalidCastException, klasa

Element zgłaszany, gdy rzutowania lub konwersji jawnej jest nieprawidłowy.

## <a name="syntax"></a>Składnia

```cpp
public ref class InvalidCastException : COMException,    IException,    IPrintable,    IEquatable
```

Aby uzyskać więcej informacji, zobacz [COMException](../cppcx/platform-comexception-class.md) klasy.
