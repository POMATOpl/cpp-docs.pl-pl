# <a name="namespaces-reference-ccx"></a>Dokumentacja przestrzeni nazw (C++/CX)

Artykuły w tej sekcji dokumentacji opisują przestrzenie nazw obsługujące kompilator języka C++/CX.

## <a name="compiler-supplied-namespaces"></a>Przestrzenie nazw dostarczone przez kompilator

Aby uprościć kodowanie programów przeznaczonych dla środowisko wykonawcze systemu Windows, kompilator C++/CX i jego pomocnicze pliki nagłówkowe zawierają przestrzenie nazw, które definiują szeroki zakres typów. Przestrzenie nazw definiują wbudowane typy liczbowe; ciągi, tablice i kolekcje; Wyjątki C++ reprezentujące błędy środowisko wykonawcze systemu Windows; i ulepszenia specyficzne dla języka dla standardowych typów środowisko wykonawcze systemu Windows.
