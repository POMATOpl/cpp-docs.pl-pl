# <a name="namespaces-reference-ccx"></a>Odwołania do przestrzeniC++nazw (/CX)

Artykuły w tej sekcji dokumentacji opisują przestrzenie nazw, które obsługują kompilator dla C++/CX.

## <a name="compiler-supplied-namespaces"></a>Przestrzenie nazw dostarczone przez kompilator

Aby uprościć kodowanie programów przeznaczonych dla środowisko wykonawcze systemu Windows, kompilator C++/CX i jego pomocnicze pliki nagłówkowe udostępniają przestrzenie nazw, które definiują szeroką gamę typów. Przestrzenie nazw definiują wbudowane typy liczbowe; ciągi, tablice i kolekcje; C++ wyjątki reprezentujące błędy środowisko wykonawcze systemu Windows; i ulepszenia specyficzne dla języka dla standardowych typów środowisko wykonawcze systemu Windows.
