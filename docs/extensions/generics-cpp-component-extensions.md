# <a name="generics--ccli-and-ccx"></a>Typy ogólne (C++/CLI i C++/CX)

Typy ogólne są sparametryzowanemi typami i metodami. W tej sekcji dowiesz się, które funkcje ogólne są obsługiwane przez środowisko wykonawcze systemu Windows i środowisko uruchomieniowe języka wspólnego, które są obsługiwane przez środowisko uruchomieniowe języka wspólnego. Ponadto Dowiedz się, jak tworzyć własne metody rodzajowe i typy w C++/CLI oraz jak używać typów ogólnych utworzonych w języku .NET Framework w C++/CLI. Na koniec Ta sekcja zawiera porównanie typów ogólnych i C++ szablonów.
