# <a name="range-of-integer-values"></a>Zakres wartości całkowitych
**ANSI 3.1.2.5** oświadczenia i zestawy wartości różnych typów całkowitych  
  
 Liczby całkowite zawiera 32-bitowy (cztery w bajtach). Oznaczone liczby całkowite są zapisywane w kodzie dopełnień do dwóch. Najbardziej znaczący bit utrzymuje znak: 1 dla wartości ujemnych, 0 dla wartości dodatnich i zera. Poniżej przedstawiono wartości:  
  
|Typ|Wartości minimalną i maksymalną|  
|----------|-------------------------|  
|**short bez znaku**|od 0 do 65535|  
|`signed short`|-32768 do 32767|  
|`unsigned long`|0 do 4294967295.|  
|**podpisana długo**|-2147483648 do 2147483647.|
