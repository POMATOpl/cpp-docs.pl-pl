# <a name="bits-per-character"></a>Bity na znak

**2.2.4.2.1 ANSI** Liczba bitów w znaku

Liczba bitów w znaku jest reprezentowana przez **CHAR_BIT** manifestu. LIMITY. Plik H definiuje **CHAR_BIT** jako 8.
