# <a name="bits-per-character"></a>Bity na znak

**ANSI 2.2.4.2.1** liczbę bitów w znaku

Liczba bitów w znaku jest reprezentowane przez stałe manifestu **CHAR_BIT**. LIMITY. Plik H definiuje **CHAR_BIT** jako 8.
