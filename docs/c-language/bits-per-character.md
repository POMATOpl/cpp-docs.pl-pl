# <a name="bits-per-character"></a>Bity na znak
**ANSI 2.2.4.2.1** liczba bitów w znaku  
  
 Liczba bitów znak jest reprezentowany przez manifestu stała **char_bit —**. LIMITY. Określa plik H **char_bit —** jako 8.
