# <a name="storage-class-specifiers-for-internal-level-declarations"></a>Specyfikatory klasy magazynowania dla deklaracji na poziomie wewnętrznym

Można użyć dowolnego z czterech *`storage-class-specifier`* terminali dla deklaracji zmiennych na poziomie wewnętrznym. W przypadku pominięcia *`storage-class-specifier`* z takiej deklaracji domyślną klasą magazynu jest **`auto`** . W związku z tym słowo kluczowe **`auto`** jest rzadko widoczne w programie C.
