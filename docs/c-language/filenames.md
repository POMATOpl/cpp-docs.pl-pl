# <a name="filenames"></a>Nazwy plików

**4.9.3 ANSI** Reguły umożliwiające redagowanie prawidłowych nazw plików

Specyfikacja pliku może zawierać opcjonalną literę dysku (zawsze dwukropek), serię opcjonalnych nazw katalogów (oddzielonych ukośnikami odwrotnymi) i nazwę pliku.

Aby uzyskać więcej informacji, zobacz [nazywanie pliku](/windows/win32/FileIO/naming-a-file) , aby uzyskać więcej informacji.
