# <a name="default-date-and-time"></a>Domyślne data i godzina

**3.8.8 ANSI** Definicje dla `__DATE__` i, w których `__TIME__` odpowiednio, Data i godzina tłumaczenia są niedostępne

Jeśli system operacyjny nie zawiera daty i godziny tłumaczenia, wartości domyślne dla `__DATE__` i `__TIME__` są `May 03 1957` i `17:00:00` .
