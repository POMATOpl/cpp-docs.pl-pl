# <a name="significant-characters-with-external-linkage"></a>Znaki znaczące z zewnętrznym połączeniem
**ANSI 3.1.2** liczba znaki znaczące z zewnętrznym powiązaniem  
  
 Identyfikatory zadeklarowany `extern` w programach skompilowane z Microsoft C są istotne dla 247 znaków. Można zmodyfikować to ustawienie domyślne mniejszą wartość przy użyciu /H (Ograniczaj długość nazw zewnętrznych) opcja.
