# <a name="character-sets"></a>Zestawy znaków

 Zestaw znaków źródła i zestaw znaków wykonania, zawierają znaki ASCII wymienione w poniższej tabeli. Sekwencje unikowe są także wyświetlane w tabeli.

|\f|Wysuw strony|12|  
|\n|Nowy wiersz|10|  
|\r|Powrót karetki|13|  
|\t|Tabulator poziomy|9|  
|\v|Tabulator pionowy|11|  
|\\"|Cudzysłowu|34|
