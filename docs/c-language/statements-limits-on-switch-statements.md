# <a name="statements-limits-on-switch-statements"></a>Instrukcje: limity w instrukcjach przełącznika

**3.6.4.2 ANSI** Maksymalna liczba wartości **wielkości liter** w `switch` instrukcji

Firma Microsoft C nie ogranicza liczby wartości **wielkości liter** w instrukcji **Switch** . Liczba jest ograniczona tylko przez dostępną pamięć.
