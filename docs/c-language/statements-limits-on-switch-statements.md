# <a name="statements-limits-on-switch-statements"></a>Instrukcje: limity w instrukcjach przełącznika

**3.6.4.2 ANSI** Maksymalna liczba **`case`** wartości w **`switch`** instrukcji

Microsoft C nie ogranicza liczby **`case`** wartości w **`switch`** instrukcji. Liczba jest ograniczona tylko przez dostępną pamięć.
