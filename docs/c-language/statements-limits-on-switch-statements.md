# <a name="statements-limits-on-switch-statements"></a>Instrukcje: Limity w instrukcjach przełącznika

**ANSI 3.6.4.2** maksymalną liczbę **przypadek** wartości w `switch` — instrukcja

Microsoft C nie ogranicza liczby **przypadek** wartości w **Przełącz** instrukcji. Liczba jest ograniczona jedynie ilością dostępnej pamięci.
