# <a name="null-macro"></a>NULL — makro

**ANSI 4.1.5** Stała wskaźnikowa null, do której zostanie rozwinięte makro o wartości NULL.

Kilka plików dołączanych definiuje makro o wartości NULL jako `((void *)0)` .
