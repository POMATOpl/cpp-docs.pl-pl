**ANSI 4.9.6.2** dane wejściowe dla **%p** konwersji w `fscanf` — funkcja  
  
 Gdy **%p** format znak jest określony, `fscanf` funkcja konwertuje wskaźniki z wartości szesnastkowe ASCII poprawny adres.
