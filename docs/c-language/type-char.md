# <a name="type-char"></a>Typ char
`char` Typ jest używany do przechowywania wartość całkowita zestawu znaków reprezentacja elementu członkowskiego. Wartość całkowita się kodu ASCII odpowiadający określony znak.  
  
 **Microsoft Specific**  
  
 Znak wartości typu `unsigned char` zakresu od 0 do 0xFF szesnastkową. A **podpisany char** zasięg 0x80 do 0x7F. Te zakresy tłumaczenie odpowiednio 0 do 255 dziesiętnych i -128 do + 127 decimal. /J — opcja kompilatora zmienia domyślne **podpisany** do `unsigned`.
