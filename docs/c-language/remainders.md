# <a name="remainders"></a>Pozostałe części

**3.3.5 ANSI** Znak reszty w oddziale liczb całkowitych

Znak reszty jest taki sam jak znak dywidendy. Przykład:

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
