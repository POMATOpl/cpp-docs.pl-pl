# <a name="remainders"></a>Pozostałe części
**ANSI 3.3.5** znak pozostałej na dzielenie liczby całkowitej  
  
 Znak pozostałej jest taka sama, jak znak dzielna. Na przykład  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
