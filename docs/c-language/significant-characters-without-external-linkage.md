# <a name="significant-characters-without-external-linkage"></a>Znaki znaczące bez zewnętrznego połączenia
**ANSI 3.1.2** liczba znaki znaczące bez zewnętrznego połączenia  
  
 Identyfikatory są istotne dla 247 znaków. Kompilator nie ogranicza liczbę znaków, które można użyć w identyfikatorze; po prostu ignoruje wszystkie znaki poza limit.
