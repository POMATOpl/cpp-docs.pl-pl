# <a name="fmod-function"></a>fmod — funkcja
**ANSI 4.5.6.4** czy występuje błąd domeny lub zero jest zwracany, gdy `fmod` funkcja ma drugi argument o wartości zero  
  
 Gdy `fmod` funkcja ma drugi argument o wartości zero, funkcja zwraca wartość zero.
