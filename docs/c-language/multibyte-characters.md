# <a name="multibyte-characters"></a>Znaki wielobajtowe
**ANSI 2.2.1.2** przesunięcia stanów znaków wielobajtowych  
  
 Znaki wielobajtowe są używane przez niektóre implementacje, łącznie z Microsoft C do reprezentowania znaków obcego języka nie jest reprezentowany w podstawowy zestaw znaków. Jednak Microsoft C nie obsługuje żadnych kodowanie zależne od stanu. W związku z tym nie istnieją żadne stany shift. Zobacz [wielobajtowe i dwubajtowe znaki](../c-language/multibyte-and-wide-characters.md) Aby uzyskać więcej informacji.
