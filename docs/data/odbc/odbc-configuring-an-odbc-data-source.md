# <a name="odbc-configuring-an-odbc-data-source"></a>ODBC: konfigurowanie źródła danych ODBC

Aby użyć [źródła danych](../../data/odbc/data-source-odbc.md) z opracowaną aplikacją, należy go skonfigurować za pomocą administratora ODBC. Odbc Administrator śledzi dostępne źródła danych i ich informacje o połączeniu w rejestrze systemu Windows. Użyj administratora ODBC, aby dodawać, modyfikować i usuwać źródła danych w oknie dialogowym **Źródła danych** oraz dodawać i usuwać sterowniki ODBC.

> [!NOTE]
> Te informacje mają zastosowanie podczas korzystania z klas obiektu dostępu do danych MFC (DAO) dla dostępu ODBC i podczas korzystania z klas MFC ODBC.

Administrator ODBC jest automatycznie instalowany z obsługą bazy danych biblioteki Microsoft Foundation Classes (MFC). Aby uzyskać więcej informacji na temat programu Administrator ODBC, zobacz [Administrator ODBC](../../data/odbc/odbc-administrator.md) i system pomocy referencyjnej interfejsu API online.

Aby uzyskać informacje dotyczące sposobu pisania programów instalacyjnych i administracyjnych ODBC dla aplikacji bazy danych MFC,[uwaga techniczna 48](../../mfc/tn048-writing-odbc-setup-and-administration-programs.md).
