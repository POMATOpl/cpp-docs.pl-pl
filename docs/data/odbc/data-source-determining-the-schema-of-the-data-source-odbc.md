# <a name="data-source-determining-the-schema-of-the-data-source-odbc"></a>Źródło danych: określanie schematu źródła danych (ODBC)

Ten temat dotyczy klas MFC ODBC.

Aby skonfigurować składowe danych w `CRecordset` obiektach, należy znać schemat źródła danych, z którym jest nawiązywane połączenie. Określenie schematu źródła danych wymaga uzyskania listy tabel w źródle danych, listy kolumn w każdej tabeli, typu danych każdej kolumny oraz istnienia wszelkich indeksów w programie.
