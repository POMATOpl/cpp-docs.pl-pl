# <a name="work-with-odbc-and-drivers"></a>Praca z ODBC i sterownikami

ODBC pozwala pisać aplikacje, które nie są zależne od tylko jeden system zarządzania bazami danych (DBMS). Można napisać jedną aplikację, która może działać na wiele systemów DBMS, dla których użytkownicy mają odpowiedni sterownik ODBC specyficznych dla systemu DBMS.
