# <a name="work-with-odbc-and-drivers"></a>Praca z ODBC i sterownikami

ODBC pozwala pisać aplikacje, które nie są zależne od tylko jednego systemu zarządzania bazami danych (DBMS). Można napisać jedną aplikację, która może działać na wielu systemach DBMS, dla których użytkownicy mają prawidłowy sterownik ODBC specyficzny dla systemu DBMS.
