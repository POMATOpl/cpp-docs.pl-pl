# <a name="working-with-documents-and-views"></a>Praca z dokumentami i widokami

Biblioteka Microsoft Foundation Classes (MFC) opiera się na architekturze dokumentów/widoków dla wielu funkcji. Zazwyczaj dokument przechowuje dane, a widok wyświetla go w obszarze klienta okna ramki i zarządza interakcją użytkownika z danymi. Widok komunikuje się z dokumentem, aby uzyskać i zaktualizować dane. Klas baz danych można używać ze strukturą lub bez niej.

Aby uzyskać więcej informacji o używaniu klas baz danych w strukturze, zobacz [MFC: Using Database classes with Documents and viewss](../../data/mfc-using-database-classes-with-documents-and-views.md).

Domyślnie Kreator aplikacji MFC tworzy aplikację szkieletową bez obsługi bazy danych. Można jednak wybrać opcje, aby uwzględnić minimalną obsługę bazy danych lub pełną obsługę opartą na formularzach. Aby uzyskać więcej informacji na temat opcji Kreatora aplikacji, zobacz [Obsługa bazy danych, Kreator aplikacji MFC](../../mfc/reference/database-support-mfc-application-wizard.md).

Można również używać klas baz danych bez korzystania z pełnej architektury dokumentu/widoku. Aby uzyskać więcej informacji, zobacz [MFC: używanie klas baz danych bez dokumentów i widoków](../../data/mfc-using-database-classes-without-documents-and-views.md).
