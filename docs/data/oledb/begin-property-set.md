# <a name="beginpropertyset"></a>BEGIN_PROPERTY_SET
Znaczniki ustawione na początku właściwości we właściwości ustaw mapy.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
BEGIN_PROPERTY_SET(  
guid   
)  
  
```  
  
#### <a name="parameters"></a>Parametry  
 `guid`  
 [in] Właściwość identyfikatora GUID.
