# <a name="beginparammap"></a>BEGIN_PARAM_MAP
Oznacza początek wpisów map parametru.  
  
## <a name="syntax"></a>Składnia  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>Parametry  
 *x*  
 [in] Nazwa klasy rekordów użytkowników.  
  
## <a name="remarks"></a>Uwagi  
 Parametry są używane przez [polecenia](https://msdn.microsoft.com/en-us/library/ms724608.aspx).

 Zobacz przykład [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md) makra.
