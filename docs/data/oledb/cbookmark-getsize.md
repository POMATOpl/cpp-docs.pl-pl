# <a name="cbookmarkgetsize"></a>CBookmark::GetSize
Pobiera rozmiar buforu zakładki.  
  
## <a name="syntax"></a>Składnia  
  
```cpp
virtual DBLENGTH GetSize() const throw();  
  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 Rozmiar buforu w bajtach.
