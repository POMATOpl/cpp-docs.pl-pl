# <a name="cnomultipleresults-class"></a>CNoMultipleResults — Klasa

Służy jako argument szablonu (*TMultiple*) do [CCommand](../../data/oledb/ccommand-class.md) , aby utworzyć zoptymalizowane polecenie, które obsługuje pojedynczy zestaw wyników.

## <a name="syntax"></a>Składnia

```cpp
class CNoMultipleResults
```

## <a name="remarks"></a>Uwagi

Jeśli chcesz, aby polecenie obsługiwało wiele zestawów wyników, zamiast tego użyj [CMultipleResults —](../../data/oledb/cmultipleresults-class.md) .

## <a name="requirements"></a>Wymagania

**Nagłówek:** atldbcli. h
