# <a name="endcolumnmap"></a>END_COLUMN_MAP
Oznacza koniec wpisów map kolumny.  
  
## <a name="syntax"></a>Składnia  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Jest ona używana z jedną metodę dostępu w zestawie wierszy. `BEGIN_COLUMN_MAP` Makro zostało zakończone z `END_COLUMN_MAP` makra.
