# <a name="cdbpropidsetaddpropertyid"></a>CDBPropIDSet::AddPropertyID
Dodaje identyfikator właściwości do zestawu właściwości Identyfikatora.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### <a name="parameters"></a>Parametry  
 `propid`  
 [in] Ustaw identyfikator właściwości, które mają zostać dodane do Identyfikatora właściwości.

 **Nagłówek:** atldbcli.h
