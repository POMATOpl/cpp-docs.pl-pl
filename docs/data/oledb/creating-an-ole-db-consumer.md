# <a name="creating-an-ole-db-consumer"></a>Tworzenie konsumenta OLE DB

::: moniker range="vs-2019"

Kreator OLE DB konsumenta ATL nie jest dostępne w programie Visual Studio 2019 r i nowszych wersjach. Można nadal ręcznie dodawać funkcje. Aby uzyskać więcej informacji, zobacz [tworzenie konsumenta bez przy użyciu kreatora](creating-a-consumer-without-using-a-wizard.md).

::: moniker-end

Można utworzyć konsumenta OLE DB szablony z lub bez **OLE DB Kreator konsumenta ATL**, ale przy użyciu kreatora jest to zalecana metoda. Użytkownik może działać w każdej aplikacji, która obsługuje ATL.
