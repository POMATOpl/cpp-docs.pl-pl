# <a name="creating-an-ole-db-consumer"></a>Tworzenie konsumenta OLE DB

::: moniker range="msvc-160"

Kreator użytkownika ATL OLE DB nie jest dostępny w programie Visual Studio 2019 i nowszych. Można nadal ręcznie dodawać funkcje. Aby uzyskać więcej informacji, zobacz [Tworzenie klienta bez korzystania z Kreatora](creating-a-consumer-without-using-a-wizard.md).

::: moniker-end

::: moniker range="<=msvc-150"

Używając kreatora programu **ATL OLE DB** , można utworzyć klienta szablonów OLE DB z użyciem kreatora lub bez niego. Konsument może działać w dowolnej aplikacji, która obsługuje ATL.

W tej sekcji omówiono następujące tematy:

- [Tworzenie konsumenta OLE DB przy użyciu kreatora](../../data/oledb/creating-an-ole-db-consumer-using-a-wizard.md)

- [Tworzenie klienta OLE DB bez użycia Kreatora](../../data/oledb/creating-a-consumer-without-using-a-wizard.md)

::: moniker-end
