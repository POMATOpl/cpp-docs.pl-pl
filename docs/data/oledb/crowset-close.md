# <a name="crowsetclose"></a>CRowset::Close
Zwalnia wierszy i obecnie [IRowset](https://msdn.microsoft.com/en-us/library/ms720986.aspx) interfejsu.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
void Close( ) throw( );  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Ta metoda zwalnia wszystkie wiersze obecnie w zestawie wierszy.

 **Nagłówek:** atldbcli.h
