# <a name="ole-db-programming"></a>Programowanie OLE DB

Szablony OLE DB są szablonami języka C++, które ułatwiają korzystanie z wysokiej wydajności OLE DB bazy danych, zapewniając klasy, które implementują wiele powszechnie używanych interfejsów OLE DB. Ta biblioteka szablonów jest podzielona na szablony klientów i szablony dostawców.

Visual C++ również ma obsługę kreatora do tworzenia OLE DB aplikacji początkowych.
