# <a name="ole-db-programming"></a>Programowanie OLE DB

Szablony OLE DB są C++ szablonami, które ułatwiają korzystanie z wysokiej wydajności OLE DBj bazy danych, zapewniając klasy, które implementują wiele powszechnie używanych interfejsów OLE DB. Ta biblioteka szablonów jest podzielona na szablony klientów i szablony dostawców.

Wizualizacja C++ ma również obsługę kreatora do tworzenia OLE DB aplikacji początkowych.
