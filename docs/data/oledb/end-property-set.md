# <a name="endpropertyset"></a>END_PROPERTY_SET

## <a name="syntax"></a>Składnia  
  
```cpp
END_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>Parametry  
 `guid`  
 [in] Właściwość identyfikatora GUID.
