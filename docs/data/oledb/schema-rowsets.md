# <a name="schema-rowsets"></a>Zestawy wierszy schematu

Schemat (lub metadane) to wszelkie informacje o strukturze lub organizacji bazy danych, takie jak informacje o dostawcy, zestawie wierszy, tabeli, kolumnach lub innych informacjach znajdujących się w rzeczywistej zawartości bazy danych. OLE DB pozwala uzyskać takie informacje bez otwierania zestawu wierszy przy użyciu zestawów wierszy schematu.

Aby uzyskać informacje o sposobach używania zestawów wierszy schematu, zobacz [Uzyskiwanie metadanych z zestawami wierszy schematu](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
