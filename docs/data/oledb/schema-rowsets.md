# <a name="schema-rowsets"></a>Zestawy wierszy schematu

Schemat (lub metadanych) jest żadnych informacji na temat struktury bazy danych lub organizacji, takich jak informacje o dostawcy, zestawu wierszy, tabel, kolumn, lub inne informacje oprócz zawartość bazy danych. OLE DB pozwala uzyskać takie informacje bez konieczności otwierania zestawu wierszy za pomocą zestawów wierszy schematu.  
  
Aby uzyskać informacje o sposobie używania zestawów wierszy schematu, zobacz [uzyskiwanie metadanych za pomocą zestawów wierszy schematu](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
