# <a name="catalog-information--mfc-data-access"></a>Informacje o katalogu (dostęp do danych MFC)

Informacje o tabelach w źródle danych mogą obejmować nazwy tabel i kolumn w ich uprawnień w tabeli, nazwy główne i obce klucze, informacje o wstępnie zdefiniowanych zapytań lub procedur przechowywanych informacji na temat indeksów w tabelach, a Statystyka tabel.

Aby uzyskać więcej informacji, zobacz [źródła danych: określanie schematu źródła danych (ODBC)](../data/odbc/data-source-determining-the-schema-of-the-data-source-odbc.md).
