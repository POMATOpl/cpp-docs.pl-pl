# <a name="catalog-information--mfc-data-access"></a>Informacje o katalogu (dostęp do danych MFC)

Informacje o tabelach w źródle danych mogą zawierać nazwy tabel i kolumn w nich, uprawnienia do tabel, nazwy kluczy podstawowych i obcych, informacje o wstępnie zdefiniowanych zapytaniach lub procedurach składowanych, informacje o indeksach tabel oraz dane statystyczne dotyczące tabel.

Aby uzyskać więcej informacji, zobacz [Źródło danych: określanie schematu źródła danych (ODBC)](../data/odbc/data-source-determining-the-schema-of-the-data-source-odbc.md).
