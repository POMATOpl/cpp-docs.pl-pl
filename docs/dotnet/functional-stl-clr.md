# <a name="functional-stlclr"></a>functional (STL/CLR)
Dołącz nagłówek STL/CLR `<cliext/functional>` do definiowania liczba szablonu klasy i delegaci pokrewne szablonu i funkcje.  
  
## <a name="syntax"></a>Składnia  
  
```  
#include <functional>  
```
