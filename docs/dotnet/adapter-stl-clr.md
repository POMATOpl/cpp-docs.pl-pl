# <a name="adapter-stlclr"></a>adapter (STL/CLR)
Nagłówek STL/CLR `<cliext/adapter>` określa dwóch szablonów klas (`collection_adapter` i `range_adapter`), a funkcja szablonu `make_collection`.  
  
## <a name="syntax"></a>Składnia  
  
```  
#include <cliext/adapter>  
```  
  
## <a name="remarks"></a>Uwagi  
  
|Class|Opis|  
|-----------|-----------------|  
|[collection_adapter (STL/CLR)](../dotnet/collection-adapter-stl-clr.md)|Opakowuje kolekcji biblioteki klasy podstawowej (BCL) jako zakres.|  
|[range_adapter (STL/CLR)](../dotnet/range-adapter-stl-clr.md)|Opakowuje zakres jako kolekcja BCL.|  
  
|Funkcja|Opis|  
|--------------|-----------------|  
|[make_collection (STL/CLR)](../dotnet/make-collection-stl-clr.md)|Tworzy kartą zakresu przy użyciu pary iteratora.|  
  
## <a name="requirements"></a>Wymagania  
 **Nagłówek:** \<cliext/karty >  
  
 **Namespace:** cliext
