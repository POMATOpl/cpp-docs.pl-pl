# <a name="native-and-net-interoperability"></a>Współdziałanie natywne i .NET

Visual C++ obsługuje funkcje współdziałania, które umożliwiają współistnienie w tym samym zestawieniu zarządzanych i niezarządzanych konstrukcji, a nawet w tym samym pliku. Niewielki podzestaw tej funkcji, taki jak P/Invoke, jest obsługiwany przez inne języki .NET, ale większość obsługi współdziałania zapewniana przez Visual C++ nie jest dostępna w innych językach.
