# <a name="native-and-net-interoperability"></a>Współdziałanie natywne i .NET

Visual C++ obsługuje funkcje współpracy, które umożliwiają zarządzane i niezarządzane konstrukcje współistnieć i współpracować w ramach tego samego zestawu, a nawet w tym samym pliku. Mały podzbiór tej funkcji, takich jak metody P/Invoke jest obsługiwana przez innych językach .NET, ale większość współdziałanie obsługę języka Visual C++ nie jest dostępna w innych językach.
