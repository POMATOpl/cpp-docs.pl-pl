# <a name="utility-stlclr"></a>utility (STL/CLR)
Dołącz nagłówek STL/CLR `<cliext/utility>` w celu zdefiniowania szablonu klasy `pair` i niektóre funkcje pomocnicze szablonu.  
  
## <a name="syntax"></a>Składnia  
  
```  
#include <utility>  
```
