# <a name="vc-directories-linux-c"></a>Katalogi VC ++ (Linux C++)

Właściwość | Opis
--- | ---
Katalogi plików nagłówkowych | Ścieżka używana podczas szukania załączonych plików podczas kompilowania projektu programu VC ++.  Odnosi się do zmiennej środowiskowej INCLUDE.
Katalogi bibliotek | Ścieżka używana podczas szukania plików bibliotek, podczas kompilowania projektu programu VC ++.  Odnosi się do zmiennej środowiskowej LIB.
{1&gt;Katalogi źródłowe&lt;1} | Ścieżka używana podczas szukania plików źródłowych dla technologii IntelliSense.
Wyklucz katalogi | Ścieżki pomijane podczas wyszukiwania zależności skanowania.
