# <a name="vc-directories-linux-c"></a>Katalogi VC ++ (Linux C++)

Właściwość | Opis
--- | ---
Dołącz katalogi | Ścieżka używana podczas wyszukiwania plików dołączanych podczas kompilowania projektu programu VC ++.  Odpowiada zmiennej środowiskowej INCLUDE.
Katalogi bibliotek | Ścieżka używana podczas szukania plików bibliotek podczas kompilowania projektu programu VC ++.  Odpowiada zmiennej środowiskowej LIB.
Katalogi źródłowe | Ścieżka używana podczas szukania plików źródłowych do użycia przez funkcję Intellisense.
Wyklucz katalogi | Ścieżka pomijana podczas wyszukiwania zależności skanowania.
