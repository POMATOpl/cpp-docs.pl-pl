# <a name="ompgetwtick"></a>omp_get_wtick
Zwraca liczbę sekund między Takty zegara procesora.  
  
## <a name="syntax"></a>Składnia  
  
```  
double omp_get_wtick( );  
```  
  
## <a name="remarks"></a>Uwagi  
 Aby uzyskać więcej informacji, zobacz [3.3.2 funkcja omp_get_wtick](../../../parallel/openmp/3-3-2-omp-get-wtick-function.md).  
  
## <a name="example"></a>Przykład  
 Zobacz [omp_get_wtime](../../../parallel/openmp/reference/omp-get-wtime.md) przykład przy użyciu `omp_get_wtick`.
