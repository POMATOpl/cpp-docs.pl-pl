# <a name="33-timing-routines"></a>3.3 Procedury chronometrażu
Funkcje opisane w tej sekcji obsługuje czasomierza przenośne zegara tablicy:  
  
-   `omp_get_wtime` Funkcja zwraca czasu czas zegara tablicy.  
  
-   `omp_get_wtick` Funkcja zwraca sekund między kolejnymi zegarowych.
