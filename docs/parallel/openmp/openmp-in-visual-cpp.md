# <a name="openmp-in-visual-c"></a>OpenMP w programie Visual C++

Interfejs aplikacji OpenMP C i C++ pozwala pisać aplikacje, które skutecznie korzystać z wielu procesorów. Visual C++ obsługuje standard OpenMP 2.0.
