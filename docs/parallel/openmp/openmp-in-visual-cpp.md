# <a name="openmp-in-visual-c"></a>OpenMP w programie Visual C++

Interfejs programu aplikacji OpenMP C i C++ pozwala pisać aplikacje, które efektywnie używają wielu procesorów. Visual C++ obsługuje standard OpenMP 2,0.
