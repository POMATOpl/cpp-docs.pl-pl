### <a name="parameters"></a>Parametry

*_V0*<br/>
Wartość do zainicjowania elementu 0.

*_V1*<br/>
Wartość do zainicjowania elementu 1.

*_V2*<br/>
Wartość można zainicjować element 2.

*_V*<br/>
Wartość inicjalizacji.

*_Inne*<br/>
Obiekt używany do inicjowania.
