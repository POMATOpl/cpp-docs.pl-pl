# <a name="nulloptt-struct"></a>nullopt_t — struktura

Typem pustą strukturą używany jako typ unikatowy do wskazania stanu nie zawiera wartości `optional` obiektów.

## <a name="syntax"></a>Składnia

```cpp
struct nullopt_t{see below};
inline constexpr nullopt_t nullopt(unspecified);
```
