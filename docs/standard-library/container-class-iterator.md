# <a name="container-classiterator"></a>Kontener Class::iterator
> [!NOTE]
>  Ten temat dotyczy w dokumentacji Visual C++ prawidłowo przykład kontenerów używanych w standardowej bibliotece C++. Aby uzyskać więcej informacji, zobacz [standardowe kontenery biblioteki C++](../standard-library/stl-containers.md).  
  
 Opis obiektu, który może służyć jako iterację w kontrolowanej sekwencji.  
  
## <a name="syntax"></a>Składnia  
  
```  
 
typedef T5 iterator;  
```  
  
## <a name="remarks"></a>Uwagi  
 Jest on opisany tutaj jako synonimem nieokreślonego typu **T5**. Obiekt typu **iterator** mogą być rzutowane na obiekt typu [const_iterator](../standard-library/container-class-const-iterator.md).
