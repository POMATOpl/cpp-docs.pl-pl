# <a name="regextraitsltwchartgt-class"></a>Klasa&lt;regex_traits&gt; wchar_t

Specjalizacja dla wchar_t  `regex_traits` .

## <a name="syntax"></a>Składnia

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Uwagi

Klasa jest jawną specjalizacją klasy szablonu [regex_traits](../standard-library/regex-traits-class.md) dla elementów typu **wchar_t** (dzięki czemu może korzystać z funkcji biblioteki, które manipulują obiektami tego typu).

## <a name="requirements"></a>Wymagania

**Nagłówek:** \<> wyrażeń regularnych

**Przestrzeń nazw:** std
