# <a name="regextraitsltwchartgt-class"></a>regex_traits —&lt;wchar_t&gt; — klasa

Specjalizacja regex_traits — dla wchar_t.

## <a name="syntax"></a>Składnia

```cpp
template <>
class regex_traits<wchar_t>
```
