# <a name="regex_traitsltwchar_tgt-class"></a>Klasa regex_traits &lt;wchar_t &gt;

Specjalizacja `regex_traits` **wchar_t**.

## <a name="syntax"></a>Składnia

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Uwagi

Klasa jest jawną specjalizacją szablonu klasy [regex_traits](../standard-library/regex-traits-class.md) dla elementów typu **wchar_t** (dzięki czemu może korzystać z funkcji biblioteki, które manipulują obiektami tego typu).

## <a name="requirements"></a>Wymagania

**Nagłówek:** \<regex >

**Przestrzeń nazw:** std
