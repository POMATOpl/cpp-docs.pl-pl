# <a name="ltdequegt"></a>&lt;deque&gt;

Definiuje szablon klasy kontenera deque i kilka szablonów pomocniczych.

## <a name="requirements"></a>Wymagania

**Nagłówek**: \<deque>

> [!NOTE]
> \<deque>Biblioteka używa również `#include <initializer_list>` instrukcji.
