# <a name="ltdequegt"></a>&lt;deque &gt;

Definiuje szablon klasy kontenera deque i kilka szablonów pomocniczych.

## <a name="requirements"></a>Wymagania

**Nagłówek**: \<deque >

> [!NOTE]
> Biblioteka > \<deque również używa instrukcji `#include <initializer_list>`.
