# <a name="ltunordered_mapgt"></a>&lt;unordered_map&gt;

Definiuje szablony klas kontenerów [unordered_map](../standard-library/unordered-map-class.md) i [unordered_multimap](../standard-library/unordered-multimap-class.md) i ich szablony pomocnicze.

## <a name="syntax"></a>Składnia

```cpp
#include <unordered_map>
```

> [!NOTE]
> \<unordered_map>Biblioteka używa również `#include <initializer_list>` instrukcji.

### <a name="classes"></a>Klasy

|Klasa|Opis|
|-|-|
|[Klasa unordered_map](../standard-library/unordered-map-class.md)|Przechowuje tabelę skrótów zawierającą pary {Key, zamapowany}.|
|[Klasa unordered_multimap](../standard-library/unordered-multimap-class.md)|Przechowuje tabelę skrótów zawierającą pary {Key, zamapowany}.|
