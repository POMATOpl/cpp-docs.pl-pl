# <a name="ltinitializerlistgt"></a>&lt;initializer_list&gt;
Definiuje klasę szablonu kontenera `initializer_list` i kilka szablonów pomocniczych.  
  
## <a name="syntax"></a>Składnia  
  
```  
#include <initializer_list>  
```
