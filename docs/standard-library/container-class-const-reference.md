# <a name="container-classconstreference"></a>Kontener Class::const_reference

> [!NOTE]
> Ten temat dotyczy w dokumentacji języka Visual C++ jako prawidłowo przykład kontenerów używanych w standardowej biblioteki języka C++. Aby uzyskać więcej informacji, zobacz [standardowych kontenerów biblioteki języka C++](../standard-library/stl-containers.md).

Opisuje obiekt, który może służyć jako stałe odwołanie do elementu w kontrolowanej sekwencji.

## <a name="syntax"></a>Składnia

```

typedef T3 const_reference;
```

## <a name="remarks"></a>Uwagi

Jest on opisany tutaj jako synonim dla nieokreślonego typu `T3` (zazwyczaj `Alloc::const_reference`).
