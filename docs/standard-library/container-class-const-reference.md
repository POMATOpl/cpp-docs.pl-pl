# <a name="container-classconst_reference"></a>Kontener Class::const_reference

> [!NOTE]
> Ten temat znajduje się w dokumentacji języka Microsoft C++ jako niefunkcjonalny przykład kontenerów używanych w standardowej bibliotece języka C++. Aby uzyskać więcej informacji, zobacz [kontenery standardowej biblioteki języka C++](../standard-library/stl-containers.md).

Opisuje obiekt, który może stanowić stałe odwołanie do elementu kontrolowanej sekwencji.

## <a name="syntax"></a>Składnia

```cpp
typedef T3 const_reference;
```

## <a name="remarks"></a>Uwagi

Jest on opisany tutaj jako synonim dla nieokreślonego typu `T3` (zazwyczaj `Alloc::const_reference` ).
