# <a name="container-classconstreference"></a>Kontener Class::const_reference
> [!NOTE]
>  Ten temat dotyczy w dokumentacji Visual C++ prawidłowo przykład kontenerów używanych w standardowej bibliotece C++. Aby uzyskać więcej informacji, zobacz [standardowe kontenery biblioteki C++](../standard-library/stl-containers.md).  
  
 Opis obiektu, który może służyć jako stałej odwołanie do elementu w kontrolowanej sekwencji.  
  
## <a name="syntax"></a>Składnia  
  
```  
 
typedef T3 const_reference;  
```  
  
## <a name="remarks"></a>Uwagi  
 Jest on opisany tutaj jako synonimem nieokreślonego typu **T3** (zazwyczaj **Alloc::const_reference**).
