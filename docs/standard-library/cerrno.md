# <a name="ltcerrnogt"></a>&lt;cerrno&gt;

Zawiera nagłówek standardowej biblioteki C \<errno.h> i dodaje skojarzone nazwy do `std` przestrzeni nazw. Dołączenie tego nagłówka zapewnia, że nazwy zadeklarowane za pomocą zewnętrznego powiązania w nagłówku standardowej biblioteki C są deklarowane w `std` przestrzeni nazw.

## <a name="syntax"></a>Składnia

```cpp
#include <cerrno>
```
