# <a name="ltstackgt"></a>&lt;stack&gt;

Definiuje stos klas szablonu i dwa szablony pomocnicze.

## <a name="requirements"></a>Wymagania

**Nagłówek:** \<> stosu

**Przestrzeń nazw:** std

> [!NOTE]
> `#include <initializer_list>` Biblioteka \<> stosu używa również instrukcji.
