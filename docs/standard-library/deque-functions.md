## <a name="swap"></a>  swap

Zamienia elementy z dwóch deques.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Parametry

*left*<br/>
Obiekt typu `deque`.

*right*<br/>
Obiekt typu `deque`.
