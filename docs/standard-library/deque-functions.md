# <a name="ltdequegt-functions"></a>&lt;deque —&gt; funkcji

## <a name="swap"></a> swap

Zamienia elementy z dwóch deques.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Parametry

*po lewej stronie*\
Obiekt typu `deque`.

*po prawej stronie*\
Obiekt typu `deque`.
