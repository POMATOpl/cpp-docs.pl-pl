##  <a name="swap"></a>  Swap  
 Zamienia deques dwa elementy.  
  
```

    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### <a name="parameters"></a>Parametry

 Obiekt typu `deque`.
