# <a name="ltcvtwstringgt"></a>&lt;CVT/wstring —&gt;

Nagłówek `<cvt/wstring>` w poprzednich wersjach programu Visual Studio zdefiniowana klasa szablonu [wstring_convert — klasa](../standard-library/wstring-convert-class.md) w przestrzeni nazw stdext::cvt. Nagłówek jest zachowywana na potrzeby zgodności z poprzednimi wersjami. Nowy kod powinien używać wersji klasy, która jest zdefiniowana w [ \<ustawień regionalnych >](../standard-library/locale.md) w `std` przestrzeni nazw

## <a name="syntax"></a>Składnia

```cpp
#include <cvt/wstring>
```
