# <a name="ltcvtwstringgt"></a>&lt;CVT/wstring&gt;

Nagłówek `<cvt/wstring>` w poprzednich wersjach programu Visual Studio określił szablon klasy [wstring_convert klasy](../standard-library/wstring-convert-class.md) w przestrzeni nazw stdext:: CVT. Nagłówek jest zachowany w celu zapewnienia zgodności z poprzednimi wersjami. Nowy kod powinien korzystać z wersji klasy, która jest zdefiniowana w [\<locale>](../standard-library/locale.md) `std` przestrzeni nazw

## <a name="syntax"></a>Składnia

```cpp
#include <cvt/wstring>
```
