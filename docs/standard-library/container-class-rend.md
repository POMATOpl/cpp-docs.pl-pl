# <a name="container-classrend"></a>Kontener Class::rend

> [!NOTE]
> Ten temat dotyczy w dokumentacji Visual C++ prawidłowo przykład kontenerów używanych w standardowej bibliotece C++. Aby uzyskać więcej informacji, zobacz [standardowe kontenery biblioteki C++](../standard-library/stl-containers.md).

Funkcja członkowska zwraca odwrotnej iteratora tego punktów w pierwszym elementem sekwencji (lub bezpośrednio po zakończeniu pustej sekwencji) wyznaczenie koniec odwrotnej kolejności.

## <a name="syntax"></a>Składnia

```

    const_reverse_iterator rend() const;


reverse_iterator rend();
```
