# <a name="ltarraygt"></a>&lt;array &gt;

Definiuje **tablicę** szablonu klasy kontenera i kilka szablonów pomocniczych.

## <a name="requirements"></a>Wymagania

**Nagłówek:** \<array >

**Przestrzeń nazw:** std

> [!NOTE]
> Biblioteka > \<array również używa instrukcji `#include <initializer_list>`.
