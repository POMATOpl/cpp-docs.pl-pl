# <a name="ltarraygt"></a>&lt;Tablica&gt;
Definiuje klasę szablonu kontenera `array` i kilka szablonów pomocniczych.  
  
## <a name="syntax"></a>Składnia  
  
```  
#include <array>  
```

|-|-|  
|[Tablica](../standard-library/array-class-stl.md)|Zapisuje sekwencję o stałej długości elementów.|  
|[tuple_element](../standard-library/tuple-element-class-tuple.md)|Opakowuje typ elementu tablicy.|  
|[tuple_size](../standard-library/tuple-size-class-tuple.md)|Opakowuje rozmiar elementu tablicy.|

|[Operator\<](../standard-library/array-operators.md#op_lt)|Tablica porównania, mniejsza niż|
