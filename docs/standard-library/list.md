# <a name="ltlistgt"></a>&lt;list&gt;

Definiuje listę szablonów klas kontenerów i kilka szablonów pomocniczych.

## <a name="syntax"></a>Składnia

```cpp
#include <list>
```

> [!NOTE]
> Biblioteka > \<list również używa instrukcji `#include <initializer_list>`.

## <a name="members"></a>Elementy członkowskie

|||
|-|-|
|[operator!=](../standard-library/list-operators.md#op_neq)|Testuje, czy obiekt listy po lewej stronie operatora nie jest równy obiektowi listy po prawej stronie.|
|[< operatora](../standard-library/list-operators.md#op_lt)|Testuje, czy obiekt listy po lewej stronie operatora jest mniejszy niż obiekt listy po prawej stronie.|
|[\< operatora =](../standard-library/list-operators.md#op_gt_eq)|Testuje, czy obiekt listy po lewej stronie operatora jest mniejszy niż lub równy obiektowi listy po prawej stronie.|
|[operator = =](../standard-library/list-operators.md#op_eq_eq)|Testuje, czy obiekt listy po lewej stronie operatora jest równy obiektowi listy po prawej stronie.|
