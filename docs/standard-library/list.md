# <a name="ltlistgt"></a>&lt;list&gt;

Definiuje listę szablonów klas kontenerów i kilka szablonów pomocniczych.

## <a name="syntax"></a>Składnia

```cpp
#include <list>
```

> [!NOTE]
> \<list > Library używa również instrukcji `#include <initializer_list>`.

## <a name="members"></a>Elementy członkowskie

|[< operatora](../standard-library/list-operators.md#op_lt)|Testuje, czy obiekt listy po lewej stronie operatora jest mniejszy niż obiekt listy po prawej stronie.|
