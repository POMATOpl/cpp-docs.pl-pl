# <a name="ltlistgt"></a>&lt;list&gt;

Definiuje listę szablonów klas kontenerów i kilka szablonów pomocniczych.

## <a name="syntax"></a>Składnia

```cpp
#include <list>
```

> [!NOTE]
> \<list>Biblioteka używa również `#include <initializer_list>` instrukcji.
