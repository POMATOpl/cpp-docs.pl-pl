# <a name="ltlistgt"></a>&lt;list&gt;

Definiuje listę kontenerów szablonu klasy i kilka szablonów pomocniczych.

## <a name="syntax"></a>Składnia

```cpp
#include <list>
```

> [!NOTE]
> \<Listy > używa również biblioteki `#include <initializer_list>` instrukcji.

## <a name="members"></a>Elementy członkowskie

### <a name="operators"></a>Operatory

|||
|-|-|
|[operator!=](../standard-library/list-operators.md#op_neq)|Sprawdza, czy obiekt listy po lewej stronie operatora nie jest równy obiektowi listy po prawej stronie.|
|[Operator <](../standard-library/list-operators.md#op_lt)|Sprawdza, czy obiekt listy, po lewej stronie operatora jest mniejszy niż obiekt listy po prawej stronie.|
|[Operator\<=](../standard-library/list-operators.md#op_gt_eq)|Sprawdza, czy lista obiektów po lewej stronie operatora jest mniejszy niż lub równy obiektowi listy po prawej stronie.|
|[operator==](../standard-library/list-operators.md#op_eq_eq)|Sprawdza, czy obiekt listy po lewej stronie operatora jest równy obiektowi listy po prawej stronie.|
