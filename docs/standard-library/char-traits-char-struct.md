# <a name="char_traitsltchargt-struct"></a>&lt;Struktura znaków &gt; char_traits

Struktura, która jest specjalizacją struktury szablonu **char_traits \<CharType> ** do elementu typu **`char`** .

## <a name="syntax"></a>Składnia

```cpp
template <>
struct char_traits<char>;
```

## <a name="remarks"></a>Uwagi

Specjalizacja umożliwia strukturze korzystanie z funkcji bibliotek, które manipulują obiektami tego typu **`char`** .
