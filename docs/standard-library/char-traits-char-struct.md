Struktury, która jest specjalizacją szablonu struktury **char_traits\<CharType >** do elementu typu `char`.

## <a name="syntax"></a>Składnia

```cpp

struct char_traits<char>;
```

## <a name="remarks"></a>Uwagi

Struktury móc korzystać z funkcji biblioteki, które manipulowania obiektów tego typu umożliwia specjalizacji `char`.
