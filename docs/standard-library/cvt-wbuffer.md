# <a name="ltcvtwbuffergt"></a>&lt;CVT/wbuffer&gt;

Nagłówek `<cvt/wstring>` w poprzednich wersjach programu Visual Studio określił szablon klasy [wbuffer_convert klasy](../standard-library/wbuffer-convert-class.md) w przestrzeni nazw stdext:: CVT. Nagłówek jest zachowany w celu zapewnienia zgodności z poprzednimi wersjami. Nowy kod powinien korzystać z wersji klasy, która jest zdefiniowana w [\<locale>](../standard-library/locale.md) `std` przestrzeni nazw

## <a name="syntax"></a>Składnia

```cpp
#include <cvt/wbuffer>
```
