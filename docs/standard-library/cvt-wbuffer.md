# <a name="ltcvtwbuffergt"></a>&lt;cvt/wbuffer &gt;

Nagłówek `<cvt/wstring>` w poprzednich wersjach programu Visual Studio określił [klasę wbuffer_convert](../standard-library/wbuffer-convert-class.md) szablonu klasy w przestrzeni nazw stdext:: CVT. Nagłówek jest zachowany w celu zapewnienia zgodności z poprzednimi wersjami. Nowy kod powinien korzystać z wersji klasy zdefiniowanej w [\<locale >](../standard-library/locale.md) w przestrzeni nazw `std`

## <a name="syntax"></a>Składnia

```cpp
#include <cvt/wbuffer>
```
