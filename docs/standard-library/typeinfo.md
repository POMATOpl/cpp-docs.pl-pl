# <a name="lttypeinfogt"></a>&lt;element TypeInfo&gt;

Dołącz nagłówek standardowy \<typeinfo > Aby zdefiniować kilka typów skojarzone z operatorem identyfikację typu [typeid — Operator](../cpp/typeid-operator.md), która daje informacji na temat typów statycznych i dynamicznych.

## <a name="syntax"></a>Składnia

```cpp
#include <typeinfo>

```
