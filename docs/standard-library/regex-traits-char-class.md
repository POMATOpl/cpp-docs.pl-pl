# <a name="regextraitsltchargt-class"></a>regex_traits&lt;char&gt; — Klasa

Specjalizacja dla char  `regex_traits` .

## <a name="syntax"></a>Składnia

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Uwagi

Klasa jest jawną specjalizacją klasy szablonu [regex_traits](../standard-library/regex-traits-class.md) dla elementów typu **char** (dzięki czemu może korzystać z funkcji biblioteki, które manipulują obiektami tego typu).

## <a name="requirements"></a>Wymagania

**Nagłówek:** \<> wyrażeń regularnych

**Przestrzeń nazw:** std
