# <a name="regextraitsltchargt-class"></a>regex_traits&lt;char&gt; klasy

Specjalizacja regex_traits char.

## <a name="syntax"></a>Składnia

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Uwagi

Klasa jest jawną specjalizacją klasy szablonu [regex_traits](../standard-library/regex-traits-class.md) dla elementów typu **char** (tak, aby go korzystać z zalet funkcji biblioteki, które manipulowania obiektów tego typu).

## <a name="requirements"></a>Wymagania

**Nagłówek:** \<wyrażenia regularnego >

**Namespace:** standardowe
