# <a name="container-classreverse_iterator"></a>Kontener Class::reverse_iterator

> [!NOTE]
> Ten temat znajduje się w dokumentacji języka Microsoft C++ jako niefunkcjonalny przykład kontenerów używanych w standardowej bibliotece języka C++. Aby uzyskać więcej informacji, zobacz [kontenery standardowej biblioteki języka C++](../standard-library/stl-containers.md).

Opisuje obiekt, który może działać jako iterator odwrotny dla kontrolowanej sekwencji.

## <a name="syntax"></a>Składnia

```cpp
typedef T7 reverse_iterator;
```

## <a name="remarks"></a>Uwagi

Jest on opisany tutaj jako synonim dla nieokreślonego typu `T7` (zazwyczaj `reverse_iterator` **\<**[iterator](../standard-library/container-class-iterator.md)**>** ).
