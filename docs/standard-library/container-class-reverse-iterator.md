# <a name="container-classreverseiterator"></a>Kontener Class::reverse_iterator
> [!NOTE]
>  Ten temat dotyczy w dokumentacji Visual C++ prawidłowo przykład kontenerów używanych w standardowej bibliotece C++. Aby uzyskać więcej informacji, zobacz [standardowe kontenery biblioteki C++](../standard-library/stl-containers.md).  
  
 Opis obiektu, który może służyć jako odwrotnej iteratora w kontrolowanej sekwencji.  
  
## <a name="syntax"></a>Składnia  
  
```  
 
typedef T7 reverse_iterator;  
```  
  
## <a name="remarks"></a>Uwagi  
 Jest on opisany tutaj jako synonimem nieokreślonego typu **T7** (zazwyczaj `reverse_iterator`  **\<**  [iterator](../standard-library/container-class-iterator.md)  **>** ).
