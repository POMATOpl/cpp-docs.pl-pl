# <a name="chartraitsltchar32tgt-struct"></a>char_traits&lt;char32_t&gt; — struktura

Struktura, która jest specjalizacją szablonu struktury **char_traits\<CharType >** do elementu typu `char32_t`.

## <a name="syntax"></a>Składnia


Specjalizacja umożliwia struktury móc korzystać z funkcji biblioteki, które manipulowania obiektów tego typu `char32_t`.
