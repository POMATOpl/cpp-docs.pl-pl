# <a name="chartraitsltchar32tgt-struct"></a>char_traits&lt;char32_t&gt; — struktura
Struktury, która jest specjalizacją szablonu struktury **char_traits\<CharType >** do elementu typu `char32_t`.  
  
## <a name="syntax"></a>Składnia  
  
```
template <>  
struct char_traits<char32_t>;
```  
  
## <a name="remarks"></a>Uwagi  
 Struktury móc korzystać z funkcji biblioteki, które manipulowania obiektów tego typu umożliwia specjalizacji `char32_t`.
