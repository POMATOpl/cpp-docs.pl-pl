# <a name="container-classdifference_type"></a>Kontener Class::difference_type

> [!NOTE]
> Ten temat znajduje się w dokumentacji języka Microsoft C++ jako niefunkcjonalny przykład kontenerów używanych w standardowej bibliotece języka C++. Aby uzyskać więcej informacji, zobacz [kontenery standardowej biblioteki języka C++](../standard-library/stl-containers.md).

Opisuje obiekt, który może reprezentować różnicę między adresami wszystkich dwóch elementów w kontrolowanej sekwencji.

## <a name="syntax"></a>Składnia

```cpp
typedef T1 difference_type;
```

## <a name="remarks"></a>Uwagi

Jest on opisany tutaj jako synonim dla nieokreślonego typu `T1` (zazwyczaj `Alloc::difference_type` ).
