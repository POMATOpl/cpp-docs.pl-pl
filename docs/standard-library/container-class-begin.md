# <a name="container-classbegin"></a>Kontener Class::begin

> [!NOTE]
> Ten temat znajduje się w dokumentacji programu C++ Visual Studio jako niefunkcjonalny przykład kontenerów używanych w C++ standardowej bibliotece. Aby uzyskać więcej informacji, zobacz [ C++ Kontenery biblioteki standardowej](../standard-library/stl-containers.md).

Zwraca iterator, który wskazuje na pierwszy element sekwencji (lub tuż poza końcem pustej sekwencji).

## <a name="syntax"></a>Składnia

```

    const_iterator begin() const;

iterator begin();
```
