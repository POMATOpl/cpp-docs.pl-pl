# <a name="container-classbegin"></a>Kontener Class::begin

> [!NOTE]
> Ten temat znajduje się w dokumentacji programu Visual Studio C++ jako niefunkcjonalny przykład kontenerów używanych w standardowej bibliotece języka C++. Aby uzyskać więcej informacji, zobacz [kontenery standardowej biblioteki języka C++](../standard-library/stl-containers.md).

Zwraca iterator, który wskazuje na pierwszy element sekwencji (lub tuż poza końcem pustej sekwencji).

## <a name="syntax"></a>Składnia

```cpp
const_iterator begin() const;

iterator begin();
```
