# <a name="ltunordered_setgt"></a>&lt;unordered_set &gt;

Definiuje szablony klas kontenerów [unordered_multiset](../standard-library/unordered-multiset-class.md) i [unordered_set](../standard-library/unordered-set-class.md) oraz ich szablony pomocnicze.

## <a name="requirements"></a>Wymagania

**Nagłówek:** \<unordered_set >

**Przestrzeń nazw:** std

> [!NOTE]
> Biblioteka > \<unordered_set również używa instrukcji `#include <initializer_list>`.
