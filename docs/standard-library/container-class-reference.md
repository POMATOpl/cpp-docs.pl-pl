# <a name="container-classreference"></a>Kontener Class::reference
> [!NOTE]
>  Ten temat dotyczy w dokumentacji Visual C++ prawidłowo przykład kontenerów używanych w standardowej bibliotece C++. Aby uzyskać więcej informacji, zobacz [standardowe kontenery biblioteki C++](../standard-library/stl-containers.md).  
  
 Opis obiektu, który może służyć jako odwołanie do elementu w kontrolowanej sekwencji.  
  
## <a name="syntax"></a>Składnia  
  
```  
 
typedef T2 reference;  
```  
  
## <a name="remarks"></a>Uwagi  
 Jest on opisany tutaj jako synonimem nieokreślonego typu **T2** (zazwyczaj **Alloc::reference**). Obiekt typu **odwołania** mogą być rzutowane na obiekt typu [const_reference](../standard-library/container-class-const-reference.md).
