# <a name="container-classconstreverseiterator"></a>Kontener Class::const_reverse_iterator

> [!NOTE]
> Ten temat dotyczy w dokumentacji Visual C++ prawidłowo przykład kontenerów używanych w standardowej bibliotece C++. Aby uzyskać więcej informacji, zobacz [standardowe kontenery biblioteki C++](../standard-library/stl-containers.md).

Opis obiektu, który może służyć jako stałej iteratora wstecznego w kontrolowanej sekwencji.

## <a name="syntax"></a>Składnia

```

typedef T8 const_reverse_iterator;
```

## <a name="remarks"></a>Uwagi

Jest on opisany tutaj jako synonimem nieokreślonego typu **T8** (zazwyczaj [reverse_iterator](../standard-library/container-class-reverse-iterator.md) <[const_iterator](../standard-library/container-class-const-iterator.md)`>`).
