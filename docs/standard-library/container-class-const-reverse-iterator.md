# <a name="container-classconst_reverse_iterator"></a>Kontener Class::const_reverse_iterator

> [!NOTE]
> Ten temat znajduje się w dokumentacji języka Microsoft C++ jako niefunkcjonalny przykład kontenerów używanych w standardowej bibliotece języka C++. Aby uzyskać więcej informacji, zobacz [kontenery standardowej biblioteki języka C++](../standard-library/stl-containers.md).

Opisuje obiekt, który może być używany jako ciągły iterator odwrotny dla kontrolowanej sekwencji.

## <a name="syntax"></a>Składnia

```cpp
typedef T8 const_reverse_iterator;
```

## <a name="remarks"></a>Uwagi

Jest on opisany tutaj jako synonim dla nieokreślonego typu `T8` (zwykle [reverse_iterator](../standard-library/container-class-reverse-iterator.md)  < [const_iterator](../standard-library/container-class-const-iterator.md) `>` ).
