# <a name="safeint-library"></a>Biblioteka SafeInt

**SafeInt** to Przenośna biblioteka, która może być używana z usługą MSVC, w zatoce lub Clang, aby zapobiec przepełnieniu liczb całkowitych, które mogą spowodować, że aplikacja wykonuje operacje matematyczne. Najnowsza wersja tej biblioteki znajduje się w [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt).
