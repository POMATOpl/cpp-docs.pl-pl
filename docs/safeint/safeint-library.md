# <a name="safeint-library"></a>Biblioteka SafeInt

**Safeint —** jest przenośnej biblioteki, który może służyć za pomocą MSVC i kompilatora GCC, Clang Aby uniknąć przepełnienia liczby całkowitej, które może spowodować, gdy aplikacja wykonuje operacje matematyczne. Najnowszą wersję tej biblioteki znajduje się w [ https://github.com/dcleblanc/SafeInt ](https://github.com/dcleblanc/SafeInt).
