# <a name="compiler-warning-level-1-c4541"></a>Kompilator ostrzeżenie (poziom 1) C4541

'Identyfikator' użyty na typie polimorficznym "type" z/gr; może spowodować nieprzewidywalne zachowanie

Próbowano użyć funkcji, która wymaga, aby informacje typu run-time bez włączania informacje typu run-time. Skompiluj ponownie z [GR](../../build/reference/gr-enable-run-time-type-information.md).
