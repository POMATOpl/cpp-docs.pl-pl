# <a name="compiler-warning-level-1-c4541"></a>Ostrzeżenie kompilatora (poziom 1) C4541

element "identifier" jest używany w typie polimorficznym "Type" z/GR-; może to spowodować nieprzewidywalne zachowanie

Podjęto próbę użycia funkcji wymagającej informacji o typie w czasie wykonywania bez włączania informacji o typie czasu wykonywania. Kompiluj ponownie z [/gr](../../build/reference/gr-enable-run-time-type-information.md).
