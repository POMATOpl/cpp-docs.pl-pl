# <a name="compiler-warning-level-1-c4541"></a>Kompilator C4541 ostrzegawcze (poziom 1)
'Identyfikator' użyty na typie polimorficznym "type" z/gr; może spowodować nieprzewidywalne zachowanie  
  
 Próbowano użyć funkcji, która wymaga informacje typu run-time bez włączania informacje typu run-time. Skompiluj przy [GR](../../build/reference/gr-enable-run-time-type-information.md).
