# <a name="compiler-warning-level-1-c4045"></a>Kompilator ostrzeżenie (poziom 1) C4045

"array": przepełnienie granic tablicy

Tablica ma zbyt wiele inicjatorów. Dodatkowe inicjatory są ignorowane.

Upewnij się, że elementy tablicy i inicjatory odpowiadają rozmiaru i liczby.
