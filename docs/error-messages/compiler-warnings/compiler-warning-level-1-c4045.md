# <a name="compiler-warning-level-1-c4045"></a>Kompilator C4045 ostrzegawcze (poziom 1)
"array": przepełnienie granic tablicy  
  
 Tablica ma zbyt wiele inicjatorów. Dodatkowe inicjatory zostały zignorowane.  
  
 Upewnij się, że elementy tablicy i inicjatory pasują do rozmiaru i ilość.
