# <a name="compiler-warning-level-1-c4045"></a>Ostrzeżenie kompilatora (poziom 1) C4045

"Array": przepełnienie granic tablicy

Tablica ma zbyt wiele inicjatorów. Dodatkowe inicjatory zostały zignorowane.

Upewnij się, że elementy tablicy i inicjatory są zgodne z rozmiarem i ilością.
