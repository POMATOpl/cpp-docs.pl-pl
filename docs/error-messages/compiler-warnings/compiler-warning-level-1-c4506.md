# <a name="compiler-warning-level-1-c4506"></a>Kompilator C4506 ostrzegawcze (poziom 1)
Brak definicji dla funkcji wbudowanej "function"  
  
 Dana funkcja została zadeklarowana i oznaczona jako przeznaczona do ze śródwierszowaniem, ale nie został zdefiniowany.  
  
 Kompilator nie nie wbudowanej funkcji.  
  
 Upewnij się, że zewnętrzne funkcje, aby był śródwierszowy został zadeklarowany z `extern` — słowo kluczowe.
