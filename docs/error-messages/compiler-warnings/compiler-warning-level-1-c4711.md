# <a name="compiler-warning-level-1-c4711"></a>Ostrzeżenie kompilatora (poziom 1) C4711

Funkcja "Function" została wybrana do rozwinięcia wbudowanego

Kompilator wykonał Tworzenie konspektu dla danej funkcji, chociaż nie został oznaczony do dekreślenia.

C4711 jest włączone, jeśli określono [/Ob2](../../build/reference/ob-inline-function-expansion.md) .

Tworzenie i składanie jest wykonywane według uznania kompilatora. To ostrzeżenie jest informacje.

To ostrzeżenie jest domyślnie wyłączone. Aby włączyć ostrzeżenie, użyj [#pragma ostrzeżenie](../../preprocessor/warning.md). Aby uzyskać więcej informacji [, zobacz ostrzeżenia kompilatora, które są domyślnie wyłączone](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
