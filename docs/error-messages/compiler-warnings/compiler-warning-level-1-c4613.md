# <a name="compiler-warning-level-1-c4613"></a>Kompilator C4613 ostrzegawcze (poziom 1)
"segment": klasa segmentu nie można zmienić.  
  
 Próbowano utworzyć segment o tej samej nazwie klasy jako segment używany przez kompilator. Nie nowa klasa segmentu został utworzony.
