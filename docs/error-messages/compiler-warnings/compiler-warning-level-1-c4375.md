# <a name="compiler-warning-level-1-c4375"></a>Kompilator C4375 ostrzegawcze (poziom 1)
Metoda niepubliczna "metoda2" nie przesłania "metoda1"  
  
 Typ, który implementuje inny typ określony to metoda przesłonięcia, ale override nie jest publiczny. W związku z tym metoda nie zastępuje metody typu podstawowego.
