# <a name="compiler-warning-level-1-c4375"></a>Ostrzeżenie kompilatora (poziom 1) C4375

Metoda niepubliczna "method2" nie przesłania elementu "Metoda1"

Typ, który implementuje inny typ zdefiniowanej metody przesłonięcia, ale zastąpienie nie było publiczne. W związku z tym metoda nie przesłania metody typu podstawowego.
