# <a name="compiler-warning-level-1-c4375"></a>Kompilator ostrzeżenie (poziom 1) C4375

Niepubliczna metoda "method2" nie przesłania "metoda1"

Typ, który implementuje inny typ zdefiniowany to metoda przesłonięcia, ale zastępowania nie jest publiczny. W związku z tym metoda nie zastępuje metody typu podstawowego.
