# <a name="compiler-warning-level-1-c4679"></a>Ostrzeżenie kompilatora (poziom 1) C4679

> "*członek*": nie można zaimportować składowej

Kompilator napotkał konstrukcję, której nie może obsłużyć, która nie może zostać zaimportowana z metadanych.

Nie próbuj używać konstrukcji.
