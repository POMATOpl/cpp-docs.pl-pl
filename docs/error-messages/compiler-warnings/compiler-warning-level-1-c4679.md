# <a name="compiler-warning-level-1-c4679"></a>Kompilator C4679 ostrzegawcze (poziom 1)
Kompilator napotkał konstrukcja, która nie może obsługiwać, których nie można importować z metadanych.  
  
 **'**   
 ***element członkowski* ": nie można zaimportować elementu członkowskiego**  
  
 Nie należy próbować użyć konstrukcji.
