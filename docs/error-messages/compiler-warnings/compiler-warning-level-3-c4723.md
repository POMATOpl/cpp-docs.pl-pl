# <a name="compiler-warning-level-3-c4723"></a>Ostrzeżenie kompilatora (poziom 3) C4723

potencjalne dzielenie przez 0

Drugi operand operacji dzielenia został oceniony jako zero w czasie kompilacji, dając niezdefiniowane wyniki.

To ostrzeżenie jest wydawane tylko w przypadku korzystania z [/og](../../build/reference/og-global-optimizations.md) lub opcji optymalizacji, która implikuje/og.

Kompilator mógł wygenerować operand zero.
