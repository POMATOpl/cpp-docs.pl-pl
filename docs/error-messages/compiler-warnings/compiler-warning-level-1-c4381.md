# <a name="compiler-warning-level-1-c4381"></a>Ostrzeżenie kompilatora (poziom 1) C4381

"function1": metoda interfejsu nie zostanie zaimplementowana przez niepubliczną metodę "function2"

Klasa musi implementować wszystkie funkcje w interfejsie. Klasa może spełnić ten warunek, jeśli jedna z jej klas podstawowych implementuje funkcję. Jednak funkcja musi być zaimplementowana jako funkcja publiczna.
