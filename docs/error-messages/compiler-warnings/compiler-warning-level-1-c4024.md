# <a name="compiler-warning-level-1-c4024"></a>Kompilator ostrzeżenie (poziom 1) C4024

'Funkcja': różne typy dla formalnego i rzeczywistego parametru "liczba"

Odpowiednie parametry formalne i rzeczywiste mają różne typy. Kompilator przekazuje rzeczywisty parametr bez zmian. Odbieranie funkcja konwertuje typ parametru oczekiwanego typu.
