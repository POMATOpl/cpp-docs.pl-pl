# <a name="compiler-warning-level-1-c4024"></a>Kompilator C4024 ostrzegawcze (poziom 1)
"Funkcja": różne typy dla formalnego i rzeczywistego parametru "number"  
  
 Odpowiednie parametry formalne i rzeczywiste mają różne typy. Kompilator przekazuje rzeczywisty parametr bez zmian. Funkcja odbierania konwertuje typ parametru na typ oczekiwany.
