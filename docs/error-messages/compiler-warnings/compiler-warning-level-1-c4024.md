# <a name="compiler-warning-level-1-c4024"></a>Ostrzeżenie kompilatora (poziom 1) C4024

"Function": różne typy dla formalnego i rzeczywistego parametru "number"

Odpowiadające im parametry formalne i rzeczywiste mają różne typy. Kompilator przekazuje rzeczywisty parametr bez zmiany. Funkcja otrzymująca konwertuje typ parametru na oczekiwany typ.
