# <a name="compiler-warning-level-3-c4283"></a>Ostrzeżenie kompilatora (poziom 3) C4283

i za poorednictwem typu "Type"

To kontynuacja ostrzeżenia [C4281](../../error-messages/compiler-warnings/compiler-warning-level-3-c4281.md) pokazuje, że wywołania **operatora->** są używane przez `type`.
