# <a name="compiler-warning-level-2-c4007"></a>Ostrzeżenie kompilatora (poziom 2) C4007

"Function": musi mieć atrybut "Attribute"

Wymagany atrybut dla funkcji nie został jawnie określony. Na przykład, **główna** funkcja musi mieć **`__cdecl`** atrybut. Kompilator wymusza atrybut.
