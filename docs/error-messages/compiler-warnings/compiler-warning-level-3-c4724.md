# <a name="compiler-warning-level-3-c4724"></a>Kompilator C4724 ostrzegawcze (poziom 3)
potencjalne dzielenie modulo przez 0  
  
 Drugi argument operacji w ramach operacji pozostałej obliczone na zero w czasie kompilacji, podając niezdefiniowane wyniki.
