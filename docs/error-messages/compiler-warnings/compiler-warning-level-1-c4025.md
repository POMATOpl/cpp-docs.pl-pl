# <a name="compiler-warning-level-1-c4025"></a>Ostrzeżenie kompilatora (poziom 1) C4025

"number": wskaźnik bazowy przeszedł do funkcji ze zmiennymi argumentami: numer parametru

Przekazanie wskaźnika wskaźnikowego do funkcji z argumentami zmiennych powoduje, że wskaźnik ma być znormalizowany, z nieprzewidywalnymi wynikami. Nie przekazuj wskaźników opartych na funkcjach za pomocą argumentów zmiennych.
