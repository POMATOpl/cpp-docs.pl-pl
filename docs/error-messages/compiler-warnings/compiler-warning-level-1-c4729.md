# <a name="compiler-warning-level-1-c4729"></a>Ostrzeżenie kompilatora (poziom 1) C4729

Funkcja za duża dla ostrzeżeń opartych na grafie Flow

To ostrzeżenie jest generowane, gdy funkcja jest zbyt duża, aby można było skompilować ją z niezawodnym sprawdzaniem w przypadku sytuacji, które generują ostrzeżenie. To ostrzeżenie jest generowane tylko w przypadku użycia opcji kompilatora [/od](../../build/reference/od-disable-debug.md) .

Aby usunąć to ostrzeżenie, Przerwij funkcję w mniejszych funkcjach.
