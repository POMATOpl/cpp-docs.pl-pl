# <a name="compiler-warning-level-1-c4098"></a>Ostrzeżenie kompilatora (poziom 1) C4098

"Function": Funkcja void zwraca wartość

Funkcja zadeklarowana z typem zwracanym [void](../../cpp/void-cpp.md) ma **`return`** instrukcję zwracającą wartość. Kompilator zakłada, że funkcja zwraca wartość typu **`int`** .
