# <a name="compiler-warning-level-1-c4652"></a>Kompilator ostrzeżenie (poziom 1) C4652

— Opcja kompilatora "opcji" niespójna z prekompilowanym nagłówkiem; Bieżąca opcja wiersza poleceń przesłoni opcję zdefiniowane we prekompilowanym nagłówku

Danej opcji wiersza polecenia różniły się od podanej podczas tworzenia prekompilowanego pliku nagłówkowego (.pch). Użyto opcji określonych w bieżącym wierszu polecenia.

Można uniknąć tego ostrzeżenia, trwa ponowne generowanie wstępnie skompilowanego nagłówka, z danej opcji wiersza polecenia.
