# <a name="compiler-warning-level-1-c4652"></a>Kompilator C4652 ostrzegawcze (poziom 1)
— Opcja kompilatora "option" niespójna z prekompilowanym nagłówkiem; Bieżąca opcja wiersza poleceń przesłoni zdefiniowaną WE prekompilowanym nagłówku  
  
 Opcja wiersza polecenia danego różnił się od podanej podczas tworzenia prekompilowanego nagłówka (.pch). Użyto opcji określony w bieżącym wierszu polecenia.  
  
 To ostrzeżenie można uniknąć przez ponowne generowanie prekompilowany nagłówek danej opcji wiersza polecenia.
