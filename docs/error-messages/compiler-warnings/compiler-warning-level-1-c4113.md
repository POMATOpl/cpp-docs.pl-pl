# <a name="compiler-warning-level-1-c4113"></a>Ostrzeżenie kompilatora (poziom 1) C4113

element "Identifier1" różni się w listach parametrów z "identifier2"

Wskaźnik funkcji jest przypisany do innego wskaźnika funkcji, ale formalne listy parametrów funkcji nie zgadzają się. Przypisanie jest kompilowane bez modyfikacji.
