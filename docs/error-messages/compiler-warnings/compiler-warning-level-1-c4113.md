# <a name="compiler-warning-level-1-c4113"></a>Kompilator ostrzeżenie (poziom 1) C4113

"identifier1" różni się listami parametrów od "identifier2"

Wskaźnik funkcji jest przypisany do innego wskaźnika funkcji, ale parametr formalny listę funkcji nie są zgodne. Przypisanie jest kompilowany bez żadnych modyfikacji.
