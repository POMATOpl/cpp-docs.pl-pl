# <a name="compiler-warning-level-1-c4113"></a>Kompilator C4113 ostrzegawcze (poziom 1)
"identifier1" różni się listami parametrów od "identifier2"  
  
 Wskaźnik funkcji jest przypisany do innego wskaźnik funkcji, ale listy formalny parametr funkcji nie są zgodne. Przypisanie jest kompilowana bez żadnych modyfikacji.
