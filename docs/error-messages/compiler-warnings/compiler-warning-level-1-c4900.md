# <a name="compiler-warning-level-1-c4900"></a>Ostrzeżenie kompilatora (poziom 1) C4900

niezgodność języka pośredniego między "Tool1" w wersji "version1" i "Tool2" w wersji "Version2"

Język pośredni używany w *Tool1* i *Tool2* nie jest zgodny. Sprawdź, czy została zainstalowana najnowsza wersja każdego narzędzia.
