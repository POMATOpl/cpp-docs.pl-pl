# <a name="compiler-warning-level-1-c4900"></a>Kompilator C4900 ostrzegawcze (poziom 1)
język pośredni niezgodność między "tool1" "version1" i "tool2" wersja "version2"  
  
 Język pośredni w *tool1* i *tool2* nie jest zgodny. Sprawdź, czy zainstalowano najnowszej wersji narzędzia.
