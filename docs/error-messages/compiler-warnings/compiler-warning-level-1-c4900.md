# <a name="compiler-warning-level-1-c4900"></a>Kompilator ostrzeżenie (poziom 1) C4900

Niezgodność języka pośredniego "tool1" wersja "version1" i "tool2" wersja "version2"

Język pośredni, używane w *tool1* i *tool2* jest niezgodny. Sprawdź, zainstalowanie najnowszej wersji każdego narzędzia.
