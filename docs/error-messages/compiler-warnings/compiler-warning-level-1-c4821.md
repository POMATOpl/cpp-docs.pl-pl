# <a name="compiler-warning-level-1-c4821"></a>Ostrzeżenie kompilatora (poziom 1) C4821

> Nie można określić typu kodowania Unicode, Zapisz plik z podpisem (BOM)

Kompilator nie może określić typu kodowania dla pliku. Aby usunąć to ostrzeżenie, Zapisz plik ze znacznikiem kolejności bajtów. Aby uzyskać więcej informacji, zobacz [Zarządzanie plikami za pomocą kodowania](/sql/ssms/solution/manage-files-with-encoding) .
