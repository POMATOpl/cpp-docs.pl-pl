# <a name="compiler-warning-level-1-c4122"></a>Kompilator C4122 ostrzegawcze (poziom 1)
"Funkcja": alloc_text można stosować tylko do funkcji z powiązaniem C  
  
 **Alloc_text** pragma ma zastosowanie tylko do funkcji zadeklarowana z **extern c**. Nie można używać z zewnętrznego funkcje języka C++.  
  
 Pragma została zignorowana.
