# <a name="compiler-warning-level-1-c4122"></a>Ostrzeżenie kompilatora (poziom 1) C4122

"Function": alloc_text stosowane tylko do funkcji z powiązaniem C

**Alloc_text** pragma dotyczy tylko funkcji zadeklarowanych za pomocą **extern c**. Nie można jej używać z zewnętrznymi funkcjami języka C++.

Pragma jest ignorowana.
