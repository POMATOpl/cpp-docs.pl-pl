# <a name="compiler-warning-level-1-c4122"></a>Kompilator ostrzeżenie (poziom 1) C4122

'Funkcja': alloc_text można stosować tylko do funkcji z powiązaniem C

**Alloc_text** pragma ma zastosowanie tylko do funkcji zadeklarowanych z **extern c**. Nie można używać z funkcji języka C++ zewnętrznej.

Pragmy jest ignorowany.
