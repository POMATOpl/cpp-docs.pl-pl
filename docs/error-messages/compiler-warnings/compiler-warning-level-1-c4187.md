# <a name="compiler-warning-level-1-c4187"></a>Kompilator C4187 ostrzegawcze (poziom 1)
\#atrybuty importu "attribute1" i "attribute2" są niezgodne; zarówno ignorowane  
  
 A [#import](../../preprocessor/hash-import-directive-cpp.md) instrukcja określona `no_implementation` i `implementation_only` atrybutów. Obie są ignorowane.
