# <a name="compiler-warning-level-1-c4187"></a>Kompilator ostrzeżenie (poziom 1) C4187

\#atrybuty importowania "attribute1" i "attribute2" są niezgodne; oba są ignorowane

A [#import](../../preprocessor/hash-import-directive-cpp.md) instrukcja określona `no_implementation` i `implementation_only` atrybutów. Obie są ignorowane.
