# <a name="compiler-warning-level-1-c4187"></a>Ostrzeżenie kompilatora (poziom 1) C4187

\#atrybuty importu "Attribute1" i "attribute2" są niezgodne; Oba zignorowano

Określona [#import](../../preprocessor/hash-import-directive-cpp.md) instrukcji `no_implementation` i `implementation_only`. Oba są ignorowane.
