# <a name="compiler-warning-level-1-c4340"></a>Ostrzeżenie kompilatora (poziom 1) C4340

"value": wartość opakowano z dodatniej do ujemnej wartości

Wartość `enum` jest większa niż największa `enum` wartość dodatnia zawiniętej do wartości ujemnej.
