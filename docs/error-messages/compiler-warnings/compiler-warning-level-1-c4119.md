# <a name="compiler-warning-level-1-c4119"></a>Ostrzeżenie kompilatora (poziom 1) C4119

określono różne podstawowe typy "Base1 —" i "Base2 —"

Dwa wskaźniki na podstawie są niezgodne, ponieważ mają różne bazy. Kompilator nie może konwertować między nimi.
