# <a name="compiler-warning-level-1-c4158"></a>Kompilator ostrzeżenie (poziom 1) C4158

> przy założeniu #pragma pointers_to_members (full_generality, dziedziczenie)

A **#pragma pointers_to_members (** *pojedynczego* &#124; *wielu* &#124; *wirtualnego* **)** został wystawiony bez towarzyszącego **#pragma pointers_to_members(full_generality)**.
