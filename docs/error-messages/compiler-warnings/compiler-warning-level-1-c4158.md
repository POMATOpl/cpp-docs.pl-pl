# <a name="compiler-warning-level-1-c4158"></a>Kompilator C4158 ostrzegawcze (poziom 1)
Zakładając, że pointers_to_members #pragma (full_generality, dziedziczenia)  
  
 A **#pragma pointers_to_members (***pojedynczy &#124; wielu &#124; wirtualnego***)** został wystawiony bez towarzyszącego **#pragma pointers_to_ members(full_generality)**.
