# <a name="compiler-warning-level-1-c4089"></a>Ostrzeżenie kompilatora (poziom 1) C4089

"Function": różne typy w rzeczywistym parametrze "number", formalny parametr "number"

Odpowiadające im parametry formalne i rzeczywiste mają różne typy. Rzeczywisty parametr jest przesyłany bez zmiany. Funkcja rzutuje rzeczywisty parametr na typ określony w definicji funkcji.
