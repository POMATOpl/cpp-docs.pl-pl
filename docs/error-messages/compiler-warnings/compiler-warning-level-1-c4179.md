# <a name="compiler-warning-level-1-c4179"></a>Kompilator ostrzeżenie (poziom 1) C4179

> "//*": analizowany jako "/" i "/\*": pomyłka ze standardowym "/ /" komentarzy

__//\*__ jest ogranicznikiem komentaraz niepoprawne. Użyj __//__ lub __/ \*__ zamiast tego.
