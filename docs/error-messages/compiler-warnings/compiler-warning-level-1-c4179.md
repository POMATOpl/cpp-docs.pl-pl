# <a name="compiler-warning-level-1-c4179"></a>Ostrzeżenie kompilatora (poziom 1) C4179

> "//*": przeanalizowane jako "/" i "/\*": pomyłka ze standardowym komentarzem "//"

__\*//__ jest nieprawidłowym ogranicznikiem komentarza. Zamiast tego użyj __//__ lub __/\*__ .
