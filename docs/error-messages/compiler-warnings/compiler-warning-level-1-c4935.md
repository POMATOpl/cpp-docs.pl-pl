# <a name="compiler-warning-level-1-c4935"></a>Ostrzeżenie kompilatora (poziom 1) C4935

specyfikator dostępu do zestawu został zmodyfikowany z "Access"

Widoczność zestawu typu została zmodyfikowana. Kompilator używa ostatniego specyfikatora, którego napotka. Na przykład widoczność zestawu deklaracji do przodu może różnić się od widoczności zestawu klasy.

C4935 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
