# <a name="compiler-warning-level-4-c4053"></a>Kompilator C4053 ostrzegawcze (poziom 4)
jeden operand void dla "?:"  
  
 `?:` Uzyskuje operator wyrażenia typu `void`. Wartość `void` argument operacji jest niezdefiniowany.
