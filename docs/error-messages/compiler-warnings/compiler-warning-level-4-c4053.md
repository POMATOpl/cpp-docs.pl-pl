# <a name="compiler-warning-level-4-c4053"></a>Ostrzeżenie kompilatora (poziom 4) C4053

jeden operand void dla "?:"

Operator `?:` ma wyrażenie typu `void`. Wartość operandu `void` jest niezdefiniowana.
