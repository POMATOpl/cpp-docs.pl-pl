# <a name="compiler-warning-level-1-c4034"></a>Kompilator C4034 ostrzegawcze (poziom 1)
sizeof zwraca 0  
  
 `sizeof` Operator jest stosowany do argumentu operacji o rozmiarze zero (pustą strukturą, Unii, klasy lub typu wyliczeniowego lub typu `void`).
