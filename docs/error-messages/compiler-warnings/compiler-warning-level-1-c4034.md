# <a name="compiler-warning-level-1-c4034"></a>Kompilator ostrzeżenie (poziom 1) C4034

Operator sizeof zwraca wartość 0

`sizeof` Operator jest stosowany do argumentu operacji o rozmiarze zero (pusty struktury, Unii, klasy lub typu wyliczeniowego, lub typu `void`).
