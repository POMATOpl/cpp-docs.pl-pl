# <a name="compiler-warning-level-1-c4034"></a>Ostrzeżenie kompilatora (poziom 1) C4034

sizeof zwraca 0

**`sizeof`** Operator jest stosowany do operandu o rozmiarze zero (pustej struktury, Unii, klasy lub typu wyliczeniowego lub typu **`void`** ).
