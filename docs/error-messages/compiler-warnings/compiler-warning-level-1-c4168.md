# <a name="compiler-warning-level-1-c4168"></a>Ostrzeżenie kompilatora (poziom 1) C4168

ograniczenie kompilatora: nie ma typów debugera, Usuń program bazy danych "Database" i Skompiluj ponownie

Plik bazy danych programu musi zostać odbudowany w celu uwzględnienia wszystkich typów w programie.
