# <a name="compiler-warning-level-1-c4168"></a>Kompilator C4168 ostrzegawcze (poziom 1)
ograniczenie kompilatora: typów debugera, Usuń program bazy danych "baza danych" i skompiluj ponownie  
  
 Plik bazy danych programu musi zostać również przebudowany w celu uwzględnienia wszystkich typów w programie.
