# <a name="compiler-warning-level-1-c4168"></a>Kompilator ostrzeżenie (poziom 1) C4168

ograniczenie kompilatora: nie ma typów debugera, Usuń program bazy danych "bazy danych" i ponownie skompiluj

Plik bazy danych programu musi zostać zrekompilowany, aby pomieścić wszystkie typy w programie.
