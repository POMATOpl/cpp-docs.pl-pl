# <a name="compiler-warning-level-3-c4622"></a>Ostrzeżenie kompilatora (poziom 3) C4622

Zastępowanie informacji o debugowaniu utworzonych podczas tworzenia prekompilowanego nagłówka w pliku obiektu: "plik"

Informacje CodeView w określonym pliku zostały utracone, gdy zostały skompilowane przy użyciu opcji [/Yu](../../build/reference/yu-use-precompiled-header-file.md) (Użyj prekompilowanych nagłówków).

Zmień nazwę pliku obiektu (przy użyciu [/fo](../../build/reference/fo-object-file-name.md)) podczas tworzenia lub używania prekompilowanego pliku nagłówkowego i Połącz przy użyciu nowego pliku obiektu.
