# <a name="compiler-warning-level-1-c4445"></a>Ostrzeżenie kompilatora (poziom 1) C4445

"Function": w WinRT lub typie zarządzanym metoda wirtualna nie może być prywatna

Jeśli wirtualna funkcja jest prywatna, nie można uzyskać do niej dostępu za pomocą typu pochodnego. Aby naprawić ten błąd, Zmień dostępność wirtualnej funkcji członkowskiej na chronioną lub publiczną.
