# <a name="compiler-warning-level-1-c4022"></a>Ostrzeżenie kompilatora (poziom 1) C4022

"Function": niezgodność wskaźnika dla rzeczywistego parametru "number"

Typ wskaźnika rzeczywistego parametru różni się od typu wskaźnika odpowiadającego parametru formalnego. Rzeczywisty parametr jest przesyłany bez zmiany.
