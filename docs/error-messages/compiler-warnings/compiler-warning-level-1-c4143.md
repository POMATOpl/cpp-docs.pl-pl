# <a name="compiler-warning-level-1-c4143"></a>Kompilator C4143 ostrzegawcze (poziom 1)
pragma "same_seg" nie jest obsługiwany; Użyj alokacji __based  
  
 **#Pragma same_seg** nie jest już obsługiwana. Użyj [__based](../../cpp/based-pointers-cpp.md) słowa kluczowego zamiast tego.
