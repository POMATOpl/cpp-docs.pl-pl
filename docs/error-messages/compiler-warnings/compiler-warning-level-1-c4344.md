# <a name="compiler-warning-level-1-c4344"></a>Ostrzeżenie kompilatora (poziom 1) C4344

zmiana zachowania: użycie argumentów jawnego szablonu w wywołaniu funkcji "Function"

Wywołanie funkcji przy użyciu jawnych argumentów szablonu wywołuje inną funkcję niż w przypadku, gdy nie określono jawnych argumentów
