# <a name="compiler-warning-level-3-c4306"></a>Ostrzeżenie kompilatora (poziom 3) C4306

> "*Identyfikator*": konwersja z "*Type1*" na "*Type2*" o większym rozmiarze

Identyfikator jest typem rzutowania na większy wskaźnik. Niewypełnione duże bity nowego typu będą wypełnione zerem.

To ostrzeżenie może wskazywać na niechcianą konwersję. Wskaźnik wyników może być nieprawidłowy.
