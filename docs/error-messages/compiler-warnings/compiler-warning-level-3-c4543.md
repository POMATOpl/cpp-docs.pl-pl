# <a name="compiler-warning-level-3-c4543"></a>Ostrzeżenie kompilatora (poziom 3) C4543

Tekst, który został pominięty przez atrybut "no_injected_text"

Atrybut [No_injected_text](../../windows/no-injected-text.md) pojawił się w kodzie źródłowym, co oznacza, że kompilator uniemożliwi wprowadzanie atrybutów do kodu.

To ostrzeżenie oznacza, że atrybuty nie będą mogły wstrzyknąć kodu.
