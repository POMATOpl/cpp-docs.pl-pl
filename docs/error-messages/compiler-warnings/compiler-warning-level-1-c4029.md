# <a name="compiler-warning-level-1-c4029"></a>Ostrzeżenie kompilatora (poziom 1) C4029

zadeklarowana formalna lista parametrów różni się od definicji

Formalne typy parametrów w deklaracji funkcji nie zgadzają się z tymi w definicji funkcji. Kompilator używa listy parametrów z definicji.
