# <a name="compiler-warning-level-1-c4166"></a>Kompilator C4166 ostrzegawcze (poziom 1)
**Niedozwolona konwencja wywołania dla konstruktora/destruktora**  
  
 Konstruktory i destruktory nie mogą mieć wywoływanie Konwencji innej niż domyślna platformy (z wyjątkiem przypadków, gdy zostaną jawnie określone **__clrcall**).
