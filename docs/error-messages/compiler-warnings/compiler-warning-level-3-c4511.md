# <a name="compiler-warning-level-3-c4511"></a>Ostrzeżenie kompilatora (poziom 3) C4511

"Class": nie można wygenerować konstruktora kopiującego

Kompilator nie może wygenerować domyślnego konstruktora kopiującego dla klasy; Klasa bazowa może mieć prywatny Konstruktor kopiujący.
