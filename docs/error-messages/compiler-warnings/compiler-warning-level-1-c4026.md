# <a name="compiler-warning-level-1-c4026"></a>Kompilator C4026 ostrzegawcze (poziom 1)
Funkcja zadeklarowana z formalną listą parametrów  
  
 Deklaracja funkcji ma formalnych parametrów, ale nie ma definicji funkcji. Kolejne wywołania tej funkcji założono, że funkcja nie przyjmuje żadnych parametrów.
