# <a name="compiler-warning-level-4-c4132"></a>Kompilator C4132 ostrzegawcze (poziom 4)
"object": obiekt const powinien być inicjowany.  
  
 Stała nie został zainicjowany. Wartość symbol **const** atrybutu nie można zmienić po jego definicji, należy wartość inicjowania.
