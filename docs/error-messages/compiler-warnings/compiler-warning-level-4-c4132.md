# <a name="compiler-warning-level-4-c4132"></a>Kompilator ostrzeżenie (poziom 4) C4132

"object": obiekt const powinien zostać zainicjowany

Stała nie został zainicjowany. Wartość symbolu za pomocą **const** atrybutu nie można zmienić po jego definicji, powinien być podawany wartość inicjalizacji.
