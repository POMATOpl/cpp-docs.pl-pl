# <a name="compiler-warning-level-4-c4132"></a>Ostrzeżenie kompilatora (poziom 4) C4132

"Object": obiekt const powinien zostać zainicjowany

Stała nie została zainicjowana. Wartości symbolu z atrybutem **const** nie można zmienić po jego definicji, należy nadać wartość inicjalizacji.
