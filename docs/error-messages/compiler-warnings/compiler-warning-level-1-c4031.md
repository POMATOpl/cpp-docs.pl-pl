# <a name="compiler-warning-level-1-c4031"></a>Kompilator ostrzeżenie (poziom 1) C4031

druga formalna lista parametrów dłuższa niż pierwsza lista

Funkcja jest ponownie zadeklarowany za pomocą różnych parametrów formalnych. Kompilator używa parametrów formalnych w pierwszej deklaracji.
