# <a name="compiler-warning-level-1-c4031"></a>Kompilator C4031 ostrzegawcze (poziom 1)
druga formalna lista parametrów dłuższa niż pierwsza lista  
  
 Funkcja jest ponownie zadeklarowany z różnych parametrów formalnych. Kompilator używa parametrów formalnych podany w pierwszej deklaracji.
