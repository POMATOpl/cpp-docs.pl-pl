# <a name="compiler-warning-level-1-c4031"></a>Ostrzeżenie kompilatora (poziom 1) C4031

Druga formalna lista parametrów dłuższa niż pierwsza lista

Funkcja jest ponownie zadeklarowana z różnymi parametrami formalnymi. Kompilator używa formalnych parametrów podanym w pierwszej deklaracji.
