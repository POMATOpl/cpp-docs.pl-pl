# <a name="compiler-warning-level-1-c4631"></a>Ostrzeżenie kompilatora (poziom 1) C4631

Program MSXML lub XPath nie jest dostępny, komentarze dokumentu XML nie zostaną przetworzone. reason

Twoja instalacja środowiska uruchomieniowego języka wspólnego nie miała plików niezbędnych do obsługi przetwarzania komentarzy do dokumentu. Ponownie zainstaluj środowisko uruchomieniowe języka wspólnego.
