# <a name="compiler-warning-level-1-c4048"></a>Kompilator ostrzeżenie (poziom 1) C4048

różne zadeklarowana indeksy dolne tablicy: "identifier1" i "identifier2"

Wyrażenie zawiera wskaźniki do tablic różnej wielkości. Wskaźniki są używane bez konwersji.

To ostrzeżenie, mogą zostać rozwiązane, jeśli jawnie rzutowane tablic do typu tej samej lub równoważnej.
