# <a name="compiler-warning-level-1-c4048"></a>Kompilator C4048 ostrzegawcze (poziom 1)
różne zadeklarowany indeksy tablicy: "identifier1" i "identifier2"  
  
 Wyrażenie zawiera wskaźniki do tablic inny rozmiar. Wskaźniki są używane bez użycia konwersji.  
  
 Może zostać ustalona to ostrzeżenie, jeśli jawne rzutowanie tablic typu tego samego lub równoważnego.
