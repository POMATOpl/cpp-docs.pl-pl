# <a name="compiler-warning-level-1-c4048"></a>Ostrzeżenie kompilatora (poziom 1) C4048

różne zadeklarowane indeksy dolne tablicy: "Identifier1" i "identifier2"

Wyrażenie obejmuje wskaźniki do tablic o różnym rozmiarze. Wskaźniki są używane bez konwersji.

To ostrzeżenie można naprawić, jeśli jawnie rzutuje tablic na ten sam lub równoważny typ.
