# <a name="compiler-warning-level-1-c4305"></a>Kompilator C4305 ostrzegawcze (poziom 1)
"identyfikator": obcięcie z "type1" na "type2"  
  
 Identyfikator jest konwertowany na mniejszy typ, co grozi utratą danych.  
  
 Jeśli używany jest Kreator punktu połączenia ATL, zobacz artykule bazy wiedzy:  
  
-   Q250847: Kreator punktu połączenia ATL wygenerować kod dla zdarzeń z argumentem typu VARIANT daje C4305 ostrzeżenie
