# <a name="compiler-warning-level-2-c4948"></a>Ostrzeżenie kompilatora (poziom 2) C4948

zwracany typ elementu "akcesor" jest niezgodny z typem ostatniego parametru odpowiadającej mu metody ustawiającej

Kompilator znalazł niezgodność między typem danych, który jest pobierany i ustawiany dla właściwości indeksowanej.

C4948 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
