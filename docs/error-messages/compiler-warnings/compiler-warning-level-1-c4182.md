# <a name="compiler-warning-level-1-c4182"></a>Ostrzeżenie kompilatora (poziom 1) C4182

\#Uwzględnij poziom zagnieżdżenia ma wartość "liczba" głębokości; możliwa Nieskończona rekursja

Kompilator zakończył miejsce na stercie ze względu na liczbę zagnieżdżonych plików dołączanych. Plik dołączany jest zagnieżdżony, gdy jest zawarty w innym pliku dołączanym.

Ten komunikat ma informacje i poprzedza błąd [C1076](../../error-messages/compiler-errors-1/fatal-error-c1076.md).
