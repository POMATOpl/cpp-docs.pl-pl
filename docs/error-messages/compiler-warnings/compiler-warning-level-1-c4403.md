# <a name="compiler-warning-level-1-c4403"></a>Kompilator C4403 ostrzegawcze (poziom 1)
Niedozwolony operator PTR  
  
 PTR operator służy niewłaściwie wbudowanego kodu asemblera.  
  
 Kompilator ignorowane PTR operator.
