# <a name="compiler-warning-level-1-c4403"></a>Kompilator ostrzeżenie (poziom 1) C4403

Niedozwolony operator PTR

PTR operator służy niewłaściwie kodem wbudowanego asemblera.

Kompilator ignorowane PTR operator.
