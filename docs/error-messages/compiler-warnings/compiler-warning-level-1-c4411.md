# <a name="compiler-warning-level-1-c4411"></a>Ostrzeżenie kompilatora (poziom 1) C4411

"Identyfikator": symbol jest rozpoznawany jako przemieszczenie rejestru

Identyfikator jest symbolem lokalnym, który jest rozpoznawany jako rejestr przemieszczenia i w związku z tym może być używany dla operandu z innym symbolem.
