# <a name="compiler-warning-level-1-c4406"></a>Ostrzeżenie kompilatora (poziom 1) C4406

Zignorowano operand dla dyrektywy

Dyrektywa nie przyjmuje żadnych operandów, ale określono operand.

Kompilator zignorował dany operand lub operandy.
