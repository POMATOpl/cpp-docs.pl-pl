# <a name="compiler-warning-c4962"></a>C4962 ostrzeżenia kompilatora
"Funkcja": profilowana Optymalizacja została wyłączona, ponieważ optymalizacje spowodowały niespójność danych profilu "  
  
 Funkcja nie został skompilowany z /LTCG:PGO, ponieważ zawodnych danych count (profil) dla funkcji. Wykonaj ponownie profilowanie, aby ponownie wygenerować plik .pgc, który zawiera dane zawodnych profilu dla tej funkcji.  
  
 To ostrzeżenie jest domyślnie wyłączone. Aby uzyskać więcej informacji, zobacz [kompilatora ostrzeżeń czy są wyłączone domyślnie](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
