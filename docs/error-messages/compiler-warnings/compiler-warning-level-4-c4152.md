# <a name="compiler-warning-level-4-c4152"></a>Ostrzeżenie kompilatora (poziom 4) C4152

niestandardowe rozszerzenie, konwersja funkcji/danych PTR w wyrażeniu

Wskaźnik funkcji jest konwertowany do lub ze wskaźnika danych. Ta konwersja jest dozwolona w ramach rozszerzeń Microsoft (/ze), ale nie poniżej ANSI C.
