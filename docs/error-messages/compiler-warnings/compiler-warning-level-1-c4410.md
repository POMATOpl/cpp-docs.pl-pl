# <a name="compiler-warning-level-1-c4410"></a>Ostrzeżenie kompilatora (poziom 1) C4410

niedozwolony rozmiar operandu

Jeden z operandów w instrukcji ma nieprawidłowy rozmiar. Został użyty najmniejszy dozwolony rozmiar operandu.
