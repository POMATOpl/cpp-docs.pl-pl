# <a name="compiler-warning-level-1-c4402"></a>Ostrzeżenie kompilatora (poziom 1) C4402

musi korzystać z operatora PTR

Typ jest używany w operandzie bez operatora PTR podczas odwoływania się do typu w kodzie zestawu wbudowanego lub do niego.

Kompilator zakłada, że jest operatorem PTR.
