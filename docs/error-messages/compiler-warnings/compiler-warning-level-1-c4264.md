# <a name="compiler-warning-level-1-c4264"></a>Ostrzeżenie kompilatora (poziom 1) C4264

"virtual_function": Brak dostępnego przesłonięcia dla wirtualnej funkcji składowej z klasy podstawowej "Class"; Funkcja jest ukryta

C4264 jest zawsze generowany po [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).

To ostrzeżenie jest domyślnie wyłączone. Aby uzyskać więcej informacji [, zobacz ostrzeżenia kompilatora, które są domyślnie wyłączone](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
