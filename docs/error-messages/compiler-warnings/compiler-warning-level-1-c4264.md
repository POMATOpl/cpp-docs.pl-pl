# <a name="compiler-warning-level-1-c4264"></a>Kompilator ostrzeżenie (poziom 1) C4264

'virtual_function': niedostępne zastąpienie dla funkcji wirtualnej składowej z klasy podstawowej 'class'; funkcja jest ukryta

C4264 był zawsze generowany po [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).

To ostrzeżenie jest domyślnie wyłączona. Zobacz [kompilatora ostrzeżenia, są wyłączone domyślnie](../../preprocessor/compiler-warnings-that-are-off-by-default.md) Aby uzyskać więcej informacji.
