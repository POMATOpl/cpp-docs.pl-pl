# <a name="compiler-warning-level-4-c4505"></a>Ostrzeżenie kompilatora (poziom 4) C4505

"Function": funkcja lokalna, do której nie istnieje odwołanie, została usunięta

Dana funkcja jest lokalna i nie jest przywoływana w treści modułu; w związku z tym funkcja jest martwym kodem.

Kompilator nie wygenerował kodu dla tej funkcji martwej.
