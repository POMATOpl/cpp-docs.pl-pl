# <a name="compiler-warning-level-1-c4088"></a>Ostrzeżenie kompilatora (poziom 1) C4088

"Function": niezgodność wskaźnika w rzeczywistym parametrze "number", formalny parametr "number"

Odpowiednie formalne i rzeczywiste parametry mają inny poziom pośredni. Rzeczywisty parametr jest przesyłany bez zmiany. Wywołana funkcja interpretuje swoją wartość jako wskaźnik.
