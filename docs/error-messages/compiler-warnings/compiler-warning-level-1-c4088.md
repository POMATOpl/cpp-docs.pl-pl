# <a name="compiler-warning-level-1-c4088"></a>Kompilator C4088 ostrzegawcze (poziom 1)
"Funkcja": niezgodność wskaźnika w rzeczywistym parametrze "number", "number" parametrów formalnych  
  
 Odpowiednie parametry formalne i rzeczywiste mają różne poziomy pośrednie. Rzeczywisty parametr jest przekazywany bez zmian. Wywoływana funkcja interpretuje jej wartość jako wskaźnik.
