# <a name="compiler-warning-level-1-c4088"></a>Kompilator ostrzeżenie (poziom 1) C4088

'Funkcja': niezgodność wskaźnika w rzeczywistym parametrze "number", parametr formalny "liczba"

Odpowiednie parametry formalne i rzeczywiste mają inny poziom pośrednictwa. Rzeczywisty parametr jest przekazywany bez zmian. Wywołana funkcja interpretuje wartość jako wskaźnik.
