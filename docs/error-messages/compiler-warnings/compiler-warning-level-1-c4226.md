# <a name="compiler-warning-level-1-c4226"></a>Ostrzeżenie kompilatora (poziom 1) C4226

> użyto niestandardowego rozszerzenia:*słowo kluczowe "Keyword*" jest przestarzałym słowem kluczowym

Bieżąca wersja Visual C++ nie używa tego słowa kluczowego.

To ostrzeżenie jest automatycznie podwyższana do błędu.
