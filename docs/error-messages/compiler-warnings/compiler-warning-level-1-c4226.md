# <a name="compiler-warning-level-1-c4226"></a>Kompilator C4226 ostrzegawcze (poziom 1)

> użyto niestandardowego rozszerzenia: "*— słowo kluczowe*" jest przestarzałym słowem kluczowym

Bieżąca wersja Visual C++ nie jest używane to słowo kluczowe.

To ostrzeżenie zostanie automatycznie podwyższony do wystąpił błąd.
