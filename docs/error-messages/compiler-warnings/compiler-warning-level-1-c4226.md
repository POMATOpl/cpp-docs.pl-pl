# <a name="compiler-warning-level-1-c4226"></a>Kompilator ostrzeżenie (poziom 1) C4226

> użyto niestandardowego rozszerzenia: "*— słowo kluczowe*" jest przestarzałym słowem kluczowym

Bieżącej wersji programu Visual C++ nie używa słowa kluczowego.

To ostrzeżenie zostanie automatycznie podwyższony do błędu.
