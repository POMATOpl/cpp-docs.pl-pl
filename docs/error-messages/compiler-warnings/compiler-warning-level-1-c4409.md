# <a name="compiler-warning-level-1-c4409"></a>Kompilator C4409 ostrzegawcze (poziom 1)
Niedozwolona instrukcja rozmiaru  
  
 Instrukcja nie miał formularza z określonym rozmiarem. Użyto najmniejszy rozmiar prawnych.
