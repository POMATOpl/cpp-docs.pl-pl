# <a name="compiler-warning-level-3-c4133"></a>Ostrzeżenie kompilatora (poziom 3) C4133

"Type": niezgodne typy — od "type1" do "type2"

To ostrzeżenie może być spowodowane próbą odejmowania dwóch wskaźników różnych typów.

Aby uniknąć tego ostrzeżenia, podaj odpowiednie rzutowanie typu.
