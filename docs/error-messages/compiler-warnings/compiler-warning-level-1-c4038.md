# <a name="compiler-warning-level-1-c4038"></a>Kompilator C4038 ostrzegawcze (poziom 1)
"modyfikator": niedozwolony otoczenia modyfikator klasy  
  
 Modyfikator nie może służyć do klas z **dllimport** lub [dllexport](../../cpp/dllexport-dllimport.md) atrybutów.
