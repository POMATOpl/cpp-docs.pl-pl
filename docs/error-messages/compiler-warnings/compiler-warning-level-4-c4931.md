# <a name="compiler-warning-level-4-c4931"></a>Ostrzeżenie kompilatora (poziom 4) C4931

> przyjęto, że biblioteka typów została skompilowana dla wskaźników *liczby*bitów

Jawne informacje nie zostały dostarczone z atrybutem **ptrsize** dyrektywy [#import](../../preprocessor/hash-import-directive-cpp.md) . Kompilator stwierdził, że rozmiar wskaźnika biblioteki typów to *Number*.

To ostrzeżenie jest domyślnie wyłączone. Aby uzyskać więcej informacji [, zobacz ostrzeżenia kompilatora, które są domyślnie wyłączone](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
