# <a name="compiler-warning-level-1-c4015"></a>Ostrzeżenie kompilatora (poziom 1) C4015

"Identyfikator": typ pola bitowego musi być całkowity

Pole bitowe nie jest zadeklarowane jako typ Integer. Kompilator zakłada, że typ podstawowy pola bitowego jest niepodpisany. Pola bitowe muszą być zadeklarowane jako typy całkowite bez znaku.
