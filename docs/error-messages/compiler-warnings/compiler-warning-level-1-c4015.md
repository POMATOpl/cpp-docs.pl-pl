# <a name="compiler-warning-level-1-c4015"></a>Kompilator C4015 ostrzegawcze (poziom 1)
'Identyfikator': typ pola bitowego musi być całkowity  
  
 Pole bitowe nie jest zadeklarowany jako typ integer. Kompilator założono typ podstawowy pola bitowego się bez znaku. Pola bitowe musi być zadeklarowany jako typami liczb całkowitych bez znaku.
