# <a name="compiler-warning-level-1-c4015"></a>Kompilator ostrzeżenie (poziom 1) C4015

'Identyfikator': typ pola bitowego musi być typu całkowitego

Pole bitowe nie został zadeklarowany jako typ liczby całkowitej. Kompilator zakłada podstawowy typ pola bitowego do będzie bez znaku. Pola bitowe musi być zadeklarowany jako typy liczb całkowitych bez znaku.
