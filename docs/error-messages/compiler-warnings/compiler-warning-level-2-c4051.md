# <a name="compiler-warning-level-2-c4051"></a>Ostrzeżenie kompilatora (poziom 2) C4051

Konwersja typu; możliwa utrata danych

Wyrażenie zawiera dwa elementy danych o różnych typach podstawowych. Konwersja jednego typu powoduje obcinanie elementu danych.

To ostrzeżenie może zostać naprawione w przypadku rzutowania elementów danych na odpowiedni typ.
