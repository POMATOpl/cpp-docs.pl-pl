# <a name="compiler-warning-level-1-c4120"></a>Ostrzeżenie kompilatora (poziom 1) C4120

niezgodność oparta/niepodstawowa

Kompilator nie może wykonać konwersji między dwoma wskaźnikami, ponieważ jeden z nich jest oparty, a drugi to nie.
