# <a name="compiler-warning-level-1-c4030"></a>Ostrzeżenie kompilatora (poziom 1) C4030

**Pierwsza formalna lista parametrów dłuższa niż druga lista**

Funkcja została ponownie zadeklarowana przy użyciu różnych parametrów formalnych. Kompilator używa formalnych parametrów podanym w pierwszej deklaracji.
