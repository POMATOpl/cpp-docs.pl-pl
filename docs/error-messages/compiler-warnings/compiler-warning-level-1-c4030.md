# <a name="compiler-warning-level-1-c4030"></a>Kompilator ostrzeżenie (poziom 1) C4030

**Pierwsza formalna lista parametrów dłuższa niż druga lista**

Funkcja została ponownie zadeklarowany za pomocą różnych parametrów formalnych. Kompilator używa parametrów formalnych w pierwszej deklaracji.
