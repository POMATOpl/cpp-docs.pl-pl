# <a name="compiler-warning-level-1-c4030"></a>Kompilator C4030 ostrzegawcze (poziom 1)
**Pierwsza formalna lista parametrów dłuższa niż druga lista**  
  
 Funkcja została ponownie zadeklarowany z różnych parametrów formalnych. Kompilator używa parametrów formalnych podany w pierwszej deklaracji.
