# <a name="compiler-warning-level-3-c4535"></a>Ostrzeżenie kompilatora (poziom 3) C4535

Wywołanie _set_se_translator () wymaga/EHa

Użycie [_set_se_translator](../../c-runtime-library/reference/set-se-translator.md) wymaga opcji kompilatora [/EHa](../../build/reference/eh-exception-handling-model.md) , a nie **/EHS**.

## <a name="example"></a>Przykład

Poniższy przykład generuje C4535.

```cpp
// C4535.cpp
// compile with: /W3 /EHsc /c
// C4535 expected
// to fix, compile with /EHa instead
#include <stdio.h>
#include <windows.h>
#include <eh.h>

void SEFunc();
void trans_func( unsigned int, EXCEPTION_POINTERS* );

class SE_Exception {
private:
   unsigned int nSE;
public:
   SE_Exception() {}
   SE_Exception( unsigned int n ) : nSE( n ) {}
   ~SE_Exception() {}
   unsigned int getSeNumber() { return nSE; }
};

int main() {
   try {
      _set_se_translator( trans_func );
      SEFunc();
   }
   catch( SE_Exception e ) {
      printf_s( "Caught a __try exception with SE_Exception.\n" );
   }
}

void SEFunc() {
   __try {
      int x, y=0;
      x = 5 / y;
   }
   __finally {
      printf_s( "In finally\n" );
   }
}

void trans_func( unsigned int u, EXCEPTION_POINTERS* pExp ) {
   printf_s( "In trans_func.\n" );
   throw SE_Exception();
}
```
