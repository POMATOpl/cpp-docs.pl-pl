# <a name="compiler-error-c3322"></a>Błąd kompilatora C3322

Właściwość "Property": nie jest prawidłową właściwością atrybutu "Attribute"

Nieprawidłowa lub nierozpoznana Właściwość (lub parametr) została przeniesiona do atrybutu. Zapoznaj się z dokumentacją dla atrybutu.
