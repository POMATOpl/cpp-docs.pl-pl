# <a name="compiler-error-c3322"></a>C3322 błąd kompilatora

"właściwość": nie jest prawidłową właściwością dla atrybutu "attribute"

Nieprawidłowy lub nierozpoznany właściwości (lub parametr) został przekazany do atrybutu. Zapoznaj się z dokumentacją dla atrybutu.
