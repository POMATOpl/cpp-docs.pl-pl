# <a name="compiler-error-c3322"></a>C3322 błąd kompilatora
"właściwość": nie jest prawidłową właściwością dla atrybutu 'atrybut'  
  
 Do atrybutu przekazano nieprawidłową lub nierozpoznaną właściwość (lub parametru). Sprawdź dokumentację dla atrybutu.
