# <a name="compiler-error-c2752"></a>Błąd kompilatora C2752

"template": więcej niż jedna specjalizacja częściowa jest zgodna z listą argumentów szablonu

Tworzenie wystąpienia było niejednoznaczne.

Poniższy przykład generuje C2752:

```cpp
// C2752.cpp
template<class T, class U>
struct A {};

template<class T, class U>
struct A<T*, U> {};

template<class T, class U>
struct A<T,U*> {};

// try the following line instead
// template<class T, class U> struct A<T*,U*> {};

int main() {
   A<char*,int*> a;   // C2752 an instantiation

   // OK
   A<char*,int> a1;
   A<char,int*> a2;
   A<char,int> a3;
}
```
