# <a name="compiler-error-c2704"></a>Błąd kompilatora C2704

'Identyfikator': __va_start dozwolone tylko w varargs wewnętrzne

`__va_start` Wewnętrzne jest używany w deklaracji funkcji ze stałą liczbą argumentów.
