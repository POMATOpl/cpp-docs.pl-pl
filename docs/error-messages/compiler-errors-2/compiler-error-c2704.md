# <a name="compiler-error-c2704"></a>C2704 błąd kompilatora
"identyfikator": __va_start dozwolone tylko w przypadku typu varargs — wewnętrzne  
  
 `__va_start` Wewnętrznej jest używany w deklaracji funkcji ze stałą liczbą argumentów.
