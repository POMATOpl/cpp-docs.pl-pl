# <a name="compiler-error-c3855"></a>C3855 błąd kompilatora
"class": parametr typu "param" jest niezgodny z deklaracją  
  
 Kompilator znaleźć szablonu stałej parametryzującej szablon lub parametrów ogólnych o różnych nazwach. Taka sytuacja może wystąpić, gdy parametr szablonu określona w definicji specjalizacja szablonu jest niezgodny z jej deklaracją.  
  
 Poniższy przykład generuje C3855:  
  
```  
// C3855.cpp  
template <int N>  
struct C {  
   void f();  
};  
  
template <char N>  
void C<N>::f() {}   // C3855  
```  
  
 Możliwe rozwiązanie:  
  
```  
// C3855b.cpp  
// compile with: /c  
template <int N>  
struct C {  
   void f();  
};  
  
template <int N>  
void C<N>::f() {}  
```  
  
 C3855 może również wystąpić, gdy użycie typów ogólnych:  
  
```  
// C3855c.cpp  
// compile with: /clr  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
generic <class V>  
ref struct GC1<T>::GC2 { };   // C3855  
```  
  
 Możliwe rozwiązanie:  
  
```  
// C3855d.cpp  
// compile with: /clr /c  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
ref struct GC1<T>::GC2 { };  
```
