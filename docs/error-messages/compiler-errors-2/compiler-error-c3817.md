# <a name="compiler-error-c3817"></a>Błąd kompilatora C3817

"Deklaracja": Właściwość może być stosowana tylko do funkcji

**`property`** Słowo kluczowe może być stosowane tylko do definicji funkcji.

C3817 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
