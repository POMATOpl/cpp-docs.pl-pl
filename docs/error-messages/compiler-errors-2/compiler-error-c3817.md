# <a name="compiler-error-c3817"></a>Błąd kompilatora C3817

"deklaracją": właściwość może zostać zastosowana tylko do funkcji

`property` — Słowo kluczowe może być tylko zastosowany do definicji funkcji.

C3817 jest dostępna, przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
