# <a name="compiler-error-c3817"></a>C3817 błąd kompilatora
"deklaracją": właściwość może być stosowany tylko do funkcji  
  
 `property` — Słowo kluczowe może być tylko zastosowany do definicji funkcji.  
  
 C3817 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
