# <a name="compiler-error-c3818"></a>Błąd kompilatora C3818

Deklaracja właściwości tablicy "właściwość1" nie powinny przeciążać właściwości indeksu "property2"

Przeciążenie nie jest możliwe dla właściwości, gdy jeden z nich jest indeksatora, a drugi to właściwości tablicy.

C3818 jest dostępna, przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
