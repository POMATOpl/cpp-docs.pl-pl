# <a name="compiler-error-c3818"></a>Błąd kompilatora C3818

Deklaracja właściwości tablicy "Property1" nie powinna przeciążać właściwości indeksu "Property2"

Przeciążenie nie jest możliwe dla właściwości, gdy jeden jest indeksatorem, a drugi jest właściwością tablicy.

C3818 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
