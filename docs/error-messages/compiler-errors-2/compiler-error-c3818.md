# <a name="compiler-error-c3818"></a>C3818 błąd kompilatora
Deklaracja właściwości tablicy "właściwości property1" nie powinny przeciążać właściwości indeksu "property2"  
  
 Przeciążenia nie jest możliwe dla właściwości, gdy jest jeden indeksator i innych właściwości tablicy. 
  
 C3818 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
