# <a name="compiler-error-c3762"></a>C3762 błąd kompilatora
Nie można przetworzyć atrybutu 'atrybut'  
  
 Zdefiniowane przez użytkownika atrybutu, która dziedziczy `System.Security.Permissions.SecurityAttribute` jest używany do definiowania atrybutu zabezpieczeń. Takiego atrybutu nie można użyć w tym samym zestawie, w którym jest zdefiniowana.
