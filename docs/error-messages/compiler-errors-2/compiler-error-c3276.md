# <a name="compiler-error-c3276"></a>Błąd kompilatora C3276

"— słowo kluczowe": skok poza __finally/blok finally ma niezdefiniowane zachowanie podczas obsługi zakończenia

Ten błąd jest taka sama jak [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) ostrzeżenie. Jednakże, korzystając z/CLR, ten warunek nie można wyłączyć za pomocą [ostrzeżenie](../../preprocessor/warning.md) pragmy.
