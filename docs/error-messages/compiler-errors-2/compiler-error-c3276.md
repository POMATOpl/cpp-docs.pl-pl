# <a name="compiler-error-c3276"></a>C3276 błąd kompilatora
"— słowo kluczowe": skok poza __finally/bloku finally ma niezdefiniowane zachowanie podczas obsługi zakończenia  
  
 Ten błąd jest taka sama jak [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) ostrzeżenie. Jednakże, korzystając z/CLR, ten warunek nie może być wyłączone z [ostrzeżenie](../../preprocessor/warning.md) pragma.
