# <a name="compiler-error-c3276"></a>Błąd kompilatora C3276

"słowo kluczowe": Wyskocz z __finally bloku/finally ma niezdefiniowane zachowanie podczas obsługi zakończenia

Ten błąd jest taki sam jak ostrzeżenie [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) . Jednak w przypadku korzystania z opcji/CLR ten warunek nie może zostać wyłączony za pomocą dyrektywy pragma [ostrzeżenia](../../preprocessor/warning.md) .
