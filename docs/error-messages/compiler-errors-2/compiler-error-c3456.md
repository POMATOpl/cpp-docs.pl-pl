# <a name="compiler-error-c3456"></a>C3456 błąd kompilatora
[source_annotation_attribute] niedozwolone w zarządzanych lub deklaracji klasy WinRT  
  
 source_annotation_attribute służy do definiowania atrybutów niestandardowych, które mają być używane przez analizę kodu.  
  
 Te atrybuty niestandardowe tylko są przydatne, jeśli istnieje odpowiedni wtyczkę, która obsługuje usługę atrybutu.
