# <a name="compiler-error-c2855"></a>Błąd kompilatora C2855

Opcja wiersza polecenia "Option" jest niespójna z prekompilowanym nagłówkiem

Dana opcja wiersza polecenia różni się od opcji używanej do tworzenia prekompilowanego nagłówka. Wygeneruj ponownie prekompilowany nagłówek z podaną opcją wiersza polecenia.
