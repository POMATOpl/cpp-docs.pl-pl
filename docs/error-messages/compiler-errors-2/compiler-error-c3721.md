# <a name="compiler-error-c3721"></a>C3721 błąd kompilatora
"sygnatura": niezgodny podpis dla zdarzenia  
  
 Zdarzenie zostało niepoprawnie zadeklarowany. Aby uzyskać więcej informacji, zobacz [__event](../../cpp/event.md).  
  
 C3721 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
