# <a name="compiler-error-c3721"></a>Błąd kompilatora C3721

"Signature": niezgodny podpis dla zdarzenia

Zdarzenie zostało zadeklarowane nieprawidłowo. Aby uzyskać więcej informacji, zobacz [__event](../../cpp/event.md).

C3721 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
