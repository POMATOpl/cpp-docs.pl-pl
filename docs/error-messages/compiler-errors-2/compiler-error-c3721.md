# <a name="compiler-error-c3721"></a>Błąd kompilatora C3721

"podpis": niezgodny podpis dla zdarzenia

Zdarzenie zostało zadeklarowane niepoprawnie. Aby uzyskać więcej informacji, zobacz [__event](../../cpp/event.md).

C3721 jest dostępna, przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
