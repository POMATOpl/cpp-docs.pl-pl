# <a name="compiler-error-c2510"></a>Błąd kompilatora C2510

"Identyfikator": po lewej stronie "::" musi być klasą/strukturą/Unią

Nazwa klasy, struktury lub Unii musi znajdować się po lewej stronie operatora rozpoznawania zakresu ( `::` ).
