# <a name="compiler-error-c3619"></a>Błąd kompilatora C3619

Nie można deklarować szablonu w ramach zarządzanej lub WinRT typu

Szablony klas nie są dozwolone w zarządzanej lub WinRT klasy lub interfejsu.

C3619 jest dostępna, przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
