# <a name="compiler-error-c3619"></a>Błąd kompilatora C3619

nie można zadeklarować szablonu w typie zarządzanym lub WinRT

Szablony klas są niedozwolone w klasie zarządzanej lub w interfejsie WinRT.

C3619 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
