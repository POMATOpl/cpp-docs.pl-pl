# <a name="compiler-error-c3619"></a>C3619 błąd kompilatora
Szablon nie może zostać zadeklarowany wewnątrz zarządzanego lub WinRT typu  
  
Szablony klas nie są dozwolone w zarządzanych lub WinRT klasy lub interfejsu.  
  
C3619 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
