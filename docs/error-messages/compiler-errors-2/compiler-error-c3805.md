# <a name="compiler-error-c3805"></a>C3805 błąd kompilatora
'token': nieoczekiwany token, oczekiwano albo "}" lub identyfikatora  
  
 Podczas definiowania właściwości, napotkano nieprawidłowy token. Usuń nieprawidłowy token.
