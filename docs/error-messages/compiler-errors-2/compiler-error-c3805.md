# <a name="compiler-error-c3805"></a>Błąd kompilatora C3805

"token": nieoczekiwany token, oczekiwano albo "}" lub identyfikator

Podczas definiowania właściwości, napotkano nieprawidłowy token. Usuń nieprawidłowy token.
