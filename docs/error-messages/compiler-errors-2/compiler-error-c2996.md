# <a name="compiler-error-c2996"></a>Błąd kompilatora C2996

'Funkcja': definicja szablonu funkcji cyklicznej

Definicja funkcji próbuje utworzyć wystąpienia klasy oparte na szablonach jego katalogu głównego. Wystąpień szablonu cykliczne są niedozwolone.
