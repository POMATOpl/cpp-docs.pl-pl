# <a name="compiler-error-c2996"></a>C2996 błąd kompilatora
"Funkcja": definicja szablonu funkcji cyklicznej  
  
 Definicja funkcji próbuje utworzyć wystąpienia klasy szablonu jego głównego. Cykliczne szablonu wystąpień nie są dozwolone.
