# <a name="compiler-error-c2839"></a>C2839 błąd kompilatora
Nieprawidłowy typ zwracany "type" dla przeciążonego "operator ->"  
  
 `->` Operator musi zwracać klasy, struktury, lub Unii lub odwołanie do jednej.
