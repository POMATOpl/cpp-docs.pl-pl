# <a name="compiler-error-c2839"></a>Błąd kompilatora C2839

Nieprawidłowy typ zwracany "type" dla przeciążonego "operator ->"

`->` Operatora musi zwracać klasy, struktury, lub Unii lub odwołanie do jednej.
