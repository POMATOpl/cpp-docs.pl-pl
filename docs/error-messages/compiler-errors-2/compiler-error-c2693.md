# <a name="compiler-error-c2693"></a>Błąd kompilatora C2693

"operator": niedozwolone porównanie dla odwołań do tablicy Managed lub WinRT

Nie można testować tablicy zarządzanej ani WinRT dla dowolnego rodzaju nierówności. Można na przykład sprawdzić, czy tablice zarządzane są równe, ale nie można przeprowadzić testu, aby zobaczyć, czy jedna tablica jest większa lub mniejsza od innej tablicy.
