# <a name="compiler-error-c2656"></a>Błąd kompilatora C2656

"Function": funkcja nie jest dozwolona jako pole bitowe

Funkcja jest zadeklarowana jako składowa pola bitowego.

Ten błąd może być spowodowany błędem składniowym na liście inicjatora konstruktora.
