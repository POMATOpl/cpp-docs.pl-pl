# <a name="compiler-error-c2830"></a>Błąd kompilatora C2830

tylko parametry umieszczania w "operator new" mogą mieć wartości domyślne

Standardowe parametry formalne dla [operatora new](../../standard-library/new-operators.md#op_new) nie mogą mieć wartości domyślnych. Tylko parametry położenia zdefiniowane przez użytkownika mogą określać wartości domyślne.
