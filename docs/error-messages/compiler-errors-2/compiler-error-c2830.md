# <a name="compiler-error-c2830"></a>C2830 błąd kompilatora
tylko umieszczenie parametrów dla "operator new" może mieć wartości domyślnej  
  
Standardowa parametrów formalnych [nowy operator](../../standard-library/new-operators.md#op_new) nie może mieć wartości domyślne. Tylko parametry umieszczania zdefiniowanych przez użytkownika można określić wartości domyślnych.
