# <a name="compiler-error-c2830"></a>Błąd kompilatora C2830

tylko umieszczenie parametrów dla "operator new" może mieć wartości domyślne

Standardowe Parametry formalne dla [nowy operator](../../standard-library/new-operators.md#op_new) nie może mieć wartości domyślne. Tylko parametry położenia zdefiniowanych przez użytkownika, można określić wartości domyślnych.
