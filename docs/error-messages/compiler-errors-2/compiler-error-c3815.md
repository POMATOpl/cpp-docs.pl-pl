# <a name="compiler-error-c3815"></a>Błąd kompilatora C3815

zwracany typ metody "get_accessor" musi być zgodny z typem ostatniego parametru metody ustawiającej

Podczas deklarowania właściwości wartość zwracana `get_accessor` metody musi być zgodna z ostatnim parametrem w deklaracji metody dostępu set.

C3815 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
