# <a name="compiler-error-c3134"></a>C3134 błąd kompilatora
"wartość": wartość argumentu atrybutu 'atrybut' nie ma prawidłowego typu "type"  
  
 Błąd składniowy został wykryty, jeśli wartość została przypisana do atrybutu.
