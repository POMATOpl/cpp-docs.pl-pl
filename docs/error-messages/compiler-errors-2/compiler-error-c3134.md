# <a name="compiler-error-c3134"></a>Błąd kompilatora C3134

"value": wartość argumentu atrybutu "Attribute" nie ma prawidłowego typu "Type"

Wykryto błąd składniowy, gdy wartość została przypisana do atrybutu.
