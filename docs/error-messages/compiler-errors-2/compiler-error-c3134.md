# <a name="compiler-error-c3134"></a>Błąd kompilatora C3134

"value": wartość argumentu atrybutu 'atrybut' nie ma prawidłowego typu "type"

Błąd składniowy zostało wykryte, gdy wartość przypisano do atrybutu.
