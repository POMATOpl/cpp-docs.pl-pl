# <a name="compiler-error-c3083"></a>Błąd kompilatora C3083

"Function": symbol po lewej stronie "::" musi być typem

Funkcja została wywołana nieprawidłowo.

## <a name="example"></a>Przykład

Poniższy przykład generuje C3083.

```cpp
// C3083.cpp
// compile with: /c
struct N {
   ~N();
};

struct N1 {
   ~N1();
};

N::N::~N() {}   // C3083
N1::~N1() {}   // OK
```
