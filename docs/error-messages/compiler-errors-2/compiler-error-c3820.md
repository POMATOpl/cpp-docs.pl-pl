# <a name="compiler-error-c3820"></a>Błąd kompilatora C3820

> "*name*": inicjatory muszą być zarządzane

## <a name="remarks"></a>Uwagi

Inicjator *nazwy* dla typu zarządzanego również musi być zarządzany. Aby naprawić ten błąd, Określ metodę inicjatora zarządzanego.
