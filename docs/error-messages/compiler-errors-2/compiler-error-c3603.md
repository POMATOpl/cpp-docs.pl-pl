# <a name="compiler-error-c3603"></a>Błąd kompilatora C3603

'Symbol': typ "Type" nie jest jeszcze obsługiwany

Podjęto próbę użycia typu danych, który nie jest jeszcze obsługiwany przez środowisko uruchomieniowe platformy .NET w zarządzanego obiektu.
