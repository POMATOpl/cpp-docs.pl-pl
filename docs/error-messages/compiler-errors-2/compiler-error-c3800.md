# <a name="compiler-error-c3800"></a>C3800 błąd kompilatora
"deklaracją": nie można mieszać właściwości i zdarzenia  
  
 Nie można zadeklarować konstrukcję zarówno właściwości i zdarzenia.  
  
 C3800 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
