# <a name="compiler-error-c3800"></a>Błąd kompilatora C3800

"deklaracją": nie można mieszać właściwości i zdarzenia

Nie można zadeklarować konstrukcję zarówno właściwości i zdarzenia.

C3800 jest dostępna, przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
