# <a name="compiler-error-c3800"></a>Błąd kompilatora C3800

"Deklaracja": nie można mieszać właściwości i zdarzeń

Nie można zadeklarować konstrukcji jako właściwości i zdarzenia.

C3800 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
