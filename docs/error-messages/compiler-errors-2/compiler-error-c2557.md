# <a name="compiler-error-c2557"></a>Błąd kompilatora C2557

"Identyfikator": prywatnych i chronionych składowych nie można zainicjować bez konstruktora

Tylko członkowie i znajomi mogą przypisywać wartość do prywatnego lub chronionego elementu członkowskiego. Niepubliczne składowe powinny być zainicjowane w konstruktorze klasy.
