# <a name="compiler-error-c2651"></a>C2651 błąd kompilatora
"typ danych": po lewej "operator" musi być klasy, struktury lub związku  
  
 Aby użyć parametru szablonu, jak w przypadku klasy, specialize szablonu klasy z klasy, nie typem całkowitym.
