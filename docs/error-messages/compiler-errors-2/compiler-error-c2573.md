# <a name="compiler-error-c2573"></a>Błąd kompilatora C2573

"Class": nie można usunąć wskaźników do obiektów tego typu; Klasa nie ma przeciążenia elementu "operator delete".

W klasie brakuje operatora usuwania bez umieszczania.
