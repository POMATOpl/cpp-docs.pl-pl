# <a name="compiler-error-c2573"></a>C2573 błąd kompilatora
"class": nie można usunąć wskaźników do obiektów tego typu; Klasa nie ma żadnych niezlokalizowanego przeciążenia dla "operator delete".  
  
 Klasa nie ma operatora delete-placement.
