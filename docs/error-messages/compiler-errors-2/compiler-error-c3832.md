# <a name="compiler-error-c3832"></a>Błąd kompilatora C3832

"Biblioteka typów": Biblioteka typów wygląda tak, jakby była skompilowana dla wskaźników 32-bitowych; Zmień kwalifikator "ptrsize"

Jawne informacje dostarczone z `ptrsize` atrybutem dyrektywy [#import](../../preprocessor/hash-import-directive-cpp.md) nie zgadzają się z elementem kompilatora znalezionym w bibliotece typów.
