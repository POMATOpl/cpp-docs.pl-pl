# <a name="compiler-error-c3832"></a>C3832 błąd kompilatora
'biblioteka typów': typ Biblioteka wygląda tak, jakby został zbudowany dla 32-bitowych wskaźników; Zmień kwalifikator "ptrsize"  
  
 Jawne informacja o `ptrsize` atrybutu [#import](../../preprocessor/hash-import-directive-cpp.md) dyrektywy nie zgadza się z kompilatora znaleziono w bibliotece typów.
