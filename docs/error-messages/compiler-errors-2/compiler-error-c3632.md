# <a name="compiler-error-c3632"></a>Błąd kompilatora C3632

"Event": niedozwolony styl zdarzenia dla konstrukcji

deklaracje [__event](../../cpp/event.md) nie są prawidłowe we wszystkich konstrukcjach.

C3632 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
