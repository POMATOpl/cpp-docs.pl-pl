# <a name="compiler-error-c2809"></a>C2809 błąd kompilatora
"operator operator" nie ma formalnych parametrów  
  
 Operator nie ma wymaganych parametrów.  
  
 Poniższy przykład generuje C2809:  
  
```  
// C2809.cpp  
// compile with: /c  
class A{};  
int operator+ ();   // C2809  
int operator+ (A);   // OK  
```
