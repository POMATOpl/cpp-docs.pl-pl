# <a name="compiler-error-c3752"></a>Błąd kompilatora C3752

"Attribute Class": nie można sklasyfikować atrybutu; słowo kluczowe "Keyword" nie powinno być używane w tym kontekście

Zdefiniowany przez użytkownika atrybut został niepoprawnie zastosowany.
