# <a name="compiler-error-c2817"></a>Błąd kompilatora C2817

typem zwracanym dla "operator delete" musi być "void"

Przeciążony [operatora delete](../../standard-library/new-operators.md#op_delete) funkcja nie zwraca wartości.
