# <a name="compiler-error-c2817"></a>Błąd kompilatora C2817

typem zwracanym dla "operator delete" musi być "void"

Funkcja usuwania przeciążonego [operatora](../../standard-library/new-operators.md#op_delete) nie może zwracać wartości.
