# <a name="compiler-error-c2532"></a>Błąd kompilatora C2532

"Identyfikator": niedozwolony modyfikator dla odwołania

Odwołanie zostało zmienione. Odwołań nie można modyfikować w celu odwoływania się do innego obiektu. Zamiast tego użyj wskaźnika.
