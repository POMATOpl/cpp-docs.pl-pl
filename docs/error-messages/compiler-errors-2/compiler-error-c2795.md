# <a name="compiler-error-c2795"></a>C2795 błąd kompilatora
"super::function" nie jest funkcją członkowską  
  
 Ten komunikat o błędzie pojawia się, gdy użytkownik próbuje użyć [super](../../cpp/super.md) uzyskać dostępu do członka innego niż funkcja elementu członkowskiego.
