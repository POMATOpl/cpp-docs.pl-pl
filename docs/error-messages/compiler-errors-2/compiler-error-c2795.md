# <a name="compiler-error-c2795"></a>Błąd kompilatora C2795

element "Super:: Function" nie jest funkcją składową

Ten komunikat o błędzie pojawia się za każdym razem, gdy spróbujesz użyć funkcji [Super](../../cpp/super.md) w celu uzyskania dostępu do elementu członkowskiego innego niż funkcja członkowska.
