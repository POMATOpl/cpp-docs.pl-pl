# <a name="compiler-error-c2795"></a>Błąd kompilatora C2795

"super::function" nie jest funkcją składową

Ten komunikat o błędzie jest wyświetlany zawsze wtedy, gdy użytkownik próbuje użyć [super](../../cpp/super.md) na uzyskiwanie dostępu do członka innego niż funkcja elementu członkowskiego.
