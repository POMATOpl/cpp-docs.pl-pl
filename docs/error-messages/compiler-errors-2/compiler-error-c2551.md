# <a name="compiler-error-c2551"></a>C2551 błąd kompilatora
"void *" typ wymaga jawnego rzutowania  
  
 A `void` wskaźnika jest przypisywany do wskaźnika nonvoid przez niejawnej konwersji. Musisz użyć jawnego rzutowania.
