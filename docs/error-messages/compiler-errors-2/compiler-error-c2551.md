# <a name="compiler-error-c2551"></a>Błąd kompilatora C2551

Typ "void *" wymaga jawnego rzutowania

`void` wskaźnik jest przypisywany do wskaźnika niebędącego nievoid przez niejawną konwersję. Musisz użyć jawnego rzutowania.
