# <a name="compiler-error-c2551"></a>Błąd kompilatora C2551

"void *" typ wymaga jawnego rzutowania

A `void` wskaźnik jest przypisywany do wskaźnika nonvoid przez niejawnej konwersji. Należy użyć jawnego rzutowania.
