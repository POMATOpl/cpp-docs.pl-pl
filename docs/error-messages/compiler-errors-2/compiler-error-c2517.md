# <a name="compiler-error-c2517"></a>Błąd kompilatora C2517

"Identyfikator": po prawej ':: ' jest niezdefiniowane

Identyfikator po prawej stronie operatora rozpoznawania zakresu (`::`) musi być zdefiniowanym członkiem klasy, struktury lub Unii po lewej stronie. Jeśli klasa, struktura lub Unia nie ma nazwy, identyfikator po prawej stronie musi być zadeklarowany z zakresem globalnym.
