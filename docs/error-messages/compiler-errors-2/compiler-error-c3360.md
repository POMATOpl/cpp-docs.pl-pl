# <a name="compiler-error-c3360"></a>Błąd kompilatora C3360

"String": nie można utworzyć nazwy

Wartość, która została przeniesiona do atrybutu [UUID](../../windows/uuid-cpp-attributes.md) , jest nieprawidłowa.

Poniższy przykład generuje C3360:

```cpp
// C3360.cpp
[ uuid("1") ]
// try this line instead
// [ uuid("12341234-1234-1234-1234-123412341234") ]
struct A   // C3360
{
};

int main()
{
}
```
