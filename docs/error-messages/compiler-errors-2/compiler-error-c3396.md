# <a name="compiler-error-c3396"></a>C3396 błąd kompilatora
'attribute': nie można odnaleźć w "namespace" atrybutu niestandardowego  
  
 C3396 wskazuje, że nie używasz aktualne środowisko uruchomieniowe języka wspólnego.  Zainstaluj ponownie Twoje CLR do rozpoznania.
