# <a name="compiler-error-c2859"></a>Błąd kompilatora C2859

Nazwa pliku nie jest typu pliku, który został użyty podczas tworzenia prekompilowanego pliku nagłówkowego, ponownie utwórz prekompilowany plik nagłówkowy.

Projekt bazy danych i prekompilowane pliki nagłówka musi zostać utworzona razem zapewnienie spójnego informacji. Ponownie skompiluj projekt, aby ponownie utwórz prekompilowany plik nagłówkowy.

Aby uzyskać informacji na temat wstępnie skompilowanych nagłówków, zobacz [/Y (prekompilowane nagłówki)](../../build/reference/y-precompiled-headers.md).
