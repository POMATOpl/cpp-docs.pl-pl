# <a name="compiler-error-c2859"></a>Błąd kompilatora C2859

Nazwa pliku nie jest plikiem typu, który został użyty podczas tworzenia prekompilowanego nagłówka, ponownie Utwórz prekompilowany nagłówek.

Aby zapewnić spójne informacje, należy utworzyć bazę danych projektu i prekompilowane pliki nagłówkowe. Skompiluj ponownie projekt, aby ponownie utworzyć prekompilowany nagłówek.

Aby uzyskać informacje na temat prekompilowanych nagłówków, zobacz [/y (prekompilowane nagłówki)](../../build/reference/y-precompiled-headers.md).
