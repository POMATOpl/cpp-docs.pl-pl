# <a name="compiler-error-c2859"></a>C2859 błąd kompilatora
Nazwa pliku nie jest plik typu, który był używany podczas tworzenia prekompilowanego nagłówka, ponownie utwórz prekompilowany nagłówek.  
  
 Projekt bazy danych i pliki prekompilowanego nagłówka musi zostać utworzony ze sobą w celu zapewnienia spójności informacji. Skompiluj ponownie projekt, aby odtworzyć prekompilowanego nagłówka.  
  
 Uzyskać informacji o prekompilowanych nagłówków, zobacz [/Y (prekompilowane nagłówki)](../../build/reference/y-precompiled-headers.md).
