# <a name="compiler-error-c2568"></a>Błąd kompilatora C2568

"Identifier1": nie można rozpoznać przeciążenia funkcji

Kompilator nie może określić, która przeciążona funkcja ma zostać wywołana. Rzeczywiste parametry przesłane do funkcji muszą być rzutowane w celu dopasowania do parametrów formalnych dla jednej ze przeciążonej funkcji, ale żadne dopasowanie nie jest jednoznacznie większe niż wszystkie pozostałe.
