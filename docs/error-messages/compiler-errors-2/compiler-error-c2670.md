# <a name="compiler-error-c2670"></a>Błąd kompilatora C2670

"Identyfikator": szablon funkcji nie może skonwertować numeru parametru z typu "Type"

Nie można przekonwertować parametru na wymagany typ.

Ten błąd może zostać rozwiązany w przypadku tworzenia jawnej konwersji.
