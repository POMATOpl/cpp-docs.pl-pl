# <a name="compiler-error-c2654"></a>C2654 błąd kompilatora
'Identyfikator': Podjęto próbę dostępu do członka poza funkcją członkowską  
  
 Element członkowski jest dostępny w deklaracji. Element członkowski danych są dostępne tylko w funkcjach Członkowskich.  
  
 Ten błąd może być spowodowany podczas próby zainicjowania zmiennych w deklaracji. W tym celu należy użyć konstruktora.
