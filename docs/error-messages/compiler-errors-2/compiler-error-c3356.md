# <a name="compiler-error-c3356"></a>C3356 błąd kompilatora
'attribute': nie można wywołać atrybutu multiemisji z w pełni kwalifikowana nazwa  
  
 Atrybut, który jest przetwarzany przez więcej niż jeden proces, na przykład, kompilatora i dostawcy ATL została określona nieprawidłowo.
