# <a name="compiler-error-c3356"></a>Błąd kompilatora C3356

"attribute": nie można wywołać atrybutu multiemisji z w pełni kwalifikowana nazwa

Atrybut, który jest przetwarzany przez więcej niż jeden proces, na przykład, kompilatora i dostawcy ATL została określona niepoprawnie.
