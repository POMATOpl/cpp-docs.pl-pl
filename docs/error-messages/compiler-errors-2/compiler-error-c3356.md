# <a name="compiler-error-c3356"></a>Błąd kompilatora C3356

"Attribute": nie można wywołać atrybutu multiemisji z w pełni kwalifikowaną nazwą

Atrybut, który jest przetwarzany przez więcej niż jeden proces, na przykład, kompilator i dostawca ATL, został niepoprawnie określony.
