# <a name="compiler-error-c2667"></a>C2667 błąd kompilatora
"Funkcja": żadne z przeciążeń numer nie posiada najlepszej konwersji  
  
 Wywołanie przeciążonej funkcji jest niejednoznaczne i nie można rozpoznać.  
  
 Konwersja musi odpowiadać rzeczywistych parametrów w wywołaniu funkcji do jednej z funkcji przeciążenia musi być ściśle większą niż konwersje wymagane przez wszystkich innych przeciążonej funkcji.  
  
 Zobacz artykuł bazy wiedzy Knowledge Base Q240869, aby uzyskać więcej informacji na częściowe porządkowanie szablonów funkcji.
