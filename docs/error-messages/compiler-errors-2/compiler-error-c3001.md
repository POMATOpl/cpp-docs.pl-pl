# <a name="compiler-error-c3001"></a>C3001 błąd kompilatora
"error_text": Oczekiwano nazwy dyrektywy OpenMP  
  
 `omp` Pragma musi występować w dyrektywie.  
  
 Poniższy przykład generuje C3001:  
  
```  
// C3001.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp   // C3001 missing token  
}  
```
