# <a name="compiler-error-c3500"></a>Błąd kompilatora C3500

Nieprawidłowy ProgID "progid"

Określono nieprawidłowy identyfikator progid z `#import` instrukcji. Sprawdź Rejestr Windows, aby upewnić się, że określono prawidłowego identyfikatora progid.
