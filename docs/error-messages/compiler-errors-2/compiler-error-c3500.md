# <a name="compiler-error-c3500"></a>Błąd kompilatora C3500

nieprawidłowy atrybut ProgID "ProgID"

Określono nieprawidłowy ProgID z `#import` instrukcją. Sprawdź rejestr systemu Windows, aby upewnić się, że określisz prawidłowy identyfikator ProgID.
