# <a name="compiler-error-c3500"></a>C3500 błąd kompilatora
Nieprawidłowy atrybut ProgID "progid"  
  
 Określono nieprawidłowy atrybut progid z `#import` instrukcji. Sprawdź w rejestrze systemu Windows, aby upewnić się, że określono prawidłową progid.
