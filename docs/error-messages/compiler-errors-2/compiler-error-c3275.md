# <a name="compiler-error-c3275"></a>C3275 błąd kompilatora
"element członkowski wyliczenia": nie można użyć tego symbolu bez kwalifikatora  
  
Korzystając z kodu zarządzanego i zawiera co najmniej dwa wyliczenia identyfikatora o takiej samej nazwie, należy określić jawnie odwołanie do identyfikatora.  
  
C3275 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
