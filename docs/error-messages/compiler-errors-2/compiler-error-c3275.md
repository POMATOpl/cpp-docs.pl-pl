# <a name="compiler-error-c3275"></a>Błąd kompilatora C3275

"składową wyliczenia": nie można użyć tego symbolu bez kwalifikatora

Korzystając z kodu zarządzanego i co najmniej dwóch wyliczenia zawiera identyfikatora o takiej samej nazwie, należy jawnie kwalifikujesz się do odwołania do identyfikatora.

C3275 jest dostępna, przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
