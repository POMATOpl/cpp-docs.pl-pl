# <a name="compiler-error-c3275"></a>Błąd kompilatora C3275

"składowa wyliczenia": nie można użyć tego symbolu bez kwalifikatora

W przypadku korzystania z kodu zarządzanego i gdy co najmniej dwa wyliczenia zawierają identyfikator o tej samej nazwie, należy jawnie zakwalifikować odwołania do identyfikatora.

C3275 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
