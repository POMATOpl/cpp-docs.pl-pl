# <a name="compiler-error-c2542"></a>Błąd kompilatora C2542

"Identyfikator": obiekt klasy nie ma konstruktora do inicjalizacji

Brak konstruktora z listą parametrów, który jest zgodny z inicjalizacją.

Ten błąd może być spowodowany przez nieprawidłowe parametry w inicjacji.
