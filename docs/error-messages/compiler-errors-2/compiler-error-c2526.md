# <a name="compiler-error-c2526"></a>Błąd kompilatora C2526

"Identifier1": funkcja powiązania C nie może zwrócić klasy "identifier2" języka C++

Funkcja zdefiniowana za pomocą powiązania C nie może zwrócić typu zdefiniowanego przez użytkownika.
