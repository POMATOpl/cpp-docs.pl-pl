# <a name="compiler-error-c2526"></a>Błąd kompilatora C2526

"identifier1": powiązanie z funkcją języka C nie można zwrócić klasy języka C++ "identifier2"

Funkcję zdefiniowaną z powiązaniem C nie może zwracać typ zdefiniowany przez użytkownika.
