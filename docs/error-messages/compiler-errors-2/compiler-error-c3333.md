# <a name="compiler-error-c3333"></a>Błąd kompilatora C3333

'biblioteka typów': nie można #import biblioteki uszkodzonego typu

Biblioteka typów określona w `#import` instrukcji nie będzie można odczytać przez kompilator. Można ponownie wygenerować biblioteki typów, jeśli jest to możliwe lub żądanie nowej biblioteki typów z dostawcą. Można użyć OLE podglądu, dostarczanego z Visual C++, aby wyświetlić plik biblioteki typów, aby zobaczyć, co to jest sprawy z nim.
