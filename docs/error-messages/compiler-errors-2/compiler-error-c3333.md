# <a name="compiler-error-c3333"></a>Błąd kompilatora C3333

"Biblioteka typów": nie można #import uszkodzonych bibliotek typów

Biblioteka typów określona w instrukcji `#import` jest nieczytelna przez kompilator. Może być konieczne ponowne wygenerowanie biblioteki typów, jeśli jest to możliwe, lub zażądanie nowej biblioteki typów od dostawcy. Możesz chcieć użyć przeglądarki OLE dostarczonej z wizualizacją C++, aby wyświetlić plik biblioteki typów, aby zobaczyć, co to jest sprawa.
