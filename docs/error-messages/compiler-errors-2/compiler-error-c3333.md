# <a name="compiler-error-c3333"></a>C3333 błąd kompilatora
'biblioteka typów': nie można #import biblioteki uszkodzonego typu  
  
 Biblioteki typów określonych w `#import` instrukcja jest niemożliwe do odczytania przez kompilator. Można ponownie wygenerować biblioteki typów, jeśli to możliwe, albo żądania nowej biblioteki typu od dostawcy. Można użyć OLE podglądu, dostarczonego z programem Visual C++, aby wyświetlić pliku biblioteki typów, aby zobaczyć, co polega z nim.
