# <a name="compiler-error-c2933"></a>Błąd kompilatora C2933

"Class": Identyfikator klasy typu został ponownie zdefiniowany jako składową typedef "identifier"

Nie można użyć klasy generycznej ani szablonu jako elementu członkowskiego `typedef`.

Poniższy przykład generuje C2933:

```cpp
// C2933.cpp
// compile with: /c
template<class T> struct TC { };
struct MyStruct {
   typedef int TC<int>;   // C2933
};

struct TC2 { };
struct MyStruct2 {
   typedef int TC2;
};
```

C2933 może również wystąpić przy użyciu typów ogólnych:

```cpp
// C2933b.cpp
// compile with: /clr /c
generic<class T> ref struct GC { };
struct MyStruct {
   typedef int GC<int>;   // C2933
};

ref struct GC2 { };
struct MyStruct2 {
   typedef int GC2;
};
```
