# <a name="compiler-error-c3631"></a>Błąd kompilatora C3631

"Function": nie można przeciążyć zdarzeń zarządzanych lub WinRT

Nie można obciążać zdarzenia zarządzanego lub WinRT.

C3631 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
