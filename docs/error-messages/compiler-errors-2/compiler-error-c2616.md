# <a name="compiler-error-c2616"></a>Błąd kompilatora C2616

"Konwersja": nie można niejawnie skonwertować elementu "type1" lvalue "na" type2 ", który nie jest wartością stałą

Nie można zainicjować odwołania z elementu innego niż lvalue.

Jest to błąd w obszarze zgodność ze standardem ANSI ([/za](../../build/reference/za-ze-disable-language-extensions.md)) i ostrzeżenie w ramach rozszerzeń Microsoft (**/ze**).
