# <a name="compiler-error-c2592"></a>Błąd kompilatora C2592

"Class": "base_class_2" jest dziedziczona po elemencie "base_class_1" i nie można go zmienić

Można określić tylko klasy bazowe, które nie dziedziczą z innych klas bazowych. W tym przypadku jest to tylko `base_class_1` potrzeba w specyfikacji, **`class`** ponieważ `base_class_1` już dziedziczy `base_class_2` .
