# <a name="compiler-error-c2592"></a>Błąd kompilatora C2592

"Class": "base_class_2" jest dziedziczona po elemencie "base_class_1" i nie można go zmienić

Można określić tylko klasy bazowe, które nie dziedziczą z innych klas bazowych. W takim przypadku w specyfikacji `class` jest wymagany tylko `base_class_1`, ponieważ `base_class_1` już dziedziczy `base_class_2`.
