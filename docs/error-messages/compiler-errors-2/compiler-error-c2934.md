# <a name="compiler-error-c2934"></a>Błąd kompilatora C2934

"class": typ klasy identyfikator ponownie definiowana jako zagnieżdżonych "item" 'Identyfikator'

Nie można użyć klasy generyczny lub szablonu jako elementy zagnieżdżone.
