# <a name="compiler-error-c2934"></a>Błąd kompilatora C2934

"Class": Identyfikator klasy typu został ponownie zdefiniowany jako zagnieżdżony element "identifier"

Nie można użyć klasy generycznej ani szablonu jako elementu zagnieżdżonego.
