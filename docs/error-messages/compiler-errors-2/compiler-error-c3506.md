# <a name="compiler-error-c3506"></a>Błąd kompilatora C3506

ma żadnych zarejestrowanych typelib dla identyfikatora LIBID "id"

Biblioteki typów nie został poprawnie zarejestrowany. Użyj regsvr32.exe, aby zarejestrować bibliotekę typów.
