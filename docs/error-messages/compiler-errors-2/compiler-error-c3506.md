# <a name="compiler-error-c3506"></a>C3506 błąd kompilatora
Brak żadnych zarejestrowanych typelib dla identyfikatora LIBID "id"  
  
 Biblioteki typów nie został prawidłowo zarejestrowany. Użyj regsvr32.exe można zarejestrować biblioteki typów.
