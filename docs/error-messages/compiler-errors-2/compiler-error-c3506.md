# <a name="compiler-error-c3506"></a>Błąd kompilatora C3506

Brak zarejestrowanej biblioteki typów dla identyfikatora LIBID "ID"

Element TypeLib nie został poprawnie zarejestrowany. Użyj regsvr32.exe, aby zarejestrować bibliotekę typów.
