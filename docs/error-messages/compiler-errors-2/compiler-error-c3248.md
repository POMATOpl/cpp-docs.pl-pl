# <a name="compiler-error-c3248"></a>Błąd kompilatora C3248

"function1": Funkcja zadeklarowana jako "__sealed" nie może być przesłonięta przez "function2"

Klasa pochodna próbowała zastąpić metodę wirtualną **__sealed** .

C3248 jest osiągalna tylko przy użyciu przestarzałej opcji kompilatora **/CLR: oldSyntax**.
