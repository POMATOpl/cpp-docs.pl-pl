# <a name="compiler-error-c3248"></a>C3248 błąd kompilatora
"function1": funkcja zadeklarowana jako __sealed nie może być zastąpione przez "function2"  
  
Klasa pochodna próbowała zastąpić **__sealed** metoda wirtualna.  
  
C3248 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
