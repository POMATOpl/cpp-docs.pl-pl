# <a name="compiler-error-c3501"></a>Błąd kompilatora C3501

Brak zarejestrowanych TypeLib dla identyfikatora ProgID "ProgID"

Identyfikator klasy dla danego identyfikatora ProgID nie ma skojarzonej biblioteki typów. W związku z tym nie można przekazać tego identyfikatora ProgID do `#import` instrukcji.
