# <a name="compiler-error-c3135"></a>Błąd kompilatora C3135

"Property": właściwość nie może mieć typu "const" lub "volatile"

Słowa kluczowe [const](../../cpp/const-cpp.md) i [volatile](../../cpp/volatile-cpp.md) są niedozwolone w przypadku właściwości.
