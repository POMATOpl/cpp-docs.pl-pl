# <a name="compiler-error-c3358"></a>C3358 błąd kompilatora
"symbol": nie odnaleziono symbolu  
  
 Nie znaleziono wymaganego symbolu.  
  
 Poniższy przykład generuje C3358:  
  
```  
// C3358.cpp  
#define __ATLEVENT_H__ 1   // remove this line to resolve the error  
#define _ATL_ATTRIBUTES 1  
#include "atlbase.h"  
#include "atlcom.h"  
  
[event_receiver(com)]  
struct A   // C3358  
{  
   void func();  
};  
  
int main()  
{  
}  
```
