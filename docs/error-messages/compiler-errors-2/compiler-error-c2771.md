# <a name="compiler-error-c2771"></a>Błąd kompilatora C2771

\#import jest dozwolony tylko w zakresie globalnym lub przestrzeni nazw

Dyrektywa `#import` nie jest dozwolona w, na przykład funkcja lub struktura.
