# <a name="compiler-error-c2948"></a>Błąd kompilatora C2948

jawne utworzenie wystąpienia; Specyfikator klasy magazynu "specyfikator" jest niedozwolony w specjalizacji

Nie można używać specyfikatorów klasy magazynu (takich jak `extern`) w specjalizacji klasy szablonu, która została wcześniej jawnie utworzona.
