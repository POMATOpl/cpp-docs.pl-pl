# <a name="compiler-error-c3157"></a>Błąd kompilatora C3157

Atrybut ParamArray może być stosowany tylko do ostatniego parametru

Ten <xref:System.ParamArrayAttribute> atrybut został zastosowany do nieprawidłowego parametru.
