# <a name="compiler-error-c3010"></a>C3010 błąd kompilatora
"etykieta": wyskok ze strukturalnego bloku OpenMP jest niedozwolony  
  
 Kod nie można przeskoczyć do lub z bloku OpenMP.  
  
 Poniższy przykład generuje C3010:  
  
```  
// C3010.c  
// compile with: /openmp  
int main() {  
   #pragma omp parallel   
   {  
      #pragma omp parallel  
      {  
         goto lbl3;  
      }  
   }  
   lbl3:;   // C3010  
}  
```
