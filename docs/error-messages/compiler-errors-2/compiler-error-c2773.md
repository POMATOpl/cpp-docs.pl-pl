# <a name="compiler-error-c2773"></a>Błąd kompilatora C2773

\#Importowanie i #using dostępne tylko w C++ kompilatorze

Kompilator języka C nie rozpoznaje dyrektywy preprocesora `#import`. Kompiluj źródło jako C++. W razie potrzeby użyj [/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) .
