# <a name="compiler-error-c2858"></a>Błąd kompilatora C2858

Opcja wiersza polecenia "/YC (/Fdfilename)" jest niespójna z prekompilowanym nagłówkiem, który użył "/Fdfilename"

Baza danych programu określona przez opcję Użyj prekompilowanego nagłówka ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) nie jest tym, co określono w poprzedniej opcji Create prekompilowany nagłówek ([/YC](../../build/reference/yc-create-precompiled-header-file.md)).
