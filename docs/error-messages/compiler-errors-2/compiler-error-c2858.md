# <a name="compiler-error-c2858"></a>Błąd kompilatora C2858

Opcja wiersza polecenia "/Yc (/ Fdfilename)" niespójna z prekompilowanym nagłówkiem, którego użyć "/ Fdfilename"

Bazy danych programu określonym przez użycie Prekompilowanego nagłówka ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) opcja nie jest określona przez poprzednie Utwórz prekompilowany nagłówek ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) opcji.
