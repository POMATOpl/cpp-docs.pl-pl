# <a name="compiler-error-c2858"></a>C2858 błąd kompilatora
Opcja wiersza polecenia "/Yc (/ Fdfilename)" niespójna z prekompilowanym nagłówkiem, który używa "/ Fdfilename"  
  
 Baza danych programu określona przez użycie Prekompilowanego nagłówka ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) opcja nie jest określona przez poprzednie Utwórz prekompilowany nagłówek ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) opcja.
