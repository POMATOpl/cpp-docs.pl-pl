# <a name="compiler-error-c3727"></a>C3727 błąd kompilatora
"event": zarządzanych zdarzeń musi być funkcją członkowską lub elementu członkowskiego danych, który jest wskaźnikiem do delegata  
  
 Zdarzenia platformy .NET musi być wskaźnikiem do typu delegata.
