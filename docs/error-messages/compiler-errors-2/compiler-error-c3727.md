# <a name="compiler-error-c3727"></a>Błąd kompilatora C3727

"Event": zdarzenie zarządzane musi być funkcją członkowską lub składową danych będącą wskaźnikiem do delegata

Zdarzenia .NET muszą być wskaźnikiem do typu delegata.
