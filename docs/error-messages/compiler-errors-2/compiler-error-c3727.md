# <a name="compiler-error-c3727"></a>Błąd kompilatora C3727

"event": zdarzenie zarządzane musi być funkcją składową lub składową danych będącą wskaźnikiem do delegata

Zdarzenia .NET musi być wskaźnikiem do typu delegata.
