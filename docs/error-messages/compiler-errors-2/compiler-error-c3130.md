# <a name="compiler-error-c3130"></a>C3130 błąd kompilatora
Wewnętrzny błąd kompilatora: nie można zapisać wstrzykniętego bloku kodu do PDB  
  
 Ten błąd występuje, gdy kompilator nie można zapisać wstrzykniętego bloku kodu do pliku PDB. Najczęstszą przyczyną błędu jest Brak miejsca na dysku.
