# <a name="compiler-error-c3130"></a>Błąd kompilatora C3130

Błąd wewnętrzny kompilatora: nie można zapisać wstrzykniętego bloku kodu do PDB

Ten błąd występuje, gdy kompilator nie może zapisać wstrzykniętego bloku kodu do pliku .pdb. Najbardziej typową przyczyną niepowodzenia jest Brak miejsca na dysku.
