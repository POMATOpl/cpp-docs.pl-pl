# <a name="compiler-error-c2829"></a>Błąd kompilatora C2829

"operator operatora" nie może mieć zmiennej listy parametrów

Tylko dwa operatory mogą przyjmować listę zmiennych parametrów:

- Wywołanie funkcji `()`

- **`new`**
