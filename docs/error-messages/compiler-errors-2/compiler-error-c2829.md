# <a name="compiler-error-c2829"></a>Błąd kompilatora C2829

"operator operator" nie może mieć zmiennej listy parametrów

Tylko dwa operatory można wykonać list parametrów zmiennych:

- Wywołanie funkcji `()`

- `new`
