# <a name="compiler-error-c2829"></a>C2829 błąd kompilatora
"operator operator" nie może mieć zmiennej listy parametrów  
  
 Tylko dwa operatory może zająć listy parametrów zmiennych:  
  
-   Wywołania funkcji `()`  
  
-   `new`
