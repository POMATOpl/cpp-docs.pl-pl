# <a name="compiler-error-c3180"></a>C3180 błąd kompilatora
"Nazwa typu": nazwa przekracza limit meta danych znaków "granica"  
  
 Kompilator obcięte nazwę typu zarządzanego w metadanych. Obcinania spowoduje typ niezdatna do użycia z `#using` — dyrektywa (lub odpowiednik w innym języku).  
  
 Limit Nazwa typu obejmuje wszystkie kwalifikacji przestrzeni nazw.
