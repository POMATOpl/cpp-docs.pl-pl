# <a name="compiler-error-c2918"></a>C2918 błąd kompilatora
"Nazwa": nie można używać właściwości indeksowanych na publikowanej powierzchni typu WinRT  
  
 Właściwości indeksowane nie są obsługiwane na publikowanej powierzchni typu WinRT.
