# <a name="compiler-error-c2918"></a>Błąd kompilatora C2918

"name": indeksowane właściwości nie mogą być używane na publikowanej powierzchni typu WinRT

Właściwości indeksowane nie są obsługiwane na publikowanej powierzchni typu WinRT.
