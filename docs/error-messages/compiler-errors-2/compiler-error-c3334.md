# <a name="compiler-error-c3334"></a>Błąd kompilatora C3334

Nie można #import uszkodzony wpisz biblioteki

Plik biblioteki w `#import` instrukcji jest uszkodzony i nie mogą być używane przez kompilator.  Uszkodzenie mogą pochodzić z nieprawidłowe generowanie modułu, w wyniku uszkodzenia dysku, lub jeśli plik binarny w jakiś sposób edytować i modyfikować.

Aby uzyskać więcej informacji, zobacz [#import Directive](../../preprocessor/hash-import-directive-cpp.md).
