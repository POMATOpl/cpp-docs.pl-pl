# <a name="compiler-error-c3334"></a>C3334 błąd kompilatora
Nie można #import uszkodzony typu biblioteki  
  
 Plik biblioteki w `#import` instrukcji jest uszkodzony i nie mogą być używane przez kompilator.  Uszkodzenie mogą pochodzić z zły generowania modułu uszkodzenie dysku, lub jeśli plik binarny został jakiś sposób edycji i modyfikować.  
  
 Aby uzyskać więcej informacji, zobacz [#import — dyrektywa](../../preprocessor/hash-import-directive-cpp.md).
