# <a name="compiler-error-c3334"></a>Błąd kompilatora C3334

nie można #import uszkodzonych bibliotek typów

Plik biblioteki w instrukcji `#import` jest uszkodzony i nie może być używany przez kompilator.  Uszkodzenie może pochodzić z nieprawidłowej generacji modułu, uszkodzenia dysku lub, jeśli plik binarny był w jakiś sposób edytowany i modyfikowany.

Aby uzyskać więcej informacji, zobacz [#import dyrektywie](../../preprocessor/hash-import-directive-cpp.md).
