# <a name="compiler-error-c3096"></a>Błąd kompilatora C3096

"Attribute": atrybut jest dozwolony tylko w składowych danych klas atrybutów

Atrybut został zastosowany nieprawidłowo.

Aby uzyskać więcej informacji, zobacz [atrybuty zdefiniowane przez użytkownika](../../extensions/user-defined-attributes-cpp-component-extensions.md).
