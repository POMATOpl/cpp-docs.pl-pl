# <a name="compiler-error-c3187"></a>Błąd kompilatora C3187

> "&#95;&#95;func&#95;&#95;": jest dostępny tylko w treści funkcji

Wstępnie zdefiniowane makro jest niedostępny poza ciałem funkcji.

Aby rozwiązać ten problem, Przenieś — makro wewnątrz treści funkcji.
