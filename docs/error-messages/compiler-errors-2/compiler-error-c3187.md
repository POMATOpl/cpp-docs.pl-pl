# <a name="compiler-error-c3187"></a>Błąd kompilatora C3187

> "&#95;&#95;&#95;Func&#95;": jest dostępny tylko w treści funkcji

Wstępnie zdefiniowane makro nie jest dostępne poza treścią funkcji.

Aby naprawić ten błąd, Przenieś makro wewnątrz treści funkcji.
