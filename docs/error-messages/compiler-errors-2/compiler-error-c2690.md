# <a name="compiler-error-c2690"></a>Błąd kompilatora C2690

'operator': nie można wykonać wskaźnik arytmetyczny na zarządzanej lub tablica WinRT

Arytmetyka wskaźnika nie jest dozwolona w zarządzanej lub WinRT tablicy. Notacja tablicy indeksu przechodzenia tablicy.
