# <a name="compiler-error-c2690"></a>Błąd kompilatora C2690

"operator": nie można wykonać arytmetyki wskaźnika dla tablicy zarządzanej lub WinRT

Arytmetyka wskaźnika nie jest dozwolona w przypadku tablicy zarządzanej ani WinRT. Użyj notacji indeksu tablicy, aby przechodzenie przez tablicę.
