# <a name="compiler-error-c2690"></a>C2690 błąd kompilatora
"operator": nie można wykonać arytmetycznego na zarządzanego wskaźnikiem lub tablicy WinRT  
  
 Arytmetyka wskaźnika nie jest dozwolona w zarządzanych lub WinRT tablicy. Umożliwia przenoszenie tablicy notacji indeksu tablicy.
