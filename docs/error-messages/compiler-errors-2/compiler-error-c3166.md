# <a name="compiler-error-c3166"></a>C3166 błąd kompilatora
"wskaźnik": nie można zadeklarować wskaźnika do wskaźnika wnętrza __gc jako członka 'type'  
  
Kompilator znaleziono nieprawidłowy wskaźnik deklaracji ( `__nogc` wskaźnik do `__gc` wskaźnik.). 
  
C3166 jest tylko przy użyciu opcji kompilatora przestarzałe **: oldsyntax**.
