# <a name="compiler-error-c2544"></a>C2544 błąd kompilatora
Oczekiwano ")" dla operatora "()"  
  
 Operator wywołania funkcji brak lewego nawiasu.  
  
 Przyczyną tego błędu może być rozwinięciu makra.
