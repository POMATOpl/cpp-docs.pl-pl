# <a name="compiler-error-c2544"></a>Błąd kompilatora C2544

Oczekiwano znaku ")" dla operatora "()"

Brak lewego nawiasu w operatorze wywołania funkcji.

Ten błąd może być spowodowany przez rozwinięcie makra.
