# <a name="compiler-error-c2544"></a>Błąd kompilatora C2544

Oczekiwano ")" dla operatora "()"

Lewy nawias okrągły Brak operatora wywołania funkcji.

Ten błąd może być spowodowany przez rozwinięciu makra.
