# <a name="compiler-error-c2543"></a>Błąd kompilatora C2543

Oczekiwano znaku "]" dla operatora "[]"

W operatorze indeksu dolnego brakuje lewego nawiasu.

Ten błąd może być spowodowany przez rozwinięcie makra.
