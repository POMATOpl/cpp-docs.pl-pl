# <a name="compiler-error-c2543"></a>Błąd kompilatora C2543

Oczekiwano "]" dla operatora "[]"

Subscripting operator brakuje lewy nawias kwadratowy.

Ten błąd może być spowodowany przez rozwinięciu makra.
