# <a name="compiler-error-c2545"></a>Błąd kompilatora C2545

"operator": nie można odnaleźć przeciążonego operatora

Nie można użyć operatora z podanymi argumentami operacji. Należy dostarczyć przeciążony operator z wymaganymi argumentami operacji.

Ten błąd może być spowodowany tym, że operandy mają niepoprawny typ.

Ten błąd może zostać naprawiony w przypadku zdefiniowania operatora konwersji lub konstruktora, który przyjmuje jeden parametr.
