# <a name="compiler-error-c2827"></a>Błąd kompilatora C2827

"operator operatora" nie może być przesłaniany globalnie za pomocą jednoargumentowego formularza

Operator nie może mieć jednoargumentowego formularza poza obiektem.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aby rozwiązać ten problem, można użyć następujących rozwiązań

1. Ustaw przeciążony operator jako lokalny dla obiektu.

1. Wybierz odpowiedni operator jednoargumentowy do przeciążenia.
