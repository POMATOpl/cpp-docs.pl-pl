# <a name="compiler-error-c3661"></a>C3661 błąd kompilatora
Lista jawnego przesłałania nie znalazła żadnych metod aby je zastąpić  
  
 Jawne przesłanianie określić co najmniej jedną nazwę typu.  Jednak nie było żadnej funkcji o sygnaturze niezbędne w co najmniej jeden typ zgodny sygnatura funkcji zastępowanie.  Próba zastąpienia opartego na nazwie typu, w określony typ, który jest zgodny z podpisem zastępowanie funkcja musi istnieć co najmniej jedną funkcję wirtualnego.  
  
 Aby uzyskać więcej informacji, zobacz [jawne zastąpienia](../../windows/explicit-overrides-cpp-component-extensions.md).
