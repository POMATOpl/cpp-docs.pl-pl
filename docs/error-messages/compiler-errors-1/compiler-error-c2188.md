# <a name="compiler-error-c2188"></a>C2188 błąd kompilatora
"numer": zbyt duży dla znaku dwubajtowego  
  
 Liczba przekracza limit rozmiaru dla typu znaków dwubajtowych. Wybierz typ większy.
