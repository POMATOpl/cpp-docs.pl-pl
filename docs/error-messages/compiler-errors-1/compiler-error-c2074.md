# <a name="compiler-error-c2074"></a>Błąd kompilatora C2074

"Identyfikator": inicjalizacja klasy wymaga nawiasów klamrowych

Brak nawiasów klamrowych wokół określonej klasy, struktury lub inicjatora Union.
