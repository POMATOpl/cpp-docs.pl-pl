# <a name="compiler-error-c2137"></a>C2137 błąd kompilatora
Stała znakowa pusty  
  
 Stała znakowa pusty ("") nie jest dozwolone.  
  
 Poniższy przykład generuje C2137:  
  
```  
// C2137.cpp  
int main() {  
   char c = '';   // C2137  
   char d = ' ';   // OK  
}  
```
