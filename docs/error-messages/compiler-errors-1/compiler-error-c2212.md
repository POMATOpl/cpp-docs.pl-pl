# <a name="compiler-error-c2212"></a>Błąd kompilatora C2212

"Identyfikator": __based niedostępne dla wskaźników do funkcji

Nie można zadeklarować wskaźników do funkcji **`__based`** . Jeśli potrzebujesz danych opartych na kodzie, użyj **`__declspec`** słowa kluczowego lub `data_seg` dyrektywy pragma.
