# <a name="compiler-error-c2052"></a>C2052 błąd kompilatora
'type': niedozwolony typ w wyrażeniu case  
  
 Wyrażenia CASE musi być stałe całkowite.  
  
 Poniższy przykład generuje C2052:  
  
```  
// C2052.cpp  
int main() {  
   int index = 0;  
   switch (index) {  
      case 1:  
         return 24;  
      case 1.0:   // C2052  
      // try the following line instead  
      // case 2:  
         return 23;  
   }  
}  
```
