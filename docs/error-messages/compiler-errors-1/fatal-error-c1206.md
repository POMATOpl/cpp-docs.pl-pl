# <a name="fatal-error-c1206"></a>Błąd krytyczny C1206

Dane dla domeny aplikacji nie są obsługiwane przez zainstalowaną wersję środowiska uruchomieniowego w trakcie wykonania

Niektóre funkcje, takie jak dane domeny aplikacji, są obsługiwane tylko przez środowisko uruchomieniowe języka wspólnego, które obsługuje tę funkcję.

C1206 wskazuje, że Najnowsza wersja środowiska uruchomieniowego nie jest zainstalowana na komputerze. Zainstaluj wersję środowiska uruchomieniowego języka wspólnego, która jest przeznaczona do użycia z kompilatorem.

Aby uzyskać więcej informacji, zobacz temat [AppDomain](../../cpp/appdomain.md) .
