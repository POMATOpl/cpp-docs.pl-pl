# <a name="compiler-error-c2020"></a>Błąd kompilatora C2020

"członek": redefinicja składowej "class"

Składową odziedziczoną z klasy bazowej lub struktury zostało przedefiniowane. Nie można przedefiniować dziedziczone elementy członkowskie, o ile nie jest zadeklarowany jako `virtual` w klasie bazowej.
