# <a name="compiler-error-c2020"></a>Błąd kompilatora C2020

"member": Ponowna definicja elementu członkowskiego "Class"

Członek Dziedziczony z klasy bazowej lub struktury jest ponownie zdefiniowany. Dziedziczone składowe nie mogą być ponownie zdefiniowane, chyba że zadeklarowano jako **`virtual`** w klasie bazowej.
