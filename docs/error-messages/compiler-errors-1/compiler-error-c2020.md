# <a name="compiler-error-c2020"></a>C2020 błąd kompilatora
"członek": zmiana definicji elementu członkowskiego "class"  
  
 Element członkowski odziedziczona z klasy podstawowej lub struktury, zostało ponownie zdefiniowane. Dziedziczone elementy członkowskie nie można ponownie zdefiniować, o ile nie jest zadeklarowany jako `virtual` w klasie podstawowej.
