# <a name="fatal-error-c1207"></a>Błąd krytyczny C1207

Zarządzane szablony nie są obsługiwane przez zainstalowaną wersję środowiska uruchomieniowego w trakcie wykonania

C1207 występuje, gdy masz kompilator dla bieżącej wersji, ale środowisko uruchomieniowe języka wspólnego z poprzedniej wersji.

Niektóre funkcje kompilatora mogą nie działać w poprzedniej wersji czasu wykonywania.

Aby rozwiązać ten problem, C1207 Zainstaluj wersję środowiska uruchomieniowego języka wspólnego, która jest przeznaczona do użycia z kompilatorem.
