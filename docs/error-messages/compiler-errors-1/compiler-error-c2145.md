# <a name="compiler-error-c2145"></a>C2145 błąd kompilatora
Błąd składni: Brak tokenu, przed identyfikatorem  
  
 Kompilator Oczekiwano `token` i zamiast tego znaleziono identyfikator.  
  
 Przyczyną tego błędu może być Brak średnika po zgłoszeniu ostatniego w bloku.
