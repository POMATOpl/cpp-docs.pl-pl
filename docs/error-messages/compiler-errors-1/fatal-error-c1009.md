# <a name="fatal-error-c1009"></a>Błąd krytyczny C1009

ograniczenie kompilatora: makra są zagnieżdżone zbyt głęboko

Kompilator próbował rozwinąć zbyt wiele makr w tym samym czasie. Kompilator ma limit 256 poziomów zagnieżdżonych makr. Dzielenie zagnieżdżonych makr na prostsze makra.
