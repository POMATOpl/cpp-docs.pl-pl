# <a name="fatal-error-c1351"></a>Błąd krytyczny C1351

> Wystąpił błąd podczas ładowania biblioteki dll "*dll*": niezgodność wersji

Znaleziono nieprawidłową wersję biblioteki DLL.

Oznacza to problem z instalacją i należy ponownie zainstalować program Visual Studio.
