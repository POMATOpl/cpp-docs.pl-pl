# <a name="fatal-error-c1351"></a>Błąd krytyczny C1351
Wystąpił błąd podczas ładowania biblioteki dll 'dll': niezgodna wersja  
  
 Znaleziono nieprawidłową wersję biblioteki DLL.  
  
 Oznacza to problem z instalacją i biblioteki DLL należy ponownie zainstalować z dysku CD produktu.
