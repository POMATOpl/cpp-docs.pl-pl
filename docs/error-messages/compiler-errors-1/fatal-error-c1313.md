# <a name="fatal-error-c1313"></a>Błąd krytyczny C1313

ograniczenie kompilatora: bloki typów nie mogą być zagnieżdżone głębiej niż poziomy liczbowe

Bloki obsługi wyjątków (lub obsługi wyjątków strukturalnych) zostały zagnieżdżone zbyt głęboko.  Uprość swój kod.
