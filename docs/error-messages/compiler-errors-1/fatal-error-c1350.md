# <a name="fatal-error-c1350"></a>Błąd krytyczny C1350

błąd podczas ładowania biblioteki dll "dll": nie znaleziono biblioteki DLL

Nie znaleziono biblioteki DLL, która obsługuje próbną operację.

Oznacza to problem z instalacją i należy ponownie zainstalować bibliotekę DLL z dysku CD produktu.
