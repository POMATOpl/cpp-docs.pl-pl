# <a name="fatal-error-c1098"></a>Błąd krytyczny C1098

Niezgodność wersji z silnikiem Edytuj i Kontynuuj

Używana wersja debugera nie jest zgodna z kompilatorem użytym do utworzenia pliku wykonywalnego. Jeśli ponowna kompilacja nie rozwiąże problemu, może być konieczne ponowne zainstalowanie Visual C++, aby upewnić się, że masz odpowiednie wersje debugera i kompilatora.
