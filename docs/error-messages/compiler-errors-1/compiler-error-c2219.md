# <a name="compiler-error-c2219"></a>Błąd kompilatora C2219

Błąd składniowy: kwalifikator typu musi być po "*"

Kwalifikator typu ( **`const`** lub **`volatile`** ) pojawia się, gdzie nie jest dozwolony.
