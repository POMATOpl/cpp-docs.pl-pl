# <a name="compiler-error-c2219"></a>Błąd kompilatora C2219

Błąd składniowy: po musi występować kwalifikator typu "*"

Kwalifikator typu (`const` lub `volatile`) jest wyświetlana, gdy nie jest dozwolone.
