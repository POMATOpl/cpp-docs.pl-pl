# <a name="compiler-error-c2219"></a>C2219 błąd kompilatora
Błąd składni: po musi występować kwalifikator typu "*"  
  
 Kwalifikator typu (`const` lub `volatile`) jest wyświetlany, gdy nie jest dozwolone.
