# <a name="compiler-error-c2152"></a>C2152 błąd kompilatora
'Identyfikator': wskaźniki do funkcji z różnymi atrybutami  
  
 Wskaźnik do funkcji z jedną konwencję wywołania (`__cdecl`, `__stdcall`, lub `__fastcall`) jest przypisany do wskaźnika do funkcji z innego konwencję wywołania.
