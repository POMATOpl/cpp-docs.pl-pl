# <a name="compiler-error-c2465"></a>Błąd kompilatora C2465

nie można zdefiniować typu anonimowego wewnątrz nawiasów

Anonimowa struktura, Unia lub typ wyliczeniowy są zdefiniowane wewnątrz wyrażenia ujętego w nawiasy. Ta wartość jest nieprawidłowa C++ w przypadku, gdy definicja nie ma znaczenia w zakresie funkcji.
