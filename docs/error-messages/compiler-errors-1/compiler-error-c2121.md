# <a name="compiler-error-c2121"></a>C2121 błąd kompilatora
"#": nieprawidłowy znak: prawdopodobnie wynik rozwinięcia makra  
  
 Nieprawidłowy #-znak zostały wstawione za pomocą nieprawidłowe makra, który używa operator wklejania tokenu (##) zamiast operatora stringizing (#).
