# <a name="compiler-error-c2121"></a>Błąd kompilatora C2121

"#": nieprawidłowy znak: prawdopodobnie wynik rozwinięcia makra

Znak # nieprawidłowy został wstawiony przez nieprawidłowe makra, która używa operator wklejania tokenu (##) zamiast operator tworzenia ciągu (#).
