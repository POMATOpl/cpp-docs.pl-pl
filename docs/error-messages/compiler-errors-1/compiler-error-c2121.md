# <a name="compiler-error-c2121"></a>Błąd kompilatora C2121

"#": nieprawidłowy znak: prawdopodobnie wynik rozwinięcia makra

Nieprawidłowy # znak mógł zostać wstawiony przez nieprawidłowe makro, które używa operatora wklejania tokenu (# #) zamiast operatora tworzenia ciągu (#).
