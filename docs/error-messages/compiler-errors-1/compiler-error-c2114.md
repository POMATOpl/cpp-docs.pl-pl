# <a name="compiler-error-c2114"></a>C2114 błąd kompilatora
"operator": wskaźnik po lewej stronie; wymaga całkowitej wartości po prawej stronie  
  
 Lewy argument operacji `operator` został wskaźnik, więc prawy operand musi być liczbą całkowitą.
