# <a name="compiler-error-c2114"></a>Błąd kompilatora C2114

"operator": wskaźnik po lewej stronie; wymaga całkowitej wartości po prawej

Lewy argument operacji `operator` jest wskaźnikiem, więc prawy operand musi być wartością całkowitą.
