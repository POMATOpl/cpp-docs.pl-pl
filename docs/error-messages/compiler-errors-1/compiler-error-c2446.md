# <a name="compiler-error-c2446"></a>Błąd kompilatora C2446

"operator": brak konwersji z "type1" na "type2"

Kompilator nie może skonwertować `type1`, aby `type2`. Konwersja może nie mieć sensu, ponieważ narusza ona C/C++ semantykę.
