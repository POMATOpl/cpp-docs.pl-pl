# <a name="compiler-error-c2446"></a>Błąd kompilatora C2446

'operator': konwersja z 'Typ1' na 'type2'

Kompilator nie może przekonwertować `type1` do `type2`. Konwersja może nie mieć sensu, ponieważ narusza semantyki C/C++.
