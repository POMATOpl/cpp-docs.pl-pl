# <a name="compiler-error-c2446"></a>Błąd kompilatora C2446

"operator": brak konwersji z "type1" na "type2"

Kompilator nie może wykonać konwersji `type1` do `type2` . Konwersja może nie mieć sensu, ponieważ narusza semantyki języka C/C++.
