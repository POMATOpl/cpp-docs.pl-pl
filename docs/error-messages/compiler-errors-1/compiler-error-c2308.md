# <a name="compiler-error-c2308"></a>Błąd kompilatora C2308

Łączenie niezgodnych ciągów

Dla łączenia nie określono obu ciągów znaków o szerokim i nieszerokim poziomie. Nie można połączyć ciągu znaków dwubajtowych i nieszerokich ciągów znaków.
