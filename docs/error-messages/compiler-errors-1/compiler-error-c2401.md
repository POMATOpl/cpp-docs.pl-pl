# <a name="compiler-error-c2401"></a>Błąd kompilatora C2401

"Identyfikator": Rejestr musi być podstawowy w "context"

Rejestr użyty w pośrednim operandzie pamięci musi być rejestrem podstawowym w tym kontekście.
