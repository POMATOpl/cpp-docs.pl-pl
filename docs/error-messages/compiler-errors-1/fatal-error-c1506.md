# <a name="fatal-error-c1506"></a>Błąd krytyczny C1506

Błąd w zakresie bloku nieodwracalny

Blok była zbyt duża, aby skompilować.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aby rozwiązać problem, sprawdzając następujące możliwe przyczyny

1. Niezgodne nawiasy klamrowe

1. Nietypowo dużej funkcji lub klasy
