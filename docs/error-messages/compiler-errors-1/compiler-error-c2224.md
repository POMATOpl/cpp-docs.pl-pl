# <a name="compiler-error-c2224"></a>Błąd kompilatora C2224

po lewej ". identyfikator" musi być typ struct/Union

Operand z lewej strony kropki (.) nie jest klasą, strukturą ani Unią.

Ten błąd może być spowodowany przez lewy argument operacji, który jest zmienną niezdefiniowaną (w związku z tym Type **`int`** ).
