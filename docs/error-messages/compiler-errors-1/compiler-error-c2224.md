# <a name="compiler-error-c2224"></a>Błąd kompilatora C2224

po lewej stronie ".identifier" musi być typ struct/union

Argument po lewej stronie kropki (.) nie jest klasy, struktury lub Unii.

Ten błąd może być spowodowany przez lewy operand, który jest niezdefiniowaną zmienną (w związku z tym wpisz `int`).
