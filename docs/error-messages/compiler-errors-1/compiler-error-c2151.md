# <a name="compiler-error-c2151"></a>C2151 błąd kompilatora
więcej niż jeden atrybut językowy  
  
 Funkcja zawiera więcej niż jedno słowo kluczowe ( `__cdecl`, `__stdcall`, lub `__fastcall`) określanie konwencji wywoływania.
