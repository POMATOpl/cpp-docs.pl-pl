# <a name="compiler-error-c2151"></a>Błąd kompilatora C2151

więcej niż jeden atrybut językowy

Funkcja ma więcej niż jedno słowo kluczowe ( `__cdecl`, `__stdcall`, lub `__fastcall`) określanie konwencji wywoływania.
