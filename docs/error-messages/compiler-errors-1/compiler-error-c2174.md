# <a name="compiler-error-c2174"></a>Błąd kompilatora C2174

"Function": rzeczywisty parametr jest typu "void": parametr Liczba1, lista parametrów liczba2

Parametr `number1` przesłany do listy parametrów `number2` jest **`void`** parametrem. Parametry nie mogą być typu **`void`** . Użyj **`void*`** zamiast tego.
