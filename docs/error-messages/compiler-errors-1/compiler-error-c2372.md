# <a name="compiler-error-c2372"></a>C2372 błąd kompilatora
"identyfikator": zmiana definicji; różne typy pośredniego wywołania  
  
 Identyfikator jest już zdefiniowany jako inny typ pochodny.  
  
 Poniższy przykład generuje C2326:  
  
```  
// C2372.cpp  
// compile with: /c  
extern int *fp;  
extern int fp[];   // C2372  
extern int fp2[];   // OK  
```
