# <a name="compiler-error-c2110"></a>C2110 błąd kompilatora
"+": nie można dodać dwóch wskaźników  
  
 Została podjęta próba dodania dwóch wartości wskaźnika za pomocą plus ( `+` ) operatora.  
  
 Poniższy przykład generuje C2110:  
  
```  
// C2110.cpp  
int main() {  
   int a = 0;  
   int *pa;  
   int *pb;  
   a = pa + pb;   // C2110  
}  
```
