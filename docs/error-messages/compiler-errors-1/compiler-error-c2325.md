# <a name="compiler-error-c2325"></a>C2325 błąd kompilatora
'type': nieoczekiwany typ po prawej "name"  
  
 Połączenie jest nawiązywane w przypadku destruktora niepoprawnego typu.  
  
 Poniższy przykład generuje C2325:  
  
```  
// C2325.cpp  
// compile with: /c  
class A {};  
typedef A* pA_t;  
void f() {  
    A** ppa = new A *;  
    ppa->~A*;   // C2325  
  
   pA_t *ppa2 = new pA_t;  
   ppa2->~pA_t();   // OK  
}  
```
