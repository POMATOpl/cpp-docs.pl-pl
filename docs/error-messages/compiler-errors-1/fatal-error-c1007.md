# <a name="fatal-error-c1007"></a>Błąd krytyczny C1007
Nierozpoznana flaga ciągu w opcji  
  
 Opcja wiersza polecenia zawiera nieprawidłowy ciąg. Sprawdź **CL** wiersz polecenia i zmienną środowiskową błędów.
