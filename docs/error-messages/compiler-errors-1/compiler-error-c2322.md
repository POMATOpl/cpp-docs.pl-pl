# <a name="compiler-error-c2322"></a>C2322 błąd kompilatora
'Identyfikator': adres dllimport "dllimport" nie jest statyczny  
  
 Niestatyczne wartość jest podana jako adresu funkcji deklarowane z `dllimport`.
