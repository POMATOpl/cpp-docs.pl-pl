# <a name="compiler-error-c2322"></a>Błąd kompilatora C2322

"Identyfikator": adres elementu dllimport "dllimport" nie jest statyczny

Wartość niestatyczna jest podawana jako adres funkcji zadeklarowanej z `dllimport`.
