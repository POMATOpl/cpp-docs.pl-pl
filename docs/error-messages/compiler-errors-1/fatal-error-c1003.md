# <a name="fatal-error-c1003"></a>Błąd krytyczny C1003

Liczba błędów przekracza liczbę; Zatrzymywanie kompilacji

Błędy w programie są zbyt liczne, aby można było je odzyskać. Kompilator musi zakończyć pracę.
