# <a name="fatal-error-c1099"></a>Błąd krytyczny C1099

Edytuj i Kontynuuj aparat kończący kompilację

Edytuj i Kontynuuj załadowanie prekompilowanego pliku nagłówkowego w celu kompilowania zmian kodu, ale kolejne akcje (takie jak zmiany kodu przed prekompilowanym nagłówkiem `#include` lub zatrzymaniem debugera) uniemożliwiły edytowanie i kontynuowanie kończenia kompilacji w ramach tego procesu. Nie trzeba podejmować żadnych działań w celu usunięcia tego błędu.
