# <a name="compiler-error-c2167"></a>C2167 błąd kompilatora
"Funkcja": zbyt wiele parametrów rzeczywistych dla wewnętrznej funkcji  
  
 Odwołanie do `intrinsic` funkcja ma za dużo parametrów.
