# <a name="compiler-error-c2167"></a>Błąd kompilatora C2167

'Funkcja': zbyt wiele parametrów rzeczywistych dla wewnętrznej funkcji

Odwołanie do `intrinsic` funkcji ma zbyt wiele parametrów.
