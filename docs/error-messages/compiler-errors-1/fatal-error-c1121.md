# <a name="fatal-error-c1121"></a>Błąd krytyczny C1121

Wywołanie CryptoAPI nie powiodło się

Kompilator wykonał wywołanie interfejsu CryptoAPI i wywołanie nie powiodło się. Zainstaluj ponownie program Visual Studio i ewentualnie Zainstaluj ponownie system operacyjny.
