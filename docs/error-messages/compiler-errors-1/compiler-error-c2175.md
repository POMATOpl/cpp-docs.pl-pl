# <a name="compiler-error-c2175"></a>C2175 błąd kompilatora
"locale": nieprawidłowe ustawienia regionalne  
  
 Określonego ustawień regionalnych jest nieprawidłowy. Zobacz [języka i ciągi Kraj/Region](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) w *odwołanie do biblioteki wykonawczej* obsługiwanych ustawień regionalnych.
