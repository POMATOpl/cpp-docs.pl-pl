# <a name="compiler-error-c2043"></a>C2043 błąd kompilatora
Niedozwolony podziału  
  
 A [podziału](../../cpp/break-statement-cpp.md) jest dozwolony tylko w obrębie `do`, `for`, `while`, lub `switch` instrukcji.
