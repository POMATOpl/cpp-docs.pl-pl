# <a name="compiler-error-c2207"></a>C2207 błąd kompilatora
"członek": elementu członkowskiego szablonu klasy nie można pobrać typu funkcji  
  
 `member` Klasy szablonu wcześniej przeanalizowany jako elementu członkowskiego danych niestatycznych. Nie można ponownie zdefiniować jako funkcję elementu członkowskiego.
