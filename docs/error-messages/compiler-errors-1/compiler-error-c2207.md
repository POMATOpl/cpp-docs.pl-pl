# <a name="compiler-error-c2207"></a>Błąd kompilatora C2207

"member": składowa szablonu klasy nie może uzyskać typu funkcji

`member`Szablon klasy został wcześniej przeanalizowany jako niestatyczny element członkowski danych. Nie można jej ponownie zdefiniować jako funkcji składowej.
