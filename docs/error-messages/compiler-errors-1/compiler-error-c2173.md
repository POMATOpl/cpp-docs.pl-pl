# <a name="compiler-error-c2173"></a>C2173 błąd kompilatora
"Funkcja": rzeczywisty parametr nie jest wskaźnikiem: parametr liczba1, liczba2 listy parametrów  
  
 Parametr `number1` przekazany do listy parametrów `number2` nie jest wskaźnikiem. Funkcja oczekuje wskaźnika.
