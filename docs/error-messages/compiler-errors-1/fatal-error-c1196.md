# <a name="fatal-error-c1196"></a>Błąd krytyczny C1196

"Identyfikator": identyfikator znaleziony w bibliotece typów "typelib" nie jest prawidłowym identyfikatorem C++

Jeden z identyfikatorów w bibliotece typów nie jest prawidłowym identyfikatorem języka C++. Biblioteka typów nie jest dostępna do użycia z [#import](../../preprocessor/hash-import-directive-cpp.md).
