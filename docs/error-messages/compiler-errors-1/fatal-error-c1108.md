# <a name="fatal-error-c1108"></a>Błąd krytyczny C1108

> nie można odnaleźć biblioteki DLL: "*nazwa biblioteki DLL*"

Nie można znaleźć podanej biblioteki DLL (*nazwa biblioteki DLL*) w ścieżce.

Aby rozwiązać ten problem, zainstaluj ponownie program Visual C++ lub skopiuj odpowiedni plik. dll z instalacji na komputer.
