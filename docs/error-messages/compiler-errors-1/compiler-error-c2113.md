# <a name="compiler-error-c2113"></a>Błąd kompilatora C2113

"-": wskaźnik może zostać odjęty tylko od innego wskaźnika

Prawy operand w operacji odejmowania był wskaźnikiem, ale lewy argument operacji nie był.
