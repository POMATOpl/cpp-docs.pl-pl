# <a name="compiler-error-c2172"></a>Błąd kompilatora C2172

"Function": rzeczywisty parametr nie jest wskaźnikiem: numer parametru

Parametr `number` nie jest wskaźnikiem. Funkcja oczekuje wskaźnika.
