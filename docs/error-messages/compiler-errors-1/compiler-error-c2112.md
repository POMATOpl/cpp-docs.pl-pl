# <a name="compiler-error-c2112"></a>C2112 błąd kompilatora
"-": wskaźnik odejmowania wymaga wartości całkowitej lub wskaźnika operandu  
  
 Nastąpiła próba do odjęcia wskaźników, które wskazują różnych typów.
