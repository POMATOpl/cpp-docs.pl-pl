# <a name="compiler-error-c2266"></a>Błąd kompilatora C2266

"Identyfikator": odwołanie do tablicy o nieustalonym zakresie jest niedozwolone

Odwołanie jest zadeklarowane dla tablicy z niestałą granicą. Tablica musi mieć granice stałe.
