# <a name="compiler-error-c2302"></a>Błąd kompilatora C2302

po lewej ". ~ identyfikator" musi być typu Class/struct/Union

Wyrażenie po lewej stronie operatora kropki (.) nie jest klasą, strukturą ani Unią.
