# <a name="compiler-error-c2302"></a>C2302 błąd kompilatora
po lewej ". ~ identyfikator" musi mieć typ klasy/struktury/Unii  
  
 Wyrażenie po lewej stronie operatora kropki (.) nie jest klasy, struktury lub związku.
