# <a name="compiler-error-c2302"></a>Błąd kompilatora C2302

po lewej ". ~ identyfikator" musi mieć typ klasy/struct/union

Wyrażenie po lewej stronie operatora kropka (.) nie jest klasy, struktury lub Unii.
