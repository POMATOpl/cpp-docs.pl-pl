# <a name="compiler-error-c2406"></a>Błąd kompilatora C2406

'Identyfikator': Nazwa niezdefiniowana w "context"

Niezdefiniowany identyfikator jest używany z `SIZE`, `LENGTH`, lub operator wyboru elementów członkowskich (.).
