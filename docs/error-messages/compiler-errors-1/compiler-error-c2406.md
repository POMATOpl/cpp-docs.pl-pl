# <a name="compiler-error-c2406"></a>C2406 błąd kompilatora
'Identyfikator': Nazwa niezdefiniowana w "context"  
  
 Niezdefiniowany identyfikator jest używany z `SIZE`, `LENGTH`, lub operatora wyboru elementu członkowskiego (.).
