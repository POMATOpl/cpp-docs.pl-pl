# <a name="fatal-error-c1077"></a>Błąd krytyczny C1077

ograniczenie kompilatora: nie może być więcej niż liczba opcji wiersza polecenia

Liczba opcji wiersza polecenia przekracza limit wewnętrzny.

Istnieje zbyt wiele symboli zdefiniowanych za pomocą [/d](../../build/reference/d-preprocessor-definitions.md). (Zamiast tego Umieść definicje w pliku nagłówkowym).
