# <a name="fatal-error-c1309"></a>Błąd krytyczny C1309

Niezgodne wersje C2.DLL i PGODB \<version> . BIBLIOTECE

Zestaw narzędzi używany do kompilowania i używania [optymalizacji](../../build/profile-guided-optimizations.md) profilowanej zawiera niezgodne składniki.  Jeśli nie możesz rozwiązać tego błędu ręcznie, zainstaluj ponownie Visual C++.
