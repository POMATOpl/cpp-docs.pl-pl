# <a name="fatal-error-c1033"></a>Błąd krytyczny C1033

nie można otworzyć programu PDB bazy danych

Przyczyną tego błędu może być błąd dysku, tymczasowa blokada utworzona przez program antywirusowy, poprzednie wystąpienie debugera, które nie zostało w pełni zamknięte, lub Parallel mspdbsrv.exe procesy, które próbują uzyskać dostęp do tego samego pliku, między innymi możliwymi przyczynami.
