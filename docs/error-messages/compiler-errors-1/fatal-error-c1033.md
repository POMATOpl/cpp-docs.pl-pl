# <a name="fatal-error-c1033"></a>Błąd krytyczny C1033

Nie można otworzyć pliku pdb bazy danych programu

Ten błąd może być spowodowany przez błąd dysku, blokadę tymczasowych utworzonych przez program antywirusowy, poprzednie wystąpienie debugera, który nie został w pełni kończy pracę, lub procesami mspdbsrv.exe równoległych kompilacji, które próba dostępu do tego samego pliku, między innymi możliwymi przyczynami.
