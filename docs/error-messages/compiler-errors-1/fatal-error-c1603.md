# <a name="fatal-error-c1603"></a>Błąd krytyczny C1603

cel rozgałęzienia zestawu wbudowanego poza zakresem przez "number" bajtów

Obliczona odległość między instrukcją JCXZ lub JECXZ i jej określona etykieta docelowa była większa niż 128 bajtów. Zaktualizuj swój kod, tak aby etykieta była bliżej instrukcji.
