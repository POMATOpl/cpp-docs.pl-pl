# <a name="compiler-error-c2022"></a>Błąd kompilatora C2022

"number": za duży na znak

Liczba ósemkowa po ukośniku odwrotnym (\\) w znaku lub stałej ciągu jest zbyt duża, aby reprezentować znak.
