# <a name="compiler-error-c2022"></a>Błąd kompilatora C2022

"numer": zbyt duży dla znaku

Liczba ósemkowa ukośniku odwrotnym (\\) znakiem lub stałą ciągu jest zbyt duży do reprezentowania znaku.
