# <a name="compiler-error-c2426"></a>C2426 błąd kompilatora
"token": niedozwolony operator w "context"  
  
 Token nie może służyć jako operator w tym kontekście. Operatory indeksu, na przykład, nie mogą być zagnieżdżone.
