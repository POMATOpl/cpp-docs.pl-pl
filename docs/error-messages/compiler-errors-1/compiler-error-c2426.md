# <a name="compiler-error-c2426"></a>Błąd kompilatora C2426

"token": niedozwolony operator w "context"

Nie można użyć tokenu jako operatora w tym kontekście. Operatory indeksowania, na przykład, nie mogą być zagnieżdżane.
