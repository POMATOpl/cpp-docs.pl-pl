# <a name="compiler-error-c2222"></a>Błąd kompilatora C2222

nieoczekiwany typ "Type": oczekiwano klasy bazowej lub składowej

Lista inicjalizatora może inicjować tylko klasy bazowe lub elementy członkowskie typu. Aby naprawić ten błąd, sprawdź, czy na liście inicjatora są inicjowane tylko klasy podstawowe lub elementy członkowskie typu.
