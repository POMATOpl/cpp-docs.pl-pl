# <a name="compiler-error-c2120"></a>C2120 błąd kompilatora
"void" niedozwolone ze wszystkimi typami  
  
 `void` Typ jest używany w deklaracji z innego typu.  
  
 Poniższy przykład generuje C2120:  
  
```  
// C2120.cpp  
int main() {  
   void int i;   // C2120  
   int j;   // OK  
}  
```
