# <a name="fatal-error-c1210"></a>Błąd krytyczny C1210
/ CLR: pure i/CLR: Safe nie są obsługiwane przez wersję środowiska uruchomieniowego zainstalowany  
  
 **/CLR: pure** i **/CLR: Safe** — opcje kompilatora zostały uznane za przestarzałe w programie Visual Studio 2015.  
  
 C1210 występuje, gdy masz kompilatora dla bieżącej wersji, ale środowisko uruchomieniowe języka wspólnego z poprzedniej wersji.  
  
 Niektóre funkcje kompilator może nie działać w poprzedniej wersji czasu wykonywania.  
  
 Aby rozwiązać typowe wersja środowiska uruchomieniowego CLR, który jest przeznaczony do użytku z kompilujący C1210 instalacji.
