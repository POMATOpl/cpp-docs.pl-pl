# <a name="compiler-error-c2154"></a>Błąd kompilatora C2154

"type": tylko typ wyliczeniowy jest dozwolony jako argument typu wewnętrznej cechy kompilatora "słowo kluczowe __underlying_type ma"

Można pobierać tylko typem podstawowym typem wyliczenia.

Aby uzyskać więcej informacji, zobacz [Obsługa cech typu w kompilatorze](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md).
