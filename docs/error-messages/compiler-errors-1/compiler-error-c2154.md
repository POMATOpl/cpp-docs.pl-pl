# <a name="compiler-error-c2154"></a>Błąd kompilatora C2154

"Type": tylko typ wyliczeniowy jest dozwolony jako argument dla cechy wewnętrznego typu "__underlying_type" kompilatora

Można uzyskać tylko typ podstawowy typu wyliczenia.

Aby uzyskać więcej informacji, zobacz [Obsługa kompilatora dla cech typu](../../extensions/compiler-support-for-type-traits-cpp-component-extensions.md).
