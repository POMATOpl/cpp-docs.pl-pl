# <a name="compiler-error-c2101"></a>Błąd kompilatora C2101

"&" na — stała

Operator address-of ( `&` ) musi mieć wartość l, jako argument operacji.

Poniższy przykład spowoduje wygenerowanie C2101:

```
// C2101.cpp
int main() {
   char test;
   test = &'a';   // C2101
   test = 'a';   // OK
}
```
