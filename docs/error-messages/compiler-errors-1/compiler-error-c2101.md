# <a name="compiler-error-c2101"></a>C2101 błąd kompilatora
"&" dla stałej  
  
 Address-of — operator ( `&` ) musi mieć wartość l jako argument.  
  
 Poniższy przykład generuje C2101:  
  
```  
// C2101.cpp  
int main() {  
   char test;  
   test = &'a';   // C2101  
   test = 'a';   // OK  
}  
```
