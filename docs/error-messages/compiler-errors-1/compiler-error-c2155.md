# <a name="compiler-error-c2155"></a>Błąd kompilatora C2155

"?": Nieprawidłowy lewy operand, oczekiwano arytmetycznego lub typu wskaźnika

Wyrażenie po lewej stronie `?` nie może być porównywane z zerem. Należy użyć wyrażenia arytmetycznego lub wskaźnika, które może być porównane z zerem.
