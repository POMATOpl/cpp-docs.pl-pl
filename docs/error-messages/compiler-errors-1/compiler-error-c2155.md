# <a name="compiler-error-c2155"></a>C2155 błąd kompilatora
"?": lewy operand nieprawidłowy, oczekiwano arytmetycznego lub typu wskaźnika  
  
 Wyrażenie po lewej stronie `?` nie można porównać wartości zero. Należy użyć wyrażenia arytmetycznego lub wskaźnikowego, który można porównać wartości zero.
