# <a name="fatal-error-c1064"></a>Błąd krytyczny C1064

ograniczenie kompilatora: token spowodował przepełnienie buforu wewnętrznego

Identyfikator przekracza długość buforu wewnętrznego używane dla identyfikatorów. Skróć nazwę.
