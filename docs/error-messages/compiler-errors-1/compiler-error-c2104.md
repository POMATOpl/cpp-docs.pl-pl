# <a name="compiler-error-c2104"></a>Błąd kompilatora C2104

"&" na pole bitowe ignorowane

Nie można przyjąć adresu polem bitowym.

Poniższy przykład spowoduje wygenerowanie C2104:

```
// C2104.cpp
struct X {
   int sb : 1;
};

int main() {
   X x;
   &x.sb;   // C2104
   x.sb;   // OK
}
```
