# <a name="compiler-error-c2223"></a>Błąd kompilatora C2223

lewo od "->identifier" musi wskazywać na struct/Union

Operand z lewej strony `->` nie jest wskaźnikiem do klasy, struktury lub Unii.

Ten błąd może być spowodowany przez lewy argument operacji, który jest zmienną niezdefiniowaną (w związku z tym Type **`int`** ).
