# <a name="fatal-error-c1903"></a>Błąd krytyczny C1903

nie można odzyskać z poprzednich błędów; Zatrzymywanie kompilacji

Kompilator znalazł zbyt wiele błędów, aby kontynuować. Usuń błędy i ponownie skompiluj.
