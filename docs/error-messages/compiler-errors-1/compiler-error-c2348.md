# <a name="compiler-error-c2348"></a>C2348 błąd kompilatora
"Nazwa typu": nie jest agregacji stylu języka C, nie może być eksportowany w osadzonych IDL  
  
 Można umieścić `struct` w pliku .idl z [wyeksportować](../../windows/export.md) atrybut `struct` musi zawierać tylko dane.  
  
 Poniższy przykład generuje C2348:  
  
```  
// C2348.cpp  
// C2348 error expected  
[ module(name="SimpleMidlTest") ];  
  
[export]  
struct Point {  
   // Delete the following two lines to resolve.  
   Point() : m_i(0), m_j(0) {}  
   Point(int i, int j) : m_i(i), m_j(j) {}  
  
   int m_i;  
   int m_j;  
};  
```
