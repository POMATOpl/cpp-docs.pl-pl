# <a name="fatal-error-c1014"></a>Błąd krytyczny C1014
zbyt wiele plików obejmują: głębokość = poziom  
  
 Zagnieżdżanie `#include` dyrektywy jest zbyt głęboka. Zagnieżdżone dyrektywy mogą obejmować otwartych plików. Plik źródłowy zawierających dyrektywy liczone jako jeden plik.
