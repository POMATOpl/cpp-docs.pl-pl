# <a name="fatal-error-c1014"></a>Błąd krytyczny C1014

zbyt wiele pliki dołączane: głębokość = poziom

Zagnieżdżanie `#include` dyrektyw jest zbyt głęboka. Zagnieżdżonych dyrektyw może zawierać otwartych plików. Plik źródłowy zawierający dyrektywę jest liczona jako jeden plik.
