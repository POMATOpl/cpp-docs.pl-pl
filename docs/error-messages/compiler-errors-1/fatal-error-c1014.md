# <a name="fatal-error-c1014"></a>Błąd krytyczny C1014

zbyt wiele plików dołączanych: Głębokość = poziom

Zagnieżdżanie `#include` dyrektyw jest zbyt głębokie. Dyrektywy zagnieżdżone mogą zawierać otwarte pliki. Plik źródłowy zawierający dyrektywę liczy się jako jeden plik.
