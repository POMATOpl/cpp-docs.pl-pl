# <a name="compiler-error-c2095"></a>Błąd kompilatora C2095

"Function": rzeczywisty parametr jest typu "void": parametr "number"

Parametr przesłany do funkcji jest typu **`void`** , co jest niedozwolone. Zamiast tego użyj wskaźnika do typu void ( `void *` ).

`number`Wskazuje, który parametr jest **`void`** .
