Wywołanie funkcji .NET nie powiodło się. *Tekst z ciągu czasu wykonywania* może lub nie być dostarczony przez środowisko uruchomieniowe com.

Więcej informacji o komunikatach o błędach systemu znajduje się w pliku systemowym Winerror. h i [FormatMessage](/windows/win32/api/winbase/nf-winbase-formatmessage).
