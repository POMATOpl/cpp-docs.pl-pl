# <a name="fatal-error-c1073"></a>Błąd krytyczny C1073
Wewnętrzny błąd dotyczący kompilacji przyrostowej (pliku kompilatora "filename", numer wiersza)  
  
 Skompiluj ponownie plik bez użycia kompilacji przyrostowej.
