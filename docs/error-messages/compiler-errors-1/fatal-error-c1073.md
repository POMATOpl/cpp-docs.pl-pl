# <a name="fatal-error-c1073"></a>Błąd krytyczny C1073

Wewnętrzny błąd dotyczący kompilacji przyrostowej (plik kompilatora 'NazwaPliku', numer wiersza)

Skompiluj ponownie plik bez korzystania z kompilacji przyrostowej.
