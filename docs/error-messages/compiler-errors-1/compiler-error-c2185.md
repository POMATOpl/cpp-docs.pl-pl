# <a name="compiler-error-c2185"></a>Błąd kompilatora C2185

"Identyfikator": niedozwolona alokacja na podstawie

Zmienna rejestru lub automatyczna (lokalna) jest zadeklarowana **`__based`** . Można zadeklarować tylko zmienne globalne **`__based`** .
