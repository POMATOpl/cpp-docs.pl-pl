# <a name="compiler-error-c2185"></a>Błąd kompilatora C2185

"Identyfikator": niedozwolona alokacja na podstawie

Zmienna rejestru lub automatyczna (lokalna) zmienna jest zadeklarowana `__based`. `__based`można zadeklarować tylko zmienne globalne.
