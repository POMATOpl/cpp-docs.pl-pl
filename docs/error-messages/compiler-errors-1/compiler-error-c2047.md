# <a name="compiler-error-c2047"></a>Błąd kompilatora C2047

niedozwolony domyślny

Słowo kluczowe `default` może występować tylko w instrukcji `switch`.

Poniższy przykład generuje C2047:

```cpp
// C2047.cpp
int main() {
   int i = 0;
   default:   // C2047
   switch(i) {
      case 0:
      break;
   }
}
```

Możliwe rozwiązanie:

```cpp
// C2047b.cpp
int main() {
   int i = 0;
   switch(i) {
      case 0:
      break;
      default:
      break;
   }
}
```
