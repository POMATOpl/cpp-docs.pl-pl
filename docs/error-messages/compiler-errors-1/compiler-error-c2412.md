# <a name="compiler-error-c2412"></a>Błąd kompilatora C2412

"label": etykieta bez uwzględniania wielkości liter jest zdefiniowana ponownie

Etykieta jest zdefiniowana więcej niż raz w bieżącej funkcji. Zmień pisownię etykiety i jej odwołań.
