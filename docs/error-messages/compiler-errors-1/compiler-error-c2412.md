# <a name="compiler-error-c2412"></a>Błąd kompilatora C2412

"etykieta": bez uwzględniania wielkości liter Etykieta zdefiniowana ponownie

Etykieta jest zdefiniowany więcej niż raz w bieżącej funkcji. Zmień pisownię etykiety i ich odwołań.
