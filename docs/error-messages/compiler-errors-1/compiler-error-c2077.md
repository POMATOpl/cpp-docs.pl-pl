# <a name="compiler-error-c2077"></a>C2077 błąd kompilatora
Inicjator pola nieskalarnego 'Identyfikator'  
  
 Próbowano zainicjować pola bitowego z nonscalar (struct, Unii, tablicy lub klasy). Użyj liczbą całkowitą lub wartość zmiennoprzecinkową.
