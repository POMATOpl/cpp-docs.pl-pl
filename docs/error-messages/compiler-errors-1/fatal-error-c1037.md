# <a name="fatal-error-c1037"></a>Błąd krytyczny C1037
Nie można otworzyć obiektu pliku filename  
  
 Plik obiektu określony przez [/Fo](../../build/reference/fo-object-file-name.md) nie może zostać otwarty.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aby rozwiązać problem, sprawdzając następujące możliwe przyczyny  
  
1.  Nieprawidłowa nazwa pliku.  
  
2.  Za mało pamięci do otwarcia pliku.  
  
3.  Inny proces używający pliku.  
  
4.  Tylko do odczytu pliku ma taką samą nazwę.
