# <a name="compiler-error-c2320"></a>Błąd kompilatora C2320

Oczekiwano ":" Aby po specyfikatorze dostępu "specyfikatora"

Słowo kluczowe `public`, `protected`, lub `private` musi być z dwukropkiem.
