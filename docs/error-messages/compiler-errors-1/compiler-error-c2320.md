# <a name="compiler-error-c2320"></a>Błąd kompilatora C2320

> Oczekiwano znaku ":" do podążania za*specyfikatorem*dostępu "

Po słowie kluczowym **`public`** **`protected`** lub **`private`** musi następować dwukropek.
