# <a name="fatal-error-c1046"></a>Błąd krytyczny C1046

ograniczenie kompilatora: Struktura zagnieżdżona zbyt głęboko

Struktura, Unia lub Klasa przekroczyła limit zagnieżdżenia, czyli 15 poziomów. Napisz ponownie definicję, aby zmniejszyć poziom zagnieżdżenia. Podziel strukturę, Unię lub klasę na dwie lub więcej części przy użyciu `typedef`, aby zdefiniować co najmniej jedną zagnieżdżoną strukturę.
