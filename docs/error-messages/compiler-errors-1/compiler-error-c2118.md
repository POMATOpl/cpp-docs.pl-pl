# <a name="compiler-error-c2118"></a>C2118 błąd kompilatora
Indeks dolny ujemna  
  
 Definiowanie rozmiaru tablicy wartości jest większy niż rozmiar maksymalny tablicy lub mniejszy od zera.  
  
 Poniższy przykład generuje C2118:  
  
```  
// C2118.cpp  
int main() {  
   int array1[-1];   // C2118  
   int array2[3];   // OK  
}  
```
