# <a name="fatal-error-c1023"></a>Błąd krytyczny C1023

"plik": nieoczekiwany błąd w PCH, spróbuj ponownie skompilować PCH

C1023 może być spowodowany przez jeden z kilku problemów, rozwiązanie, do którego jest odbudowa prekompilowanego pliku nagłówkowego.

C1023 może być spowodowany próbą użycia prekompilowanego pliku nagłówkowego na komputerze innym niż ten, na którym został skompilowany.
