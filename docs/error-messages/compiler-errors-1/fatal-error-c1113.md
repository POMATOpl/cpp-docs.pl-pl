# <a name="fatal-error-c1113"></a>Błąd krytyczny C1113

\#Użycie polecenia "File" nie powiodło się

Do dyrektywy [#using](../../preprocessor/hash-using-directive-cpp.md) można przesłać tylko plik w formacie języka pośredniego firmy Microsoft (MSIL). Opcja kompilatora [/CLR](../../build/reference/clr-common-language-runtime-compilation.md) umożliwia utworzenie pliku wyjściowego MSIL. Inne języki programu Visual Studio generują również pliki MSIL.
