# <a name="compiler-error-c2241"></a>Błąd kompilatora C2241

"Identyfikator": dostęp do składowej jest ograniczony

Kod próbuje uzyskać dostęp do prywatnego lub chronionego elementu członkowskiego.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aby rozwiązać ten problem, można użyć następujących rozwiązań

1. Zmień poziom dostępu elementu członkowskiego.

1. Zadeklaruj element członkowski a **`friend`** funkcji dostępu.
