# <a name="fatal-error-c1053"></a>Błąd krytyczny C1053

" \<identifier> ": funkcja zbyt duża

Funkcja jest zbyt duża do skompilowania.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aby rozwiązać ten problem, można użyć następujących rozwiązań

1. Wypróbuj kompilację bez optymalizacji.

1. Podziel funkcję na mniejsze funkcje.

1. Ogranicz wywołania funkcji wbudowanych.
