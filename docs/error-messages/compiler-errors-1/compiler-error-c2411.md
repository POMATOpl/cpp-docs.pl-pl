# <a name="compiler-error-c2411"></a>C2411 błąd kompilatora
"identyfikator": niedozwolony członkowski struct/union w "context"  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aby rozwiązać problem, sprawdzając następujące możliwe przyczyny  
  
1.  Identyfikator nie jest elementem członkowskim widoczne struktury lub Unii, w tym kontekście.  
  
2.  Identyfikator nie jest elementem członkowskim struktury lub Unii określony za pomocą operatora wyboru elementu członkowskiego (.).
