# <a name="compiler-error-c2411"></a>Błąd kompilatora C2411

'Identyfikator': niedozwolony składowa struktury/Unii w "context"

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aby rozwiązać problem, sprawdzając następujące możliwe przyczyny

1. Identyfikator nie jest członkiem widoczne struktury lub Unii w tym kontekście.

1. Identyfikator nie jest elementem członkowskim struktury lub Unii, określony za pomocą operatora wyboru składowej (.).
