# <a name="compiler-error-c2411"></a>Błąd kompilatora C2411

"Identyfikator": niedozwolona składowa struktury/Unii w "context"

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aby rozwiązać ten problem, sprawdzając następujące możliwe przyczyny

1. Identyfikator nie jest elementem członkowskim widocznej struktury lub Unii w tym kontekście.

1. Identyfikator nie jest elementem członkowskim struktury lub Unii określonym za pomocą operatora wyboru elementu członkowskiego (.).
