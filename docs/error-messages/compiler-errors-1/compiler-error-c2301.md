# <a name="compiler-error-c2301"></a>Błąd kompilatora C2301

po lewej "-> ~ identyfikator" musi wskazywać na Class/struct/Union

Wyrażenie po lewej stronie operatora `->` nie jest oceniane jako wskaźnik do klasy, struktury lub Unii.
