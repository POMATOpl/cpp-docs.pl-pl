# <a name="compiler-error-c2301"></a>Błąd kompilatora C2301

po lewej "-> ~ Identyfikator ' musi wskazywać na klasy/struct/union

Wyrażenie po lewej stronie `->` operator nie zwraca wskaźnik do klasy, struktury lub Unii.
