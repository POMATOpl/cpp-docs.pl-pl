# <a name="compiler-error-c2444"></a>C2444 błąd kompilatora
'Identyfikator': używany prototyp ANSI, znaleziono "type", oczekiwano "{" lub ";"  
  
 Prototyp funkcji następuje typu.  
  
 Przyczyną tego błędu może być Brak średnika lub nawias klamrowy.
