# <a name="compiler-error-c2168"></a>C2168 błąd kompilatora
"Funkcja": zbyt mało rzeczywistych parametrów w wewnętrznej funkcji  
  
 Odwołanie do `intrinsic` funkcja ma za mało parametrów.
