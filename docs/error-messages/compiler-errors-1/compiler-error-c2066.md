# <a name="compiler-error-c2066"></a>Błąd kompilatora C2066

Rzutowanie na typ funkcji jest niedozwolone

W standardzie ANSI C nie jest to dozwolone rzutowanie między wskaźnikiem funkcji a wskaźnikiem danych.
