# <a name="compiler-error-c2169"></a>C2169 błąd kompilatora
"Funkcja": wewnętrzna funkcja nie może być zdefiniowana  
  
 Definicja funkcji pojawia się dla funkcji już zadeklarowana `intrinsic`.
