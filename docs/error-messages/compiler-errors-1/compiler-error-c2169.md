# <a name="compiler-error-c2169"></a>Błąd kompilatora C2169

"Function": wewnętrzna funkcja, nie może być zdefiniowana

Zostanie wyświetlona definicja funkcji dla funkcji, która jest już zadeklarowana `intrinsic`.
