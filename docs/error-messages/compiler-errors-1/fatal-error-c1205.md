# <a name="fatal-error-c1205"></a>Błąd krytyczny C1205

Typy ogólne nie są obsługiwane przez zainstalowaną wersję środowiska uruchomieniowego w trakcie wykonania

Wersja środowiska uruchomieniowego języka wspólnego używanego przez kompilator nie jest wersją środowiska uruchomieniowego obsługiwaną przez bieżący kompilator.  Na przykład funkcja generyczna wymaga środowiska uruchomieniowego dopasowanego do kompilatora.

Może być konieczne zmodyfikowanie specyfikacji ścieżki.
