# <a name="fatal-error-c1085"></a>Błąd krytyczny C1085

Nie można zapisać pliku z plikiem: "plik": komunikat

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aby rozwiązać ten problem, sprawdzając następujące możliwe przyczyny

1. Dysk jest tylko do odczytu.

1. Dysk jest pełny.

1. Naruszenie zasad współużytkowania.

1. Jeśli komunikat brzmi "zły numer pliku", plik mógł zostać zamknięty na pierwszym planie podczas kompilowania w tle.
