# <a name="compiler-error-c2318"></a>C2318 błąd kompilatora
nie try blok skojarzonych z tym obsługi catch  
  
 A `catch` obsługi jest zdefiniowany, ale nie jest poprzedzona `try` bloku.  
  
 Poniższy przykład generuje C2318:  
  
```  
// C2318.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   // no try block  
   catch( int ) {}   // C2318  
}  
```  
  
 Możliwe rozwiązanie:  
  
```  
// C2318b.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   try{}  
   catch( int ) {}  
}  
```
