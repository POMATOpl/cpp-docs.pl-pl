# <a name="compiler-error-c2018"></a>Błąd kompilatora C2018

nieznany znak "HexNumber"

Plik źródłowy zawiera nieoczekiwany znak ASCII, który jest identyfikowany przez jego liczbę szesnastkową. Aby rozwiązać ten problem, Usuń znak.
