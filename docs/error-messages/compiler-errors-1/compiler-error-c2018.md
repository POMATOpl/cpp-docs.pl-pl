# <a name="compiler-error-c2018"></a>Błąd kompilatora C2018

nieznany znak "hexnumber"

Plik źródłowy zawiera nieoczekiwany znak ASCII, który jest identyfikowany przez jego liczba szesnastkowa. Aby naprawić błąd, należy usunąć znak.
