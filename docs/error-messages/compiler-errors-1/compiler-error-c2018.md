# <a name="compiler-error-c2018"></a>C2018 błąd kompilatora
nieznany znak "hexnumber"  
  
 Plik źródłowy zawiera nieoczekiwany znak ASCII, który jest identyfikowany przez numer szesnastkowych. Aby rozwiązać problem, Usuń znak.
