# <a name="compiler-error-c2487"></a>Błąd kompilatora C2487

"Identyfikator": składowa klasy interfejsu dll nie może być zadeklarowana z interfejsem dll

Można zadeklarować całą klasę lub niektórych elementów członkowskich klasy interfejsu innej niż DLL z interfejsem DLL. Nie można zadeklarować klasy z interfejsem DLL, a następnie zadeklarować składową tej klasy z interfejsem DLL.
