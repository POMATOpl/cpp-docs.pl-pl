# <a name="compiler-error-c2134"></a>Błąd kompilatora C2134

'Funkcja': nie powoduje wywołanie w wyrażeniu stałym

Funkcja zadeklarowana jako constexpr można wywoływać tylko inne funkcje zadeklarowany jako constexpr.

Poniższy przykład spowoduje wygenerowanie C2134:

```
// C2134.cpp
// compile with: /c
int A() {
    return 42;
};

constexpr int B() {
    return A();  // Error C2134: 'A': call does not result in a constant expression.
}
```

Możliwe rozwiązanie:

```
// C2134b.cpp
constexpr int A() {  // add constexpr to A, since it meets the requirements of constexpr.
    return 42;
};

constexpr int B() {
    return A();  // No error
}
```
