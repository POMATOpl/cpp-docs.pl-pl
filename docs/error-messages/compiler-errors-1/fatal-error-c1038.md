# <a name="fatal-error-c1038"></a>Błąd krytyczny C1038
ograniczenie kompilatora: funkcja: zbyt złożone, stan przepływu sterowania Uprość funkcję  
  
 Funkcja ma więcej stanów przepływu sterowania niż kompilator może obsłużyć. Uproszczenie przepływu sterowania lub Podziel funkcji na mniejsze funkcji.
