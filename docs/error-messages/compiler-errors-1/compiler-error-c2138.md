# <a name="compiler-error-c2138"></a>Błąd kompilatora C2138

niedozwolone zdefiniowanie wyliczenia bez składowych

Wyliczenie musi mieć co najmniej jeden element członkowski, gdy jest zaznaczone/za (Wyłącz rozszerzenia Microsoft).
