# <a name="fatal-error-c1008"></a>Błąd krytyczny C1008

nie określono pliku wejściowego

Kompilator nie otrzymał pliku C lub C++ źródłowego do skompilowania. Sprawdź, czy wiersz polecenia **CL** i zmienna środowiskowa są zgodne z specyfikacjami nazw.
