# <a name="fatal-error-c1057"></a>Błąd krytyczny C1057
Nieoczekiwany koniec pliku w rozwinięciu makra  
  
 Kompilator osiągnięto koniec pliku źródłowego podczas zbierania argumentów wywołanie makra, prawdopodobnie z powodu Brak nawiasu zamykającego w wywołaniu makra.
