# <a name="fatal-error-c1045"></a>Błąd krytyczny C1045

ograniczenie kompilatora: wymagania dotyczące konsolidacji są zagnieżdżone zbyt głęboko

Zagnieżdżone wartości zewnętrzne przekraczają limit kompilatora. Zagnieżdżone zewnętrzne są dozwolone z zewnętrznym typem powiązania, takim jak **`extern`** "C++". Zmniejsz liczbę zagnieżdżonych elementów zewnętrznych, aby rozwiązać ten problem.
