# <a name="expression-evaluator-error-cxx0020"></a>Błąd CXX0020 programu Expression Evaluator

typy argumentów operacji są nieprawidłowe dla tej operacji

Operator został zastosowany do wyrażenia z nieprawidłowym typem dla tego operatora.

Na przykład nie jest prawidłowy, aby pobrać adres rejestru lub indeksować tablicę z wyrażeniem zmiennoprzecinkowym.

Ten błąd jest identyczny z CAN0020.
