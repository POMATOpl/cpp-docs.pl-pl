# <a name="resource-compiler-error-rc2135"></a>Błąd kompilatora zasobów RC2135

nie znaleziono pliku: filename

Nie znaleziono pliku określonego w wierszu polecenia kompilatora zasobów. Sprawdź, czy plik został przeniesiony do innego katalogu i czy nazwa pliku i ścieżka zostały wpisane prawidłowo.

Pliki są wyszukiwane przy użyciu zmiennej środowiskowej **include** lub C++ ustawienia Visual **include** .
