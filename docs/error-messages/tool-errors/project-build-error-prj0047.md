# <a name="project-build-error-prj0047"></a>Błąd PRJ0047 kompilacji projektu

Nie można wznowić procesu wstrzymania.  Kompilacja nie powiodła się.

Wystąpił błąd w środowisku deweloperskim.

Aby rozwiązać ten problem, Zamknij i Załaduj ponownie projekt. W razie potrzeby Zamknij i uruchom ponownie program Visual Studio.
