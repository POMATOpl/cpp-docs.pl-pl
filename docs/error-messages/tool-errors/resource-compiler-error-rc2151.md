# <a name="resource-compiler-error-rc2151"></a>Błąd kompilatora zasobów RC2151

nie można ponowne użyć stałych ciągu

Używasz tej samej wartości dwukrotnie w instrukcji **String** . Upewnij się, że nie Mieszasz nakładających się wartości dziesiętnych i szesnastkowych.

Każdy identyfikator w tabeli **ciągów** musi być unikatowy. Aby uzyskać maksymalną wydajność, użyj ciągłych stałych, które zaczynają się od wielokrotności 16.
