# <a name="linker-tools-warning-lnk4076"></a>Ostrzeżenie LNK4076 narzędzi konsolidatora

Nieprawidłowy przyrostowy plik stanu "filename"; Łączenie nieprzyrostowe

LINK nie może zapisywać do pliku stanu przyrostowego (. ilk). `filename`Jest uszkodzony lub nie jest przyrostową bazą danych łączenia. Usuń plik i Połącz ponownie.
