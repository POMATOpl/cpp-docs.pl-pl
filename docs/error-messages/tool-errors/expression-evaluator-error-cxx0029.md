# <a name="expression-evaluator-error-cxx0029"></a>Błąd CXX0029 programu Expression Evaluator

nie jest wskaźnikiem struktury

Operator wyboru elementu członkowskiego ( **->** ) został zastosowany do wyrażenia, które nie jest wskaźnikiem do struktury.

Sprawdź, czy całe wyrażenie jest prawidłowo ujęte w nawiasy, lub wpisz Cast wyrażenie adresu do odpowiedniego typu wskaźnika struktury.

Ten błąd jest identyczny z CAN0029.
