# <a name="expression-evaluator-error-cxx0028"></a>Błąd CXX0028 programu Expression Evaluator

nie jest elementem struct/Union

Określono wyrażenie "struct. member" lub "pStruct->member", ale \<member> nie jest to element struktury.

Wyrażenie nie może być prawidłowo ujęte w nawiasy.

Ten błąd jest identyczny z CAN0028.
