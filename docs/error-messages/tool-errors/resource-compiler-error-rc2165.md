# <a name="resource-compiler-error-rc2165"></a>Błąd kompilatora zasobów RC2165

Ciąg nie został znaleziony w instrukcji DLGINCLUDE

Wykonywanie instrukcji nie określiła nieprawidłowy dołączanego pliku.

**DLGINCLUDE** instrukcji należy użyć następującej składni:

**DLGINCLUDE "**<em>filename</em>**.h"**
