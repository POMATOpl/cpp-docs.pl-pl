# <a name="resource-compiler-error-rc2165"></a>Błąd kompilatora zasobów RC2165

nie znaleziono ciągu w instrukcji DLGINCLUDE

Instrukcja nie określiła prawidłowego pliku dołączanego.

Instrukcja **DLGINCLUDE** musi mieć następującą składnię:

**DLGINCLUDE "**<em>filename</em>**. h"**
