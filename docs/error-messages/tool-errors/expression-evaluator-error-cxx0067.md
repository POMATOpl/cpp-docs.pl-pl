# <a name="expression-evaluator-error-cxx0067"></a>Błąd CXX0067 programu Expression Evaluator
Obliczanie funkcji nie jest obsługiwane.  
  
 Wyrażenie zawierał wywołania funkcji. Niektóre ewaluatorów wyrażeń nie obsługują wywołań funkcji.  
  
 Ten błąd jest taki sam jak CAN0067.
