# <a name="expression-evaluator-error-cxx0058"></a>Błąd CXX0058 programu Expression Evaluator

nie znaleziono przeciążonego operatora

Typ klasy został określony jako lewy argument operacji w wyrażeniu, ale nie zdefiniowano przeciążonego operatora dla klasy.

Ten błąd jest identyczny z CAN0058.
