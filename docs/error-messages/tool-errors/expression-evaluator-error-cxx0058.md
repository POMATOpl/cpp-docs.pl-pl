# <a name="expression-evaluator-error-cxx0058"></a>Błąd CXX0058 programu Expression Evaluator
Nie znaleziono przeciążonego operatora  
  
 Określono typ klasy jako lewego operandu w wyrażeniu, ale Przeciążony operator nie został zdefiniowany dla klasy.  
  
 Ten błąd jest taki sam jak CAN0058.
