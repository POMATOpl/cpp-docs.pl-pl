# <a name="expression-evaluator-error-cxx0058"></a>Błąd CXX0058 programu Expression Evaluator

Przeciążony operator nie znaleziono

Typ klasy został określony jako lewy operand w wyrażeniu, ale Przeciążony operator nie został zdefiniowany dla klasy.

Ten błąd jest taka sama jak CAN0058.
