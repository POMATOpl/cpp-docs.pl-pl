# <a name="cvtres-fatal-error-cvt1105"></a>Błąd krytyczny CVTRES CVT1105

nie można wyszukać w pliku

CVTRES nie może przejść do lokalizacji w pliku.

Ten błąd może być spowodowany uszkodzeniem pliku.
