# <a name="cvtres-fatal-error-cvt1105"></a>Błąd krytyczny CVTRES CVT1105

Nie można wyszukać w pliku

BŁĄD nie można przejść do lokalizacji w pliku.

Ten błąd może być spowodowany przez uszkodzenie pliku.
