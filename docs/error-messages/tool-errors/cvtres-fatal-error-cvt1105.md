# <a name="cvtres-fatal-error-cvt1105"></a>Błąd krytyczny CVTRES CVT1105
Nie można wyszukać w pliku  
  
 CVTRES nie można przejść do lokalizacji pliku.  
  
 Przyczyną tego błędu może być uszkodzenie pliku.
