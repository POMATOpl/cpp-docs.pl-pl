# <a name="resource-compiler-warning-rc4002"></a>Ostrzeżenie RC4002 kompilatora zasobów

zbyt wiele parametrów rzeczywistych dla makra "identifier"

Liczba rzeczywistych parametrów określona za pomocą podanego identyfikatora była większa niż liczba parametrów formalnych podanych w definicji makra identyfikatora.

Dodatkowe rzeczywiste parametry zostały zebrane, ale zignorowane podczas rozszerzania makra.
