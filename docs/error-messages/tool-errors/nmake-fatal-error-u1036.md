# <a name="nmake-fatal-error-u1036"></a>Błąd krytyczny NMAKE U1036

Błąd składniowy: za dużo nazw na lewo od znaku "="

Tylko jeden ciąg jest dozwolone po lewej stronie w definicji makra.
