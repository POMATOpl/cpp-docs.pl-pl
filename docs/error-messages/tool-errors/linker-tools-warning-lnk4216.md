# <a name="linker-tools-warning-lnk4216"></a>Ostrzeżenie LNK4216 narzędzi konsolidatora

Wpis wyeksportowanego punktu wejścia

Punkt wejścia został wyeksportowany z biblioteki DLL. Nie trzeba eksportować punktu wejścia biblioteki DLL.
