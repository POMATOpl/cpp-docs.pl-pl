# <a name="linker-tools-warning-lnk4216"></a>Ostrzeżenie LNK4216 narzędzi konsolidatora

Wpis punktu wejścia wyeksportowanej

Punkt wejścia został wyeksportowany z biblioteki DLL. Punkt wejścia biblioteki DLL nie musi być eksportowane.
