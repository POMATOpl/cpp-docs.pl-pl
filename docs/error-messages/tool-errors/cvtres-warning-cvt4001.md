# <a name="cvtres-warning-cvt4001"></a>Ostrzeżenie CVTRES CVT4001
**Typ urządzenia nie jest określona; założono, że**   
 ***Typ***  
  
 CVTRES nie znaleziono specyfikacji maszyny. Zakłada typu danego komputera. Jeśli wartość domyślna jest niepoprawny, uruchom ponownie CVTRES przy użyciu opcji/Machine.
