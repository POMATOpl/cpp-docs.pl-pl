# <a name="cvtres-warning-cvt4001"></a>Ostrzeżenie CVTRES CVT4001

> Typ maszyny nie jest określona; założono, że *typu*

BŁĄD nie znaleziono specyfikacji maszyny. Przyjmuje typ danego komputera. Jeśli wartość domyślna to CVTRES niepoprawny, ponownie uruchom za pomocą opcji/Machine.
