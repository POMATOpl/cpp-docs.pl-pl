# <a name="cvtres-warning-cvt4001"></a>Ostrzeżenie CVTRES CVT4001

> nie określono typu maszyny; *Typ* przyjęty

CVTRES nie znalazł specyfikacji maszyny. Przyjęto założenie, że dany typ maszyny. Jeśli wartość domyślna jest nieprawidłowa, należy ponownie uruchomić CVTRES przy użyciu opcji/MACHINE.
