# <a name="linker-tools-warning-lnk4229"></a>Ostrzeżenie LNK4229 narzędzi konsolidatora

Znaleziono nieprawidłową dyrektywę/Directive; Ignoruj

Dyrektywa przeniesiona za pośrednictwem dyrektywy pragma [komentarza](../../preprocessor/comment-c-cpp.md) jest nieprawidłowa. Konsolidator ignoruje `/directive` .
