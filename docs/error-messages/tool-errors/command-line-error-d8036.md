# <a name="command-line-error-d8036"></a>Błąd D8036 wiersza polecenia

element "Option" nie jest dozwolony z wieloma plikami źródłowymi

Tych opcji kompilatora nie można używać z wieloma plikami źródłowymi:

- Nazwa pliku zestawu nazw (/fa)

- Zmień nazwę pliku obiektu (/fo)

- Utwórz informacje o przeglądarce źródłowej bez zmiennych lokalnych (/fr)

- Utwórz informacje o przeglądarce źródłowej zawierające zmienne lokalne (/FR)
