# <a name="resource-compiler-fatal-error-rc1205"></a>Błąd krytyczny kompilatora zasobów RC1205

Nieprawidłowa strona kodowa

Po opcji określ stronę kodową (/c) nastąpiła nieprawidłowa strona kodowa. Aby uzyskać więcej informacji, zobacz [strony kodowe](../../c-runtime-library/code-pages.md) w *odwołaniu do biblioteki wykonawczej* .
