# <a name="expression-evaluator-error-cxx0044"></a>Błąd CXX0044 programu Expression Evaluator

Użycie wskaźnika _based (void) wymaga: operator >

Wskaźnika opartego na **`void`** nie można używać bezpośrednio. Należy utworzyć kompletny wskaźnik przy użyciu operatora **: >** .

Ten błąd jest identyczny z CAN0044.
