# <a name="resource-compiler-error-rc2169"></a>Błąd kompilatora zasobów RC2169

Nazwa pliku zasobu nie jest w formacie 2,03

Określony zasób używał formatu wcześniejszego niż wersja 2,03. Plik zasobów musi być konwertowany lub ponownie utworzony przy użyciu formatu w wersji 3,00 lub nowszej.
