# <a name="linker-tools-error-lnk1248"></a>Błąd narzędzi konsolidatora LNK1248

> rozmiar obrazu ("*rozmiar*") przekracza maksymalny dozwolony rozmiar (80000000)

Konsolidator ustalić, czy rozmiar pliku danych wyjściowych przekroczy największy rozmiar obrazu program 32-bitowy. Możesz przekształcić program wiele bibliotek DLL.
