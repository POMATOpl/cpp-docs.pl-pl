# <a name="linker-tools-error-lnk1248"></a>Błąd narzędzi konsolidatora LNK1248

> rozmiar obrazu ("*size*") przekracza maksymalny dozwolony rozmiar (80000000)

Konsolidator ustalił, że rozmiar pliku wyjściowego będzie przekroczyć największy możliwy rozmiar obrazu programu 32-bitowego. Możesz chcieć wprowadzić program do wielu bibliotek DLL.
