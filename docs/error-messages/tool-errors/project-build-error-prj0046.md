# <a name="project-build-error-prj0046"></a>Błąd PRJ0046 kompilacji projektu

Nie można zduplikować wiersza polecenia, ponieważ określony był pusty.

Dla konfiguracji pliku reguł programu make określono pusty wiersz polecenia. wymagany jest wiersz polecenia.
