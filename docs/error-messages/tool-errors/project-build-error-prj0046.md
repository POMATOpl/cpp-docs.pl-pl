# <a name="project-build-error-prj0046"></a>Błąd PRJ0046 kompilacji projektu
Nie można zduplikować wiersza polecenia, ponieważ określona była pusta.  
  
 Określono pusty wiersz polecenia dla konfiguracji pliku reguł programu make. Wiersz polecenia jest wymagana.
