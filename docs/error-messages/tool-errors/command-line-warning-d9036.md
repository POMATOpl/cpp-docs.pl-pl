# <a name="command-line-warning-d9036"></a>Ostrzeżenie D9036 dla wiersza polecenia

> "*option\_2*" zamiast "*Option\_1*"

## <a name="remarks"></a>Uwagi

Ostrzeżenie D9036 jest kontynuacją [ostrzeżenia wiersza polecenia D9035](../../error-messages/tool-errors/command-line-warning-d9035.md). Jeśli jedna opcja kompilatora jest preferowana zamiast opcji przestarzałe, zostanie wyświetlona w tym miejscu. Aby uzyskać linki do dodatkowych informacji, zobacz [Opcje kompilatora przestarzałe i usunięte](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options).
