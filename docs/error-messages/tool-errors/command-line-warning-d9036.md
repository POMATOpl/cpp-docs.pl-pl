# <a name="command-line-warning-d9036"></a>Ostrzeżenie D9036 dla wiersza polecenia

> "*opcji\_2*"zamiast z"*opcji\_1*"

## <a name="remarks"></a>Uwagi

Ostrzeżenie d9036 dla stanowi kontynuację [wiersza polecenia ostrzeżenie d9035 dla](../../error-messages/tool-errors/command-line-warning-d9035.md). Czy jedną z opcji kompilatora nie jest opcją przestarzałe, są one dostępne w tym miejscu. Aby uzyskać linki do szczegółowych informacji, zobacz [opcje kompilatora usunięte i przestarzałe](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options).
