# <a name="project-build-error-prj0028"></a>Błąd PRJ0028 kompilacji projektu
Plik tymczasowy 'Plik' zawiera znaki unikodowe, które nie mogą być przekonwertowana na stronę kodową ANSI użytkownika.  
  
 Określono wartość z [/MIDL (Określanie MIDL wiersza polecenia opcji)](../../build/reference/midl-specify-midl-command-line-options.md) opcji konsolidatora, które nie zostały rozpoznane przez stronę kodową systemu.  
  
 Strona kodowa używany do określania polecenia MIDL (strona wejściowego kodu) musi być taka sama jak stronę kodową systemu.
