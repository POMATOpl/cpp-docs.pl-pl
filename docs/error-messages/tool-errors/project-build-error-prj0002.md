# <a name="project-build-error-prj0002"></a>Błąd PRJ0002 kompilacji projektu
wynik błędu zwrócony z "wiersza polecenia".  
  
 W poleceniu ***wiersza polecenia***, który został utworzony z danych wejściowych użytkownika w **strony właściwości** wyświetli się okno dialogowe, zwrócony kod błędu, ale żadne informacje, w oknie danych wyjściowych.  
  
 Rozwiązania z tym błędem zależy od tego, które narzędzie wygenerował błąd. Dla MIDL uzyskasz informacje o tym, co poszło źle, jeśli zdefiniowano /o (Przekieruj dane wyjściowe).  
  
 Plik wsadowy, takich jak niestandardowego kroku budowania lub zdarzenia kompilacji, która nie jest informacji o warunkach błędu mogą być również przyczynę tego błędu.
