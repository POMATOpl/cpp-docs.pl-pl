# <a name="expression-evaluator-error-cxx0010"></a>Błąd CXX0010 programu Expression Evaluator

Brak lewego nawiasu

Wyrażenie zawiera niezrównoważone nawiasy kwadratowe. Wpisz ponownie wyrażenie z wyrównanymi nawiasami kwadratowymi.

Ten błąd jest identyczny z nie0,010.
