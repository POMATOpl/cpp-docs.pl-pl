# <a name="math-error-m6101"></a>Błąd matematyczny M6101
nieprawidłowe  
  
 Nieprawidłowa operacja.  
  
 Ten błąd może być spowodowany, gdy argument jest wartością typu NaN (nieliczbową) lub nieskończonością.  
  
 Program kończy się z kodem zakończenia 129.
