# <a name="math-error-m6101"></a>Błąd matematyczny M6101

nieprawidłowe

Nieprawidłowa operacja.

Ten błąd może być spowodowany tym, że argument operacji to NaN (nie liczba) lub nieskończoność.

Program kończy pracę z kodem zakończenia 129.
