# <a name="project-build-error-prj0004"></a>Błąd PRJ0004 kompilacji projektu
Nie można wygenerować wiersza polecenia dla narzędzia "narzędzia".  
  
 Co najmniej jednej właściwości zostały określone w taki sposób, aby składnie wywołanie ***narzędzie*** niedozwolone:  
  
-   Być może określono makra źle sformułowana lub jest nieznany.  
  
-   Komputer może być za mało wolnego miejsca na dysku.  
  
 Warto przejrzeć ustawienia narzędzia analizując [strony właściwości wiersza polecenia](../../ide/command-line-property-pages.md).
