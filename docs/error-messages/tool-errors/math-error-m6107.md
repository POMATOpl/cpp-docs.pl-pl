# <a name="math-error-m6107"></a>Błąd matematyczny M6107

unemulated

Próbowano wykonać instrukcję Koprocesor, który jest nieprawidłowy lub nie jest obsługiwana przez emulator.

Program kończy się z kodem zakończenia 135.
