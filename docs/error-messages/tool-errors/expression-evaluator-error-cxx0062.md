# <a name="expression-evaluator-error-cxx0062"></a>Błąd CXX0062 programu Expression Evaluator

wywołania konstruktora nie są obsługiwane

Wyrażenie wykonał wywołanie konstruktora. Wyrażenia nie mogą wprowadzać jawnych wywołań do konstruktorów lub dokonać konwersji, które wymagają wywołania konstruktora.

Ten błąd jest identyczny z CAN0062.
