# <a name="resource-compiler-error-rc2163"></a>Błąd kompilatora zasobów RC2163
wymagany typ akceleratora [ASCII lub VIRTKEY]  
  
 `type` w **AKCELERATORÓW** instrukcja musi zawierać wartość ASCII lub VIRTKEY.
