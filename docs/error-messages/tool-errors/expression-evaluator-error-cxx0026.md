# <a name="expression-evaluator-error-cxx0026"></a>Błąd CXX0026 programu Expression Evaluator

nieprawidłowy ciąg formatu

Ciąg formatu został nieprawidłowo określony. Sprawdź składnię wyrażenia.

Ten błąd jest identyczny z CAN0026.

Aby uzyskać więcej informacji, zobacz [specyfikatory formatu w języku C++](/visualstudio/debugger/format-specifiers-in-cpp).
