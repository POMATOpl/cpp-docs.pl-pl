# <a name="resource-compiler-fatal-error-rc1021"></a>Błąd krytyczny kompilatora zasobów RC1021

Nieprawidłowe polecenie preprocesora "String"

Znaki następujące po znaku numeru ( **#** ) nie tworzą prawidłowej dyrektywy preprocesora.

Nie można użyć znaku numeru jako pierwszego znaku w identyfikatorze.
