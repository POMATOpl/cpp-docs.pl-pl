# <a name="expression-evaluator-error-cxx0012"></a>Błąd CXX0012 programu Expression Evaluator

Brak lewy nawias klamrowy

Wyrażenie zawiera niezrównoważone nawiasu klamrowego. Ponownie wpisz wyrażenie za pomocą o zrównoważonym obciążeniu nawiasów klamrowych.

Ten błąd jest taka sama jak CAN0012.
