# <a name="expression-evaluator-error-cxx0012"></a>Błąd CXX0012 programu Expression Evaluator
Brak lewego nawiasu klamrowego  
  
 Wyrażenie zawiera nawias klamrowy równowagi. Wpisz wyrażenie w nawiasach klamrowych zrównoważone.  
  
 Ten błąd jest taki sam jak CAN0012.
