# <a name="expression-evaluator-error-cxx0012"></a>Błąd CXX0012 programu Expression Evaluator

Brak lewego nawiasu klamrowego

Wyrażenie zawiera niezrównoważone nawiasy klamrowe. Wpisz ponownie wyrażenie z wyrównanymi nawiasami klamrowymi.

Ten błąd jest identyczny z CAN0012.
