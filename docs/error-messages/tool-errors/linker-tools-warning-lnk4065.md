# <a name="linker-tools-warning-lnk4065"></a>Ostrzeżenie LNK4065 narzędzi konsolidatora

Nie można porządkować 'Funkcja'; ignorowane

Daną funkcję nie został skompilowany w spakowanych funkcji. Skompiluj ponownie przy użyciu [/Gy](../../build/reference/gy-enable-function-level-linking.md).
