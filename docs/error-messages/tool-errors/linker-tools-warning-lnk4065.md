# <a name="linker-tools-warning-lnk4065"></a>Ostrzeżenie LNK4065 narzędzi konsolidatora

 Dana funkcja nie został skompilowany jako spakowanych funkcji. Skompiluj ponownie przy użyciu [/Gy](../../build/reference/gy-enable-function-level-linking.md).
