# <a name="resource-compiler-error-rc2127"></a>Błąd kompilatora zasobów RC2127

oczekiwane wyrazy w wersji oddzielone przecinkami

Numery wersji w zasobie wersji powinny być typu **Word** i oddzielone przecinkami.
