# <a name="resource-compiler-error-rc2127"></a>Błąd kompilatora zasobów RC2127
Wersja wyrazy oddzielone przecinkami, oczekiwano  
  
 Numery wersji w zasobie wersji powinien być typu **WORD**i oddzielone przecinkami.
