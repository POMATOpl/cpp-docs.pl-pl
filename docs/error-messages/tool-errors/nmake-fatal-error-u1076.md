# <a name="nmake-fatal-error-u1076"></a>Błąd krytyczny NMAKE U1076

Nazwa jest za długa

Ciąg przekroczenie jednego z następujących ograniczeń:

- 1024 znaki dla nazwy makra.

- 256 znaków w nazwie ścieżki docelowej.

- 2048 znaków dla polecenia.
