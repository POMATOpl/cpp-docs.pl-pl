# <a name="nmake-fatal-error-u1076"></a>Błąd krytyczny NMAKE U1076

zbyt długa nazwa

Ciąg przekracza jeden z następujących limitów:

- 1024 znaków dla nazwy makra.

- 256 znaków dla docelowej nazwy ścieżki.

- 2048 znaków dla polecenia.
