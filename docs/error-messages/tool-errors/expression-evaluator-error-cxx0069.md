# <a name="expression-evaluator-error-cxx0069"></a>Błąd CXX0069 programu Expression Evaluator

Zmienna wymaga ramki stosu

Ewaluatora wyrażeń nie może obliczyć zmiennej, ponieważ nie występuje ona w ramce stosu. Przyczyną może być zmienne zadeklarowane jako część wbudowanej funkcji.
