# <a name="cvtres-fatal-error-cvt1103"></a>Błąd krytyczny CVTRES CVT1103

nie można odczytać nazwy pliku

Wystąpił nieodwracalny błąd podczas próby odczytania danego pliku przez CVTRES.

Ten błąd może być spowodowany uszkodzeniem pliku.
