# <a name="expression-evaluator-error-cxx0032"></a>Błąd CXX0032 programu Expression Evaluator

dzielenie przez 0

Wyrażenie zawiera dzielnik równy zero, co jest niedozwolone. Ten dzielnik może być numerem literału równym zero lub może być wyrażeniem, którego wynikiem jest zero.

Ten błąd jest identyczny z CAN0032.
