# <a name="resource-compiler-fatal-error-rw1009"></a>Błąd krytyczny kompilatora zasobów RW1009

Wystąpił błąd podczas tworzenia nazwy zasobu

Nie można utworzyć plik .res określony. Upewnij się, że nie jest on tworzony na dysku tylko do odczytu. Użyj /V, aby dowiedzieć się, czy plik jest tworzona.
