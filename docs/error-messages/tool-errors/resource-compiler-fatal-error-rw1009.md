# <a name="resource-compiler-fatal-error-rw1009"></a>Błąd krytyczny kompilatora zasobów RW1009
Wystąpił błąd podczas tworzenia nazwy zasobu  
  
 Nie można utworzyć .res określonego pliku. Upewnij się, że nie został on utworzony na dysku tylko do odczytu. Użyj /V, aby dowiedzieć się, czy plik jest tworzony.
