# <a name="nmake-fatal-error-u1077"></a>Błąd krytyczny NMAKE U1077

"program": Kod powrotu "value"

Podanym poleceniem lub programie wywoływanym przez NMAKE nie powiodło się i zwróciło dany kod zakończenia.

Aby pominąć ten błąd i kontynuować sesję NMAKE, użyj opcji/I **. Ignoruj** dyrektywę z kropką lub modyfikator poleceń łącznika ( **-** ). Aby kontynuować sesję NMAKE dla niepowiązanych części drzewa zależności, użyj opcji/K.
