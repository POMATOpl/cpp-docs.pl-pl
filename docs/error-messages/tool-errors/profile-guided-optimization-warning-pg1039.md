# <a name="profile-guided-optimization-warning-pg1039"></a>Ostrzeżenie PG1039 dotyczące optymalizacji sterowania profilem

Niepowodzenie podczas oczekiwania na odczyszczenie czasu.

Aplikacja nie zezwala na uruchomienie [pgosweep](../../build/pgosweep.md) .  Jeśli to możliwe, uruchom pgosweep, gdy aplikacja jest w stanie bezczynności.
