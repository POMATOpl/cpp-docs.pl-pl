# <a name="profile-guided-optimization-warning-pg1039"></a>Ostrzeżenie PG1039 dotyczące optymalizacji sterowania profilem

Aplikacja nie zezwala na [pgosweep](../../build/reference/pgosweep.md) do uruchomienia.  Jeśli to możliwe należy uruchomić pgosweep, gdy aplikacja jest w stanie bezczynności.
