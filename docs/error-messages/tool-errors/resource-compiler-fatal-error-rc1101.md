nie zasobów binarnych nie określono nazwy pliku  
  
 Zmiana nazwy wyjściowego (/ fo) opcja nie nastąpiła nazwę pliku.  
  
 /Fo opcji, należy użyć następującej składni:  
  
 **RC /fo**
