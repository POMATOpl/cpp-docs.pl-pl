# <a name="linker-tools-warning-lnk4073"></a>Ostrzeżenie LNK4073 narzędzi konsolidatora

Nie można utworzyć mapy dla pliku .ilk; zostanie wykonana Konsolidacja nieprzyrostowa

Nie ma dużą wystarczającej ilości miejsca ciągły w pamięci współdzielonej, łącza utworzyć plik stanu przyrostowego (.ilk). LINK wykonywane nieprzyrostowa kompilacji.
