# <a name="linker-tools-warning-lnk4073"></a>Ostrzeżenie LNK4073 narzędzi konsolidatora

nie można utworzyć mapy dla pliku. ilk; Łączenie nieprzyrostowe

W pamięci współdzielonej nie ma wystarczająco dużo miejsca, aby można było utworzyć LINK do tworzenia przyrostowego pliku stanu (. ilk). LINK wykonał kompilację nieprzyrostową.
