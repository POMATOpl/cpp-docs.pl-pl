# <a name="project-build-error-prj0005"></a>Błąd PRJ0005 kompilacji projektu

Nie można utworzyć pliku tymczasowego w katalogu "Directory".

Wywołanie tworzenia tymczasowego pliku nie powiodło się. Przyczyny niepowodzenia obejmują:

- Zabrakło tymczasowych nazw plików.

- Katalog Temp jest tylko do odczytu.

- Brak katalogu tymczasowego lub zmiennej środowiskowej TMP.

- Brak wolnego miejsca na dysku komputera.
