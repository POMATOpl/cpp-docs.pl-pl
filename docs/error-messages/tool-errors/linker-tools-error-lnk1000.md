# <a name="linker-tools-error-lnk1000"></a>Błąd narzędzi konsolidatora LNK1000
Nieznany błąd; zobacz dokumentację, aby uzyskać informacje o opcjach pomocy technicznej  
  
 Uwaga okoliczności błąd, spróbuj ustalić przyczynę problemu, a następnie utwórz odtworzenia przypadek testowy, następnie skontaktuj się z `Microsoft Product Support Services`. Aby uzyskać informacje na temat zbadania i raportuje te błędy, zobacz [http://support.microsoft.com/default.aspx?scid=kb;en-us;134650](http://support.microsoft.com/default.aspx?scid=kb;en-us;134650).  
  
 Błąd ten może wystąpić, jeśli mieszać plików standardowy nagłówek (na przykład dos.h) i pliki. `#include`standardowych nagłówków po pliki nagłówka.
