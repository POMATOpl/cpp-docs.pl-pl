# <a name="c-runtime-errors-r6002-through-r6035"></a>C Runtime błędy R6002 po R6035
Ta sekcja ma odwołanie do błędów, które mogą być generowane przez biblioteki środowiska uruchomieniowego C (CRT), gdy aplikacja jest załadowany lub nie działają. Są to błędy wskazujące usterki w kodzie aplikacji lub warunek, który nie może obsługiwać Biblioteka środowiska uruchomieniowego, takie jak brak pamięci. Jeśli nie kod aplikację, aby zapobiec warunki, które powodują ich użytkowników końcowych w aplikacji mogą zostać wyświetlone te błędy.
