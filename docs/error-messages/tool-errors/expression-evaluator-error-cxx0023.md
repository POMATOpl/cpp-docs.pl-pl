# <a name="expression-evaluator-error-cxx0023"></a>Błąd CXX0023 programu Expression Evaluator

zła podstawy

Ewaluatora wyrażeń języka C nie rozpoznaje określonego podstawy. Prawidłowe są tylko wartości dziesiętne, szesnastkowe i ósemkowe radixes.

Ten błąd jest identyczny z CAN0023.
