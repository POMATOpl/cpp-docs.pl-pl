# <a name="resource-compiler-fatal-error-rc1018"></a>Błąd krytyczny kompilatora zasobów RC1018

Nieoczekiwany element "#elif"

`#elif`Dyrektywa nie została wyświetlona w `#if` konstrukcji, **#ifdef** lub **#ifndef** .

`#if`Przed tą instrukcją upewnij się, że jest stosowana instrukcja, **#ifdef** lub **#ifndef** .
