# <a name="resource-compiler-fatal-error-rc1018"></a>Błąd krytyczny kompilatora zasobów RC1018

Nieoczekiwany element "#elif"

Dyrektywa `#elif` nie była wyświetlana w konstrukcji `#if`, **#ifdef**lub **#ifndef** .

Upewnij się, że instrukcja `#if`, **#ifdef**lub **#ifndef** obowiązuje przed tą instrukcją.
