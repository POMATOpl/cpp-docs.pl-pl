# <a name="expression-evaluator-error-cxx0061"></a>Błąd CXX0061 programu Expression Evaluator

przekroczono głębokość zagnieżdżania funkcji

Wyrażenie zawiera głębokość zagnieżdżenia funkcji większą niż limit. Zmodyfikuj wyrażenie, aby zmniejszyć głębokość zagnieżdżenia.

Ten błąd jest identyczny z CAN0061.
