# <a name="bscmake-error-bk1516"></a>Błąd BSCMAKE BK1516

bscfile uszkodzony; nie można zaktualizować przyrostowo

Plik. bsc został uszkodzony, prawdopodobnie z powodu błędu systemu podczas kompilacji. Usuń plik BSC, Skompiluj ponownie wszystkie pliki. sbr, a następnie Skompiluj ponownie plik. BSC.
