# <a name="bscmake-error-bk1517"></a>Błąd BSCMAKE BK1517

plik źródłowy dla sbrfile skompilowany z/Yc i/Yu

Plik. sbr odwołuje się do samego siebie. Prawdopodobnie został ponownie skompilowany przy użyciu/Yu po kompilacji z/Yc. Zresetuj opcję kompilatora dla pliku źródłowego na/YC, a następnie wybierz pozycję **Skompiluj ponownie** , aby wygenerować nowe pliki. sbr. Nie Kompiluj ponownie pliku źródłowego z/Yu.
