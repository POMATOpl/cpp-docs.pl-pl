# <a name="expression-evaluator-error-cxx0072"></a>Błąd CXX0072 programu Expression Evaluator

Błąd: brak informacji o typie lub są one nieznane

Plik. PCH nie został połączony w programie lub kod zawiera odwołanie do typu, który znajduje się w module nieskompilowanym za pomocą/zi.

Ten błąd jest identyczny z CAN0067.
