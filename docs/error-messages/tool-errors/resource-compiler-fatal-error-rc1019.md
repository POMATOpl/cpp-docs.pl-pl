# <a name="resource-compiler-fatal-error-rc1019"></a>Błąd krytyczny kompilatora zasobów RC1019

Nieoczekiwany element "#else"

`#else`Dyrektywa nie została wyświetlona w `#if` konstrukcji, **#ifdef** lub **#ifndef** .

`#if`Przed tą instrukcją upewnij się, że jest stosowana instrukcja, **#ifdef** lub **#ifndef** .
