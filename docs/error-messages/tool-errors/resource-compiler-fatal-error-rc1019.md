# <a name="resource-compiler-fatal-error-rc1019"></a>Błąd krytyczny kompilatora zasobów RC1019

Nieoczekiwany element "#else"

Dyrektywa `#else` nie była wyświetlana w konstrukcji `#if`, **#ifdef**lub **#ifndef** .

Upewnij się, że instrukcja `#if`, **#ifdef**lub **#ifndef** obowiązuje przed tą instrukcją.
