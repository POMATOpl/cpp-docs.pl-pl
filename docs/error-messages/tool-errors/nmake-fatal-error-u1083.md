# <a name="nmake-fatal-error-u1083"></a>Błąd krytyczny NMAKE U1083

docelowe makro "target" jest rozwijane do Nothing

Podanym elementem docelowym jest wywołanie makra, które nie zostało zdefiniowane lub ma wartość null. NMAKE nie może przetworzyć elementu docelowego o wartości null.
