# <a name="expression-evaluator-error-cxx0063"></a>Błąd CXX0063 programu Expression Evaluator

Przeciążony operator -> nie obsługiwane

Wyrażenie używany operator dostępu do składowej klasy przeciążona (**->**).

Ten błąd jest taka sama jak CAN0063.
