# <a name="expression-evaluator-error-cxx0063"></a>Błąd CXX0063 programu Expression Evaluator

przeciążony operator — > nie jest obsługiwana

Wyrażenie używało operatora dostępu do składowej klasy przeciążonej ( **->** ).

Ten błąd jest identyczny z CAN0063.
