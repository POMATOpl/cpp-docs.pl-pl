# <a name="linker-tools-error-lnk1218"></a>Błąd narzędzi konsolidatora LNK1218

Ostrzeżenie traktowane jako błąd; nie Wygenerowano pliku wyjściowego

Po połączeniu z **/WX** wszystkie ostrzeżenia konsolidatora będą traktowane jako błędy i nie zostanie utworzony żaden plik wyjściowy.

Aby uzyskać więcej informacji [, zobacz/WX (Traktuj ostrzeżenia konsolidatora jako błędy)](../../build/reference/wx-treat-linker-warnings-as-errors.md) .
