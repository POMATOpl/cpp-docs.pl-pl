# <a name="resource-compiler-fatal-error-rc1121"></a>Błąd krytyczny kompilatora zasobów RC1121

Błąd we/wy podczas odczytywania pliku

Kompilator zasobów nie może odczytać pliku. Sprawdź, czy dysk zawierający plik jest dostępny i czy plik jest prawidłowy.
