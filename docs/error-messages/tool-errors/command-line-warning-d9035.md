# <a name="command-line-warning-d9035"></a>Ostrzeżenie D9035 dla wiersza polecenia

opcji "opcji" jest przestarzała i zostanie usunięta w przyszłej wersji

Określono opcję kompilatora, która zostanie usunięta w przyszłej wersji kompilatora Visual C++. To ostrzeżenie występuje ostrzeżenie [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) przypadku sugerowane zastępuje *opcji*.

Określona opcja nadal działa, ale należy zaktualizować teraz konfiguracji kompilacji. Co w efekcie projekt jest bardziej prawdopodobne kontynuować kompilacji po uaktualnieniu kompilator.
