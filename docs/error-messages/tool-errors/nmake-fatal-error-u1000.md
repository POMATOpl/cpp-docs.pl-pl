# <a name="nmake-fatal-error-u1000"></a>Błąd krytyczny NMAKE U1000

> Błąd składniowy: Brak znaku ")" w wywołaniu makra

Lewy nawias **(** ,, który pojawił się bez pasującego prawego nawiasu **)** , w wywołaniu makra. Prawidłowym formularzem jest **$ (** <em>name</em> **)** ; **$** <em>n</em> jest dozwolony dla jednoznakowych nazw.
