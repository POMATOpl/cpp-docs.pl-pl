# <a name="nmake-warning-u4011"></a>Ostrzeżenie NMAKE U4011
'target': nie wszystkie elementy zależne, które są dostępne; Nie utworzono docelowego  
  
 Nie istnieje lub została nieaktualne zależne od danego obiektu docelowego, a polecenia aktualizowania zależnego zwróciło kod zakończenia różną od zera. Opcję/k informację NMAKE kontynuować przetwarzanie niepowiązanych części kompilacji i wydania kod zakończenia 1, po zakończeniu sesji NMAKE.  
  
 To ostrzeżenie jest poprzedzony ostrzeżenie [U4010](../../error-messages/tool-errors/nmake-warning-u4010.md) dla poszczególnych zależnych, że nie można utworzyć lub zaktualizować.
