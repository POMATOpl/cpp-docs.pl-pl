# <a name="linker-tools-error-lnk1314"></a>Błąd narzędzi konsolidatora LNK1314

uszkodzona lub nieprawidłowa tabela symboli COFF (niezdefiniowany znak statyczny lub symbol etykiety)

Zawartość wygenerowanego przez kompilator dla sekcji w danym obiekcie prawdopodobnie jest uszkodzona.

Ten błąd może reprezentować błąd kompilatora.
