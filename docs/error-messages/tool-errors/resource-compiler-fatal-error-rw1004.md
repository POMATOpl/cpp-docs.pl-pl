# <a name="resource-compiler-fatal-error-rw1004"></a>Błąd krytyczny kompilatora zasobów RW1004

Nieoczekiwany koniec pliku

Ten błąd może być spowodowany brakiem znaku wysuwu wiersza i znakami powrotu karetki w ostatnim wierszu pliku tekstowego.
