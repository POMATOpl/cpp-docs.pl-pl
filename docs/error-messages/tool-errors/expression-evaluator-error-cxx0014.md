# <a name="expression-evaluator-error-cxx0014"></a>Błąd CXX0014 programu Expression Evaluator

Brak operandu

Operator został określony bez wymaganego operandu. Sprawdź składnię wyrażenia.

Ten błąd jest identyczny z CAN0014.
