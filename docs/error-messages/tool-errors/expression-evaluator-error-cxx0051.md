# <a name="expression-evaluator-error-cxx0051"></a>Błąd CXX0051 programu Expression Evaluator

w punkcie przerwania określono więcej niż jeden przeciążony symbol

Ewaluatora wyrażeń nie może określić, który z więcej niż jeden przeciążony symbol ma być używany jako punkt przerwania.

Ten błąd jest identyczny z CAN0051.
