# <a name="linker-tools-warning-lnk4071"></a>Ostrzeżenie LNK4071 narzędzi konsolidatora
Nie można konsolidować przyrostowo podczas kolejnych konsolidacji  
  
 LINK odnaleziono wiele definicji dla jednego lub więcej symboli, ale [/FORCE](../../build/reference/force-force-file-output.md) lub **/FORCE:MULTIPLE** został użyty do utworzenia pliku wyjściowego, bez względu na błędy. ŁĄCZE usunięto plik stanu przyrostowego (.ilk).
