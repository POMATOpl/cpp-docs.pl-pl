# <a name="linker-tools-warning-lnk4071"></a>Ostrzeżenie LNK4071 narzędzi konsolidatora

Nie można konsolidować przyrostowo podczas kolejnych konsolidacji

LINK znaleziono wiele definicji dla jednego lub wielu symboli, ale [/FORCE](../../build/reference/force-force-file-output.md) lub **/FORCE:MULTIPLE** został użyty do utworzenia pliku wyjściowego, niezależnie od tego, błędy. LINK usunięto plik stanu przyrostowego (.ilk).
