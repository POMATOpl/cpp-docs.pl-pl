# <a name="linker-tools-warning-lnk4071"></a>Ostrzeżenie LNK4071 narzędzi konsolidatora

nie można łączyć przyrostowo z kolejnymi łączami

LINK znalazł wiele definicji dla co najmniej jednego symbolu, ale [/Force](../../build/reference/force-force-file-output.md) lub **/Force: wielokrotność** została użyta do utworzenia pliku wyjściowego bez względu na błędy. LINK usunął plik stanu przyrostowego (. ilk).
