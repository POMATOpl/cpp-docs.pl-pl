# <a name="resource-compiler-error-rc2171"></a>Błąd kompilatora zasobów RC2171
Nieznany format nagłówka DIB  
  
 Nagłówek mapy bitowej nie jest **BITMAPCOREHEADER** lub **BITMAPINFOHEADER** struktury.
