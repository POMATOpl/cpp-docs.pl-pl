# <a name="nmake-fatal-error-u1055"></a>Błąd krytyczny NMAKE U1055

Brak miejsca w środowisku

W systemie operacyjnym zabrakło miejsca na zmienne środowiskowe.

Zwiększ ilość miejsca w środowisku lub ustaw mniejszą liczbę zmiennych środowiskowych.
