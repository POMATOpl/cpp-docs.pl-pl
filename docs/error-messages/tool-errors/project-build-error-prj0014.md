# <a name="project-build-error-prj0014"></a>Błąd PRJ0014 kompilacji projektu

Obiekt zadania służący do kontrolowania procesów duplikowanych nie powiódł się.  Kompilacja nie może być kontynuowana.

Wystąpił błąd w środowisku deweloperskim.

Aby rozwiązać ten problem, Zamknij i Załaduj ponownie projekt. W razie potrzeby Zamknij i uruchom ponownie program Visual Studio.
