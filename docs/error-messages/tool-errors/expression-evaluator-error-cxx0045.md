# <a name="expression-evaluator-error-cxx0045"></a>Błąd CXX0045 programu Expression Evaluator

nie jest funkcją

Podano listę argumentów dla symbolu w programie, który nie jest nazwą funkcji.

## <a name="example"></a>Przykład

```
queue( alpha, beta )
```

gdy `queue` nie jest funkcją.

Ten błąd jest identyczny z CAN0045.
