# <a name="resource-compiler-fatal-error-rc1208"></a>Błąd krytyczny kompilatora zasobów RC1208

. Rozszerzenie RES jest używane dla danych wyjściowych kompilatora zasobów. . Należy używać rozszerzeń RC dla skryptów kompilator zasobów.
