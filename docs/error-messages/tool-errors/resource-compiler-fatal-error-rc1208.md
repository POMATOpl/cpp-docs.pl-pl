# <a name="resource-compiler-fatal-error-rc1208"></a>Błąd krytyczny kompilatora zasobów RC1208

plik wejściowy ma. Rozszerzenie zasobu

Polu. Rozszerzenie RES jest używane na potrzeby danych wyjściowych kompilatora zasobów. Polu. Rozszerzenia RC należy używać w skryptach kompilatora zasobów.
