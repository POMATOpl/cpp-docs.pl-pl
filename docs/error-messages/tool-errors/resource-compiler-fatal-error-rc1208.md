# <a name="resource-compiler-fatal-error-rc1208"></a>Błąd krytyczny kompilatora zasobów RC1208
Plik wejściowy ma. Rozszerzenie RES  
  
 . Rozszerzenie RES jest używane dla danych wyjściowych kompilatora zasobów. . Rozszerzeń RC powinien być używany dla kompilatora zasobów skryptów.
