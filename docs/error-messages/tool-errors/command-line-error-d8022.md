# <a name="command-line-error-d8022"></a>Błąd D8022 wiersza polecenia

nie można otworzyć "messagefile"

Podany plik nie znajduje się w bieżącym katalogu lub w katalogu określonym w zmiennej środowiskowej PATH. Plik wiadomości zawiera krótkie podsumowanie składni i opcji wiersza polecenia kompilatora.

Przenieś ten plik do bieżącego katalogu lub katalogu w bieżącej ścieżce. Jeśli nie można znaleźć tego pliku, uruchom program instalacyjny, aby skopiować go z dysków dystrybucji.
