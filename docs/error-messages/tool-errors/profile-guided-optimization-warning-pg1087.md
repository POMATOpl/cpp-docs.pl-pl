# <a name="profile-guided-optimization-warning-pg1087"></a>Ostrzeżenie PG1087 dotyczące optymalizacji sterowania profilem

Plik. PGC jest uszkodzony, prawdopodobnie z powodu nietypowego zakończenia sesji profilowania.  Uruchom ponownie scenariusz profilowania i Zakończ normalne działanie.
