# <a name="expression-evaluator-error-cxx0060"></a>Błąd CXX0060 programu Expression Evaluator
Rejestr jest niedostępny  
  
 Wyrażenie określone rejestru, której nie można użyć. Ten błąd może być spowodowany przez próby uzyskania dostępu do rejestru, który nie istnieje na komputerze, na którym działa.  
  
 Ten błąd jest taki sam jak CAN0060.
