# <a name="nmake-warning-u4001"></a>Ostrzeżenie NMAKE U4001

> plik polecenia można wywołać tylko z wiersza polecenia

Plik poleceń, który jest wywoływany przez specyfikator at znaku ( **\@** ), nie może zawierać specyfikacji dla innego pliku polecenia. Takie zagnieżdżenie jest niedozwolone. Specyfikacja została zignorowana.
