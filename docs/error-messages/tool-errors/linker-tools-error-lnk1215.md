# <a name="linker-tools-error-lnk1215"></a>Błąd narzędzi konsolidatora LNK1215
Operacja metadanych nie powiodła się (wynik HRESULT): błąd  
  
 Konsolidator odebrała błąd od środowiska uruchomieniowego .NET podczas próby aktualizacji metadanych przez środowisko uruchomieniowe platformy .NET.  
  
 `HRESULT`jest HRESULT z metody środowiska uruchomieniowego .NET. `error`jest. Podana sieć tekst.  
  
 Prawdopodobnie niedopasowanych konsolidatora i środowiska uruchomieniowego .NET; Zainstaluj ponownie Visual C++.
