# <a name="resource-compiler-error-rc2017"></a>Błąd kompilatora zasobów RC2017

niedozwolona sekwencja ucieczki

Sekwencja ucieczki pojawiła się, gdzie nie jest oczekiwana.

Sekwencja ucieczki — ukośnik odwrotny ( **\\** ), po którym następuje cyfra lub litera — może wystąpić tylko w stałej znakowej lub ciągu.
