# <a name="linker-tools-warning-lnk4002"></a>Ostrzeżenie LNK4002 narzędzi konsolidatora

symbol zdefiniowany w obiekcie

Symbol, wyświetlany w jego postaci dekoracyjnej, został określony w postaci niedekoracyjnej w `object` , ale nie można znaleźć unikatowego dopasowania do symbolu dekoracyjnego. To ostrzeżenie jest zawsze poprzedzane przez ostrzeżenie [LNK4022 narzędzi konsolidatora](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) i następuje błąd krytyczny [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
