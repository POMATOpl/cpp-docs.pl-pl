# <a name="expression-evaluator-error-cxx0025"></a>Błąd CXX0025 programu Expression Evaluator

Operator wymaga struktury/Unii

Operator, który przyjmuje wyrażenie `struct` lub typu **złożenia** , został zastosowany do wyrażenia, które nie jest elementem `struct` lub **Union**.

Składniki klasy, struktury lub Unii muszą mieć w pełni kwalifikowaną nazwę. Nie można wprowadzać składników bez pełnej specyfikacji.

Ten błąd jest identyczny z CAN0025.
