# <a name="expression-evaluator-error-cxx0025"></a>Błąd CXX0025 programu Expression Evaluator
operator wymaga określenia struktury/Unii  
  
 Operator, który przyjmuje wyrażenia `struct` lub **Unii** typu została zastosowana do wyrażenia, który nie jest `struct` lub **Unii**.  
  
 Składniki klasy, struktury lub Unii zmiennych muszą mieć w pełni kwalifikowana nazwa. Nie można wprowadzić składniki bez pełnego specyfikacji.  
  
 Ten błąd jest taki sam jak CAN0025.
