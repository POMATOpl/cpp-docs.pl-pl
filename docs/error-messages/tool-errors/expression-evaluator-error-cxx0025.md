# <a name="expression-evaluator-error-cxx0025"></a>Błąd CXX0025 programu Expression Evaluator

Operator potrzebuje struct/union

Operator, który przyjmuje wyrażenia `struct` lub **Unii** typ został zastosowany do wyrażenia, który nie jest `struct` lub **Unii**.

Elementy klasy, struktury lub Unii zmiennych muszą mieć w pełni kwalifikowana nazwa. Nie można wprowadzić składniki bez pełną specyfikację.

Ten błąd jest taka sama jak CAN0025.
