# <a name="resource-compiler-error-rc2170"></a>Błąd kompilatora zasobów RC2170

Nazwa pliku mapy bitowej nie jest w formacie 3,00

Mapy bitowe używające formatu systemu Windows w wersji 2. x nie mogą być używane w plikach zasobów w wersji 3. x.

Aby rozwiązać ten problem, ponownie narysuj lub przekonwertuj mapę bitową na format 3. x.
