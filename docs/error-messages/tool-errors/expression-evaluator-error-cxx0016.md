# <a name="expression-evaluator-error-cxx0016"></a>Błąd CXX0016 programu Expression Evaluator
stała jest za duży  
  
 Ewaluator wyrażeń C nie może zaakceptować stała liczba całkowita bez znaku większych niż 4 294 967 295 (0FFFFFFFF szesnastkowym) lub stała zmiennoprzecinkowa, którego wartość jest większa niż około 1.8E + 308.  
  
 Ten błąd jest taki sam jak CAN0016.
