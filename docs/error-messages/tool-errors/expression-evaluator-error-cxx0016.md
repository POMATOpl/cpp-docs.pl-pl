# <a name="expression-evaluator-error-cxx0016"></a>Błąd CXX0016 programu Expression Evaluator

stała za duża

Ewaluatora wyrażeń języka C nie może akceptować stałej całkowitej bez znaku większego niż 4 294 967 295 (0FFFFFFFF w formacie szesnastkowym) lub stałej zmiennoprzecinkowej, której rozmiar jest większy niż około 1.8 E + 308.

Ten błąd jest identyczny z CAN0016.
