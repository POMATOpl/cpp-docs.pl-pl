# <a name="linker-tools-error-lnk1141"></a>Błąd narzędzi konsolidatora LNK1141
Błąd podczas kompilowania pliku eksportów  
  
 [ŁĄCZE](../../build/reference/linker-command-line-syntax.md) nie można utworzyć pliku eksportu (.exp).  
  
 Przyczyny tego błędu obejmować problemy systemu, np. za mało pamięci i składni błędy w opcjach lub instrukcji definicji modułu.  
  
 Ten błąd jest poprzedzony inny błąd, który zapewnia dodatkowe informacje.
