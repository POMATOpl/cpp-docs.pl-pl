# <a name="linker-tools-error-lnk1141"></a>Błąd narzędzi konsolidatora LNK1141

Wystąpił błąd podczas kompilowania pliku eksportów

[LINK](../../build/reference/linking.md) nie można utworzyć pliku eksportu (.exp).

Wystąpienia tego błędu przyczyny problemów z systemem np. za mało pamięci i składnię błędy w opcji lub instrukcji definicji modułu.

Ten błąd jest poprzedzony przez inny błąd, który zapewnia dodatkowe informacje.
