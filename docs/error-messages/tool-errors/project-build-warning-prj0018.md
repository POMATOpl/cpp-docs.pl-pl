# <a name="project-build-warning-prj0018"></a>Ostrzeżenie PRJ0018 dotyczące kompilacji projektu

Nie znaleziono następujących zmiennych środowiskowych:

Zmienna środowiskowa nie jest zdefiniowana. Ten błąd zawiera listę zmiennych środowiskowych, które nie zostały zdefiniowane.

Zobacz [makra dla niestandardowych poleceń kompilacji](../../build/reference/common-macros-for-build-commands-and-properties.md) , aby uzyskać informacje dotyczące makr w procesie kompilacji.
