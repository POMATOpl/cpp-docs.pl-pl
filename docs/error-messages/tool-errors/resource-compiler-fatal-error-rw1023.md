# <a name="resource-compiler-fatal-error-rw1023"></a>Błąd krytyczny kompilatora zasobów RW1023

Błąd we/wy podczas zapisywania pliku, dysk jest pełny

Wolne miejsce musi mieć co najmniej dwa razy większy rozmiar tworzonego pliku wykonywalnego.
