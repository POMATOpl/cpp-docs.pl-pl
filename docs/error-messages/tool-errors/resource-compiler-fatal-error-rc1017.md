# <a name="resource-compiler-fatal-error-rc1017"></a>Błąd krytyczny kompilatora zasobów RC1017
Nieprawidłowa liczba całkowita stałego wyrażenia  
  
 Wyrażenie w `#if` dyrektywy nie istnieje lub nie zostało oszacowane jako stała.
