# <a name="linker-tools-warning-lnk4044"></a>Ostrzeżenie LNK4044 narzędzi konsolidatora

Nierozpoznana opcja "Option"; Ignoruj

Dana opcja nie jest prawidłową opcją dla tego narzędzia. Narzędzie zignoruje tę opcję.

Aby wyświetlić listę prawidłowych opcji, uruchom narzędzie bez argumentów lub opcji.
