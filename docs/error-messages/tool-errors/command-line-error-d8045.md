# <a name="command-line-error-d8045"></a>Błąd D8045 wiersza polecenia
Nie można skompilować pliku C "file" z opcją/CLR  
  
 Tylko C++ — pliki kodu źródłowego, mogą zostać przekazane do kompilacji, która używa **/CLR**.  Użyj **/TP** skompilować pliku .c jako plik .cpp; zobacz [/TC, / TP, / TC, /TP (Określ źródłowy plik typ)](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) Aby uzyskać więcej informacji.  
  
 Aby uzyskać więcej informacji, zobacz [/CLR (kompilacja języka wspólnego środowiska uruchomieniowego)](../../build/reference/clr-common-language-runtime-compilation.md).  
  
 D8045 polecenia może również wystąpić, jeśli kompilacja aplikacji ATL przy użyciu programu Visual C++. Zobacz [porady: Migracja do/CLR](../../dotnet/how-to-migrate-to-clr.md) Aby uzyskać więcej informacji.
