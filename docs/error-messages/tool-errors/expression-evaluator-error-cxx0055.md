# <a name="expression-evaluator-error-cxx0055"></a>Błąd CXX0055 programu Expression Evaluator

niejednoznaczny symbol w operatorze kontekstu

Symbol w operatorze kontekstu ( **{}** ) odnosi się do więcej niż jednego symbolu w programie.

Operator rozpoznawania zakresu ( `::` ) może być w stanie rozpoznać niejednoznaczność.

Ten błąd jest identyczny z CAN0055.
