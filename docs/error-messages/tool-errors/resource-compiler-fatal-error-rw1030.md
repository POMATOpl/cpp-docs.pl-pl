# <a name="resource-compiler-fatal-error-rw1030"></a>Błąd krytyczny kompilatora zasobów RW1030

Błąd danych wyjściowych

Ten błąd może być spowodowany tym, że kompilator zasobów nie mógł utworzyć pliku docelowego. Upewnij się, że jest wystarczająca ilość miejsca na dysku i że masz uprawnienia do zapisu w woluminie.
