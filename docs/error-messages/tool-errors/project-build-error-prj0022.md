# <a name="project-build-error-prj0022"></a>Błąd PRJ0022 kompilacji projektu

> Nieznane narzędzie, właściwość '*właściwość*"zawiera niepoprawną nazwę pliku"*pliku*".

Nazwa pliku *pliku* określony we właściwości *właściwość* był nieprawidłowy.

Być może użyto nieznanego lub nieprawidłowego — makro.

Zobacz [ustawienie właściwości projektu Visual C++](../../ide/working-with-project-properties.md) informacji na temat uzyskiwania dostępu do właściwości projektu.
