# <a name="project-build-error-prj0022"></a>Błąd PRJ0022 kompilacji projektu
Nieznane narzędzie, właściwość "property" zawiera nieprawidłowy "pliku".  
  
 Nazwa pliku ***pliku***, określonego we właściwości ***właściwości***, jest nieprawidłowy.  
  
 Użyto nieznanego lub nieprawidłowego makra.  
  
 Zobacz [Ustawianie właściwości projektu Visual C++](../../ide/working-with-project-properties.md) informacji na temat sposobu dostępu do właściwości projektu.
