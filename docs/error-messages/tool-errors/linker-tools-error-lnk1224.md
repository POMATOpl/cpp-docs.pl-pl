# <a name="linker-tools-error-lnk1224"></a>Błąd narzędzi konsolidatora LNK1224

Nieprawidłowy adres podstawowy obrazu

Określono nieprawidłowy adres podstawowy obrazu. Adresy podstawowe muszą być wyrównane do 64 KB (ostatnie cztery cyfry szesnastkowe muszą być równe zero), a podstawa obrazu musi być zgodna z wartością 32-bitową podpisaną lub bez znaku.
