# <a name="resource-compiler-fatal-error-rc1109"></a>Błąd krytyczny kompilatora zasobów RC1109

błąd podczas tworzenia nazwy zasobu

Nie można utworzyć określonego pliku. res. Upewnij się, że nie jest on tworzony na dysku tylko do odczytu. Użyj/V opcji, aby dowiedzieć się, czy plik jest tworzony.
