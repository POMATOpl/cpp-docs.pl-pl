# <a name="bscmake-error-bk1515"></a>Błąd BSCMAKE BK1515
bscfile: niezgodna wersja: nie można zaktualizować przyrostowo  
  
 Plik .bsc został utworzony z inną wersją BSCMAKE. Utwórz ponownie pliku .bsc.
