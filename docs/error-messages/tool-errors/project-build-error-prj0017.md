# <a name="project-build-error-prj0017"></a>Błąd PRJ0017 kompilacji projektu

Bieżący katalog roboczy jest nieprawidłowy.

Ścieżka do bieżącego katalogu roboczego jest dłuższa niż _MAXPATH. Aby rozwiązać ten problem, nie należy tworzyć projektu na tym poziomie.
