# <a name="linker-tools-warning-lnk4039"></a>Ostrzeżenie LNK4039 narzędzi konsolidatora

sekcja "name" określona w opcji/SECTION nie istnieje

[Polecenia DUMPBIN](../../build/reference/dumpbin-reference.md) lub [polecenia EDITBIN](../../build/reference/editbin-reference.md) nie może znaleźć sekcji o nazwie `name` w pliku wejściowym.

Aby wyświetlić sekcje w pliku, uruchom polecenia DUMPBIN z opcją [/Headers](../../build/reference/headers.md) .
