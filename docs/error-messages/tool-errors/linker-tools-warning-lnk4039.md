# <a name="linker-tools-warning-lnk4039"></a>Ostrzeżenie LNK4039 narzędzi konsolidatora

sekcja "name" określony za pomocą opcji/Section nie istnieje

[DUMPBIN](../../build/reference/dumpbin-reference.md) lub [EDITBIN](../../build/reference/editbin-reference.md) nie można znaleźć sekcję o nazwie `name` w pliku wejściowym.

Aby zobaczyć sekcje w pliku, uruchom DUMPBIN z [/HEADERS](../../build/reference/headers.md) opcji.
