# <a name="linker-tools-warning-lnk4039"></a>Ostrzeżenie LNK4039 narzędzi konsolidatora
sekcja "name" określony w opcji/Section nie istnieje  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) lub [polecenia EDITBIN](../../build/reference/editbin-reference.md) nie można odnaleźć sekcji o nazwie `name` w pliku wejściowym.  
  
 Aby wyświetlić sekcji w pliku, Uruchom polecenia DUMPBIN z [/HEADERS](../../build/reference/headers.md) opcji.
