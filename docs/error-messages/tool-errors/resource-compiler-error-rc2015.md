# <a name="resource-compiler-error-rc2015"></a>Błąd kompilatora zasobów RC2015
zbyt wiele znaków w stałej  
  
 Stała znakowa zawiera więcej niż dwa znaki.  
  
 Stałe znakowe są ograniczone do jednego znaku (stałe znakowe standardowe) lub dwóch znaków (stałe znakowe długa).  
  
 Należy pamiętać, że sekwencja (na przykład \t karta) jest konwertowana na pojedynczym znakiem.
