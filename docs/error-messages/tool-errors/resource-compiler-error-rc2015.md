# <a name="resource-compiler-error-rc2015"></a>Błąd kompilatora zasobów RC2015

zbyt wiele znaków w stałej

Stała znakowa zawiera więcej niż dwa znaki.

Stałe znakowe są ograniczone do jednego znaku (stałe znaku standardowego) lub dwóch znaków (stałych znaków długich).

Należy zauważyć, że sekwencja ucieczki (na przykład \t for Tab) jest konwertowana na pojedynczy znak.
