# <a name="resource-compiler-error-rc2015"></a>Błąd kompilatora zasobów RC2015

zbyt wiele znaków w stałej

Stałej znakowej zawiera więcej niż dwa znaki.

Stałe znaków są ograniczone do jednego znaku (stałych znaków standardowych) lub dwa znaki (stałe znaków długie).

Należy pamiętać, że sekwencja ucieczki (na przykład, \t karta) jest konwertowana na pojedynczy znak.
