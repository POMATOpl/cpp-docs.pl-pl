# <a name="math-error-m6205"></a>Błąd matematyczny M6205
"Funkcja": _tloss — błąd  
  
 Wystąpił całkowitej utraty istotności (precyzja).  
  
 Ten błąd może być spowodowany przez nadanie bardzo dużą liczbą jako argument sin, cos lub tan, ponieważ argument musi być ograniczone do liczbą z zakresu od 0 do 2 * pi.
