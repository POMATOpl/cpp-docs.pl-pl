# <a name="resource-compiler-error-rc2152"></a>Błąd kompilatora zasobów RC2152

nieprawidłowy znak kontrolny

Znak kontrolny w instrukcji **akceleratorów** jest nieprawidłowy. Prawidłowy znak kontrolny składa się z jednej litery (tylko) po znaku daszka ( **^** ).
