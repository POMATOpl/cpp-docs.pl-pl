# <a name="nmake-fatal-error-u1086"></a>Błąd krytyczny NMAKE U1086

**Reguła wnioskowania nie może mieć elementów zależnych**

Następujący dwukropek ( **:** ) w regule wnioskowania musi mieć jedną z następujących wartości:

- Znak nowego wiersza

- Średnik ( **;** ), do którego może następować polecenie

- Znak numeru ( **#** ), do którego może następować komentarz
