# <a name="expression-evaluator-error-cxx0066"></a>Błąd CXX0066 programu Expression Evaluator

Statyczny element członkowski jest nieobecny

Statyczny element członkowski klasy nie został odnaleziony lub nie został zdefiniowany. Ten błąd może wynikać ze statycznej składowej klasy, która jest zadeklarowana, ale nie jest zdefiniowana, lub jest tylko zdefiniowana i przywoływana w modułach, które nie zawierają informacji debugowania.

Ten błąd jest identyczny z CAN0066.
