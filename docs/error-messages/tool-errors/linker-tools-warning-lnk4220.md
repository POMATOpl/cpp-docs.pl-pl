# <a name="linker-tools-warning-lnk4220"></a>Ostrzeżenie LNK4220 narzędzi konsolidatora

Nieprawidłowa "wartość" opcji konsolidatora "value"; przyjęto wartość domyślną

Wartość poza zakresem została określona za pomocą opcji [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) . Wartość domyślna dla **/TLBID** to 1.
