# <a name="resource-compiler-fatal-error-rc1203"></a>Błąd krytyczny kompilatora zasobów RC1203

określono nieprawidłowy szesnastkowy identyfikator języka domyślnego.

Po opcji określ język domyślny (/l) następują nieprawidłowe identyfikatory języka szesnastkowego. Aby zapoznać się z listą prawidłowych identyfikatorów języka, zobacz [ciągi języka i kraju/regionu](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) w dokumentacji dotyczącej *biblioteki wykonawczej* .
