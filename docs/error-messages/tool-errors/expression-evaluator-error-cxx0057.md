# <a name="expression-evaluator-error-cxx0057"></a>Błąd CXX0057 programu Expression Evaluator

Brak kodu w numerze wiersza

Nie Wygenerowano żadnego kodu dla określonego numeru wiersza. Nie może być używany jako punkt przerwania.

Ten błąd jest identyczny z CAN0057.
