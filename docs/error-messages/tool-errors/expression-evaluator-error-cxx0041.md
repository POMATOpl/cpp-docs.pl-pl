# <a name="expression-evaluator-error-cxx0041"></a>Błąd CXX0041 programu Expression Evaluator
element klasy musi być statycznym elementem członkowskim lub funkcją członkowską  
  
 Niestatycznego elementu członkowskiego klasy (lub struktury lub Unii) została użyta bez określania, które podczas tworzenia wystąpienia klasy do użycia.  
  
 Bez określania wystąpienia mogą służyć tylko statyczne elementy członkowskie danych lub funkcji elementów członkowskich.  
  
 Ten błąd jest taki sam jak CAN0041.
