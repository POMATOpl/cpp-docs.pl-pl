# <a name="expression-evaluator-error-cxx0041"></a>Błąd CXX0041 programu Expression Evaluator

element klasy musi być statycznym elementem członkowskim lub funkcją członkowską

Niestatyczny element członkowski klasy (lub struktury lub Unii) został użyty bez określenia, które wystąpienie klasy ma być używane.

Tylko statyczne elementy członkowskie danych lub funkcje składowe mogą być używane bez określania wystąpienia.

Ten błąd jest identyczny z CAN0041.
