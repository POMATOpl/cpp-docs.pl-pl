# <a name="expression-evaluator-error-cxx0041"></a>Błąd CXX0041 programu Expression Evaluator

element klasy musi być statyczny element członkowski lub funkcji składowej

Niestatycznej składowej klasy (lub struktury lub Unii) użyto bez określania, które podczas tworzenia wystąpienia klasy do użycia.

Bez określania egzemplarzem można tylko statyczne elementy członkowskie danych ani funkcji elementów członkowskich.

Ten błąd jest taka sama jak CAN0041.
