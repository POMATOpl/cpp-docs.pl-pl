# <a name="expression-evaluator-error-cxx0007"></a>Błąd CXX0007 programu Expression Evaluator

Brak prawego nawiasu klamrowego

W wyrażeniu znaleziono niezrównoważone nawiasy. Wpisz ponownie wyrażenie z wyrównanymi nawiasami.

Ten błąd jest identyczny z CAN0007.
