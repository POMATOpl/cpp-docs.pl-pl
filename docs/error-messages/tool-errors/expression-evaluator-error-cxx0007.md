# <a name="expression-evaluator-error-cxx0007"></a>Błąd CXX0007 programu Expression Evaluator
Brak nawiasu zamykającego  
  
 Nieparzysta liczba nawiasów znaleziono w wyrażeniu. Wpisz wyrażenie o zrównoważonym nawiasów.  
  
 Ten błąd jest taki sam jak CAN0007.
