# <a name="expression-evaluator-error-cxx0007"></a>Błąd CXX0007 programu Expression Evaluator

Brak prawy nawias okrągły

Znaleziono niezrównoważone nawiasów w wyrażeniu. Ponownie wpisz wyrażenie w nawiasach o zrównoważonym obciążeniu.

Ten błąd jest taka sama jak CAN0007.
