# <a name="expression-evaluator-error-cxx0006"></a>Błąd CXX0006 programu Expression Evaluator

Brak lewego nawiasu

W wyrażeniu znaleziono niezrównoważone nawiasy. Wpisz ponownie wyrażenie z wyrównanymi nawiasami.

Ten błąd jest identyczny z CAN0006.
