# <a name="expression-evaluator-error-cxx0006"></a>Błąd CXX0006 programu Expression Evaluator
Brak lewego nawiasu  
  
 Nieparzysta liczba nawiasów znaleziono w wyrażeniu. Wpisz wyrażenie o zrównoważonym nawiasów.  
  
 Ten błąd jest taki sam jak CAN0006.
