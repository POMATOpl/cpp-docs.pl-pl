# <a name="expression-evaluator-error-cxx0006"></a>Błąd CXX0006 programu Expression Evaluator

Brak lewy nawias okrągły

Znaleziono niezrównoważone nawiasów w wyrażeniu. Ponownie wpisz wyrażenie w nawiasach o zrównoważonym obciążeniu.

Ten błąd jest taka sama jak CAN0006.
