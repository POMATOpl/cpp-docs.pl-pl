# <a name="linker-tools-error-lnk2026"></a>Błąd narzędzi konsolidatora LNK2026

Moduł niebezpieczny dla obrazu SAFESEH

[/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) został określony, ale moduł nie był zgodny z funkcją obsługi wyjątków bezpiecznych. Jeśli chcesz użyć tego modułu z **/SAFESEH**, należy ponownie skompilować moduł.
