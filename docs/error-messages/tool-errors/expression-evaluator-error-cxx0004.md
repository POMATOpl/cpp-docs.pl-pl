# <a name="expression-evaluator-error-cxx0004"></a>Błąd CXX0004 programu Expression Evaluator

Błąd składniowy

Składnia wyrażenia jest niepoprawna.

Wpisz ponownie wyrażenie, używając poprawnej składni.

Ten błąd jest taki sam jak w przypadku programu.
