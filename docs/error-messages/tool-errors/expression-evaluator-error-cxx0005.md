# <a name="expression-evaluator-error-cxx0005"></a>Błąd CXX0005 programu Expression Evaluator

nieobsługiwany operator

W wyrażeniu określono nieobsługiwany operator języka C. Napisz równoważne wyrażenie przy użyciu obsługiwanych operatorów języka C.

Ten błąd jest identyczny z nierówną.
