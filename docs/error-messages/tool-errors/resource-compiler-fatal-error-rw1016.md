# <a name="resource-compiler-fatal-error-rw1016"></a>Błąd krytyczny kompilatora zasobów RW1016

Kończenie działania RC po błędach preprocesora

Kompilator zasobów został zatrzymany z powodu innych błędów. Usuń inne błędy i ponownie skompiluj.
