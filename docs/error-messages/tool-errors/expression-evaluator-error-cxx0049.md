# <a name="expression-evaluator-error-cxx0049"></a>Błąd CXX0049 programu Expression Evaluator
przestarzały element OMF — Połącz ponownie program  
  
 Program używany starego OMF (Format obiektu modułu).  
  
 Połącz ponownie program przy użyciu bieżącej wersji konsolidatora.  
  
 Ten błąd jest taki sam jak CAN0049.
