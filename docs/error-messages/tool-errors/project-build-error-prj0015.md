# <a name="project-build-error-prj0015"></a>Błąd PRJ0015 kompilacji projektu

W systemie brakuje urządzenia o wartości NULL. Nie można uruchomić kompilacji.

Przyczyną tego błędu może być mała z zasobów systemowych. Zamknij niektóre aplikacje lub Uruchom ponownie.

Może być również przyczyną, że nie masz wystarczających uprawnień do urządzenia o wartości NULL na komputerze.
