# <a name="bscmake-error-bk1509"></a>Błąd BSCMAKE BK1509

za mało miejsca na stercie

BSCMAKE zabrakło pamięci, w tym pamięci wirtualnej.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aby rozwiązać ten problem, można użyć następujących rozwiązań

1. Zwolnij trochę miejsca na dysku.

1. Zwiększ rozmiar pliku wymiany.

1. Zwiększ rozmiar pliku wymiany systemu Windows.

1. Ogranicz BSCMAKE pamięci wymaganej przez użycie/EI lub/ES, aby wyeliminować niektóre pliki wejściowe lub/em w celu wyeliminowania treści makr.
