# <a name="linker-tools-error-lnk1136"></a>Błąd narzędzi konsolidatora LNK1136
nieprawidłowy lub uszkodzony plik  
  
 Plik wejściowy ma uszkodzony nagłówek lub zero wyjątkowo małych lub rozmiar.  
  
 Jeśli plik został niepoprawnie określony jako dodatkowe zależności, może być również spowodowane LNK1136.
