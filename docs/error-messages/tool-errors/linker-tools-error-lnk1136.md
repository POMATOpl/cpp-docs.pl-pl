# <a name="linker-tools-error-lnk1136"></a>Błąd narzędzi konsolidatora LNK1136

nieprawidłowy lub uszkodzony plik

Plik wejściowy zawiera nagłówek w uszkodzona lub jest równe zeru, rozmiar lub jest nieprawidłowo małe.

Jeśli plik został niepoprawnie określony jako dodatkową zależność, może być także spowodowane LNK1136.
