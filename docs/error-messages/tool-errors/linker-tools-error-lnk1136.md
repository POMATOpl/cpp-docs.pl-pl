# <a name="linker-tools-error-lnk1136"></a>Błąd narzędzi konsolidatora LNK1136

nieprawidłowy lub uszkodzony plik

Plik wejściowy ma uszkodzony nagłówek lub ma rozmiar zerowy lub nietypowy.

LNK1136 może również być przyczyną, jeśli plik został niepoprawnie określony jako zależność dodatkowa.
