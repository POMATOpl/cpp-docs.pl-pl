# <a name="project-build-error-prj0025"></a>Błąd PRJ0025 kompilacji projektu
Plik wsadowy 'Plik' zawiera znaki unikodowe, które nie mogą być przekonwertowana na stronę kodową ANSI użytkownika.  
  
 ***Zawartość pliku UNICODE***  
  
 System projektu znaleziono zawartości Unicode w niestandardowej reguły kompilacji lub utworzyć zdarzenie, którego nie można przetłumaczyć prawidłowo na stronę kodową ANSI bieżącego użytkownika.  
  
 Rozwiązanie dotyczące tego błędu jest aby zaktualizować zawartość reguły kompilacji lub utworzyć zdarzenia użyj ANSI lub Zainstaluj stronę kodową na komputerze i ustaw go jako domyślnego systemu.  
  
 Aby uzyskać więcej informacji na temat niestandardowe kroki procesu kompilacji i zdarzenia kompilacji, zobacz [niestandardowych krokach kompilacji zrozumienia i tworzenie zdarzenia](../../ide/understanding-custom-build-steps-and-build-events.md).
