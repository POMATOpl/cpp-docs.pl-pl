# <a name="expression-evaluator-error-cxx0008"></a>Błąd CXX0008 programu Expression Evaluator

**Brak "na końcu ciągu**

Brakuje podwójnego cudzysłowu na końcu literału ciągu. Wpisz ponownie wyrażenie, umieszczając literał ciągu w podwójnym cudzysłowie.

Ten błąd jest identyczny z CAN0008.
