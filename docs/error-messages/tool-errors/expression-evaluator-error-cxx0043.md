# <a name="expression-evaluator-error-cxx0043"></a>Błąd CXX0043 programu Expression Evaluator

Ten wskaźnik używany poza funkcją składowej

**`this`** Wskaźnika można używać tylko dla niestatycznych funkcji składowych.

Ten błąd jest identyczny z CAN0043.
