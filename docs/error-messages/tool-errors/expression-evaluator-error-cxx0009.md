# <a name="expression-evaluator-error-cxx0009"></a>Błąd CXX0009 programu Expression Evaluator

Brak "po znaku stałej"

Brakuje cudzysłowu pojedynczego na końcu stałej znakowej. Wpisz ponownie wyrażenie, umieszczając znak stałej w pojedynczym cudzysłowie.

Ten błąd jest identyczny z CAN0009.
