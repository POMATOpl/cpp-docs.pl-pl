# <a name="expression-evaluator-error-cxx0013"></a>Błąd CXX0013 programu Expression Evaluator
Brak operatora  
  
 Operator oczekiwany wyrażenie, ale nie został znaleziony. Sprawdź składnię wyrażenia.  
  
 Ten błąd jest taki sam jak CAN0013.
