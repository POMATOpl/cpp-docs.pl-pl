# <a name="project-build-error-prj0040"></a>Błąd PRJ0040 kompilacji projektu
Błąd wewnętrzny podczas kompilacji. Nie można kontynuować. Załaduj projekt ponownie i spróbuj ponownie.  
  
 System projektu nie może przetworzyć projektu.  
  
 Spróbuj załadować ponownie projekt.
