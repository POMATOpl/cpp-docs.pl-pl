# <a name="project-build-error-prj0040"></a>Błąd PRJ0040 kompilacji projektu

Wystąpił błąd wewnętrzny podczas kompilowania. Nie można kontynuować. Załaduj ponownie projekt i spróbuj ponownie.

System projektu nie mógł przetworzyć Twojego projektu.

Spróbuj ponownie załadować projekt.
