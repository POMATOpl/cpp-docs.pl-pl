# <a name="writecr8"></a>__writecr8
**Dotyczące firmy Microsoft**  
  
 Zapisuje wartość `Data` do rejestru CR8.  
  
## <a name="syntax"></a>Składnia  
  
```  
void writecr8(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 [in]`Data`  
 Wartość do zapisu do rejestru CR8.
