# <a name="int2c"></a>__int2c

**Microsoft Specific**

Generuje `int 2c` instrukcji, co powoduje wyzwolenie `2c` przerwania.

## <a name="syntax"></a>Składnia

```
void __int2c(void);
```

## <a name="requirements"></a>Wymagania

|Wewnętrzne|Architektura|
|---------------|------------------|
|`__int2c`|x86, x64|
