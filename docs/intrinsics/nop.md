# <a name="nop"></a>__nop
**Microsoft Specific**  
  
 Generuje kod maszynowy specyficzne dla platformy, który wykonuje żadnych operacji.  
  
## <a name="syntax"></a>Składnia  
  
```  
void __nop();  
```
