# <a name="outdword"></a>__outdword
**Microsoft Specific**  
  
 Generuje `out` instrukcji można wysłać z bitowego `Data` z portu `Port`.  
  
## <a name="syntax"></a>Składnia  
  
```  
void __outdword(   
   unsigned short Port,   
   unsigned long Data   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 [in] `Port`  
 Port do wysyłania danych do.  
  
 [in] `Data`  
 Bitowego do wysłania.
