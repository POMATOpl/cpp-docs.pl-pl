# <a name="__m128"></a>__m128

**Specyficzne dla firmy Microsoft**

Typ **danych __m128** , używany z instrukcjami Streaming SIMD Extensions i Streaming SIMD Extensions 2, jest zdefiniowany w \<xmmintrin. h >.

```cpp
// data_types__m128.cpp
#include <xmmintrin.h>
int main() {
   __m128 x;
}
```

## <a name="remarks"></a>Uwagi

Nie należy bezpośrednio uzyskiwać dostępu do pól **__m128** . Można jednak wyświetlić te typy w debugerze. Zmienna typu **__m128** jest mapowana na rejestry XMM [0-7].

Zmienne typu **__m128** są automatycznie wyrównane na granicach 16-bajtowych.

Typ danych **__m128** nie jest obsługiwany w procesorach ARM.
