# <a name="m128"></a>__m128

## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft

**Danych __m128** typ do użytku z programem Streaming SIMD Extensions i Streaming SIMD Extensions 2 funkcje wewnętrzne instrukcji, jest zdefiniowany w \<xmmintrin.h >.

```cpp
// data_types__m128.cpp
#include <xmmintrin.h>
int main() {
   __m128 x;
}
```

## <a name="remarks"></a>Uwagi

Nie mają do **__m128** pól bezpośrednio. Można jednak zobaczyć te typy w debugerze. Zmienna typu **__m128** mapy do rejestrów XMM [0-7].

Zmienne typu **__m128** będą automatycznie wyrównywane do 16-bajtowych granicach.

**__M128** typ danych nie jest obsługiwany w procesorach ARM.

**END specyficzny dla Microsoft**
