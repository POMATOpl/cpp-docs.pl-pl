# <a name="statements-c"></a>Instrukcje (C++)
Instrukcje C++ są określające, jak i w jakiej kolejności obiekty są manipulować elementów programu. Ta sekcja zawiera:  
  
-   [Omówienie](../cpp/overview-of-cpp-statements.md)  
  
-   [Labeled — instrukcje](../cpp/labeled-statements.md)  
  
-   Kategorie — instrukcje  
  
    -   [Instrukcje wyrażeń](../cpp/expression-statement.md). Te instrukcje obliczyć wyrażenia, dla jej efekty uboczne lub jego wartości zwracanej.  
  
    -   [Wartość null, instrukcje](../cpp/null-statement.md). Instrukcje te można podać w przypadku, gdy instrukcja jest wymagany przez składni języka C++, ale nie są podejmowane żadne akcje.  
  
    -   [Instrukcje złożone](../cpp/compound-statements-blocks.md). Te instrukcje są grup instrukcji ujętą w nawiasy klamrowe ({}). Mogą one używane wszędzie tam, gdzie można stosować jednej instrukcji.  
  
    -   [Instrukcje wyboru](../cpp/selection-statements-cpp.md). Te instrukcje wykonać test; następnie realizują jedną sekcję kodu, jeśli test zwraca wartość true (różną od zera). Mogą one wykonać innej sekcji kodu, gdy test jest spełniony.  
  
    -   [Instrukcje iteracji](../cpp/iteration-statements-cpp.md). Te instrukcje Podaj wielokrotnego wykonywania blok kodu do momentu spełnienia kryterium określonego rozwiązania.  
  
    -   [Instrukcje skoku](../cpp/jump-statements-cpp.md). Te instrukcje natychmiast transfer kontroli do innej lokalizacji w funkcji lub zwrócić kontrolkę z funkcji.  
  
    -   [Instrukcje deklaracji](http://msdn.microsoft.com/en-us/14538558-356f-450e-9e1e-3cd62ba952b9). Deklaracje wprowadzić nazwę programu. ([Deklaracje i definicje](declarations-and-definitions-cpp.md) zawiera bardziej szczegółowe informacje o deklaracjach.)
