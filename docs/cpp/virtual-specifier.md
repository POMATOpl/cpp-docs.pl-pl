# <a name="virtual-specifier"></a>virtual — specyfikator

[Wirtualne](../cpp/virtual-cpp.md) słowo kluczowe można zastosować tylko do funkcji niestatycznej klasy członkowskiej. Oznacza to, że powiązanie wywołań funkcji jest odroczone do czasu wykonywania. Aby uzyskać więcej informacji, zobacz [funkcje wirtualne](../cpp/virtual-functions.md).
