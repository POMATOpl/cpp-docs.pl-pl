# <a name="__m128i"></a>__m128i

**Microsoft Specific**

Typ danych **__m128i** , używany z instrukcjami Streaming SIMD Extensions 2 (SSE2), jest zdefiniowany w \<emmintrin. h >.

```cpp
// data_types__m128i.cpp
#include <emmintrin.h>
int main() {
   __m128i x;
}
```

## <a name="remarks"></a>Uwagi

Nie należy bezpośrednio uzyskiwać dostępu do pól **__m128i** . Możesz jednak zobaczyć te typy w debugerze. Zmienna typu **__m128i** jest mapowana na rejestry XMM [0-7].

Zmienne typu **_m128i** są automatycznie wyrównane na granicach 16-bajtowych.

> [!NOTE]
>  Użycie zmiennych typu **__m128i** spowoduje wygenerowanie instrukcji SSE2 `movdqa` przez kompilator. Ta instrukcja nie powoduje błędów w procesorach Pentium III, ale spowoduje niepowodzenie ciche, z możliwymi efektami ubocznymi spowodowanymi przez dowolnych instrukcji `movdqa` przekładają się na procesory Pentium III.

Typ danych **__m128i** nie jest obsługiwany w procesorach ARM.
