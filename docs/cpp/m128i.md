# <a name="__m128i"></a>__m128i

**Specyficzne dla firmy Microsoft**

Typ danych **__m128i,** do użytku z instrukcjami wewnętrznego emmintrin.h extensions 2 (SSE2) jest zdefiniowany w \<> emmintrin.h.

```cpp
// data_types__m128i.cpp
#include <emmintrin.h>
int main() {
   __m128i x;
}
```

## <a name="remarks"></a>Uwagi

Nie należy uzyskiwać bezpośredniego dostępu do pól **__m128i.** Można jednak zobaczyć te typy w debugerze. Zmienna typu **__m128i** jest mapowana do rejestrów XMM[0-7].

Zmienne typu **__m128i** są automatycznie wyrównywane na granicach 16-bajtowych.

> [!NOTE]
> Przy użyciu zmiennych typu **__m128i** spowoduje kompilator do generowania instrukcji SSE2. `movdqa` Ta instrukcja nie powoduje usterki procesorów Pentium III, ale spowoduje cichą awarię, z możliwymi skutkami ubocznymi spowodowanymi `movdqa` przez wszelkie instrukcje przekładające się na procesory Pentium III.

Typ danych **__m128i** nie jest obsługiwany na procesorach ARM.
