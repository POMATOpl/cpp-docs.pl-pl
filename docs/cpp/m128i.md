# <a name="m128i"></a>__m128i
## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft  
 `__m128i` Typ danych do użycia z funkcje wewnętrzne instrukcje Streaming SIMD Extensions 2 (SSE2), jest zdefiniowany w emmintrin.h.  
  
```  
// data_types__m128i.cpp  
#include <emmintrin.h>  
int main() {  
   __m128i x;  
}  
```  
  
## <a name="remarks"></a>Uwagi  
 Nie powinien uzyskiwać dostęp **__m128i** pola bezpośrednio. Można jednak zobaczyć te typy w debugerze. Zmienna typu `__m128i` mapy w rejestrach [0-7] XMM.  
  
 Zmienne typu **_m128i** automatycznie są wyrównane do 16-bajtowych granicach.  
  
> [!NOTE]
>  Za pomocą zmiennych typu `__m128i` spowoduje, że kompilator, aby wygenerować SSE2 **movdqa** instrukcji. Ta instrukcja nie powoduje błąd w procesorach Pentium III, ale spowoduje niepowodzenie dyskretnej skutki uboczne możliwe spowodowane niezależnie od instrukcje **movdqa** przekłada się na na procesorach Pentium III.  
  
 `__m128i` Typ danych nie jest obsługiwany na procesorów ARM.
