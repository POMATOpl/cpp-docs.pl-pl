# <a name="m128i"></a>__m128i
## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft  
 **__M128i** typu danych, do użytku z programem funkcje wewnętrzne instrukcji Streaming SIMD Extensions 2 (SSE2), jest zdefiniowany w \<emmintrin.h >.  
  
```cpp 
// data_types__m128i.cpp  
#include <emmintrin.h>  
int main() {  
   __m128i x;  
}  
```  
  
## <a name="remarks"></a>Uwagi  
 Nie mają do **__m128i** pól bezpośrednio. Można jednak zobaczyć te typy w debugerze. Zmienna typu **__m128i** mapy do rejestrów XMM [0-7].  
  
 Zmienne typu **_m128i** będą automatycznie wyrównywane do 16-bajtowych granicach.  
  
> [!NOTE]
>  Za pomocą zmiennych typu **__m128i** spowoduje, że kompilator, aby wygenerować SSE2 `movdqa` instrukcji. Ta instrukcja nie powoduje awarii na procesorach Pentium III, ale spowoduje niepowodzenie dyskretnej, za pomocą możliwe efekty uboczne, które spowodowało niezależnie od zgodnie z instrukcjami opisanymi `movdqa` przekłada się na na procesorach Pentium III.  
  
 **__M128i** typ danych nie jest obsługiwany w procesorach ARM.  
  
**END specyficzny dla Microsoft**
