# <a name="jump-statements-c"></a>Instrukcje skoku (C++)
Instrukcja skok C++ wykonuje natychmiastowego lokalnego przekazanie sterowania.  
  
## <a name="syntax"></a>Składnia  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
