# <a name="jump-statements-c"></a>Instrukcje skoku (C++)

Instrukcja skoku C++ wykonuje natychmiastowego lokalnego transfer kontroli.

## <a name="syntax"></a>Składnia

```
break;
continue;
return [expression];
goto identifier;
```
