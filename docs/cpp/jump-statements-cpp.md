# <a name="jump-statements-c"></a>Instrukcje skoku (C++)

Instrukcja C++ skoku wykonuje natychmiastowy lokalny transfer kontroli.

## <a name="syntax"></a>Składnia

```
break;
continue;
return [expression];
goto identifier;
```
