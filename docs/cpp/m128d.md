# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft  
 `__m128d` Typ danych do użycia z funkcje wewnętrzne instrukcje Streaming SIMD Extensions 2, jest zdefiniowany w emmintrin.h.  
  
```  
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>Uwagi  
 Nie powinien uzyskiwać dostęp `__m128d` pola bezpośrednio. Można jednak zobaczyć te typy w debugerze. Zmienna typu `__m128` mapy w rejestrach [0-7] XMM.  
  
 Zmienne typu **_m128d** automatycznie są wyrównane do 16-bajtowych granicach.  
  
 `__m128d` Typ danych nie jest obsługiwany na procesorów ARM.
