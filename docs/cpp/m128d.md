# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft  
 **__M128d** typu danych, do użytku z programem funkcje wewnętrzne instrukcji Streaming SIMD Extensions 2, jest zdefiniowany w \<emmintrin.h >.  
  
```cpp 
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>Uwagi  
 Nie mają do **__m128d** pól bezpośrednio. Można jednak zobaczyć te typy w debugerze. Zmienna typu **__m128** mapy do rejestrów XMM [0-7].  
  
 Zmienne typu **_m128d** będą automatycznie wyrównywane do 16-bajtowych granicach.  
  
 **__M128d** typ danych nie jest obsługiwany w procesorach ARM.  
  
**END specyficzny dla Microsoft**
