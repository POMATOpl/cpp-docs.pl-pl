# <a name="numerical-limits-c"></a>Limity wartości liczbowych (C++)

Dwa standardowe pliki dołączane \<limits.h> oraz \<float.h> definiują limity liczbowe lub wartości minimalne i maksymalne, które mogą być przechowywane w zmiennej danego typu. Te minimalne i maksymalne wartości są gwarantowane, aby można było je przenosić do dowolnego kompilatora języka C++, który używa tych samych danych reprezentacji co ANSI C. \<limits.h> Plik dołączany definiuje [limity liczbowe dla typów całkowitych](../cpp/integer-limits.md)i \<float.h> definiuje [limity liczbowe dla typów zmiennoprzecinkowych](../cpp/floating-limits.md).
