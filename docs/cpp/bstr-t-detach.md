# <a name="bstrtdetach"></a>_bstr_t::Detach

**Microsoft Specific**

Zwraca `BSTR` opakowane przez `_bstr_t` oraz odłącza `BSTR` z `_bstr_t`.

## <a name="syntax"></a>Składnia

```
BSTR Detach( ) throw;
```

## <a name="return-value"></a>Wartość zwracana

`BSTR` Opakowane przez `_bstr_t`.
