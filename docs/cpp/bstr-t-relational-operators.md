# <a name="bstrt-relational-operators"></a>_bstr_t — Operatory relacyjne
**Dotyczące firmy Microsoft**  
  
 Porównuje dwa `_bstr_t` obiektów.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      bool operator!( ) const throw( );   
bool operator==(  
   const _bstr_t& str   
) const throw( );  
bool operator!=(  
   const _bstr_t& str   
) const throw( );  
bool operator<(  
   const _bstr_t& str   
) const throw( );  
bool operator>(  
   const _bstr_t& str   
) const throw( );  
bool operator<=(  
   const _bstr_t& str   
) const throw( );  
bool operator>=(  
   const _bstr_t& str   
) const throw( );  
```  
  
## <a name="remarks"></a>Uwagi  
 Te operatory porównania dwóch `_bstr_t` lexicographically obiektów. Operatory zwracać **true** w przypadku przechowywania porównań, w przeciwnym razie zwraca **false**.
