# <a name="__m64"></a>__m64

**Microsoft Specific**

Typ danych **__m64** jest używany z MMX i 3DNow! elementy wewnętrzne i są zdefiniowane w \<xmmintrin. h >.

```cpp
// data_types__m64.cpp
#include <xmmintrin.h>
int main()
{
   __m64 x;
}
```

## <a name="remarks"></a>Uwagi

Nie należy bezpośrednio uzyskiwać dostępu do pól **__m64** . Można jednak wyświetlić te typy w debugerze. Zmienna typu **__m64** Maps do rejestrów mm [0-7].

Zmienne typu **_m64** są automatycznie wyrównane na granicach 8-bajtowych.

Typ danych **__m64** nie jest obsługiwany w przypadku procesorów x64. Aplikacje, które używają __m64 jako części wewnętrznych funkcji MMX, muszą zostać przetworzone w celu użycia równoważnej instrukcji SSE i SSE2.
