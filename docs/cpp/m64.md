# <a name="m64"></a>__m64

## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft

**__M64** typ danych jest do użytku z MMX i 3DNow! funkcje wewnętrzne i jest definiowany w \<xmmintrin.h >.

```cpp
// data_types__m64.cpp
#include <xmmintrin.h>
int main()
{
   __m64 x;
}
```

## <a name="remarks"></a>Uwagi

Nie mają do **__m64** pól bezpośrednio. Można jednak zobaczyć te typy w debugerze. Zmienna typu **__m64** mapy do rejestrów MM [0-7].

Zmienne typu **_m64** będą automatycznie wyrównywane do 8-bajtowych granicach.

**__M64** typ danych nie jest obsługiwany na x64 procesorów. Aplikacje, które używają __m64, jako część wewnętrzne MMX muszą zostać przepisane, aby używać równoważnej funkcji wewnętrznych rozszerzenia SSE i SSE2.

**END specyficzny dla Microsoft**
