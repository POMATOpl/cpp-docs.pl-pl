# <a name="m64"></a>__m64
## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft  
 `__m64` — Typ danych jest przeznaczona do użytku z MMX i 3DNow! funkcje wewnętrzne i jest zdefiniowany w \<xmmintrin.h >.  
  
```  
// data_types__m64.cpp  
#include <xmmintrin.h>  
int main()  
{  
   __m64 x;  
}  
```  
  
## <a name="remarks"></a>Uwagi  
 Nie powinien uzyskiwać dostęp `__m64` pola bezpośrednio. Można jednak zobaczyć te typy w debugerze. Zmienna typu `__m64` mapy w rejestrach [0-7] MM.  
  
 Zmienne typu **typu _m64** automatycznie są wyrównane do 8-bajtowych granicach.  
  
 `__m64` Typ danych nie jest obsługiwany na [!INCLUDE[vcprx64](../assembler/inline/includes/vcprx64_md.md)] procesorów. Aby użyć równoważne funkcje wewnętrzne SSE i SSE2, należy ponownie zapisać aplikacji, które używają __m64 jako część MMX funkcje wewnętrzne.
