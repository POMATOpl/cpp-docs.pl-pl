# <a name="types-of-expressions"></a>Typy wyrażeń
Wyrażeń C++ są podzielone na kilka kategorii:  
  
-   [Wyrażenia podstawowe](../cpp/primary-expressions.md). Są to bloki konstrukcyjne, z których wszystkie inne wyrażenia są tworzone.  
  
-   [Wyrażenia przyrostków](../cpp/postfix-expressions.md). Są to następuje operator wyrażenia podstawowe — na przykład tablicy indeks dolny operatory przyrostka inkrementacji operatora or.  
  
-   [Sformatowany wyrażenia z operatorami jednoargumentowymi](../cpp/expressions-with-unary-operators.md). Operatory jednoargumentowe działa tylko jedno operandu w wyrażeniu.  
  
-   [Sformatowany wyrażenia z operatorami dwuargumentowymi](../cpp/expressions-with-binary-operators.md). Operatory dwuargumentowe oddziałują na dwa operandy w wyrażeniu.  
  
-   [Wyrażenia z operator warunkowy](../cpp/conditional-operator-q.md). Operator warunkowy jest trójargumentowy — tylko takie operator w języku C++ — i ma trzy operandy.  
  
-   [Wyrażenia stałe](../cpp/cpp-constant-expressions.md). Wyrażenia stałe są tworzone wyłącznie z danych stałej.  
  
-   [Wyrażenia z jawne konwersje typów](http://msdn.microsoft.com/en-us/060ad6b4-9592-4f3e-8509-a20ac84a85ae). Jawne konwersje typów lub "rzutowania," można używać w wyrażeniach.  
  
-   [Wyrażenia z operatorami wskaźnika do elementu członkowskiego](../cpp/pointer-to-member-operators-dot-star-and-star.md).  
  
-   [Rzutowanie](../cpp/casting.md). Bezpieczne "rzutowania" można używać w wyrażeniach.  
  
-   [Informacje typu Run-Time](../cpp/run-time-type-information.md). Określ typ obiektu podczas wykonywania programu.
