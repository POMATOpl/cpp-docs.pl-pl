# <a name="types-of-expressions"></a>Typy wyrażeń

C++wyrażenia są podzielone na kilka kategorii:

- [Wyrażenia podstawowe](../cpp/primary-expressions.md). Są to bloki konstrukcyjne, z których tworzone są wszystkie inne wyrażenia.

- [Wyrażenia przyrostkowe](../cpp/postfix-expressions.md). Są to wyrażenia podstawowe, po których następuje operator — na przykład indeks dolny tablicy lub operator przyrostu.

- [Wyrażenia utworzone za pomocą operatorów jednoargumentowych](../cpp/expressions-with-unary-operators.md). Operatory jednoargumentowe działają tylko dla jednego operandu w wyrażeniu.

- [Wyrażenia utworzone za pomocą operatorów binarnych](../cpp/expressions-with-binary-operators.md). Operatory dwuargumentowe oddziałują na dwa operandy w wyrażeniu.

- [Wyrażenia z operatorem warunkowym](../cpp/conditional-operator-q.md). Operator warunkowy jest operatorem Trzyelementowy — jedynym tym operatorem w C++ języku — i pobiera trzy operandy.

- [Wyrażenia stałe](../cpp/cpp-constant-expressions.md). Wyrażenia stałe są tworzone w całości z danych stałych.

- [Wyrażenia z jawnymi konwersjemi typów](explicit-type-conversion-operator-parens.md). Jawne konwersje typów lub "casts" mogą być używane w wyrażeniach.

- [Wyrażenia z operatorami wskaźnika do składowej](../cpp/pointer-to-member-operators-dot-star-and-star.md).

- [Rzutowanie](../cpp/casting.md). W wyrażeniach można używać bezpiecznych "rzutowania".

- [Informacje o typie czasu wykonywania](../cpp/run-time-type-information.md). Określ typ obiektu podczas wykonywania programu.
