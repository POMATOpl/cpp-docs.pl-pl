# <a name="indirection-on-array-types"></a>Pośrednie w typach tablic

Korzystanie z operatora pośredniego (<strong>\*</strong>) na *n*— daje dwuwymiarowa tablica typu *n*-1-wymiarowej tablicy. Jeśli *n* wynosi 1, jest uzyskane skalarnych (lub element tablicy).
