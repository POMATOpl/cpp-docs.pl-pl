# <a name="program-and-linkage--c"></a>Program i połączenie (C++)
Program składa się z co najmniej jednej jednostki tłumaczenia ze sobą powiązane. Wykonywanie programu (koncepcyjnie) rozpoczyna się w jednostce tłumaczenia, która zawiera funkcję **głównego**. (Aby uzyskać więcej informacji dotyczących jednostek tłumaczenia, zobacz [fazy tłumaczenia](../preprocessor/phases-of-translation.md)w *odwołania preprocesora*.) Aby uzyskać więcej informacji na temat **głównego** funkcji, zobacz [uruchamianie programu: funkcja main](../cpp/main-program-startup.md).)
