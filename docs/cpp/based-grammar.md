# <a name="based-grammar"></a>__based — Gramatyka
## <a name="microsoft-specific"></a>Specyficzne dla firmy Microsoft  
 Adresowanie na podstawie jest przydatne, gdy konieczne jest ścisła kontrola nad segment, w których obiekty są przydzielane (statycznych i dynamicznych na podstawie danych).  
  
 Tylko formularza na podstawie adresowania dopuszczalne w kompilacjach kodu 32-bitowe i 64-bitowych jest "na podstawie wskaźnik" definiuje typ, który zawiera 32-bitowy lub 64-bitowe przesunięcie z podstawową 32-bitowy lub 64-bitowe lub oparte na `void`.
