# <a name="writing-an-exception-handler"></a>Pisanie programu do obsługi wyjątku

Programy obsługi wyjątków są zwykle używane do reagowania na określone błędy. Można użyć składni obsługi wyjątków, aby odfiltrować wszystkie wyjątki inne niż te, które są znane. Inne wyjątki powinny być przesyłane do innych programów obsługi (prawdopodobnie w bibliotece wykonawczej lub w systemie operacyjnym), aby szukać tych konkretnych wyjątków.

Procedury obsługi wyjątków używają instrukcji try-except.
