# <a name="selection-statements-c"></a>Instrukcje wyboru (C++)

Instrukcje wyboru języka C++, [if](../cpp/if-else-statement-cpp.md) i [Switch](../cpp/switch-statement-cpp.md)umożliwiają warunkowe wykonywanie sekcji kodu.

Instrukcje [__if_exists](../cpp/if-exists-statement.md) i [__if_not_exists](../cpp/if-not-exists-statement.md) umożliwiają warunkowe dołączenie kodu w zależności od istnienia symbolu.

Zapoznaj się z poszczególnymi tematami dotyczącymi składni każdej instrukcji.
