**Specyficzne dla firmy Microsoft**

Zwraca liczbę znaków w `_bstr_t`, bez uwzględnienia kończącej wartości null `BSTR`hermetyzowane.

## <a name="syntax"></a>Składnia

```
unsigned int length ( ) const throw( );
```
