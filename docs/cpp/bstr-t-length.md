# <a name="bstrtlength"></a>_bstr_t::length
**Microsoft Specific**  
  
 Zwraca liczbę znaków w `_bstr_t`, nie wliczając końcowej wartości null z zhermetyzowany `BSTR`.  
  
## <a name="syntax"></a>Składnia  
  
```  
unsigned int length ( ) const throw( );  
```
