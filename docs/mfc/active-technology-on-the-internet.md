# <a name="active-technology-on-the-internet"></a>Technologia Active w Internecie
Technologia Active jest otwartej platformie, która umożliwia deweloperom tworzenie atrakcyjnych, dynamicznej zawartości, jak i aplikacje globalne internetowe lub sieci wewnętrznej firmy, znany jako intranet. Poniżej opisano głównych technologie obsługiwane przez firmę Microsoft do programowania Internetu.  
  
## <a name="activex-controls"></a>Kontrolki ActiveX  
 Kontrolki ActiveX (dawniej formanty OLE) są obiektami, które można wstawiać do stron sieci Web ani żadnych innych aplikacji, która jest kontenera formantu ActiveX. Przykładami przycisków, giełdowych i kontrolek wykresów. Aby uzyskać więcej informacji, zobacz [formantów ActiveX w Internecie](../mfc/activex-controls-on-the-internet.md).  
  
## <a name="active-documents"></a>Dokumenty aktywne  
 Dokumenty aktywne mogą być wyświetlane przez przeglądarki sieci Web lub przeglądarki dokumentów. Tradycyjne obiekty osadzone były ograniczone do jednej strony i zostały pokazane osadzony w dokumencie. Technologia Active można wyświetlić dokument pełne ramki w oknie Obszar całego klienta. Aby uzyskać więcej informacji, zobacz [dokumenty aktywne w Internecie](../mfc/active-documents-on-the-internet.md).  
  
## <a name="internet-data-download-services"></a>Usługi pobierania danych w Internecie  
 Można pobrać danych za pośrednictwem Internetu za pomocą wspólnych protokołów: HTTP, FTP i gopher. Klasy MFC WinInet ułatwiają transferu danych za pomocą protokołów HTTP, FTP i gopher przez abstrakcyjność protokołów TCP/IP i WinSock. Asynchroniczne krótkiej nazwy klas MFC umożliwiają do pobierania plików bez blokowania i do renderowania asynchronicznie dużych obiektów. Aby uzyskać więcej informacji, zobacz [rozszerzenia internetowe Win32 (WinInet)](../mfc/win32-internet-extensions-wininet.md).  
  
## <a name="active-scripts"></a>Aktywnych skryptów  
 VBScript i innych języków skryptowych łączenie formantów i dodać funkcje interakcyjne do stron sieci Web. Wykonywanie skryptów przenosi przetwarzanie z serwera do klienta. Na przykład wpisy formularza można sprawdzić poprawności na kliencie i następnie wysyłane do serwera.  
  
## <a name="html-extensions"></a>Rozszerzenia HTML  
 Rozszerzenia języka HTML, takie jak object tag zostały dodane do obsługi kontrolki i obsługę skryptów.
