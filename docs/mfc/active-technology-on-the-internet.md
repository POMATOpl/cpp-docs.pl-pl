# <a name="active-technology-on-the-internet"></a>Technologia Active w Internecie

Technologia Active to otwarta platforma, która umożliwia deweloperom tworzenie atrakcyjnej, dynamicznej zawartości i aplikacji dla globalnego Internetu, a także dla sieci wewnętrznej firmy, znanej jako intranet. Poniżej opisano główne technologie udostępniane przez firmę Microsoft do programowania w Internecie.

>[!IMPORTANT]
> Kontrolka ActiveX to Starsza technologia, która nie powinna być używana do nowych celów programistycznych. Aby uzyskać więcej informacji na temat nowoczesnych technologii, które zastępują ActiveX, zobacz [kontrolki ActiveX](activex-controls.md).

## <a name="activex-controls"></a>Kontrolki ActiveX

Formanty ActiveX (dawniej formanty OLE) to obiekty, które można wstawiać do stron sieci Web lub innej aplikacji, która jest kontenerem kontrolki ActiveX. Przykłady obejmują przyciski, znaczniki giełdowe i kontrolki wykresu. Aby uzyskać więcej informacji, zobacz [kontrolki ActiveX w Internecie](activex-controls-on-the-internet.md).

## <a name="internet-data-download-services"></a>Usługi pobierania danych internetowych

Dane można pobrać za pośrednictwem Internetu przy użyciu popularnych protokołów: HTTP, FTP i gopher. Klasy MFC WinInet umożliwiają łatwe przenoszenie danych przy użyciu protokołów HTTP, FTP i gopher przez abstrakcję protokołów TCP/IP i WinSock. Asynchroniczne klasy monikerów MFC umożliwiają pobieranie plików bez blokowania i renderowanie dużych obiektów w sposób asynchroniczny. Aby uzyskać więcej informacji, zobacz [rozszerzenia internetowe Win32 (WinInet)](win32-internet-extensions-wininet.md).

## <a name="active-scripts"></a>Skrypty aktywne

Skrypty VBScript i inne języki skryptów łączą formanty i dodają funkcje interaktywne do stron sieci Web. Skrypt przenosi przetwarzanie z serwera na klienta. Na przykład wpisy formularza mogą być zweryfikowane na kliencie, a następnie wysyłane do serwera.

## <a name="html-extensions"></a>Rozszerzenia HTML

Rozszerzenia HTML, takie jak tag Object, zostały dodane do obsługi formantów i skryptów.
