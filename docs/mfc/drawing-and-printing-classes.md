# <a name="drawing-and-printing-classes"></a>Klasy związane z rysowaniem i drukowaniem

W systemie Windows wszystkie grafiki wyjściowe są rysowane w wirtualnym obszarze rysowania nazywanym kontekstem urządzenia (DC). MFC udostępnia klasy do hermetyzowania różnych typów kontrolerów domen, a także hermetyzacji dla narzędzi do rysowania systemu Windows, takich jak mapy bitowe, pędzle, palety i pióra.
