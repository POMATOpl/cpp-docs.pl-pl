# <a name="setting-the-dialog-boxs-background-color"></a>Ustawianie koloru tła okna dialogowego

Dzięki obsłudze wiadomości wm_ctlcolor — okno dialogowe, można ustawić kolor tła z okien dialogowych. Gdy ustawiasz kolor jest używany tylko określonego okna dialogowego.

Zobacz [codexpert blog](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) przykład.
