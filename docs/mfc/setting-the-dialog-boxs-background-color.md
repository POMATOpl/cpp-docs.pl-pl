# <a name="setting-the-dialog-boxs-background-color"></a>Ustawianie koloru tła okna dialogowego

Możesz ustawić kolor tła okien dialogowych, przenosząc WM_CTLCOLOR komunikaty dla okna dialogowego. Ustawiony kolor jest używany tylko dla określonego okna dialogowego.

Przykład można znaleźć na [blogu codexpert](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) .
