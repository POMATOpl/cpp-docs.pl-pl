# <a name="tab-controls-and-property-sheets"></a>Formanty karty i arkusze właściwości

Arkusze właściwości to okna dialogowe z wieloma stronami lub okna dialogowe z kartami, które mogą wyświetlać do użytkownika do 24 zasobów szablonów okien dialogowych. Aby zapoznać się z przykładami arkuszy właściwości, zobacz okno dialogowe właściwości wyświetlania systemu Windows lub Następująca aplikacja Przykładowa MFC:

[CMNCTRL1: demonstruje wspólne klasy formantów, część 1](../overview/visual-cpp-samples.md)

[CMNCTRL2: demonstruje wspólne klasy formantów, część 2](../overview/visual-cpp-samples.md)

Arkusze właściwości można łatwo zaimplementować przy użyciu klasy MFC [CPropertySheet](../mfc/reference/cpropertysheet-class.md).
