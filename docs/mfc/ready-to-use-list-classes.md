# <a name="ready-to-use-list-classes"></a>Gotowe do użycia klasy list
Następujące klasy są gotowe do użycia klasy list.  
  
 [CObList](../mfc/reference/coblist-class.md)  
 Wskaźniki do obiektów klasy przechowuje `CObject` lub do obiektów klasy pochodzące od `CObject` połączonej listy.  
  
 [CPtrList](../mfc/reference/cptrlist-class.md)  
 Wskaźniki do przechowuje `void` (wskaźniki ogólnego) połączonej listy.  
  
 [CStringList](../mfc/reference/cstringlist-class.md)  
 Magazyny `CString` obiektów w połączonej listy.
