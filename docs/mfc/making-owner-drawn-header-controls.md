# <a name="making-owner-drawn-header-controls"></a>Tworzenie formantów nagłówka rysowanych przez właściciela

Można zdefiniować poszczególne elementy kontrolki nagłówka ([CHeaderCtrl](reference/cheaderctrl-class.md)) jako elementy rysowane przez właściciela. Aby uzyskać więcej informacji, zobacz [kontrolki nagłówka rysowane przez właściciela](/windows/win32/Controls/header-controls) w Windows SDK.
