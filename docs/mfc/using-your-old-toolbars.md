# <a name="using-your-old-toolbars"></a>Używanie swoich starych pasków narzędzi

Jeśli używasz poprzednich wersji Visual C++ do tworzenia niestandardowych pasków narzędzi, nowa implementacja klasy [CToolBar](../mfc/reference/ctoolbar-class.md) może spowodować problemy. Aby nie trzeba było wykazywać starych pasków narzędzi do korzystania z nowych funkcji, stara implementacja jest nadal obsługiwana.

W przykładzie DOCKTOOL nie są używane paski narzędzi starego stylu, a tylko paski narzędzi New-Style.

Nie można edytować pasków narzędzi starego stylu przy użyciu edytora zasobów paska narzędzi.
