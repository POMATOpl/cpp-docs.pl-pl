# <a name="ole-classes"></a>Klasy OLE
Klasy OLE współpracować z innych aplikacji framework klas aby zapewnić łatwy dostęp do interfejsu API ActiveX, podając programy łatwy sposób możliwości ActiveX dla użytkowników. Przy użyciu ActiveX, można:  
  
-   Tworzenie dokumentów złożonych, które umożliwiają użytkownikom tworzenie i edytowanie dokumentów zawierających dane utworzone przez wiele aplikacji, w tym tekst, grafiki, arkusze kalkulacyjne, dźwięk, lub inne typy danych.  
  
-   Tworzenie obiektów OLE, które mogą być osadzone w dokumentów złożonych.  
  
-   Umożliwia przeciąganie i upuszczanie OLE kopiowanie danych między aplikacjami.  
  
-   Używanie automatyzacji do kontrolowania jeden program na inny.  
  
-   Tworzenie formantów ActiveX i kontenery formantów ActiveX (wcześniej nazywanych formantów OLE i kontenery formantów OLE, odpowiednio).
