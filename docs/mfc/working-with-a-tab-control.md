# <a name="working-with-a-tab-control"></a>Praca z formantem karty

Najprostszym sposobem użycia formantu karty ([CTabCtrl](../mfc/reference/ctabctrl-class.md)) jest dodanie go do zasobu szablonu okna dialogowego z edytorem okien dialogowych. Możesz również użyć kontrolki karta. Wywołania MFC `InitCommonControls` . Najważniejsze zadania są następujące:

- [Tworzenie kontrolki karty](../mfc/creating-the-tab-control.md)

- [Dodawanie kart do kontrolki karty](../mfc/adding-tabs-to-a-tab-control.md)

- [Przetwarzanie komunikatów powiadomień dotyczących kontrolki karty](../mfc/processing-tab-control-notification-messages.md)

Jeśli obiekt formantu karty jest osadzony w widoku nadrzędnym lub klasie okna dialogowego, formant zostanie zniszczony po zniszczeniu elementu nadrzędnego.
