# <a name="containers-for-activex-controls"></a>Kontenery dla formantów ActiveX

Można użyć kontrolek ActiveX opracowanych w Visual C++ w innych aplikacjach, o ile obsługują one zawieranie formantów ActiveX. Różne aplikacje firmy Microsoft, począwszy od wersji wymienionych, obsługują kontrolki ActiveX.

>[!IMPORTANT]
> Kontrolka ActiveX to Starsza technologia, która nie powinna być używana do nowych celów programistycznych. Aby uzyskać więcej informacji na temat nowoczesnych technologii, które zastępują ActiveX, zobacz [kontrolki ActiveX](activex-controls.md).

> [!NOTE]
> Poniższa lista nie jest kompletną listą aplikacji, które obsługują kontrolki ActiveX, ale reprezentuje zestaw najczęściej używany do testowania:

Aplikacje obsługujące formanty ActiveX zawierają:

- Internet Explorer 3. x i nowsze

- Visual Basic 4. x i więcej

- Visual C++ 4. x i więcej

- Dostęp do 95 i nowsze

- Excel 97 i nowsze

- Word 97 i nowsze

- Dostęp do 97 i nowsze

- FrontPage 97 i nowsze

- PowerPoint 97 i nowsze

- Visual InterDev 97 i nowsze

Poniżej znajdują się aplikacje inne niż firmy Microsoft, które obsługują zawieranie formantów ActiveX:

- PowerBuilder

- Delphi

- Konstruktor języka C++

- Wtyczka NCompass dla programu Netscape Navigator
