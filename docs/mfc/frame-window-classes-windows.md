# <a name="frame-window-classes-windows"></a>Klasy okien ramowych (Windows)
Okna ramowe są systemu windows, które ramki aplikacji lub jej część aplikacji. Okna ramowe zwykle zawierają inne systemu windows, takich jak widoki, pasków narzędzi i paski stanu. W przypadku liczby `CMDIFrameWnd`, mogą zawierać `CMDIChildWnd` pośrednio obiektów.  
  
 [Cframewnd —](../mfc/reference/cframewnd-class.md)  
 Klasa podstawowa aplikacja SDI głównego okna ramowego. Również klasę podstawową dla wszystkich innych klasy okien ramowych.  
  
 [Cmdiframewnd —](../mfc/reference/cmdiframewnd-class.md)  
 Klasa podstawowa dla aplikacji MDI głównego okna ramowego.  
  
 [CMDIChildWnd](../mfc/reference/cmdichildwnd-class.md)  
 Klasa podstawowa dla okien ramowych dokumentu aplikacji MDI.  
  
 [CMiniFrameWnd](../mfc/reference/cminiframewnd-class.md)  
 Okno ramowe wysokości zazwyczaj występuje wokół przestawne paski narzędzi.  
  
 [COleIPFrameWnd](../mfc/reference/coleipframewnd-class.md)  
 Udostępnia ramkę okna w widoku, gdy dokument serwera jest edytowany w miejscu.  
  
## <a name="related-class"></a>Klasy pokrewne  
 Klasa `CMenu` udostępnia interfejs, za pomocą którego można uzyskać dostępu do menu aplikacji. Jest przydatne w przypadku operowanie menu dynamicznie w czasie wykonywania. na przykład podczas dodawania lub usuwania elementów menu w zależności od kontekstu. Mimo że menu są najczęściej używane w systemie windows ramki, ich można również z okien dialogowych i innych oknach nonchild.  
  
 [Cmenu —](../mfc/reference/cmenu-class.md)  
 Hermetyzuje `HMENU` dojścia do paska menu i menu podręczne aplikacji.
