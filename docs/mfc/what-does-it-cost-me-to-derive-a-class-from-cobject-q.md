# <a name="what-does-it-cost-me-to-derive-a-class-from-cobject"></a>Jaki jest koszt wyprowadzenia klasy z obiektu CObject?

Obciążenie wynikające z klasy [CObject](../mfc/reference/cobject-class.md) jest minimalne. Klasa pochodna dziedziczy tylko cztery funkcje wirtualne i pojedynczy obiekt [CRuntimeClass](../mfc/reference/cruntimeclass-structure.md) .
