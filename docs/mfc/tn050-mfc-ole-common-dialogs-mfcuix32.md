# <a name="tn050-mfcole-common-dialogs-mfcuix32"></a>TN050: Wspólne okna dialogowe MFC/OLE (MFCUIx32)

Ta uwaga objęte pierwotnie niektóre problemy dotyczące oraz przyszłość OLE wspólne okna dialogowe dostarczane i używane przez MFC. Wspólne okna dialogowe OLE są obecnie dostarczany jako składnik wbudowane w system (OLEDLG. Biblioteka DLL) i są w pełni udokumentowane w dokumentacji produktu.
