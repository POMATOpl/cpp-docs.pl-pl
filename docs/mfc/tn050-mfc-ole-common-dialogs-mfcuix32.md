# <a name="tn050-mfcole-common-dialogs-mfcuix32"></a>TN050: wspólne okna dialogowe MFC/OLE (MFCUIx32)

Ta Uwaga pierwotnie dotyczyła niektórych problemów i przyszłości wspólnych okien dialogowych OLE dostarczonych i używanych przez MFC. Wspólne okna dialogowe OLE są teraz udostępniane jako składnik wbudowany w system (OLEDLG.DLL) i są w pełni udokumentowane w dokumentacji produktu.
