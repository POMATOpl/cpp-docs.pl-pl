# <a name="property-sheets-mfc"></a>Arkusze właściwości (MFC)

W tej rodzinie artykułów wyjaśniono, jak zaimplementować obsługę arkuszy właściwości w aplikacjach MFC. Arkusz właściwości, znany również jako okno dialogowe karty, umożliwia zarządzanie dużą liczbą kontrolek w oknie dialogowym. Arkusz właściwości zawiera strony właściwości, z których każda jest oparta na osobnym zasobie szablonu okna dialogowego. Kontrolki okna dialogowego można podzielić na grupy logiczne i umieścić każdą grupę na własnej stronie właściwości.
