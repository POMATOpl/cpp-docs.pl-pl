# <a name="file-and-database-classes"></a>Klasy plików i baz danych
Te klasy pozwalają na przechowywanie informacji z bazą danych lub pliku na dysku. Istnieją trzy zestawy klasy baz danych, OLE DB, ODBC i DAO — które zapewniają podobne funkcje. Grupy OLE DB jest implementowane za pomocą OLE DB i działa z szablonami konsumentów OLE DB, grupy DAO jest implementowane za pomocą obiektu dostępu do danych i grupy ODBC jest implementowane za pomocą Otwórz połączenie z bazą danych. Istnieją także zestaw klas do manipulowania standardowe pliki strumieni Active i strumieni HTML.
