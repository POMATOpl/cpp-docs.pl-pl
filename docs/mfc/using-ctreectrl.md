# <a name="using-ctreectrl"></a>Korzystanie z CTreeCtrl
Formant drzewa, reprezentowany przez klasę [CTreeCtrl](../mfc/reference/ctreectrl-class.md), jest oknem, które wyświetla listę hierarchiczną elementów, takich jak nagłówki w dokumencie, wpisy w indeksie lub plików i katalogów na dysku. Każdy element składa się z etykiet oraz opcjonalny obraz mapy bitowej, a każdy element może mieć listy elementów podrzędnych skojarzonych z nim. Kliknięcie pozycji użytkownika można zwijać i rozwijać skojarzonej listy elementów podrzędnych. Drzewo katalogów w okienku po lewej stronie Eksploratora plików jest przykładem formantu drzewa.
