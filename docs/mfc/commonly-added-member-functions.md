# <a name="commonly-added-member-functions"></a>Powszechnie dodawane funkcje członkowskie

Jeśli Twoje okno dialogowe zawiera przyciski innym niż OK lub przycisk Anuluj, należy napisać program obsługi komunikatów funkcje Członkowskie w swojej klasy okien dialogowych, aby odpowiadać na komunikaty powiadamianie kontrolki, które generują. Aby uzyskać przykład, zobacz [Bazgroły](../overview/visual-cpp-samples.md) przykładowy program. Może również obsługiwać formant powiadomienia z innych formantów w oknie dialogowym.
