# <a name="active-document-classes"></a>Klasy dokumentów aktywnych

Aktywne dokumenty można wyświetlić w całym oknie klienta przeglądarki sieci Web, na przykład w programie Internet Explorer 5,5 lub w aktywnym kontenerze, takim jak spinacz Microsoft Office, który obsługuje dokumenty aktywne.

[CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)<br/>
Mapuje aktywne interfejsy dokumentów i inicjuje i uaktywnia obiekt aktywnego dokumentu.

[CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)<br/>
Implementuje zlecenia serwera OLE przeznaczone dla serwerów dokumentów aktywnych.

[Metody COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)<br/>
Implementuje zawieranie dokumentów aktywnych.
