# <a name="walkthroughs-mfc"></a>Wskazówki (MFC)
Ta sekcja zawiera artykuły, które umożliwia przeprowadzenie różnych zadań związanych z nowych funkcji biblioteki MFC.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [Wskazówki: Używanie nowych formantów powłoki MFC](../mfc/walkthrough-using-the-new-mfc-shell-controls.md)  
 W tym przewodniku utworzysz aplikację, która jest podobny Eksploratora plików. Utworzysz okna, który zawiera dwa okienka. W okienku po lewej stronie będzie zawierać [CMFCShellTreeCtrl](../mfc/reference/cmfcshelltreectrl-class.md) obiektu, która będzie wyświetlana na pulpicie w hierarchiczny widok. Okienku po prawej stronie będzie zawierać [CMFCShellListCtrl](../mfc/reference/cmfcshelllistctrl-class.md) który wyświetli pliki w folderze, który wybrano w okienku po lewej stronie.  
  
 [Wskazówki: Umieszczanie formantów na paskach narzędzi](../mfc/walkthrough-putting-controls-on-toolbars.md)  
 Paski narzędzi MFC nowoczesnych może obsługiwać formantów innego niż prosty przycisków. W tym artykule wyjaśniono, jak to zrobić.  
  
 [Wskazówki: Dodawanie obiektu D2D do projektu MFC](../mfc/walkthrough-adding-a-d2d-object-to-an-mfc-project.md)  
 Pokazuje, jak dodawanie obiektu D2D do projektu i przedstawiono sposób użycia D2D.  
  
 [Wskazówki: Dodawanie animacji do projektu MFC](../mfc/walkthrough-adding-animation-to-an-mfc-project.md)  
 Przedstawiono sposób dodawania obiektu graficznego do projektu i zawiera wprowadzenie do animowania go.
