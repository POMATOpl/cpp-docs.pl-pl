# <a name="win32-internet-extensions-wininet"></a>Rozszerzenia internetowe Win32 (WinInet)

Klienckich aplikacji internetowych to program, który uzyskuje dostęp do informacji ze źródła danych sieci (serwer) przy użyciu protokołów internetowych, takich jak gopher, FTP lub HTTP. Klienckich aplikacji internetowych, mogą uzyskiwać dostęp do serwera, na przykład pobrać dane, takie jak mapy pogody, cen akcji lub nagłówki gazet. Klienta internetowego można korzystać z serwera za pośrednictwem sieci zewnętrznej (Internet) lub sieci wewnętrznej (nazywane dalej w sieci intranet).

MFC obejmuje rozszerzenia internetowe Win32 lub WinInet, do tworzenia klienckich aplikacji internetowych. MFC hermetyzuje rozszerzeń Internet w zestaw klas, standard i łatwy w użyciu. Można napisać aplikację kliencką interfejsu WinInet, przez wywołanie funkcji Win32 bezpośrednio lub przy użyciu klas MFC WinInet.

Funkcje programu Microsoft Internet Win32 (WinInet) ułatwić wprowadzanie Internet integralną częścią dowolnej aplikacji. Nowe funkcje, zawarte w WININET. Biblioteki DLL, Uprość dostęp do Internetu za pomocą protokołu HTTP (Hypertext Transfer Protocol), FTP (File Transfer Protocol) i gopher.
