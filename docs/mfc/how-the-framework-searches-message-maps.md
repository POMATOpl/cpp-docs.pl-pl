# <a name="how-the-framework-searches-message-maps"></a>Jak struktura wyszukuje mapy komunikatów

Struktura przeszukuje tabelę map komunikatów w celu dopasowania do komunikatów przychodzących. Po zapisaniu wpisu mapy wiadomości dla każdego komunikatu, który ma być obsługiwany przez klasę i napisać odpowiednie programy obsługi, struktura automatycznie wywołuje programy obsługi. W poniższych tematach opisano wyszukiwanie w mapie komunikatów:

- [Gdzie można znaleźć mapy komunikatów](where-to-find-message-maps.md)

- [Pochodne mapy komunikatów](derived-message-maps.md)

- [Mapowanie zakresów komunikatów, identyfikatorów poleceń lub identyfikatorów kontrolek na jedną procedurę obsługi](handlers-for-message-map-ranges.md)
