# <a name="servers-user-interface-issues"></a>Serwery: Problemy z interfejsem użytkownika

Aplikacja serwera zawiera pewną liczbę funkcji, które muszą zostać dodane do interfejsu użytkownika, które umożliwiają określanie wartości elementów OLE do aplikacji kontenera. Aby uzyskać więcej informacji na temat menu i dodatkowe zasoby, które należy dodać do aplikacji serwera, zobacz [menu i zasoby: Dodatki do serwera](../mfc/menus-and-resources-server-additions.md).
