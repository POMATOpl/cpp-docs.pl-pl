# <a name="menus-mfc"></a>Menu (MFC)

MFC udostępnia dwa elementy, które ułatwiają współpracę z menu:

- Klasa [CMenu](reference/cmenu-class.md) do manipulowania menu programu w czasie wykonywania. Skorzystaj z dokumentacji programu `CMenu` i przykładu, aby dowiedzieć się, jak efektywnie korzystać z programu `CMenu` .

- Mechanizm aktualizowania menu i przycisków paska narzędzi: Włączanie lub wyłączanie na bieżąco z bieżącymi warunkami programu.

Visual C++ udostępnia również [Edytor menu](../windows/menu-editor.md) służący do tworzenia i edytowania zasobów menu programu.
