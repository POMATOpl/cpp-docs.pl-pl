# <a name="using-ccomboboxex"></a>Korzystanie z CComboBoxEx

Formantu rozszerzonego pola kombi jest rozszerzeniem formant pola kombi standardowa, która zapewnia macierzystą obsługę obrazów elementów. Te obrazy może służyć do wskazania stanu poszczególnych elementów w polu kombi, takie jak aktualnie zaznaczone i niezaznaczone elementy. Aby obrazów elementów były łatwo dostępne, formant zapewnia obsługę list obrazów.

Aby zapewnić funkcjonalność pola kombi bez konieczności ręcznego rysowania grafiki elementu, należy użyć tej kontrolki.
