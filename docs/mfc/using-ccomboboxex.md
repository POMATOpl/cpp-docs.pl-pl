# <a name="using-ccomboboxex"></a>Korzystanie z CComboBoxEx

Kontrolka rozszerzonego pola kombi jest rozszerzeniem standardowej kontrolki pola kombi, która zapewnia natywną obsługę obrazów elementów. Te obrazy mogą służyć do wskazywania stanu poszczególnych elementów w polu kombi, takich jak aktualnie zaznaczone i niezaznaczone elementy. Aby obrazy elementów były łatwo dostępne, formant udostępnia obsługę listy obrazów.

Użyj tego formantu, aby zapewnić funkcjonalność pola kombi bez konieczności ręcznego rysowania grafiki elementu.
