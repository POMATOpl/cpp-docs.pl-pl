# <a name="sdi-and-mdi"></a>SDI i MDI
MFC ułatwia działać z interfejsu pojedynczego dokumentu (SDI) i aplikacje interfejsu wielu dokumentów (MDI).  
  
 SDI — aplikacje Zezwalaj tylko jedno okno ramowe otwartych dokumentów w czasie. MDI — aplikacje Zezwalaj wielu dokumentów, okien ramowych były otwarte w tym samym wystąpieniu aplikacji. Aplikacja MDI ma okna w ramach których wiele MDI okno podrzędne, które okien ramowych, się, mogą być otwarte, zawierających osobne dokumentu. W niektórych aplikacjach okno podrzędne może być różnych typów, takie jak windows wykresu i arkusza kalkulacyjnego systemu windows. W takim przypadku na pasku menu można zmieniać okien podrzędnych MDI różnych typów są uaktywnione.  
  
> [!NOTE]
>  W systemie Windows 95 lub nowszego oraz aplikacje są często SDI ponieważ system operacyjny przyjęła widok "wyśrodkowany dokumentu".
