# <a name="afxdll-versions"></a>Wersje AFXDLL
Zamiast tworzenia aplikacji, łącząc statycznie do biblioteki MFC kodów obiektu, należy skompilować aplikację użycia jednej z bibliotek AFXDLL, które zawierają MFC w bibliotece DLL, które można udostępniać wielu uruchamianie aplikacji. Dla tabeli AFXDLL nazw, zobacz [biblioteki DLL: konwencje nazewnictwa](../build/naming-conventions-for-mfc-dlls.md).  
  
> [!NOTE]
>  Domyślnie Kreator aplikacji MFC tworzy projekt AFXDLL. Aby użyć statycznego łączenia kodu MFC zamiast niego, ustaw **Użyj MFC w bibliotece statycznej** opcji Kreatora aplikacji MFC. Statyczne połączenie nie jest dostępna w standardowej wersji programu Visual C++.
