# <a name="status-bars"></a>Paski stanu

Paski stanu zapewniają aplikacji miejsce do wyświetlenia wiadomości i przydatnych informacji do użytkownika bez przerywania pracy użytkownika. Zazwyczaj są wyświetlane w dolnej części okna, paski stanu mają "okienka" obejmujących "wskaźniki" i "wiersz wiadomości". Wskaźniki nadać stan takich zadań jak SCROLL LOCK, czy jest włączone rejestrowanie makra, na, i tak dalej. Wiersz wiadomości na pasku stanu można wyświetlić informacje o stanie programu lub elementem paska narzędzi przycisku lub menu, użytkownik jest wskazywane za pomocą myszy.

Utwórz pasek stanu w programie, wybierając **pasek stanu początkowego** opcja w Kreatorze aplikacji MFC.
