# <a name="ole-automation-classes"></a>Klasy automatyzacji OLE

Te klasy obsługuje klientów automatyzacji (aplikacje, które kontrolują inne aplikacje). Serwery automatyzacji (aplikacje, które mogą być kontrolowane przez inne aplikacje) są obsługiwane za pośrednictwem [mapy wysyłania](../mfc/reference/dispatch-maps.md).

[COleDispatchDriver](../mfc/reference/coledispatchdriver-class.md)<br/>
Używane do wywoływania serwerów automatyzacji z klienta usługi automation. Dodawanie klasy, ta klasa służy do tworzenia klas bezpieczny dla serwerów automatyzacji, które udostępniają biblioteki typów.

[COleDispatchException](../mfc/reference/coledispatchexception-class.md)<br/>
Wyjątek wynikającego z błędem podczas automatyzacji OLE. Automation wyjątki są zgłaszane przez serwery automatyzacji i przechwycony przez klientów automatyzacji.
