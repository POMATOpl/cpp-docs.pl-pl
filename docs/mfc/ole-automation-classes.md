# <a name="ole-automation-classes"></a>Klasy automatyzacji OLE
Te klasy obsługuje klientów automatyzacji (aplikacje, które kontrolują inne aplikacje). Serwery automatyzacji (aplikacje, które mogą być kontrolowane przez inne aplikacje) są obsługiwane za pośrednictwem [mapy wysyłania](../mfc/reference/dispatch-maps.md).  
  
 [COleDispatchDriver](../mfc/reference/coledispatchdriver-class.md)  
 Używane do wywoływania serwery automatyzacji z klienta automatyzacji. Podczas dodawania klasy, ta klasa jest używana do tworzenia klas bezpieczne w przypadku serwerów automatyzacji, które zapewniają bibliotekę typów.  
  
 [COleDispatchException](../mfc/reference/coledispatchexception-class.md)  
 Wystąpił wyjątek wynikającego z wystąpił błąd podczas automatyzacji OLE. Automatyzacja wyjątki są zgłaszane przez serwery automatyzacji i przechwycony przez klientów automatyzacji.
