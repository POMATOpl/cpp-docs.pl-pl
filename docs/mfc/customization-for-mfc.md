# <a name="customization-for-mfc"></a>Dostosowywanie na potrzeby MFC
Ten temat zawiera wskazówki dotyczące dostosowywania aplikacji MFC.  
  
## <a name="general-customizations"></a>Dostosowania ogólne  
 Można zapisywać i ładowania stanu aplikacji do rejestru. Po włączeniu tej opcji aplikacja zostanie załadowany stanu początkowego z rejestru. Jeśli zmienisz początkowy układ dokowania dla aplikacji, należy wyczyścić dane rejestru dla aplikacji. W przeciwnym razie dane w rejestrze zastąpią wszelkie zmiany wprowadzone w układzie początkowej.
