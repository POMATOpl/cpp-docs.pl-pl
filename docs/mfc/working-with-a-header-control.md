# <a name="working-with-a-header-control"></a>Praca z formantem nagłówka

Łatwy sposób używania kontrolki nagłówka ([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)) jest w połączeniu z kontrolką listy; Zobacz [Korzystanie z CListCtrl](../mfc/using-clistctrl.md) w dalszej części tego tematu. Możesz również użyć kontrolki nagłówka. Wywołania MFC `InitCommonControls` . Najważniejsze zadania są następujące:

- [Tworzenie kontrolki nagłówka](../mfc/creating-the-header-control.md)

- [Dodawanie elementów do kontrolki nagłówka](../mfc/adding-items-to-the-header-control.md)

- [Określanie kolejności elementów w kontrolce nagłówka](../mfc/ordering-items-in-the-header-control.md)

- [Przetwarzanie powiadomień dotyczących kontrolki nagłówka](../mfc/processing-header-control-notifications.md)

Jeśli obiekt formantu nagłówka jest osadzony w widoku nadrzędnym lub klasie okna dialogowego, formant zostanie zniszczony po zniszczeniu elementu nadrzędnego.
