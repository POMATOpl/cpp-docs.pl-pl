# <a name="creating-an-mfc-activex-control-container"></a>Tworzenie kontenera kontrolek ActiveX MFC
Kontenera formantu ActiveX to dostarczającego środowiska dla formantu ActiveX (dawniej OLE) uruchomić program nadrzędnej. Można utworzyć aplikacji może zawierać kontrolki ActiveX z lub bez MFC, ale jest łatwiej wykonać za pomocą MFC.  
  
 Tworzenie MFC kontenera programu za pomocą [Kreator aplikacji MFC](../../mfc/reference/mfc-application-wizard.md) umożliwia dostęp do wielu funkcji formantów ActiveX i automatyzacji implementowane za pomocą klasy MFC i ActiveX. Te funkcje obejmują visual edycję automatyzacji tworzenia pliki złożone i obsługę formantów. Kreator aplikacji MFC visual edycji opcje obsługujące program nadrzędnego obejmują tworzenie kontenera, mini serwer pełny serwer i program, który jest kontenerem i serwer.  
  
-   **Nowa aplikacja MFC**. Aby utworzyć nowy program MFC, która obejmuje automatyzacji, edycja wizualna złożone pliki, lub sterować pomocy technicznej, użyj Kreatora aplikacji MFC i wybierz odpowiednie opcje automatyzacji.  
  
-   **Istniejącej aplikacji MFC**. Jeśli dodajesz zawierania kontrolek do istniejącej aplikacji MFC, zobacz [kontenery formantów OLE: ręczne Włączanie OLE zawierania formantów](../../mfc/activex-control-containers-manually-enabling-activex-control-containment.md).
