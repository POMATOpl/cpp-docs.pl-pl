# <a name="adding-an-mfc-class"></a>Dodawanie klasy MFC
Aby dodać klasy pochodnej z klasy biblioteki Microsoft Foundation Class (MFC) do projektu, należy użyć **Dodaj klasę** dostępne z polecenie [widoku klasy](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925). Określ nazwę nowej klasy, wybierz klasę podstawową, a następnie wybierz identyfikator okna dialogowego, z którym jest skojarzona (jeśli istnieje). Kreator kod tworzy plik nagłówka i pliku z implementacją i dodaje je do projektu.  
  
> [!NOTE]
>  Możesz dodawać klasy MFC do aplikacji ATL COM, jeśli można początkowo [tworzenia aplikacji z obsługą MFC](../../atl/reference/mfc-support-in-atl-projects.md). Klasy MFC mogą również dodać do projektów Win32, których obsługa MFC.  
  
### <a name="to-add-an-mfc-class-to-your-project"></a>Aby dodać do projektu klasy MFC  
  
1.  Z widoku klasy kliknij prawym przyciskiem myszy nazwę projektu. Kliknij przycisk **Dodaj** , a następnie kliknij przycisk **Dodaj klasę** otworzyć [Dodaj klasę](../../ide/add-class-dialog-box.md) okno dialogowe.  
  
2.  W okienku szablonów wybierz **klasy MFC** i naciśnij klawisz **Dodaj** przycisku.  
  
3.  Zdefiniuj ustawienia dla nowej klasy w [Kreator klas MFC](../../mfc/reference/mfc-add-class-wizard.md) okno dialogowe.  
  
4.  Kliknij przycisk **Zakończ** aby zamknąć kreatora, aby wyświetlić nową klasę w widoku klas. Możesz również wyświetlić pliki tworzone przez kreatora w **Eksploratora rozwiązań**.
