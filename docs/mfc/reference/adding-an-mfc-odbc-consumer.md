# <a name="adding-an-mfc-odbc-consumer"></a>Dodawanie klienta MFC ODBC

Odbiorca MFC ODBC składa się z klasy zestawu rekordów ODBC i powiązań danych niezbędnych do uzyskania dostępu do źródła danych.

### <a name="to-add-an-mfc-odbc-consumer"></a>Aby dodać użytkownika MFC ODBC

1. W Widok klasy kliknij prawym przyciskiem myszy projekt. W menu skrótów kliknij pozycję **Dodaj** , a następnie kliknij pozycję **Dodaj klasę**.

1. W folderze Visual C++ kliknij dwukrotnie ikonę **użytkownika MFC ODBC** lub wybierz ją, a następnie kliknij przycisk **Otwórz**.

   Zostanie otwarty Kreator użytkownika MFC ODBC.

1. Zdefiniuj ustawienia zgodnie z opisem w [Kreatorze interfejsu użytkownika MFC ODBC](../../mfc/reference/mfc-odbc-consumer-wizard.md).

1. Kliknij przycisk **Zakończ** , aby zamknąć kreatora, co spowoduje wstawienie nowo utworzonego kodu klienta ODBC w projekcie.
