    HICON hIcon,

 [in]`hIcon`

##  <a name="addimage"></a>CMFCToolBarImages::AddImage  
 Dodaje mapy bitowej do obrazy pasków narzędzi.  
  
```  
int AddImage(
    HBITMAP hbmp,  
    BOOL bSetBitPerPixel=FALSE);

int AddImage(
    const CMFCToolBarImages& imageList,  
    int nIndex);
```  
  
### <a name="parameters"></a>Parametry  
 [in]`hbmp`  
 Dojście do mapy bitowej do dodania.  
  
 [in]`bSetBitPerPixel`  
 `TRUE`Jeśli `CMFCToolBarImages` obiekt używa głębi kolorów (bity na piksel) nowego obrazu. `FALSE` Jeśli `CMFCToolbarImages` obiektu zachowuje bieżący głębi kolorów.  
  
 [in]`imageList`  
 Odwołanie do `CMFCToolbarImages` obiekt, który zawiera obraz, aby dodać.  
  
 [in]`nIndex`  
 Indeks w źródle `CMFCToolbarImages` obiektu obrazu do dodania.  
  
### <a name="return-value"></a>Wartość zwracana  
 Liczba narzędzi obrazów `CMFCToolBarImages` obiekt zachowuje po nowej mapy bitowej został dodany pomyślnie; -1, jeśli operacja nie powiodła się.

BOOL PrepareDrawImage(

    0)  
    BOOL bFadeInactive=FALSE);

### <a name="parameters"></a>Parametry
