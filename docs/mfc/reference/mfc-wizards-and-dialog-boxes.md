# <a name="mfc-wizards-and-dialog-boxes"></a>Kreatory i okna dialogowe MFC

Kreatory Microsoft Foundation Class (MFC) wygenerować standardowy kod dla różnych rodzajów składników i obiektów. Kreatory można uruchomić, otwierając menu skrótów dla projektu w **Eksploratora rozwiązań** i wybierając pozycję **Dodaj**, **klasy**.
