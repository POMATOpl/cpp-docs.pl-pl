## <a name="remarks"></a>Uwagi  
 Formant kalendarza miesięcznego zapewnia użytkownikowi z interfejsem prostego kalendarza, z którego użytkownik może wybrać datę. Użytkownik może zmienić wyświetlaną przez:  
  
-   Przewijanie do tyłu i do przodu w miesiąca miesiąc.  
  
-   Kliknięcie przycisku dzisiaj tekst do wyświetlenia bieżącego dnia (Jeśli **MCS_NOTODAY** nie jest używany styl).  
  
-   Pobrania miesiąca lub roku z menu podręcznego.  
  
 Można dostosować miesiąca w formancie kalendarza przez zastosowanie różnych stylów do obiektu po jego utworzeniu. Te style są opisane w [stylów formantu kalendarza miesięcznego](http://msdn.microsoft.com/library/windows/desktop/bb760919) w zestawie Windows SDK.  
  
 Formant kalendarza miesięcznego można wyświetlić więcej niż jeden miesiąc i może on wskazywać dni specjalne (takie jak dni wolnych) przez pogrubienie daty.  
  
 Aby uzyskać więcej informacji na temat używania formant kalendarza miesięcznego, zobacz [przy użyciu CMonthCalCtrl](../../mfc/using-cmonthcalctrl.md).

##  <a name="create"></a>  CMonthCalCtrl::Create  
 Tworzy formant kalendarza miesięcznego i dołącza go do `CMonthCalCtrl` obiektu.  
  
```  
virtual BOOL Create(
    DWORD dwStyle,  
    const RECT& rect,  
    CWnd* pParentWnd,  
    UINT nID);

 
virtual BOOL Create(
    DWORD dwStyle,  
    const POINT& pt,  
    CWnd* pParentWnd,  
    UINT nID);
```  
  
### <a name="parameters"></a>Parametry  
 `dwStyle`  
 Określa kombinację style Windows stosowane do formant kalendarza miesięcznego. Zobacz [stylów formantu kalendarza miesięcznego](http://msdn.microsoft.com/library/windows/desktop/bb760919) w zestawie SDK systemu Windows, aby uzyskać więcej informacji na temat style.  
  
 `rect`  
 Odwołanie do [RECT](http://msdn.microsoft.com/library/windows/desktop/dd162897) struktury. Zawiera położenie i rozmiar formant kalendarza miesięcznego.  
  
 `pt`  
 Odwołanie do [punktu](http://msdn.microsoft.com/library/windows/desktop/dd162805) strukturę, która identyfikuje lokalizację w formancie kalendarza miesięcznego.  
  
 `pParentWnd`  
 Wskaźnik do [CWnd](../../mfc/reference/cwnd-class.md) obiekt, który jest okno nadrzędne w formancie kalendarza miesięcznego. Nie może być **NULL**.  
  
 `nID`  
 Określa identyfikator formantu formant kalendarza miesięcznego.  
  
### <a name="return-value"></a>Wartość zwracana  
 Różna od zera, jeśli inicjowanie zakończyło się pomyślnie; w przeciwnym razie 0.  
  
### <a name="remarks"></a>Uwagi  
 Utwórz miesiąc kalendarza kontroli w dwóch krokach:  
  
1.  Wywołanie [CMonthCalCtrl](../../mfc/reference/cmonthcalctrl-class.md) do skonstruowania `CMonthCalCtrl` obiektu.  
  
2.  Wywołanie funkcji członkowskiej, która tworzy kontrolkę kalendarza miesięcznego i dołącza go do `CMonthCalCtrl` obiektu.  
  
 Podczas wywoływania **Utwórz**, są inicjowane wspólne kontrolki. Wersja **Utwórz** możesz wywołania Określa, jak jest o rozmiarze:  
  
-   Aby automatycznie rozmiar formantu do jednego miesiąca MFC, wywołaj zastąpienia, która używa `pt` parametru.  
  
-   Aby samodzielnie rozmiar formantu, należy wywołać zastąpienie tej funkcji, która używa `rect` parametru.
