## <a name="remarks"></a>Uwagi  
 Formant kalendarza miesięcznego zapewnia użytkownika przy użyciu interfejsu kalendarza prosty, z którego użytkownik może wybrać wartość typu date. Użytkownik może zmienić wyświetlane dane według:  
  
-   Przewijanie do tyłu i do przodu z miesiąca na miesiąc.  
  
-   Klikając tekst dzisiaj, aby wyświetlić bieżący dzień (jeśli jest to styl MCS_NOTODAY nie jest używany).  
  
-   Pobrania miesiąc lub rok z menu podręcznego.  
  
 Można dostosować miesiąca kontrolki kalendarza, stosując różne style do obiektu podczas jego tworzenia. Te style są opisane w [style kontrolki kalendarza miesięcznego](http://msdn.microsoft.com/library/windows/desktop/bb760919) w zestawie Windows SDK.  
  
 Formant kalendarza miesięcznego można wyświetlić więcej niż jeden miesiąc, a dni specjalnych (takich jak dni wolnych od pracy) może sugerować przez pogrubienie daty.  
  
 Aby uzyskać więcej informacji na temat korzystania z formant kalendarza miesięcznego, zobacz [korzystanie z CMonthCalCtrl](../../mfc/using-cmonthcalctrl.md).

##  <a name="create"></a>  CMonthCalCtrl::Create  
 Tworzy formant kalendarza miesięcznego i dołącza je do `CMonthCalCtrl` obiektu.  
  
```  
virtual BOOL Create(
    DWORD dwStyle,  
    const RECT& rect,  
    CWnd* pParentWnd,  
    UINT nID);

 
virtual BOOL Create(
    DWORD dwStyle,  
    const POINT& pt,  
    CWnd* pParentWnd,  
    UINT nID);
```  
  
### <a name="parameters"></a>Parametry  
 *dwStyle*  
 Określa kombinację style Windows, które dotyczą kontrolki kalendarza miesięcznego. Zobacz [style kontrolki kalendarza miesięcznego](http://msdn.microsoft.com/library/windows/desktop/bb760919) w zestawie SDK Windows, aby uzyskać więcej informacji o stylach.  
  
 *Rect*  
 Odwołanie do [Prostokąt](http://msdn.microsoft.com/library/windows/desktop/dd162897) struktury. Zawiera położenie i rozmiar kontrolki kalendarza miesięcznego.  
  
 *(czas pacyficzny)*  
 Odwołanie do [punktu](http://msdn.microsoft.com/library/windows/desktop/dd162805) strukturę, która określa lokalizację formant kalendarza miesięcznego.  
  
 *pParentWnd*  
 Wskaźnik do [CWnd](../../mfc/reference/cwnd-class.md) obiekt, który jest okno nadrzędne kontrolki kalendarza miesięcznego. Nie może być równa NULL.  
  
 *nID*  
 Określa identyfikator kontrolki kalendarza miesięcznego kontrolki.  
  
### <a name="return-value"></a>Wartość zwracana  
 Wartość różną od zera, jeśli inicjowanie się powiodła. w przeciwnym razie 0.  
  
### <a name="remarks"></a>Uwagi  
 Utwórz miesiąc kalendarza kontroli w dwóch etapach:  
  
1.  Wywołaj [CMonthCalCtrl](../../mfc/reference/cmonthcalctrl-class.md) do konstruowania `CMonthCalCtrl` obiektu.  
  
2.  Wywołanie tej funkcji elementu członkowskiego, która tworzy formant kalendarza miesięcznego i dołącza je do `CMonthCalCtrl` obiektu.  
  
 Gdy wywołujesz `Create`, typowe formanty są inicjowane. Wersja `Create` możesz wywołanie określa, jak ma rozmiar:  
  
-   Aby MFC automatycznie rozmiar kontrolki na jeden miesiąc, wywołaj zastępowania, która używa *pt* parametru.  
  
-   Aby samodzielnie rozmiar kontrolki, należy wywołać przesłonięcie tej funkcji, która korzysta z *prostokąt* parametru.
