# <a name="cmfcribbonpanel-class"></a>Klasa CMFCRibbonPanel

Implementuje panel, który zawiera zestaw elementów wstążki. Po narysowaniu panelu wyświetla możliwie jak najwięcej elementów, przy czym rozmiar panelu.

Aby uzyskać więcej szczegółów, zobacz kod źródłowy znajdujący się w folderze **VC \\ atlmfc \\ src \\ MFC** instalacji programu Visual Studio.

## <a name="syntax"></a>Składnia

```
class CMFCRibbonPanel : public CObject
```

## <a name="remarks"></a>Uwagi

Panele wstążki są logicznymi grupami powiązanych zadań tworzonych w ramach kategorii wstążki. Po zmianie rozmiaru wstążki układ panelu automatycznie dostosowuje się do wyświetlania możliwie największej liczby elementów.

Możesz uzyskać panele wstążki, które znajdują się w kategorii wstążki, wywołując metodę [CMFCRibbonCategory:: GetPanel](../../mfc/reference/cmfcribboncategory-class.md#getpanel) .

## <a name="cmfcribbonpaneladdtoolbar"></a><a name="addtoolbar"></a> CMFCRibbonPanel:: AddToolBar

Dodaje pasek narzędzi do panelu wstążki.

```
CMFCRibbonButtonsGroup* AddToolBar(
UINT uiToolbarResID,
UINT uiColdResID = 0,
UINT uiHotResID = 0,
UINT uiDisabledResID = 0);
```

### <a name="parameters"></a>Parametry

*uiToolbarResID*<br/>
podczas Określa identyfikator zasobu paska narzędzi, który ma zostać dodany.

*uiColdResID*<br/>
podczas Określa identyfikator zasobu zimnych obrazów paska narzędzi.

*uiHotResID*<br/>
podczas Określa identyfikator zasobu dla gorąca obrazów paska narzędzi.

*uiDisabledResID*<br/>
podczas Określa identyfikator zasobu wyłączonych obrazów paska narzędzi.

### <a name="return-value"></a>Wartość zwracana

Wywołaj tę metodę, aby dodać pasek narzędzi do panelu wstążki. Pasek narzędzi zostanie dodany obok elementu wstążki dodanego przez poprzednie wywołanie do [CMFCRibbonPanel:: Add](#add).

### <a name="remarks"></a>Uwagi

Aby uzyskać więcej informacji o paskach narzędzi, aktywnych obrazach, zimnych obrazach i wyłączonych obrazach, zobacz [Klasa CMFCToolBar](../../mfc/reference/cmfctoolbar-class.md).

### <a name="example"></a>Przykład

Poniższy przykład pokazuje, jak używać `SetElementRTCByID` metody:

```

// Load and add toolbar with standard buttons. This toolbar
// should display a custom color button with id ID_CHAR_COLOR:

pPanel->AddToolBar(IDR_MAINFRAME,
    IDB_MAINFRAME256);

CMFCRibbonColorButton* pColorButton =
(CMFCRibbonColorButton*)pPanel->SetElementRTCByID(
ID_CHAR_COLOR,
    RUNTIME_CLASS (CMFCRibbonColorButton));

// SetElementRTCByID sets runtime class and returns a pointer
// to the newly created custom button,
    which can be set up immediately:
pColorButton->EnableAutomaticButton(_T("Automatic"),
    RGB (0,
    0,
```

## <a name="cmfcribbonpanelsetjustifycolumns"></a><a name="setjustifycolumns"></a> CMFCRibbonPanel::SetJustifyColumns

```cpp

```

### <a name="parameters"></a>Parametry

*bSet*<br/>
podczas Wartość TRUE, aby dostosować szerokość elementów wstążki w tej samej kolumnie do szerokości największego elementu wstążki w kolumnie; Wartość FALSE powoduje wyłączenie tego dopasowania szerokości.
