# <a name="cmfcribbonpanel-class"></a>Klasa CMFCRibbonPanel

Implementuje panel, który zawiera zestaw elementów wstążki. Po narysowaniu panelu wyświetla się jak najwięcej elementów, biorąc pod uwagę rozmiar panelu.

Aby uzyskać więcej informacji, zobacz kod źródłowy znajdujący się w folderze **vc\\\\atlmfc src\\mfc** instalacji programu Visual Studio.

## <a name="syntax"></a>Składnia

```
class CMFCRibbonPanel : public CObject
```

## <a name="cmfcribbonpaneladdtoolbar"></a><a name="addtoolbar"></a>CMFCRibbonPanel::Pasek dodawania

Dodaje pasek narzędzi do panelu wstążki.

```
CMFCRibbonButtonsGroup* AddToolBar(
UINT uiToolbarResID,
UINT uiColdResID = 0,
UINT uiHotResID = 0,
UINT uiDisabledResID = 0);
```

### <a name="parameters"></a>Parametry

*interfejs użytkownika uiToolbarResID*<br/>
[w] Określa identyfikator zasobu paska narzędzi do dodania.

*identyfikator uiColdResID*<br/>
[w] Określa identyfikator zasobu zimnych obrazów paska narzędzi.

*uiHotResID*<br/>
[w] Określa identyfikator zasobu gorących obrazów paska narzędzi.

*identyfikator uiDisabledResID*<br/>
[w] Określa identyfikator zasobu wyłączonych obrazów paska narzędzi.

### <a name="return-value"></a>Wartość zwracana

Wywołanie tej metody, aby dodać pasek narzędzi do panelu wstążki. Pasek narzędzi zostanie dodany obok elementu wstążki dodanego przez poprzednie wywołanie do [CMFCRibbonPanel::Dodaj](#add).

### <a name="remarks"></a>Uwagi

Aby uzyskać więcej informacji na temat pasków narzędzi, obrazów gorących, obrazów zimnych i obrazów niepełnosprawnych, zobacz [CMFCToolBar Class](../../mfc/reference/cmfctoolbar-class.md).

### <a name="example"></a>Przykład

W poniższym przykładzie `SetElementRTCByID` pokazano, jak używać metody:

```

// should display a custom color button with id ID_CHAR_COLOR:

    IDB_MAINFRAME256);

(CMFCRibbonColorButton*)pPanel->SetElementRTCByID(
ID_CHAR_COLOR,
    RUNTIME_CLASS (CMFCRibbonColorButton));

// SetElementRTCByID sets runtime class and returns a pointer
// to the newly created custom button,
    which can be set up immediately:
pColorButton->EnableAutomaticButton(_T("Automatic"),
    RGB (0,
    0,
    0));
```

## <a name="cmfcribbonpanelsetjustifycolumns"></a><a name="setjustifycolumns"></a>CMFCRibbonPanel::SetJustifyColumns

Włącza lub wyłącza dopasowanie szerokości elementów wstążki w tej samej kolumnie.

```
void SetJustifyColumns(BOOL bSet = TRUE);
```

### <a name="parameters"></a>Parametry

*bStaw*<br/>
[w] PRAWDA, aby dostosować szerokość elementów wstążki w tej samej kolumnie do szerokości największego elementu wstążki w kolumnie; FALSE, aby wyłączyć tę regulację szerokości.

### <a name="remarks"></a>Uwagi

Gdy ta funkcja jest włączona w panelu wstążki, szerokości elementów wstążki w tej samej kolumnie są dostosowywane do szerokości największego elementu wstążki w tej samej kolumnie.
