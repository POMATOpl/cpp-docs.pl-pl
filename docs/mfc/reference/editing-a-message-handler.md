# <a name="editing-a-message-handler"></a>Edytowanie programu obsługi komunikatów

Po zdefiniowaniu programu obsługi komunikatów można przejść do definicji funkcji elementu członkowskiego, aby dodać lub zmodyfikować kod.

Aby przejść do definicji funkcji składowej z [edytora okien dialogowych](../../windows/dialog-editor.md), kliknij dwukrotnie formant, dla którego program obsługi jest już zdefiniowany. Spowoduje to przejście do pliku, w którym zdefiniowano procedurę obsługi komunikatów wybranego formantu.

Aby przejść do definicji funkcji składowej z [Widok klasy](/visualstudio/ide/viewing-the-structure-of-code), kliknij dwukrotnie nazwę funkcji w widok klasy.
