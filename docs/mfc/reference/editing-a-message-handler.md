# <a name="editing-a-message-handler"></a>Edytowanie programu obsługi komunikatów
Po zdefiniowaniu obsługi wiadomości, można przejść do definicji funkcji członkowskiej do dodawania lub modyfikowania kodu.  
  
 Aby przejść do definicji funkcji członkowskiej, z [Edytor okien dialogowych](../../windows/dialog-editor.md), kliknij dwukrotnie formant, dla którego program obsługi jest już zdefiniowany. To przechodzi do pliku, w którym zdefiniowana jest zaznaczony formant obsługi wiadomości.  
  
 Aby przejść do definicji funkcji członkowskiej, z [widoku klasy](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), kliknij dwukrotnie nazwę funkcji w widoku klasy.
