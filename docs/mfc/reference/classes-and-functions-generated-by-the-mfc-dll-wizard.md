# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>Klasy i funkcje generowane przez kreatora biblioteki MFC DLL

Kod generowany przez kreatora MFC DLL zależy od rodzaju tworzonego pliku DLL oraz wybranych opcji. Kreator biblioteki MFC DLL generuje ten sam kod dla obu form zwykłych bibliotek DLL MFC.

|Rodzaj biblioteki DLL|Opcja|Klasy|Funkcje|
|-----------------|------------|-------------|---------------|
|[Rozszerzenie](../../build/extension-dlls-overview.md)|Brak|Brak|`DllMain`|
|[Zwykły](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Brak|Klasa aplikacji pochodna `CWinApp`|Brak|
|[Zwykły](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Automatyzacja|Klasa aplikacji pochodna `CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[Rozszerzenie](../../build/extension-dlls-overview.md)|Gniazda okien|Brak|`DllMain`|
|[Zwykły](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Gniazda okien|Klasa aplikacji pochodna `CWinApp`|`InitInstance` zawiera wywołanie `AfxSocketInit`|
