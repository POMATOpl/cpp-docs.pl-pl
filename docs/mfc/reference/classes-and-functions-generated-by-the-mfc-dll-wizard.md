# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>Klasy i funkcje generowane przez kreatora biblioteki MFC DLL

Kod generowany przez Kreatora biblioteki DLL MFC zależy od rodzaju biblioteki DLL, którą tworzysz i wybranych opcji. Kreator bibliotek DLL MFC generuje ten sam kod dla obu form zwykłych bibliotek DLL MFC.

|Rodzaj biblioteki DLL|Opcja|Klasy|Funkcje|
|-----------------|------------|-------------|---------------|
|[Wewnętrzny](../../build/extension-dlls-overview.md)|Brak|Brak|`DllMain`|
|[Zwykłe](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Brak|Klasa aplikacji pochodząca z`CWinApp`|Brak|
|[Zwykłe](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Automatyzacja|Klasa aplikacji pochodząca z`CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[Wewnętrzny](../../build/extension-dlls-overview.md)|Gniazda okienne|Brak|`DllMain`|
|[Zwykłe](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Gniazda okienne|Klasa aplikacji pochodząca z`CWinApp`|`InitInstance`zawiera połączenie z`AfxSocketInit`|
