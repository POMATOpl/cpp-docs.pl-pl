# <a name="mapping-messages-to-functions"></a>Mapowanie komunikatów na funkcje

[Kreator klas](mfc-class-wizard.md) umożliwia powiązanie obsługi komunikatów (funkcji członkowskich klas interfejsu użytkownika MFC) z komunikatami generowanymi przez zasoby aplikacji. Używają [map komunikatów MFC](../../mfc/messages-and-commands-in-the-framework.md) do utworzenia powiązania.

Korzystając z widoku klasy do utworzenia nowej klasy pochodnej jednej z klas framework, automatycznie umieszcza pełną i funkcjonalną klasę w plikach nagłówka (.h) i implementacji (.cpp), które określisz.

> [!NOTE]
> Aby dodać nową klasę, która nie obsługuje wiadomości, należy utworzyć klasę bezpośrednio w edytorze tekstu.

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>Aby zdefiniować lub usunąć program obsługi wiadomości za pomocą Kreatora klas

1. W **widoku klasy**kliknij prawym przyciskiem myszy klasę.

1. W menu kontekstowym wybierz pozycję [Kreator zajęć](mfc-class-wizard.md).
