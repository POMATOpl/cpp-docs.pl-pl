# <a name="manipulating-menus-during-program-execution"></a>Operowanie menu podczas wykonywania programu

Użyj klasy `CMenu` , aby manipulować menu i elementami menu na bieżąco. `CMenu` hermetyzuje obsługę HMENU systemu Windows i dostarcza funkcje członkowskie do pracy z menu.

Zobacz Omówienie klasy [CMenu](reference/cmenu-class.md) , aby uzyskać szczegółowe informacje.
