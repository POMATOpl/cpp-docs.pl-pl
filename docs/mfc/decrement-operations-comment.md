# <a name="-operations-comment"></a>Komentarz // Operations

`// Operations` Część deklaracji klasy MFC zawiera funkcje Członkowskie, które można wywołać obiektu do ułatwiają wykonywanie czynności lub wykonywania akcji (operacji). Te funkcje są zazwyczaj non -**const** ponieważ zazwyczaj mają skutki uboczne. Mogą być wirtualne ani niewirtualne w zależności od potrzeb klasy. Zazwyczaj te elementy członkowskie są publiczne.

W przykładzie z klasy `CStdioFile`w [przykład komentarzy](../mfc/an-example-of-the-comments.md), lista zawiera dwie funkcje Członkowskie w ramach tego komentarza: `ReadString` i `WriteString`.

Podobnie jak w przypadku atrybutów, operacji może być podzielona.
