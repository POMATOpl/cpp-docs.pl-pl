# <a name="window-dialog-and-control-classes"></a>Klasy okien, okien dialogowych i formantów

Klasa `CWnd` i jej klasy pochodne hermetyzowają `HWND` uchwyt do okna systemu Windows. `CWnd` może być używany przez siebie lub jako podstawa do wyprowadzania nowych klas. Klasy pochodne dostarczone przez bibliotekę klas reprezentują różne rodzaje systemu Windows.

[CWnd](../mfc/reference/cwnd-class.md)<br/>
Klasa bazowa dla wszystkich okien. Można użyć jednej z klas pochodnych `CWnd` lub samodzielnie utworzyć własne klasy.
