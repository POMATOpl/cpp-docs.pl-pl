# <a name="activex-controls"></a>Kontrolki ActiveX
W programie Visual C++ można utworzyć kontrolek ActiveX przy użyciu kontenera MFC ani ATL.  

>[!IMPORTANT]
> ActiveX jest technologią starszą, która nie powinny być używane w przypadku nowych wdrożeń. Wiele możliwości formantów ActiveX mogą być wykonywane w prostszy i bardziej bezpieczny sposób za pomocą nowoczesnych technologii, takich jak HTML5 i JavaScript, rozszerzenia nowoczesne przeglądarki lub format WebAssembly modułów. Aby uzyskać więcej informacji, zobacz [podziału z przeszłości, część 2: usuwanie attachEvent ActiveX, VBScript,...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) i [natywnej obsługi komunikatów](/microsoft-edge/extensions/guides/native-messaging) i [rozszerzenia Microsoft Edge](/microsoft-edge/extensions) i [format WebAssembly](https://webassembly.org/).
