# <a name="activex-controls"></a>Kontrolki ActiveX

W Visual C++ można tworzyć kontrolki ActiveX przy użyciu biblioteki MFC lub ATL.

>[!IMPORTANT]
> Kontrolka ActiveX to Starsza technologia, która nie powinna być używana do nowych celów programistycznych. Wiele możliwości formantów ActiveX może być wykonywanych w prostszy i bardziej bezpieczny sposób dzięki nowoczesnym technologiom, takim jak HTML5 i JavaScript, nowoczesne rozszerzenia przeglądarki lub moduły webassembly.
