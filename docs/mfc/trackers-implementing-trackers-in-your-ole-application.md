# <a name="trackers-implementing-trackers-in-your-ole-application"></a>Trackery: implementowanie trackerów w aplikacji OLE

Trackery zapewniają interfejs graficzny umożliwiający użytkownikom interakcję z elementami klienta OLE. Przy użyciu różnych śledzenie stylów, OLE klienta, które elementy mogą być wyświetlane z kreskowanym obramowaniem, zmienić rozmiar dojścia lub szereg innych efektów wizualnych. W tym artykule opisano:

- [Jak: Implementowanie śledzenia w kodzie](../mfc/how-to-implement-tracking-in-your-code.md).

- [Gumka i Trackery](../mfc/rubber-banding-and-trackers.md).

Artykuł obejmuje również użycie style o śledzeniu. Ponadto zapewnia kilka odwołań do przykładu MFC OLE [OCLIENT](../visual-cpp-samples.md).
