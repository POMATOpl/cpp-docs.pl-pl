# <a name="document-and-view-classes-created-by-the-mfc-application-wizard"></a>Klasy dokumentów i widoków tworzone przez kreatora aplikacji MFC
Kreator aplikacji MFC zapewnia ułatwiają rozpoczęcie na komputerze program za tworzenie klasy szkieletowych dokumentów i widoków. Następnie możesz [mapy poleceń i komunikatów do tych klas](../mfc/reference/mapping-messages-to-functions.md) i umożliwia ich funkcje Członkowskie zapisu Edytor kodu źródłowego języka Visual C++.  
  
 Klasa dokumentu tworzone przez Kreatora aplikacji MFC pochodzi od klasy [CDocument](../mfc/reference/cdocument-class.md). Klasa widoku pochodzi od [CView](../mfc/reference/cview-class.md). Nazwy tych klas zawiera Kreatora aplikacji, a pliki zawierające je są oparte na nazwę projektu, należy podać w oknie dialogowym Kreatora aplikacji. W Kreatorze aplikacji strona wygenerowane klasy służy do zmiany nazwy domyślnej.  
  
 Niektóre aplikacje mogą wymagać więcej niż jedną klasę dokumentu, klasa widoku lub klasy okien ramowych. Aby uzyskać więcej informacji, zobacz [wiele typów dokumentów, widoków i okien ramowych](../mfc/multiple-document-types-views-and-frame-windows.md).
