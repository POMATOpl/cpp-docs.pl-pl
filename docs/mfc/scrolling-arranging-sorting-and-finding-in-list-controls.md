# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>Przewijanie, porządkowanie, sortowanie i znajdowanie w kontrolkach listy
Formanty listy ([CListCtrl](../mfc/reference/clistctrl-class.md)) są przewijanego domyślnie. Aby uzyskać więcej informacji, zobacz [pozycji przewijania](http://msdn.microsoft.com/library/windows/desktop/bb774736) w zestawie Windows SDK i [przewijania](../mfc/reference/clistctrl-class.md#scroll) funkcję elementu członkowskiego.  
  
 Możesz wywołać `CListCtrl` funkcji elementów członkowskich, aby uporządkować listę elementów w formancie, sortowanie elementów i Znajdź określonych elementów. Aby uzyskać więcej informacji, zobacz [Arranging, sortowanie i wyszukiwanie](http://msdn.microsoft.com/library/windows/desktop/bb774736) w zestawie Windows SDK i [CListCtrl](../mfc/reference/clistctrl-class.md) członków [Rozmieść](../mfc/reference/clistctrl-class.md#arrange), [SortItems](../mfc/reference/clistctrl-class.md#sortitems) , a [FindItem](../mfc/reference/clistctrl-class.md#finditem).
