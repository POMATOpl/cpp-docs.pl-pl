# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>Przewijanie, porządkowanie, sortowanie i znajdowanie w kontrolkach listy

Kontrolki listy ([CListCtrl](../mfc/reference/clistctrl-class.md)) są przewijany domyślnie. Aby uzyskać więcej informacji, zobacz [położenie przewijania](/windows/desktop/Controls/using-list-view-controls) w zestawie Windows SDK i [przewijania](../mfc/reference/clistctrl-class.md#scroll) funkcja elementu członkowskiego.

Możesz wywołać `CListCtrl` funkcji elementów członkowskich do Rozmieść elementy listy w kontrolce sortowania elementów i wyszukiwania określonych elementów. Aby uzyskać więcej informacji, zobacz [Arranging, sortowanie i wyszukiwanie](https://msdn.microsoft.com/library/windows/desktop/bb774736) w zestawie Windows SDK i [CListCtrl](../mfc/reference/clistctrl-class.md) członków [Rozmieść](../mfc/reference/clistctrl-class.md#arrange), [SortItems](../mfc/reference/clistctrl-class.md#sortitems) , a [FindItem](../mfc/reference/clistctrl-class.md#finditem).
