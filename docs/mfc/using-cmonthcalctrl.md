# <a name="using-cmonthcalctrl"></a>Korzystanie z CMonthCalCtrl

Kontrola kalendarza miesiąca ([CMonthCalCtrl](../mfc/reference/cmonthcalctrl-class.md)) implementuje interfejs użytkownika podobny do kalendarza. Zapewnia to użytkownikowi bardzo intuicyjną i rozpoznawalną metodę wprowadzania lub wybierania daty. Formant zapewnia również aplikacji środki do uzyskania i ustawić informacje o dacie w formancie przy użyciu istniejących typów danych. Domyślnie formant kalendarza miesiąca wyświetla bieżący dzień i miesiąc. Jednak użytkownik jest w stanie przewinąć do poprzedniego i następnego miesiąca i wybrać określony miesiąc i/lub rok.

> [!NOTE]
> Aby użyć `CMonthCalCtrl` klasy w projekcie, należy dołączyć AFXDTCTL. H, zwykle w *stdafx.h*.
