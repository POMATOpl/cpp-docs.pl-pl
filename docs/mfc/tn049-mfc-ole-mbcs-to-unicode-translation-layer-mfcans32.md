# <a name="tn049-mfcole-mbcs-to-unicode-translation-layer-mfcans32"></a>TN049: warstwa tłumaczenia MFC/OLE MBCS na Unicode (MFCANS32)

Ta Uwaga pierwotnie opisano sposób, w jaki MFCANS32.DLL udostępnia interfejsy ANSI w głównie świecie Unicode 32-bitowym. Ta biblioteka DLL nie jest już używana przez MFC.
