# <a name="tn037-multithreaded-mfc-21-applications"></a>TN037: wielowątkowe aplikacje MFC 2.1

Ta Uwaga techniczna została pierwotnie opisana w temacie ograniczenia dotyczące programów wielowątkowych z MFC 2,1, które zostały pierwotnie udostępnione Visual C++ 1,0 dla systemu Windows NT. MFC 3,0 obsługuje wielowątkowość bezpośrednio i jest udokumentowana. Aby uzyskać więcej informacji, zobacz to odwołanie.
