# <a name="real4"></a>REAL4

Przydziela i opcjonalnie inicjuje liczba zmiennoprzecinkowa pojedynczej precyzji (4-bajtową) dla każdego *inicjatora*.

## <a name="syntax"></a>Składnia

> Inicjator real4 — nazwa [[, inicjator]]...
