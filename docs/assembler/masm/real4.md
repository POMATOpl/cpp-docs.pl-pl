# <a name="real4"></a>REAL4

Alokuje i opcjonalnie inicjuje liczbę zmiennoprzecinkową o pojedynczej precyzji (4-bajtową) dla każdego *inicjatora*.

## <a name="syntax"></a>Składnia

> **REAL4** *inicjatora* nazw ⟦ __,__ *inicjator* ... ⟧
