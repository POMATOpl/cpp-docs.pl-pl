# <a name="real4"></a>REAL4
Przydziela i opcjonalnie inicjuje liczbą zmiennoprzecinkowych pojedynczej precyzji (4-bajtowych) dla każdego *inicjatora*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name REAL4 initializer [[, initializer]]...  
```
