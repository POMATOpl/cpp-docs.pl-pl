# <a name="struct"></a>STRUCT

Deklaruje typ struktury *z określonymi deklaracjami pól*. Każde pole musi być prawidłową definicją danych. Analogicznie jak [STRUC](struc.md).

## <a name="syntax"></a>Składnia

> *Nazwa* **struct** ⟦*wyrównanie*⟧ ⟦ __,__ **nieunikatowe**⟧ *pól-deklaracje*\
> **koniec** nazwy

## <a name="remarks"></a>Uwagi

Argument *name* musi być taki sam w instrukcji otwierającej i zamykającej.
