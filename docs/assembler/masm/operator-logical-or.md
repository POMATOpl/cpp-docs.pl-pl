# <a name="operator-"></a>operator ||

Logiczne `OR`. Stosowane tylko w [. Jeśli](../../assembler/masm/dot-if.md), [. GDY](../../assembler/masm/dot-while.md), lub [. Powtórz](../../assembler/masm/dot-repeat.md) blokuje i oceniane w czasie wykonywania, a nie w czasie zestawu.

## <a name="syntax"></a>Składnia

> expression1 || expression2
