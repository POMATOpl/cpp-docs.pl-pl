# <a name="operator-"></a>operator ||

Logiczne OR. Używany tylko w ramach programu [. Jeśli](dot-if.md), [. WHILE](dot-while.md)lub [. POWTARZAj](dot-repeat.md) bloki i oceniaj w czasie wykonywania, a nie w czasie montażu.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **||** *wyrażenie2*
