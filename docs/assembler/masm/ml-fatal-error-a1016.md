# <a name="ml-fatal-error-a1016"></a>Błąd krytyczny ML A1016

**Błąd wewnętrzny asemblera**

Sterownik MASM o nazwie ML.exe wygenerował błąd systemu.

Zanotuj sytuacje błędu i zawiadom firmę Microsoft Corporation. Usługi pomocy technicznej są dostępne pod adresem [https://support.microsoft.com/](https://support.microsoft.com/) .
