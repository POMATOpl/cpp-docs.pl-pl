# <a name="sqword"></a>SQWORD
Przydziela i opcjonalnie inicjuje 8 bajtów podpisem magazynu dla każdej `initializer`. Można także używać jako Specyfikator typu dowolnym typem jest dozwolony.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
