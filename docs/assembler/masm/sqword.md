# <a name="sqword"></a>SQWORD

Przydziela i opcjonalnie inicjuje 8 bajtów podpisem magazynu dla każdego `initializer`. Może również służyć jako specyfikatora typu dowolnym typem jest dozwolony.

## <a name="syntax"></a>Składnia

> Inicjator sqword — nazwa [[, inicjator]]...
