# <a name="real8"></a>REAL8

Alokuje i opcjonalnie inicjuje liczbę zmiennoprzecinkową podwójnej precyzji (8-bajtową) dla każdego *inicjatora*.

## <a name="syntax"></a>Składnia

> **REAL8** *inicjatora* nazw ⟦ __,__ *inicjator* ... ⟧
