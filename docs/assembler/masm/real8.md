# <a name="real8"></a>REAL8

Przydziela i opcjonalnie inicjuje liczba zmiennoprzecinkowa podwójnej precyzji (8-bajtową) dla każdego *inicjatora*.

## <a name="syntax"></a>Składnia

> Inicjator real8 — nazwa [[, inicjator]]...
