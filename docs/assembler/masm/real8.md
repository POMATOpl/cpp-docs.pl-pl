# <a name="real8"></a>REAL8
Przydziela i opcjonalnie inicjuje liczbą zmiennoprzecinkowe podwójnej precyzji (8-bajtowych) dla każdego *inicjatora*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name REAL8 initializer [[, initializer]]...  
```
