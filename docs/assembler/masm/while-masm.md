# <a name="while"></a>WHILE

Powtarza zestaw *instrukcji* Block, tak długo, jak *constantExpression* pozostaje true.

## <a name="syntax"></a>Składnia

> **WHILE** *ConstantExpression*\
> *instrukcje*\
> **ENDM**
