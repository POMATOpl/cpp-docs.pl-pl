# <a name="operator-highword"></a>operator HIGHWORD
Zwraca 16 bitów niski 32-bitowy *wyrażenia*. Wyrażenia MASM są 64-bitowej wartości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
HIGHWORD expression  
```
