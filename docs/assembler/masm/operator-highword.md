# <a name="operator-highword"></a>operator HIGHWORD

Zwraca 16 bitów o niskiej 32 bitów *wyrażenia*. Wyrażenia MASM są 64-bitowe wartości.

## <a name="syntax"></a>Składnia

> *Wyrażenie* HIGHWORD
