# <a name="operator-eq"></a>operator EQ

Zwraca wartość true (-1) Jeśli *wyrażenie1* jest równa *wyrażenie2*, lub zwraca wartość false (0), jeśli nie jest.

## <a name="syntax"></a>Składnia

> wyrażenie1 EQ wyrażenie2
