# <a name="operator-eq"></a>operator EQ
Zwraca wartość true (-1) Jeśli *wyrażenie1* jest równe *wyrażenie2*, lub zwraca wartość false (0), jeśli jej nie ma.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
expression1 EQ expression2  
```
