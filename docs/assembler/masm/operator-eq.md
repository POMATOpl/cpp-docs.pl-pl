# <a name="operator-eq"></a>operator EQ

Zwraca wartość true (-1), jeśli *wyrażenie1* równa *wyrażenie2*lub zwraca wartość false (0), jeśli nie.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **EQ** *wyrażenie2*
