# <a name="tbyte"></a>TBYTE

Przydziela i opcjonalnie inicjuje 10 bajtów magazynu dla każdego *inicjatora*. Może być również używany jako specyfikator typu, gdziekolwiek typ jest dozwolony. **DT** jest synonimem **TBYTE**.

## <a name="syntax"></a>Składnia

> ⟦*name*⟧ **TBYTE** *inicjator* ⟦ __,__ *inicjator* ... ⟧
