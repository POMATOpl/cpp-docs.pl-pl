# <a name="386-32-bit-masm"></a>.386 (32-bitowy MASM)

Włącza zestaw nieuprzywilejowanych instrukcji dla procesora 80386; wyłącza zestaw instrukcji wprowadzonych w nowszych procesorach. (tylko 32-bitowy MASM).

## <a name="syntax"></a>Składnia

> **.386**

## <a name="remarks"></a>Uwagi

Włącza również instrukcje 80387.
