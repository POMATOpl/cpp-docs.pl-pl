# <a name="386"></a>.386
Włącza zestawu nieuprzywilejowanym instrukcje dotyczące procesora 80386; Wyłącza zestaw instrukcji wprowadzone w systemie nowsze procesorów.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.386  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Umożliwia również 80387 instrukcje.
