# <a name="dd"></a>DD

Przydziela i opcjonalnie inicjuje podwójne słowo (4 bajty) magazynu dla każdego `initializer`. `DD` jest synonimem [DWORD](../../assembler/masm/dword.md).

## <a name="syntax"></a>Składnia

> [[name]] Dodaj inicjator [[, inicjator]]...

## <a name="remarks"></a>Uwagi

Może również służyć jako specyfikatora typu dowolnym typem jest dozwolony.
