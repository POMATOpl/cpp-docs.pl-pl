# <a name="dd"></a>DD

Przydziela i opcjonalnie inicjuje podwójne słowo (4 bajty) magazynu dla każdego *inicjatora*. **DD** jest synonimem [typu DWORD](dword.md).

## <a name="syntax"></a>Składnia

> ⟦*Nazwa*⟧ **DD** *inicjator* ⟦ __,__ *inicjator* ... ⟧

## <a name="remarks"></a>Uwagi

Może być również używany jako specyfikator typu, gdziekolwiek typ jest dozwolony.
