# <a name="ml-warning-a4012"></a>Ostrzeżenie ML A4012
**informacje o numerach wierszy dla segmentu bez klasy "CODE"**  
  
 Brak instrukcji w segmencie, który nie ma nazwy klasy, która kończy się wyrazem "Kod". Asembler nie spowodowała wygenerowania informacje CodeView te instrukcje.  
  
 CodeView nie może przetworzyć modułów z kodu w segmentach z nazwą klasy, które nie kończą się znakiem 'CODE'.
