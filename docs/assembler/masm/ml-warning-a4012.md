# <a name="ml-warning-a4012"></a>Ostrzeżenie ML A4012

**informacje o numerze wiersza dla segmentu bez klasy "CODE"**

Wystąpiły instrukcje w segmencie, który nie ma nazwy klasy, która kończy się ciągiem "Kod". Asembler nie wygenerować informacje CodeView dotyczące tych instrukcji.

CodeView nie może przetworzyć modułów przy użyciu kodu w segmentach z nazwą klasy, które nie kończą się znakiem "Kod".
