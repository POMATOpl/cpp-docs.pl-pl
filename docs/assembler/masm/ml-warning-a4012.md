# <a name="ml-warning-a4012"></a>Ostrzeżenie ML A4012

**Informacje o numerze wiersza dla segmentu bez klasy "CODE"**

Wystąpiły instrukcje w segmencie, który nie ma nazwy klasy kończącej się znakiem "CODE". Asembler nie wygenerował informacji CodeView dla tych instrukcji.

CodeView nie może przetwarzać modułów z kodem w segmentach z nazwami klas, które nie kończą się znakiem "CODE".
