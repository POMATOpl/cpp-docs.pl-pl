# <a name="substr"></a>SUBSTR
Zwraca podciąg *textitem*, rozpoczynając od *pozycji*. *Textitem* może być literałem stałej poprzedzony `%`, lub długość ciągu zwróconego przez funkcję makro.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name SUBSTR textitem, position [[, length]]  
```
