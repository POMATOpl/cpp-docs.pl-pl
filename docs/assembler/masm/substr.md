# <a name="substr"></a>SUBSTR

Zwraca podciąg *elementu textitem*, rozpoczynając od *pozycji*. *Element textitem* może być ciągiem literału, stałą poprzedzoną przez `%` lub ciągiem zwracanym przez funkcję makro.

## <a name="syntax"></a>Składnia

> *Nazwa* **elementu substr** __,__ *pozycja* ⟦__,__ *Długość*⟧
