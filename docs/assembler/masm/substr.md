# <a name="substr"></a>SUBSTR

Zwraca podciąg *textitem*, począwszy od stawki *pozycji*. *Textitem* może być literałem stałej poprzedzone `%`, lub ciąg zwracany przez funkcję makro.

## <a name="syntax"></a>Składnia

> Nadaj nazwę SUBSTR textitem, pozycja [[, długość]]
