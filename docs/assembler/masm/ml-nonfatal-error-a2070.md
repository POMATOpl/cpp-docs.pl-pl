# <a name="ml-nonfatal-error-a2070"></a>Błąd niekrytyczny ML A2070

**Nieprawidłowa instrukcja operandów**

Jednego lub większej liczbie operandów nie jest nieprawidłowa dla instrukcji za pomocą którego zostały określone.
