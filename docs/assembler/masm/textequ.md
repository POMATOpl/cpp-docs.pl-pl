# <a name="textequ"></a>TEXTEQU

Assigns *textitem* to *name*. The *textitem* can be a literal string, a constant preceded by a `%`, or the string returned by a macro function.

## <a name="syntax"></a>Składnia

> *name* **TEXTEQU** ⟦*textitem*⟧
