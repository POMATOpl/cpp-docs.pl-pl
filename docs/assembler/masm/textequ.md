# <a name="textequ"></a>TEXTEQU

Przypisuje *textitem* do *nazwa*. *Textitem* może być literałem stałej poprzedzone `%`, lub ciąg zwracany przez funkcję makro.

## <a name="syntax"></a>Składnia
