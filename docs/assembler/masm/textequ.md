# <a name="textequ"></a>TEXTEQU
Przypisuje *textitem* do *nazwa*. *Textitem* może być literałem stałej poprzedzony `%`, lub długość ciągu zwróconego przez funkcję makro.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name TEXTEQU [[textitem]]  
```
