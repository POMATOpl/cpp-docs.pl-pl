# <a name="ml-nonfatal-error-a2060"></a>Błąd niekrytyczny ML A2060

**wymuszony błąd: ciągi nie są równe**

Dyrektywa błędu warunkowego [. ERRDIF](../../assembler/masm/dot-errdif.md) lub **. ERRDIFI** został użyty do wygenerowania tego błędu.
