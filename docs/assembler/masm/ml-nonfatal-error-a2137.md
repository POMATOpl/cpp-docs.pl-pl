# <a name="ml-nonfatal-error-a2137"></a>Błąd niekrytyczny ML A2137

**za mało argumentów do wywołania**

Liczba argumentów przesłanych za pomocą dyrektywy [Invoke](invoke.md) była mniejsza niż liczba wymaganych parametrów określonych w prototypie dla wywoływanej procedury.
