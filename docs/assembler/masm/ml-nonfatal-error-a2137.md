# <a name="ml-nonfatal-error-a2137"></a>Błąd niekrytyczny ML A2137

**za mało argumentów dla INVOKE**

Liczba argumentów przekazana, za pomocą [INVOKE](../../assembler/masm/invoke.md) dyrektywy była mniejsza niż wymagana określona liczba parametrów w prototypie procedury wywoływanej.
