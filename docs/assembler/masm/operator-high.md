# <a name="operator-high"></a>operator HIGH
Zwraca wysokiej 8 bitów niski 16 bitów *wyrażenia*. Wyrażenia MASM są 64-bitowej wartości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
HIGH expression  
```
