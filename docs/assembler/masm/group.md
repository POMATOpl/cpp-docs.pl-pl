# <a name="group"></a>GRUPA

(tylko 32-bitowy MASM). Dodaj określony *segmenty* do grupy o nazwie *Nazwa*.

## <a name="syntax"></a>Składnia

> *Nazwa* *segmentu* grupy ⟦ __,__ *segment* ... ⟧
