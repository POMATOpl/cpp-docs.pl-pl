# <a name="fword"></a>FWORD
Przydziela i opcjonalnie inicjuje 6 bajtów magazynu dla każdej `initializer`.  
  
## <a name="syntax"></a>Składnia  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Uwagi  
 Można także używać jako Specyfikator typu dowolnym typem jest dozwolony.
