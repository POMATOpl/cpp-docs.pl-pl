# <a name="fword"></a>FWORD

Przydziela i opcjonalnie inicjuje 6 bajtów magazynu dla każdego `initializer`.

## <a name="syntax"></a>Składnia

> [[name]] Inicjator fword — [[, inicjator]]...

## <a name="remarks"></a>Uwagi

Może również służyć jako specyfikatora typu dowolnym typem jest dozwolony.
