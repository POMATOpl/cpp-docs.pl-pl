# <a name="ml-nonfatal-error-a2064"></a>Błąd niekrytyczny ML A2064
**Wyrównanie struktury musi być 1, 2, 4, 8 lub 16**  
  
 Wyrównanie określone w definicji struktury jest nieprawidłowy.
