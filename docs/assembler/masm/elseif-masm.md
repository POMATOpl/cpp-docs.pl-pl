# <a name="elseif"></a>ELSEIF

Łączy [inne](else-masm.md) i [Jeśli](if-masm.md) w jednej instrukcji.

## <a name="syntax"></a>Składnia

> \ **ElseIf** *constantExpression*
> *instrukcje*\
> ⟦**ELSE**\
> *else-instrukcje*⟧ \
> **ENDIF**
