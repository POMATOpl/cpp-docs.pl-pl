# <a name="if2"></a>IF2
**Jeśli** bloku jest oceniana w każdym przebiegu zestawu, jeśli **opcja: SETIF2** jest **TRUE**.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
IF2   
expression  
  
```
