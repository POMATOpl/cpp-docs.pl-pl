# <a name="exit-32-bit-masm"></a>. EXIT (32-bitowy MASM)

Generuje kod zakończenia. (tylko 32-bitowy MASM).

## <a name="syntax"></a>Składnia

> . EXIT [[wyrażenie]]

## <a name="remarks"></a>Uwagi

Zwraca *wyrażenie* opcjonalne do powłoce.
