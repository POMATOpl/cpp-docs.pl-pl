# <a name="exit"></a>.EXIT
Generuje kod zakończenia.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.EXIT [[expression]]  
```  
  
## <a name="remarks"></a>Uwagi  
 Zwraca opcjonalne *wyrażenie* powłoki.
