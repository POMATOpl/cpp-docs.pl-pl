# <a name="operator-xor"></a>operator XOR

Zwraca wynik operacji bitowego **XOR** dla elementu *wyrażenie1* i *wyrażenie2*.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **XOR** *wyrażenie2*
