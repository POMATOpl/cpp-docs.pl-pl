# <a name="ml-nonfatal-error-a2037"></a>Błąd niekrytyczny ML A2037
**Instrukcja nie jest dozwolona wewnątrz definicji struktury**  
  
 Definicja struktury zawierała instrukcję nieprawidłowy.  
  
 Struktury nie mogą zawierać instrukcji, etykiet, procedury, dyrektywy przepływu sterowania [. URUCHAMIANIE](../../assembler/masm/dot-startup.md), lub [. Zakończ](../../assembler/masm/dot-exit.md).
