# <a name="ml-nonfatal-error-a2037"></a>Błąd niekrytyczny ML A2037

**Instrukcja nie jest dozwolona wewnątrz definicji struktury**

Definicja struktury zawiera nieprawidłową instrukcję.

Struktura nie może zawierać instrukcji, etykiet, procedur, dyrektyw przepływu sterowania, [. Uruchamianie](../../assembler/masm/dot-startup.md)lub [. Zakończ](../../assembler/masm/dot-exit.md).
