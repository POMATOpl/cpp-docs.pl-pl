# <a name="ml-nonfatal-error-a2037"></a>Błąd niekrytyczny ML A2037

**Instrukcja nie jest dozwolona wewnątrz definicji struktury**

Nieprawidłowa instrukcja znajdujących się w definicji struktury.

Struktura nie może zawierać instrukcje, etykiety, procedur, dyrektywy przepływ sterowania [. URUCHAMIANIE](../../assembler/masm/dot-startup.md), lub [. Zakończ](../../assembler/masm/dot-exit.md).
