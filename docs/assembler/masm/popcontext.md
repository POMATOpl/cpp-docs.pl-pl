# <a name="popcontext"></a>POPCONTEXT

Przywraca część lub całość bieżącego `context` (zapisane przez [pushcontext —](../../assembler/masm/pushcontext.md) dyrektywy). `context` Może być **ASSUMES**, `RADIX`, **OFERCIE**, **Procesora**, lub **wszystkich**.

## <a name="syntax"></a>Składnia

> Popcontext — kontekst
