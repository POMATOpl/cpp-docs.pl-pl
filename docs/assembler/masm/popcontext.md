# <a name="popcontext"></a>POPCONTEXT
Przywraca części lub całej bieżącej `context` (zapisane przez [pushcontext —](../../assembler/masm/pushcontext.md) dyrektywy). `context` Może być **ZAKŁADA**, `RADIX`, **WYŚWIETLANIA**, **Procesora**, lub **wszystkich**.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
POPCONTEXT context  
```
