# <a name="popcontext"></a>POPCONTEXT

Przywraca część lub wszystkie bieżące *kontekstu* (zapisane przez dyrektywę [PUSHCONTEXT](pushcontext.md) ). *Kontekst* można **założeniu** (tylko 32-bitowy MASSM), **podstawy**, **Lista**, **procesor CPU** (32-bitowy MASSM) lub **wszystkie**.

## <a name="syntax"></a>Składnia

>  *Kontekst* POPCONTEXT
