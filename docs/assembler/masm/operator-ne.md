# <a name="operator-ne"></a>Operator NE

Zwraca wartość true (-1) Jeśli *wyrażenie1* nie jest równa *wyrażenie2*, lub zwraca wartość false (0), jeśli istnieje.

## <a name="syntax"></a>Składnia

> wyrażenie1 wyrażenie2 NE
