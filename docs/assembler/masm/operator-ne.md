# <a name="operator-ne"></a>Operator NE

Zwraca wartość true (-1), jeśli element *wyrażenie1* nie jest równy *wyrażenie2* lub zwraca wartość false (0), jeśli jest to możliwe.

## <a name="syntax"></a>Składnia

> *wyrażenie1* , **ne** *wyrażenie2*
