# <a name="operator-high32"></a>operator HIGH32
Zwraca wysokiej 32-bitowy *wyrażenia*. Wyrażenia MASM są 64-bitowej wartości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
HIGH32 expression  
```
