# <a name="operator-high32"></a>operator HIGH32

Zwraca wysoko 32 bitów *wyrażenia*. Wyrażenia MASM są 64-bitowe wartości.

## <a name="syntax"></a>Składnia

> *Wyrażenie* HIGH32
