# <a name="continue"></a>.CONTINUE
Generuje kod, aby przejść do góry [. GDY](../../assembler/masm/dot-while.md) lub [. Powtórz](../../assembler/masm/dot-repeat.md) zablokować, jeśli `condition` ma wartość true.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.CONTINUE [[.IF condition]]  
```
