# <a name="continue-32-bit-masm"></a>. Kontynuuj (32-bitowy MASM)

Generuje kod, aby przeskoczyć do góry elementu [. WHILE](dot-while.md) lub [. Powtórz](dot-repeat.md) blok, jeśli *warunek* ma wartość true. (tylko 32-bitowy MASM).

## <a name="syntax"></a>Składnia

> **. Kontynuuj** ⟦ **. Jeśli** *warunek*⟧
