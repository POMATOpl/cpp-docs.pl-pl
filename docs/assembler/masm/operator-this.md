# <a name="operator-this"></a>operator THIS
Zwraca wartość określonego argumentu operacji `type` których wartości przesunięcia i segmentów są równe bieżącą wartość licznika lokalizacji.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
THIS type  
```
