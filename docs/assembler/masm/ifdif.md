# <a name="ifdif"></a>IFDIF]

Przyznaje zestawu, jeśli elementy tekstowe są różne.

## <a name="syntax"></a>Składnia

> IFDIF [[I]] textitem1, textitem2

## <a name="remarks"></a>Uwagi

Jeśli **I** jest podana, wynikiem porównania jest uwzględniana wielkość liter. Zobacz [IF](../../assembler/masm/if-masm.md) dla pełnej składni.
