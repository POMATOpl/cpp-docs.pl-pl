# <a name="ifdif-ifdifi"></a>IFDIF, IFDIFI

Przyznaje zestawu, jeśli elementy tekstowe są różne.

## <a name="syntax"></a>Składnia

> **IFDIF**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Uwagi

Jeśli **IFDIFI** jest podana, wynikiem porównania jest uwzględniana wielkość liter. Zobacz [IF](../../assembler/masm/if-masm.md) dla pełnej składni.
