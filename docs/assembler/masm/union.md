# <a name="union"></a>UNION

Deklaruje Unię składającą się z co najmniej jednego typu danych. *Deklaracje pól* muszą być prawidłowymi definicjami danych. Pomiń etykietę [](ends-masm.md) *nazwy* zakończenia dla zagnieżdżonych definicji **Unii** .

## <a name="syntax"></a>Składnia

> *Nazwa* **Union** ⟦*wyrównanie*⟧ ⟦ __,__ **nieunikatowy**⟧ \
> \ *deklaracji pola*
> ⟦*Nazwa*⟧
