# <a name="ml-nonfatal-error-a2079"></a>Błąd niekrytyczny ML A2079

**Instrukcja nie zezwala na daleko bezpośrednie adresowanie**

Warunkowego skoku lub pętli nie może należeć do innego segmentu lub grupy.
