# <a name="ml-nonfatal-error-a2063"></a>Błąd niekrytyczny ML A2063

**można wyrównać tylko do potęgi 2: wyrażenie**

Wyrażenie określone za pomocą dyrektywy [align](align-masm.md) jest nieprawidłowe.

Wyrażenie **align** musi być potęgą liczby 2 z przedziału od 2 do 256 i musi być mniejsze niż lub równe wyrównaniu bieżącego segmentu, struktury lub związku.
