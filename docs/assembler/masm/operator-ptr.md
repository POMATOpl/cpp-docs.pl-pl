# <a name="operator-ptr"></a>operator PTR

Pierwszy wymusza operator *wyrażenie* powinien być traktowany jako posiadające określonego *typu*. Drugi operator określa wskaźnik *typu*.

## <a name="syntax"></a>Składnia

> *Typ* PTR *wyrażenia*

> [[*odległość*]] PTR *typu*
