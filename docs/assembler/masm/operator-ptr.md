# <a name="operator-ptr"></a>operator PTR

Pierwszy operator wymusza, aby *wyrażenie* było traktowane jako mające określony *Typ*. Drugi operator określa wskaźnik do *typu*.

## <a name="syntax"></a>Składnia

> *wyrażenie* *typu* **PTR**
>
> *typ* \[*odległość*] **PTR**
