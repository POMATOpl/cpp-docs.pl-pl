# <a name="ml-nonfatal-error-a2119"></a>Błąd niekrytyczny ML A2119
**należy określić typ języka**  
  
 Definicja procedury lub prototypu nie podano typ języka.  
  
 Typ języka musi zostać zadeklarowany w każdej definicji procedury lub prototyp, jeśli nie określono domyślnego języka typu. Domyślny typ język jest ustawiony za pomocą [. MODEL](../../assembler/masm/dot-model.md) dyrektywy **opcji LANG**, ML opcji wiersza polecenia lub **/Gc** lub **/Gd**.
