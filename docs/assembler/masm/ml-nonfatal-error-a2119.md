# <a name="ml-nonfatal-error-a2119"></a>Błąd niekrytyczny ML A2119

**należy określić typ języka**

Definicja procedury lub prototyp nie otrzymał typu języka.

Jeśli typ języka domyślnego nie zostanie określony, typ języka musi być zadeklarowany w każdej definicji procedury lub prototypie. Domyślny typ języka jest ustawiany przy użyciu [. Dyrektywa MODEL](../../assembler/masm/dot-model.md) , **Option lang**lub opcje wiersza polecenia ml **/GC** lub **/GD**.
