# <a name="align-masm"></a>ALIGN (MASM)
Wyrównuje dalej zmiennej lub instrukcji na bajt, który jest wielokrotnością liczby `number`.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
ALIGN [[  
number  
]]  
  
```
