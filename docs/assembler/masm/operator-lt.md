# <a name="operator-lt"></a>operator LT
Zwraca wartość true (-1) Jeśli *wyrażenie1* jest mniejsza niż *wyrażenie2*, lub zwraca wartość false (0), jeśli nie jest.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
expression1 LT expression2  
```
