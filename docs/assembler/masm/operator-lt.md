# <a name="operator-lt"></a>operator LT

Zwraca wartość true (-1), jeśli element *wyrażenie1* jest mniejszy niż *wyrażenie2*lub zwraca wartość false (0), jeśli nie jest.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **lt** *wyrażenie2*
