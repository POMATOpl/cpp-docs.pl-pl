# <a name="const"></a>.CONST

Gdy jest używane z [. MODEL](../../assembler/masm/dot-model.md), rozpoczyna się segmentu danych stała (z segmentem nazwy CONST).

## <a name="syntax"></a>Składnia

> .CONST

## <a name="remarks"></a>Uwagi

Ten segment ma atrybut tylko do odczytu.
