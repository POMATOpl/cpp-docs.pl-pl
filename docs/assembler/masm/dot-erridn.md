# <a name="perioderridn-perioderridni"></a>&period;ERRIDN, &period;ERRIDNI

Generuje błąd, jeśli elementy tekstowe są identyczne.

## <a name="syntax"></a>Składnia

> **. ERRIDN**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *komunikat*]

## <a name="remarks"></a>Uwagi

Jeśli  **&period;ERRIDNI** jest podana, wynikiem porównania jest uwzględniana wielkość liter.
