# <a name="erridn"></a>. ERRIDN]

Generuje błąd, jeśli elementy tekstowe są identyczne.

## <a name="syntax"></a>Składnia

> . ERRIDN [[I]] *textitem1*, *textitem2* [[, *komunikat*]]

## <a name="remarks"></a>Uwagi

Jeśli **I** jest podana, wynikiem porównania jest uwzględniana wielkość liter.
