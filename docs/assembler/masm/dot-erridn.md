# <a name="erridn"></a>. ERRIDN]
Generuje błąd, jeśli elementy tekstowe są identyczne.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.ERRIDN[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>Uwagi  
 Jeśli **I** jest podany, wynik porównania ma bez uwzględniania wielkości liter.
