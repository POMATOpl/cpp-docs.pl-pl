# <a name="while"></a>.WHILE
Generuje kod, który wykonuje bloku `statements` podczas `condition` pozostaje wartość true.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
