# <a name="while-32-bit-masm"></a>. WHILE (32-bitowy MASM)

Generuje kod, który wykonuje blok *instrukcji* , gdy *warunek* jest prawdziwy. (tylko 32-bitowy MASM).

## <a name="syntax"></a>Składnia

> **.**  *Warunek*\
> *instrukcje*\
> **.ENDW**
