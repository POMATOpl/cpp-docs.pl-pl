# <a name="comment"></a>COMMENT

Traktuje cały *tekst* między lub w tym samym wierszu co ogranicznik jako komentarz.

## <a name="syntax"></a>Składnia

> *Ogranicznik* **komentarza** ⟦*Text*⟧ \
> ⟦*Text*⟧ \
> ⟦*Text*⟧ *ogranicznik* ⟦*Text*⟧
