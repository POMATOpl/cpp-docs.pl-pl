# <a name="comment-masm"></a>COMMENT (MASM)
Traktuje wszystkie *tekst* między lub w tym samym wierszu jako ograniczników jako komentarz.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
