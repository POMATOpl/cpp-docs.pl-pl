# <a name="operator-lowword"></a>operator LOWWORD
Zwraca niski 16 bitów *wyrażenia*. Wyrażenia MASM są 64-bitowej wartości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
LOWWORD expression  
```
