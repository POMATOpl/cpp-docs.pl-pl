# <a name="operator-lowword"></a>operator LOWWORD

Zwraca mniej 16 bitów *wyrażenia*. Wyrażenia MASM są 64-bitowe wartości.

## <a name="syntax"></a>Składnia

> *Wyrażenie* LOWWORD
