# <a name="listall"></a>.LISTALL

Rozpoczyna się lista wszystkich instrukcji.

## <a name="syntax"></a>Składnia

> .LISTALL

## <a name="remarks"></a>Uwagi

Równoważne z kombinacją [. Lista](../../assembler/masm/dot-list.md), [. LISTIF](../../assembler/masm/dot-listif.md), i [. LISTMACROALL](../../assembler/masm/dot-listmacroall.md).
