# <a name="errdef"></a>.ERRDEF
Generuje błąd, jeśli *nazwa* uprzednio zdefiniowanej etykiety, zmiennej lub symbol.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.ERRDEF name [[, message]]  
```
