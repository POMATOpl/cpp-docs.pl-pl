# <a name="errdef"></a>.ERRDEF

Generuje błąd, jeśli *Nazwa* jest wcześniej zdefiniowaną etykietą, zmienną lub symbolem.

## <a name="syntax"></a>Składnia

> **. ERRDEF** *Nazwa* ⟦__,__ *komunikat*⟧
