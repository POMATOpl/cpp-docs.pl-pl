# <a name="qword"></a>QWORD

Przydziela i opcjonalnie inicjuje 8 bajtów magazynu dla każdego *inicjatora*. Można też użyć jako specyfikatora typu, gdziekolwiek typ jest dozwolony. Wartość **Qword** jest synonimem [elemencie DQ](dq.md).

## <a name="syntax"></a>Składnia

> ⟦*Nazwa*⟧  *inicjatora* Qword ⟦__,__ *inicjator* ... ⟧
