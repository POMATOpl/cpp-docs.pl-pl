# <a name="page"></a>STRONA
Pierwsza dyrektywa ustawia wiersz *długość* i znaku *szerokość* listy programu. Jeśli są podane bez argumentów, generuje podziałów strony. Druga dyrektywa zwiększa numer sekcji i resetuje numer strony do 1.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
