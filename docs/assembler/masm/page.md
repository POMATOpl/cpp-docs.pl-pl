# <a name="page"></a>STRONA

Pierwszy dyrektywą Ustawia wiersz *długość* znaków i *szerokość* listy programów. Jeśli nie podano argumentów, generuje podziału strony. Druga dyrektywa zwiększa liczbę sekcji, a ponadto Resetuje numer strony do 1.

## <a name="syntax"></a>Składnia

> Strona [[*długość*]] [[, *szerokość*]]<br/><br/>
> STRONA +
