# <a name="operator-gt"></a>operator GT

Zwraca wartość true (-1), jeśli element *wyrażenie1* jest większy niż *wyrażenie2*lub zwraca wartość false (0), jeśli nie jest.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **gt** *wyrażenie2*
