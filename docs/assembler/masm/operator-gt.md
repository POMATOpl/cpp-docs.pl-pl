# <a name="operator-gt"></a>operator GT
Zwraca wartość true (-1) Jeśli *wyrażenie1* jest większa niż *wyrażenie2*, lub zwraca wartość false (0), jeśli nie jest.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
expression1 GT expression2  
```
