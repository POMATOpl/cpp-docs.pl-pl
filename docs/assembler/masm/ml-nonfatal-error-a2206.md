# <a name="ml-nonfatal-error-a2206"></a>Błąd niekrytyczny ML A2206
**Brak operator w wyrażeniu**  
  
 Nie można obliczyć wyrażenia, ponieważ nie ma operatora. Ten komunikat o błędzie mogą być również ubocznym poprzedni błąd programu.  
  
 Następujący wiersz wygeneruje błąd to:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
