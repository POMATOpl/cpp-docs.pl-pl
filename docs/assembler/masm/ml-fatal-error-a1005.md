# <a name="ml-fatal-error-a1005"></a>Błąd krytyczny ML A1005

**limit asemblera: tabela nazw parametrów makro pełna**

Zdefiniowano zbyt wiele parametrów, ustawień lokalnych lub etykiet makr dla makra. Nie ma więcej miejsca w tabeli nazw makr.

Zdefiniuj krótsze lub mniejsze nazwy lub usuń niepotrzebne makra.
