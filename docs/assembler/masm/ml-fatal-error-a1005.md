# <a name="ml-fatal-error-a1005"></a>Błąd krytyczny ML A1005

**limit asemblera: pełna tabela nazw parametrów — makro**

Zbyt wiele parametrów, zmiennych lokalnych lub makro etykiety zostały zdefiniowane dla makra. Nie można było ma więcej miejsca w tabeli nazw makr.

Zdefiniuj krótszy lub mniejszą liczbę nazw, lub usuń niepotrzebne makra.
