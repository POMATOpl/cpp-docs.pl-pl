# <a name="ml-fatal-error-a1005"></a>Błąd krytyczny ML A1005
**Asembler limit: pełna tabela nazw parametru makra**  
  
 Zbyt wiele parametrów, zmienne lokalne lub etykiety makro zostały zdefiniowane dla makr. Nie można było ma więcej miejsca w tabeli nazw makra.  
  
 Zdefiniuj krótszy lub mniej nazwy lub usuń niepotrzebne makra.
