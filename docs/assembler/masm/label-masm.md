# <a name="label-masm"></a>ETYKIETA (MASM)
Tworzy nową etykietę, przypisując bieżącą wartość licznika lokalizacji i danego `type` do *nazwa*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
