# <a name="label-masm"></a>ETYKIETA (MASM)

Tworzy nową etykietę, przypisując bieżącą wartość licznika lokalizacji i danym *typu* do *nazwa*.

## <a name="syntax"></a>Składnia

> *Nazwa* etykiety *typu*

> *Nazwa* etykiety [[NEAR | ZNACZNIE | PTR PROC]] [[*typu*]]
