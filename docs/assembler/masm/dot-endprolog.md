# <a name="endprolog"></a>.ENDPROLOG

Sygnalizuje koniec deklaracje prologu.

## <a name="syntax"></a>Składnia

> .ENDPROLOG

## <a name="remarks"></a>Uwagi

Jest to błąd, aby użyć dowolnego z deklaracji prologu poza regionem między [PROC](../../assembler/masm/proc.md) ramki i. ENDPROLOG.

Aby uzyskać więcej informacji, zobacz [MASM x64 (ml64.exe)](../../assembler/masm/masm-for-x64-ml64-exe.md).
