# <a name="endprolog"></a>.ENDPROLOG

Sygnalizuje koniec deklaracji prologu.

## <a name="syntax"></a>Składnia

> **.ENDPROLOG**

## <a name="remarks"></a>Uwagi

Wystąpił błąd podczas używania którejkolwiek z deklaracji prologu spoza regionu między **ramką** [proces](proc.md) i **. ENDPROLOG**.

Aby uzyskać więcej informacji, zobacz [MASM for x64 (ml64. exe)](masm-for-x64-ml64-exe.md).
