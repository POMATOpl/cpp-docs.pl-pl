# <a name="dt"></a>DT

Przydziela i opcjonalnie inicjuje 10 bajtów magazynu dla każdego *inicjatora*. Może być również używany jako specyfikator typu, gdziekolwiek typ jest dozwolony. **DT** jest synonimem [TBYTE](tbyte.md).

## <a name="syntax"></a>Składnia

> ⟦*Nazwa*⟧  *inicjatora* DT ⟦__,__ *inicjator* ... ⟧
