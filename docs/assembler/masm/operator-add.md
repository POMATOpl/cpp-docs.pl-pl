# <a name="operator-"></a>operator +
Zwraca pierwszy operator *wyrażenie1* plus *wyrażenie2*. Drugi operator zmieni znak *wyrażenia*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
      expression1 + expression2  
-expression  
```
