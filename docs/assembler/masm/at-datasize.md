# <a name="datasize"></a>\@DataSize

0 w przypadku modeli **bardzo** dużych, **małych**, **średnich** i **płaskich** , 1 dla modeli **kompaktowych** i **dużych** oraz 2 dla **dużego** modelu (wartość liczbowa równa się).

## <a name="syntax"></a>Składnia

> **\@DataSize**
