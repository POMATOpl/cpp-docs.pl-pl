# <a name="ml-nonfatal-error-a2019"></a>Błąd niekrytyczny ML A2019

**argument operacji musi być typem rekordu lub polem**

Operand następujący po [szerokości](operator-width.md) lub operatorze [maski](operator-mask.md) jest nieprawidłowy.

Operator [Width](operator-width.md) przyjmuje operand, który jest nazwą pola lub rekordem. Operator [Mask](operator-mask.md) przyjmuje operand, który jest nazwą pola lub typu rekordu.
