# <a name="ml-nonfatal-error-a2019"></a>Błąd niekrytyczny ML A2019
**Argument musi być typu REKORDU lub pola**  
  
 Następujący argument [SZEROKOŚĆ](../../assembler/masm/operator-width.md) lub [MASKI](../../assembler/masm/operator-mask.md) operatora jest nieprawidłowy.  
  
 [SZEROKOŚĆ](../../assembler/masm/operator-width.md) operator przyjmuje operand jest nazwą pola lub rekordu. [MASKI](../../assembler/masm/operator-mask.md) operator przyjmuje operand jest nazwą pola lub typu rekordu.
