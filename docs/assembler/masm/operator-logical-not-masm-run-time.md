# <a name="operator--masm-run-time"></a>operator ! (MASM czasu wykonywania)

Negacja logiczna. Używany tylko w ramach programu [. Jeśli](dot-if.md), [. WHILE](dot-while.md)lub [. POWTARZAj](dot-repeat.md) bloki i oceniaj w czasie wykonywania, a nie w czasie montażu.

## <a name="syntax"></a>Składnia

> **!** *expression*
