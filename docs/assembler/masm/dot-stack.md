# <a name="stack"></a>.STACK

Gdy jest używane z [. MODEL](../../assembler/masm/dot-model.md), definiuje segment stosu (z nazwą segmentu STOSU). Opcjonalny `size` określa liczbę bajtów stosu (ustawienie domyślne 1024). `.STACK` Dyrektywy automatycznie zamyka instrukcję stosu.

> . STOS [[rozmiar]]
