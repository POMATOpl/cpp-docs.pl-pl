# <a name="operator--"></a>operator-

Zwraca *wyrażenie1* minus *wyrażenie2*. Drugi operator odwraca znak *wyrażenie*.

## <a name="syntax"></a>Składnia

> *wyrażenie1* - *wyrażenie2*
>
> -*Wyrażenie*
