# <a name="operator--"></a>operator -

Zwraca wartość *wyrażenie1* minus *wyrażenie2*. Drugi operator odwraca znak *wyrażenia*.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **-** *wyrażenie2*
>
> *wyrażenie*-
