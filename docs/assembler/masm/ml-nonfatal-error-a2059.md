# <a name="ml-nonfatal-error-a2059"></a>Błąd niekrytyczny ML A2059

**wymuszony błąd: ciągi równe**

Dyrektywa błędu warunkowego [. ERRIDN](../../assembler/masm/dot-erridn.md) lub **. ERRIDNI** został użyty do wygenerowania tego błędu.
