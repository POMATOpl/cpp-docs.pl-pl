# <a name="ml-nonfatal-error-a2038"></a>Błąd niekrytyczny ML A2038

**Brak operandu dla operatora makra**

Asembler znalazł koniec listy parametrów makra bezpośrednio po [!](operator-logical-not-masm-run-time.md) or — [%](operator-percent.md) operator.
