# <a name="ml-nonfatal-error-a2069"></a>Błąd niekrytyczny ML A2069

**nie argumentów dozwolone dla tej instrukcji**

Jeden lub więcej argumentów została określona w instrukcji, która przyjmuje nie argumentów.
