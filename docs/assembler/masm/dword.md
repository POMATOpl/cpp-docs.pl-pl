# <a name="dword"></a>DWORD
Przydziela i opcjonalnie inicjuje wyraz o podwójnej precyzji (4 bajty) magazynu dla każdej `initializer`.  
  
## <a name="syntax"></a>Składnia  
  
```  
[[name]] DWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Uwagi  
 Może także służyć jako Specyfikator typu dowolnym typem jest dozwolony.
