# <a name="386p"></a>.386P
Włącza zestawu wszystkich instrukcji (łącznie z uprawnieniami) dla procesora 80386; Wyłącza zestaw instrukcji wprowadzone w systemie nowsze procesorów.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.386P  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Umożliwia również 80387 instrukcje.
