# <a name="operator-"></a>operator :

Zastępuje domyślny segment *wyrażenie* z *segmentu*. *Segmentu* może być rejestru segmentu, nazwę grupy, nazwą segmentu lub wyrażenie segmentu. *Wyrażenie* musi być stałą.

## <a name="syntax"></a>Składnia

> *segment* : *wyrażenia*
