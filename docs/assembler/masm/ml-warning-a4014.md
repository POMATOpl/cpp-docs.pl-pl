# <a name="ml-warning-a4014"></a>Ostrzeżenie ML A4014

instrukcje i zainicjowane dane nie są obsługiwane w segmentach BSS

Podjęto próbę zdefiniowania danych zainicjowanych w sekcji BSS.  Sekcja BSS jest definiowana jako Klasa, której nazwa to BSS.  Obejmuje to uproszczony segment `.data?`.
