# <a name="ml-warning-a4004"></a>Ostrzeżenie ML A4004

**Nie można ZAŁOŻYĆ CS**

Nastąpiła próba przyjmuje wartość rejestru CS. CS jest zawsze ustawiony na bieżącym segmencie lub grupy.
