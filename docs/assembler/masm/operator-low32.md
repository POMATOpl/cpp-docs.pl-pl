# <a name="operator-low32"></a>operator LOW32

Zwraca mniej 32 bitów *wyrażenia*. Wyrażenia MASM są 64-bitowe wartości.

## <a name="syntax"></a>Składnia

>  *Wyrażenie* LOW32
