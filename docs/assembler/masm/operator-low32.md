# <a name="operator-low32"></a>operator LOW32
Zwraca niski 32-bitowy *wyrażenia*. Wyrażenia MASM są 64-bitowej wartości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
LOW32 expression  
```
