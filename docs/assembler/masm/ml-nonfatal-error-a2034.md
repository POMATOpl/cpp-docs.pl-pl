- Instrukcja

- Definicja etykiety

- A [niniejszej](../../assembler/masm/operator-this.md) — operator

- A [ $ ](../../assembler/masm/dollar.md) — operator

- Definicja procedury

- [Organizacji](../../assembler/masm/org.md) — dyrektywa
