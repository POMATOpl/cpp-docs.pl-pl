# <a name="ml-nonfatal-error-a2034"></a>Błąd niekrytyczny ML A2034

**musi znajdować się w bloku segmentu**

Znaleziono jedną z następujących elementów poza blokiem segmentu:

- Instrukcja

- Definicja etykiety

- [Ten](../../assembler/masm/operator-this.md) operator

- Operator [$](../../assembler/masm/dollar.md)

- Definicja procedury

- Dyrektywa [align](../../assembler/masm/align-masm.md)

- Dyrektywa [org](../../assembler/masm/org.md)
