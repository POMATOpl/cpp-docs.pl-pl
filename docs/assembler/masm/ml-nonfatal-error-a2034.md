# <a name="ml-nonfatal-error-a2034"></a>Błąd niekrytyczny ML A2034
**musi być w segmencie bloku**  
  
 Jedną z następujących znaleziono poza blokiem segmentu:  
  
-   Instrukcja  
  
-   Definicja etykiety  
  
-   A [ta](../../assembler/masm/operator-this.md) — operator  
  
-   A [ $ ](../../assembler/masm/dollar.md) — operator  
  
-   Definicja procedury  
  
-   [WYRÓWNAJ](../../assembler/masm/align-masm.md) — dyrektywa  
  
-   [ORG](../../assembler/masm/org.md) — dyrektywa
