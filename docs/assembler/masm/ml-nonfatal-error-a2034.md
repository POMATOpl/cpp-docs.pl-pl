# <a name="ml-nonfatal-error-a2034"></a>Błąd niekrytyczny ML A2034

**musi być w segmencie bloku**

Znaleziono się jedną z następujących czynności poza blokiem segmentu:

- Instrukcja

- Definicja etykiety

- A [niniejszej](../../assembler/masm/operator-this.md) — operator

- A [ $ ](../../assembler/masm/dollar.md) — operator

- Definicja procedury

- [WYRÓWNAJ](../../assembler/masm/align-masm.md) — dyrektywa

- [Organizacji](../../assembler/masm/org.md) — dyrektywa
