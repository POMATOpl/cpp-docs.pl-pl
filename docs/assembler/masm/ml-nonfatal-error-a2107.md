# <a name="ml-nonfatal-error-a2107"></a>Błąd niekrytyczny ML A2107
**nie może mieć niejawne skok daleko lub wywołanie pobliżu etykiety**  
  
 Próbowano utworzyć niejawnego daleko szybkiego dostępu lub zadzwoń do najbliższej etykiety w innym segmencie.
