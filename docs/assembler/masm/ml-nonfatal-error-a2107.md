# <a name="ml-nonfatal-error-a2107"></a>Błąd niekrytyczny ML A2107

**nie może być niejawne skok daleko lub wywołanie do umieszczonej blisko etykiety**

Nastąpiła próba się niejawny daleko szybkiego dostępu lub zadzwoń do niemal etykiety w innym segmencie.
