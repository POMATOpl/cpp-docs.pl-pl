# <a name=""></a>\@\@:

Definiuje etykietę kodu rozpoznawalną tylko między *Label1* i *etykiety 2*, gdzie *Label1* jest początkiem kodu lub poprzednia `@@:` etykieta, a *etykiety 2* jest końcem kodu lub następną `@@:` etykietą. Zobacz [ \@ B](at-b.md) i [ \@ F](at-f.md).

## <a name="syntax"></a>Składnia

> **\@\@:**
