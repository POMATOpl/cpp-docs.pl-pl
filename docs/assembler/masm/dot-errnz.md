# <a name="errnz"></a>**.ERRNZ**

Generuje błąd, jeśli *wyrażenie* jest prawdziwe (niezerowe).

## <a name="syntax"></a>Składnia

> **.**  *Wyrażenie* ERRNZ ⟦ __,__ *Message*⟧
