# <a name="operator-mod"></a>operator MOD

Zwraca wartość całkowitą resztę (modulo) w przypadku dzielenia *wyrażenie1* przez *wyrażenie2*.

## <a name="syntax"></a>Składnia

> wyrażenie1 MOD wyrażenie2
