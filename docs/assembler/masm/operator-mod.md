# <a name="operator-mod"></a>operator MOD

Zwraca wartość całkowitą reszty (modulo) podczas dzielenia elementu *wyrażenie1* przez *wyrażenie2*.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **mod** *wyrażenie2*
