# <a name="operator-mod"></a>operator MOD
Zwraca wartość całkowitą pozostałej (modulo) w przypadku dzielenia *wyrażenie1* przez *wyrażenie2*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
expression1 MOD expression2  
```
