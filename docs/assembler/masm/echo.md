# <a name="echo"></a>ECHO

Wyświetla *komunikat* na urządzeniu standardowe dane wyjściowe (domyślnie ekranu).

## <a name="syntax"></a>Składnia

> ECHO *wiadomości*

## <a name="remarks"></a>Uwagi

Taki sam jak [% OUT](../../assembler/masm/percent-out.md).
