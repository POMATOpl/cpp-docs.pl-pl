# <a name="echo"></a>ECHO

Displays *message* to the standard output device (by default, the screen).

## <a name="syntax"></a>Składnia

> **ECHO** *message*

## <a name="remarks"></a>Uwagi

Same as [%OUT](../../assembler/masm/percent-out.md).
