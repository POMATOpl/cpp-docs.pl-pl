# <a name="ifidn-ifidni"></a>IFIDN, IFIDNI

Przyznaje zestaw, jeśli elementy tekstowe są identyczne.

## <a name="syntax"></a>Składnia

> **IFIDN** *textitem1*__,__ *textitem2*\
> **IFIDNI** *textitem1*__,__ *textitem2*

## <a name="remarks"></a>Uwagi

Jeśli **IFIDNI** jest określony, porównanie uwzględnia wielkość liter. Aby uzyskać pełną składnię [, zobacz.](if-masm.md)
