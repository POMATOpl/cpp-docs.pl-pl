# <a name="ifidn-ifidni"></a>IFIDN, IFIDNI

Przyznaje zestawu, jeśli elementy tekstowe są identyczne.

## <a name="syntax"></a>Składnia

> **IFIDN**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Uwagi

Jeśli **IFIDNI** jest podana, wynikiem porównania jest uwzględniana wielkość liter. Zobacz [IF](../../assembler/masm/if-masm.md) dla pełnej składni.
