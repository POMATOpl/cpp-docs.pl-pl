# <a name="ifidn"></a>IFIDN]

Przyznaje zestawu, jeśli elementy tekstowe są identyczne.

## <a name="syntax"></a>Składnia

> IFIDN [[I]] textitem1, textitem2

## <a name="remarks"></a>Uwagi

Jeśli **I** jest podana, wynikiem porównania jest uwzględniana wielkość liter. Zobacz [IF](../../assembler/masm/if-masm.md) dla pełnej składni.
