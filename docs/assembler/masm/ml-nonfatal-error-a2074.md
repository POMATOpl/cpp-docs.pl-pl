# <a name="ml-nonfatal-error-a2074"></a>Błąd niekrytyczny ML A2074

**nie można uzyskać dostępu do etykiety za poorednictwem rejestrów segmentów**

Podjęto próbę uzyskania dostępu do etykiety za pomocą rejestru segmentów, który nie został przyjęty do jego segmentu lub grupy.
