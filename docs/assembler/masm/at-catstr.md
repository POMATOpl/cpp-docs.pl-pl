# <a name="catstr"></a>\@CatStr —

Funkcja makro, która łączy jeden lub więcej ciągów. Zwraca ciąg.

## <a name="syntax"></a>Składnia

> **\@ CatStr — (** *ciąg1* ⟦__,__ *ciąg2*... ⟧ **)**
