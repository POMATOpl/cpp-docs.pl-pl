# <a name="repeat-32-bit-masm"></a>. Powtórz (32-bitowy MASM)

Generuje kod, który powtarza wykonywanie bloku *instrukcji* do momentu, gdy *warunek* stanie się prawdziwy. [. UNTILCXZ](dot-untilcxz.md), który ma wartość true, jeśli CX wynosi zero, może zostać zastąpiony przez [. DO](dot-until.md). *Warunek* jest opcjonalny z **. UNTILCXZ**. (tylko 32-bitowy MASM).

## <a name="syntax"></a>Składnia

> **. Powtórz**\
> *instrukcje*\
> **. UNTIL** — *warunek*
