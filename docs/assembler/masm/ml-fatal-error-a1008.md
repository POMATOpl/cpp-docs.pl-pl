# <a name="ml-fatal-error-a1008"></a>Błąd krytyczny ML A1008

**niedopasowane zagnieżdżenie makr**

Makro nie zostało zakończone przed końcem pliku lub [ENDM](../../assembler/masm/endm.md) dyrektywy kończącej poza blok makra.

Jedną z przyczyn tego błędu jest pominięcie kropki przed [. Powtórz](../../assembler/masm/dot-repeat.md) lub [. WHILE](../../assembler/masm/dot-while.md).
