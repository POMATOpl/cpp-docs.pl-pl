# <a name="ml-fatal-error-a1008"></a>Błąd krytyczny ML A1008

**niedopasowane zagnieżdżenie makr**

Makro nie zostało zakończone przed końcem pliku lub [ENDM](endm.md) dyrektywy kończącej poza blok makra.

Jedną z przyczyn tego błędu jest pominięcie kropki przed [. Powtórz](dot-repeat.md) lub [. WHILE](dot-while.md).
