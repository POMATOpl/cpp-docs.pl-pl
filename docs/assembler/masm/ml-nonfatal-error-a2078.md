# <a name="ml-nonfatal-error-a2078"></a>Błąd niekrytyczny ML A2078

**Instrukcja nie umożliwia pośredniego adresowania**

Warunkowe skoki lub pętla nie może przyjmować operandu pamięci. Musi mieć adres względny lub etykieta.
