# <a name="byte-masm"></a>BYTE (MASM)
Przydziela i opcjonalnie inicjuje bajt magazynu dla każdej `initializer`.  
  
## <a name="syntax"></a>Składnia  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>Uwagi  
 Może także służyć jako Specyfikator typu dowolnym typem jest dozwolony.
