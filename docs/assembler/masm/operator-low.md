# <a name="operator-low"></a>operator LOW
Zwraca niski 8 bitów *wyrażenia*. Wyrażenia MASM są 64-bitowej wartości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
LOW expression  
```
