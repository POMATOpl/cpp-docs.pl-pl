# <a name="operator-"></a>operator ;;

Traktuje *tekstu* jako komentarz w makro, które pojawia się tylko w definicji makra. Nie są wyświetlane w liście *tekstu* gdzie rozwinięciu makra.

## <a name="syntax"></a>Składnia

> ;; *tekstu*
