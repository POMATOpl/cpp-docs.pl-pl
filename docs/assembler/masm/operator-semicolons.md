# <a name="operator-"></a>operator ;;
Traktuje *tekst* jako komentarz w makrze, która jest wyświetlana tylko w definicji makra. Lista nie pokazuje *tekst* gdzie rozwinięciu makra.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
;;  
text  
  
```
