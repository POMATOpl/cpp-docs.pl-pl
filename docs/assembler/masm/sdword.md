# <a name="sdword"></a>SDWORD

Przydziela i opcjonalnie inicjuje podpisem podwójne słowo (4 bajty) magazynu dla każdego `initializer`. Może również służyć jako specyfikatora typu dowolnym typem jest dozwolony.

## <a name="syntax"></a>Składnia

> Inicjator sdword — nazwa [[, inicjator]]...
