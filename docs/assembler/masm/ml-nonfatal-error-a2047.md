# <a name="ml-nonfatal-error-a2047"></a>Błąd niekrytyczny ML A2047

**pusty ciąg (null)**

Ciąg obejmowały ograniczająca parę znaki cudzysłowu i żadnych znaków w ciągu.

Dla ciągu był prawidłowy musi zawierać 1 – 255 znaków.
