# <a name="ml-nonfatal-error-a2047"></a>Błąd niekrytyczny ML A2047
**pusty ciąg (null)**  
  
 Ciąg obejmował rozdzielające pary znaków cudzysłowu i znaków w ciągu.  
  
 Ciąg był prawidłowy musi ona zawierać 1 – 255 znaków.
