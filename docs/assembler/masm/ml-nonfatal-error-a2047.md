# <a name="ml-nonfatal-error-a2047"></a>Błąd niekrytyczny ML A2047

**pusty ciąg (null)**

Ciąg składający się z ograniczającej pary znaków cudzysłowu i bez znaków w.

Aby ciąg był prawidłowy, musi zawierać 1-255 znaków.
