# <a name="operator-ge"></a>operator GE

Zwraca wartość true (-1) Jeśli *wyrażenie1* jest większa niż lub równa *wyrażenie2*, lub zwraca wartość false (0), jeśli nie jest.

## <a name="syntax"></a>Składnia

> wyrażenie1 wyrażenie2 GE
