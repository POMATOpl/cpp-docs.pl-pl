# <a name="public-masm"></a>PUBLICZNY (MASM)
Sprawia, że każdej zmiennej, etykiety lub bezwzględny symbol określony jako *nazwa* dostępne dla wszystkich modułów w programie.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
