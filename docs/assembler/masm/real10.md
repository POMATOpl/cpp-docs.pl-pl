# <a name="real10"></a>REAL10
Przydziela i opcjonalnie inicjuje 10-bajtowa liczba zmiennoprzecinkowa dla każdego *inicjatora*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```
