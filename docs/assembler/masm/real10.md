# <a name="real10"></a>REAL10

Przydziela i opcjonalnie inicjuje 10-bajtową liczbę zmiennoprzecinkową dla każdego *inicjatora*.

## <a name="syntax"></a>Składnia

>  **REAL10** *inicjatora* nazw ⟦__,__ *inicjator* ... ⟧
