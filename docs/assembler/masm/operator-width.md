# <a name="operator-width"></a>operator WIDTH

Zwraca szerokość w bitach bieżącego *recordfieldname* lub *rekordu*.

## <a name="syntax"></a>Składnia

> SZEROKOŚĆ {recordfieldname | rekordu}
