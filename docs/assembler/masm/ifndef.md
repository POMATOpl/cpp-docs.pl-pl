# <a name="ifndef"></a>IFNDEF
Przyznaje zestawu, jeżeli *nazwa* nie został zdefiniowany.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
IFNDEF name  
```  
  
## <a name="remarks"></a>Uwagi  
 Zobacz [IF](../../assembler/masm/if-masm.md) dla pełnej składni.
