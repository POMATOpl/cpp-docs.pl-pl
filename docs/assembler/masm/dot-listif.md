# <a name="listif"></a>.LISTIF
Rozpoczyna się lista instrukcje w blokach warunkowego wartość false.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.LISTIF  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Taki sam jak [. LFCOND](../../assembler/masm/dot-lfcond.md).
