# <a name="ml-nonfatal-error-a2004"></a>Błąd niekrytyczny ML A2004

**Konflikt typów symboli: Identyfikator**

Dyrektywa [EXTERNDEF](../../assembler/masm/externdef.md) lub [Label](../../assembler/masm/label-masm.md) została użyta na zmiennej, symbolu, strukturze danych lub etykiecie, która została zdefiniowana w tym samym module, ale z innym typem.
