# <a name="ifdef"></a>IFDEF
Przyznaje zestawu, jeżeli *nazwa* uprzednio zdefiniowanej etykiety, zmiennej lub symbol.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
IFDEF name  
```  
  
## <a name="remarks"></a>Uwagi  
 Zobacz [IF](../../assembler/masm/if-masm.md) dla pełnej składni.
