# <a name="nolistif"></a>.NOLISTIF

Pomija listy bloków warunkowych, której warunek ma wartość FAŁSZ (0).

## <a name="syntax"></a>Składnia

> .NOLISTIF

## <a name="remarks"></a>Uwagi

Domyślnie włączone. Taki sam jak [. SFCOND](../../assembler/masm/dot-sfcond.md).
