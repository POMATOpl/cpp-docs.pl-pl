# <a name="nolistif"></a>.NOLISTIF
Pomija Lista bloków warunkowych, której warunek jest spełniony (0).  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.NOLISTIF  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Domyślnie włączone. Taki sam jak [. SFCOND](../../assembler/masm/dot-sfcond.md).
