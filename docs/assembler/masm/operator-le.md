# <a name="operator-le"></a>operator LE

Zwraca wartość true (-1) Jeśli *wyrażenie1* jest mniejsza niż lub równa *wyrażenie2*, lub zwraca wartość false (0), jeśli nie jest.

## <a name="syntax"></a>Składnia

> wyrażenie1 LE wyrażenie2
