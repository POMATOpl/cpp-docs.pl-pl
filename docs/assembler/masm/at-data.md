# <a name="data"></a>\@Data

Nazwa domyślnej grupy danych. Oblicza DGROUP dla wszystkich modeli, z wyjątkiem **Flat**. Oblicza **płaskie** w modelu pamięci **płaskiej** (makro tekstowe).

## <a name="syntax"></a>Składnia

> **\@Data**
