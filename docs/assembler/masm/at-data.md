# <a name="data"></a>@data

Nazwa domyślnej grupy danych. Daje w wyniku DGROUP dla wszystkich modeli, z wyjątkiem **PROSTEGO**. Daje w wyniku **PROSTEGO** w obszarze **PROSTEGO** model pamięci (makro tekstu).

## <a name="syntax"></a>Składnia

> @data
