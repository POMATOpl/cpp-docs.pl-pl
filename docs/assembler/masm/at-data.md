# <a name="data"></a>@data
Nazwa domyślnej grupy danych. Daje w wyniku DGROUP dla wszystkich modeli z wyjątkiem **PŁASKIEJ**. Daje w wyniku **PŁASKIEJ** w obszarze **PŁASKIEJ** modelu pamięci (tekst makro).  
  
## <a name="syntax"></a>Składnia  
  
```  
  
@data  
  
```
