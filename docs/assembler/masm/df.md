# <a name="df"></a>DF

Przydziela i opcjonalnie inicjuje 6 bajtów magazynu dla każdego *inicjatora*. **DF** jest synonimem [FWORD](fword.md).

## <a name="syntax"></a>Składnia

> ⟦*Nazwa*⟧ *inicjatora* DF ⟦ __,__ *inicjator* ... ⟧

## <a name="remarks"></a>Uwagi

Można też użyć jako specyfikatora typu, gdziekolwiek typ jest dozwolony.
