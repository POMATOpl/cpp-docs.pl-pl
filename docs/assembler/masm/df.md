# <a name="df"></a>DF
Przydziela i opcjonalnie inicjuje 6 bajtów magazynu dla każdej `initializer`. `DF` jest synonimem [fword —](../../assembler/masm/fword.md).  
  
## <a name="syntax"></a>Składnia  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Uwagi  
 Można także używać jako Specyfikator typu dowolnym typem jest dozwolony.
