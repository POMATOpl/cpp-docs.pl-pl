# <a name="typedef"></a>TYPEDEF

Definiuje nowy typ lub **proto** o nazwie *name*, który jest odpowiednikiem *typu* lub *protoDefinition*.

## <a name="syntax"></a>Składnia

> *Nazwa* **typedef** *Type* | **proto** *protoDefinition*
