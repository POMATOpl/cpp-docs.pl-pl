# <a name="typedef-masm"></a>TYPEDEF (MASM)
Definiuje nowy typ o nazwie *nazwa*, który jest odpowiednikiem `type`.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name TYPEDEF type  
```
