# <a name="oword"></a>OWORD

Używany jako specyfikator typu, gdy wymagany jest 16-bajtowy typ danych.

## <a name="syntax"></a>Składnia

> ⟦*name*⟧ **OWORD**  *inicjator* ⟦ __,__ *inicjator* ... ⟧
