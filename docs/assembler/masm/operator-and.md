# <a name="operator-and"></a>operator AND

Zwraca wynik wartości bitowej i operacji dla elementu *wyrażenie1* i *wyrażenie2*.

## <a name="syntax"></a>Składnia

> *wyrażenie1* **i** *wyrażenie2*
