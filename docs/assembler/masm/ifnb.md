# <a name="ifnb"></a>IFNB
Przyznaje zestawu, jeżeli *textitem* nie jest pusty.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
IFNB textitem  
```  
  
## <a name="remarks"></a>Uwagi  
 Zobacz [IF](../../assembler/masm/if-masm.md) dla pełnej składni.
