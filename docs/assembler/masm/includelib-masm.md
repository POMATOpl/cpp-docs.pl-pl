# <a name="includelib-masm"></a>INCLUDELIB (MASM)
Informuje konsolidator, że bieżący moduł powinna być łączona z *libraryname*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
INCLUDELIB libraryname  
```  
  
## <a name="remarks"></a>Uwagi  
 *Libraryname* musi być ujęta w nawiasy, jeśli zawiera ukośnik odwrotny, średnik, większa-niż symbol, mniej-niż symbol, pojedynczego cudzysłowu lub podwójny cudzysłów.
