# <a name="includelib-masm"></a>INCLUDELIB (MASM)

Informuje konsolidator, że bieżący moduł powinna być połączona z *libraryname*.

## <a name="syntax"></a>Składnia

> Includelib — libraryname

## <a name="remarks"></a>Uwagi

*Libraryname* muszą być ujęte w nawiasy ostre zawiera ukośnik odwrotny średnika, większą-niż symbol, mniej-niż symbol, cudzysłowem pojedynczym lub podwójnym.
