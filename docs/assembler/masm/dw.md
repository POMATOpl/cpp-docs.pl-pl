# <a name="dw"></a>DW
Przydziela i opcjonalnie inicjuje word (2 bajty) magazynu dla każdej `initializer`. Może także służyć jako Specyfikator typu dowolnym typem jest dozwolony. `DW`jest synonimem [WORD](../../assembler/masm/word.md).  
  
## <a name="syntax"></a>Składnia  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
