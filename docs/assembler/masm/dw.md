# <a name="dw"></a>DW

Przydziela i opcjonalnie inicjuje wyraz (2 bajty) magazynu dla każdego *inicjatora*. Może być również używany jako specyfikator typu, gdziekolwiek typ jest dozwolony. **DW** jest synonimem [wyrazu](word.md).

## <a name="syntax"></a>Składnia

> ⟦*Nazwa*⟧ *inicjatora* DW ⟦ __,__ *inicjator* ... ⟧
