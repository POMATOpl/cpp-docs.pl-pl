# <a name="codesize"></a>@CodeSize
0 dla **bardzo MAŁA**, **małych**, **COMPACT**, i **PŁASKIEJ** modeli i 1 dla **średni**,  **DUŻE**, i **HUGE** modeli (liczbowe są równoważne).

```  
  
@CodeSize  
  
```
