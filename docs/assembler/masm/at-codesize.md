# <a name="codesize"></a>\@CodeSize

0 dla modeli bardzo **niewielkich, małych**, **kompaktowych** i **płaskich** , a 1 dla **średnich**, **dużych** i **ogromnych** modeli (wartość liczbowa jest równa).

## <a name="syntax"></a>Składnia

> **\@CodeSize**
