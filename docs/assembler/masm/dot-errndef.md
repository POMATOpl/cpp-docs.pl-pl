# <a name="errndef"></a>.ERRNDEF

Generuje błąd, jeśli *Nazwa* nie została zdefiniowana.

## <a name="syntax"></a>Składnia

> **. ERRNDEF** *Nazwa* ⟦__,__ *komunikat*⟧
