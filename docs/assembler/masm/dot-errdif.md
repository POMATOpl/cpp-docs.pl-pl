# <a name="errdifi"></a>. ERRDIF [[I]]

Generuje błąd, jeśli elementy tekstowe są różne.

## <a name="syntax"></a>Składnia

> . ERRDIF [[I]] *textitem1*, *textitem2* [[, *komunikat*]]

## <a name="remarks"></a>Uwagi

Jeśli **I** jest podana, wynikiem porównania jest uwzględniana wielkość liter.
