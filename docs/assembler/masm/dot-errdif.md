# <a name="errdifi"></a>. ERRDIF [[I]]
Generuje błąd, jeśli są różne elementy tekstu.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>Uwagi  
 Jeśli **I** jest podany, wynik porównania ma bez uwzględniania wielkości liter.
