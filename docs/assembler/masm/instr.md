# <a name="instr"></a>INSTR

Znajduje pierwsze wystąpienie *textitem2* w *textitem1*.

## <a name="syntax"></a>Składnia

> *Nazwa* INSTR [[*pozycji*,]] *textitem1*, *textitem2*

## <a name="remarks"></a>Uwagi

Początkowy *pozycji* jest opcjonalne. Każdy element tekst może być literałem stałej poprzedzone `%`, lub ciąg zwracany przez funkcję makro.
