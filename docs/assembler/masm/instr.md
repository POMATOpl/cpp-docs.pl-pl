# <a name="instr"></a>INSTR
Znajduje pierwsze wystąpienie *textitem2* w *textitem1*.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
name  
 INSTR [[position,]] textitem1, textitem2  
```  
  
## <a name="remarks"></a>Uwagi  
 Początkowe *pozycji* jest opcjonalna. Każdy element tekst może być literałem stałej poprzedzony `%`, lub długość ciągu zwróconego przez funkcję makro.
