# <a name="equ"></a>EQU

Pierwszy dyrektywą przypisuje wartość liczbową *wyrażenie* do *nazwa*.

## <a name="syntax"></a>Składnia

> *Nazwa* EQU *wyrażenia*

> *Nazwa* EQU \< *tekstu*>

## <a name="remarks"></a>Uwagi

*Nazwa* nie można zdefiniować jej ponownie później.

Drugi przypisuje dyrektywy określony *tekstu* do *nazwa*. *Nazwa* można przypisać inną *tekstu* później. Zobacz [textequ —](../../assembler/masm/textequ.md).
