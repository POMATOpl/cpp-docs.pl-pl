# <a name="equ"></a>EQU

Pierwsza dyrektywa przypisuje wartość numeryczną *wyrażenia* do *nazwy*.

## <a name="syntax"></a>Składnia

>   *wyrażenie* EQU Name
>
>  **EQU** nazwy __\<__*text*__>__

## <a name="remarks"></a>Uwagi

Nie można ponownie zdefiniować *nazwy* .

Druga dyrektywa przypisuje określony *tekst* do *nazwy*. *Nazwę* można przypisać do innego *tekstu* później. Zobacz [TEXTEQU](textequ.md).
