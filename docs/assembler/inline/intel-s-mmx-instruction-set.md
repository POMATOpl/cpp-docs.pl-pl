# <a name="intel39s-mmx-instruction-set"></a>Zestaw instrukcji MMX Intel&#39;s

**Specyficzne dla firmy Microsoft**

Kompilator języka Microsoft C++ umożliwia użycie zestawu instrukcji MMX (multimedialnej) firmy Intel w asemblerze wbudowanym. Instrukcje MMX są również obsługiwane przez demontaż debugera. Kompilator generuje komunikat ostrzegawczy, jeśli funkcja zawiera instrukcje MMX, ale nie zawiera instrukcji EMMS, aby opróżnić stan multimediów. Aby uzyskać więcej informacji, zobacz witrynę sieci Web firmy Intel.
