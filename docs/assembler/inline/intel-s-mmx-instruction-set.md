# <a name="intel39s-mmx-instruction-set"></a>Zestaw&#39;instrukcji Intel s MMX

**Specyficzne dla firmy Microsoft**

Kompilator firmy C++ Microsoft umożliwia użycie zestawu instrukcji MMX (multimedialnej) firmy Intel w asemblerze wbudowanym. Instrukcje MMX są również obsługiwane przez demontaż debugera. Kompilator generuje komunikat ostrzegawczy, jeśli funkcja zawiera instrukcje MMX, ale nie zawiera instrukcji EMMS, aby opróżnić stan multimediów. Aby uzyskać więcej informacji, zobacz witrynę sieci Web firmy Intel.
