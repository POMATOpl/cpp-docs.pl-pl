# <a name="auto_search-import-attribute"></a>auto_search atrybut importowania

**Specyficzne dla języka C++**

Określa, że gdy istnieje odwołanie do biblioteki typów `#import` i sama odwołuje się do innej biblioteki typów, kompilator może wykonać niejawną `#import` dla innej biblioteki typów.

## <a name="syntax"></a>Składnia

> **#import** **auto_search** *biblioteki typów*
