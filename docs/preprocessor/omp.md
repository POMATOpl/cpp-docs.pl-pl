# <a name="omp-pragma"></a>omp, pragma

Przyjmuje co najmniej jedną dyrektywę OpenMP wraz z wszelkimi opcjonalnymi klauzulami dyrektywy.

## <a name="syntax"></a>Składnia

> **#pragma OMP** — *dyrektywa*

Aby uzyskać więcej informacji, zobacz [dyrektywy OpenMP](../parallel/openmp/reference/openmp-directives.md).
