# <a name="omp"></a>omp
Trwa co najmniej jeden dyrektywy OpenMP, wraz z dowolnym opcjonalna klauzula w dyrektywie.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
#pragma omp directive  
```
