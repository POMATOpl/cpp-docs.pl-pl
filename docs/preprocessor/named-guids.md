# <a name="named_guids-import-attribute"></a>named_guids atrybut importowania

**Specyficzne dla języka C++**

Instruuje kompilator, aby definiuje i inicjować zmienne GUID w starym stylu, w postaci,, `LIBID_MyLib` `CLSID_MyCoClass` `IID_MyInterface` i `DIID_MyDispInterface` .

## <a name="syntax"></a>Składnia

> **#import** **named_guids** *biblioteki typów*
