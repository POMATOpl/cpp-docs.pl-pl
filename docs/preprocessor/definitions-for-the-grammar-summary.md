# <a name="definitions-for-the-grammar-summary"></a>Definicje dla podsumowania dotyczącego gramatyki

Terminale są punktami końcowymi w definicji składni. Inne rozwiązanie nie jest możliwe. Terminale obejmują zestaw słów zarezerwowanych i identyfikatorów zdefiniowanych przez użytkownika.

Nieterminale są symbolami zastępczymi w składni. Większość jest definiowana w innym miejscu w tym podsumowaniu składni. Definicje mogą być cykliczne. Następujące elementy niebędące terminalami są zdefiniowane [](../cpp/lexical-conventions.md) w sekcji konwencje leksykalne w  *C++ dokumentacji języka*:

*stała*, *stała — wyrażenie*, *Identyfikator*, *słowo kluczowe*, *operator*, *punctuator*

Opcjonalny składnik jest wskazywany przez <sub>wybór</sub>z indeksu. Na przykład, poniżej wskazuje opcjonalne wyrażenie ujęte w nawiasy klamrowe:

*wyrażenie*{<sub>opt</sub> **}**
