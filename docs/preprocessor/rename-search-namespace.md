# <a name="rename_search_namespace-import-attribute"></a>rename_search_namespace atrybut importowania

**Specyficzne dla języka C++**

Ma takie same funkcje jak atrybut [rename_namespace](../preprocessor/rename-namespace.md) , ale jest używany w bibliotekach typów, w których jest używana `#import` dyrektywa wraz z atrybutem [auto_search](../preprocessor/auto-search.md) .

## <a name="syntax"></a>Składnia

> **#import** rename_search_namespace *biblioteki typów* **(** "*newname*" **)**

### <a name="parameters"></a>Parametry

*NewName*\
Nowa nazwa przestrzeni nazw.
