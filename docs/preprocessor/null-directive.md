# <a name="null-directive"></a>Null — dyrektywa
Dyrektywy preprocesora o wartości null jest pojedynczy znak numeru (**#**) wyłącznie w wierszu. Nie ma wpływu.

## <a name="syntax"></a>Składnia

```
#
```
