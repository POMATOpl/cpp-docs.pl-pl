# <a name="null-directive"></a>Null, dyrektywa

Dyrektywa preprocesora o wartości null jest pojedynczą cyfrą ( **#** ) w wierszu. Nie ma żadnego wpływu.

## <a name="syntax"></a>Składnia

> **#**
