# <a name="null-directive"></a>Null — dyrektywa
Dyrektywy preprocesora wartość null jest jednym znakiem numeru (**#**) tylko w wierszu. Nie ma wpływu.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
#  
  
```
