# <a name="autorename"></a>auto_rename
**Określonego języka C++**  
  
 Zmienia nazwę słowa zastrzeżone w języku C++ przez dołączenie do nazwy zmiennej, aby rozwiązać potencjalne konflikty nazw dwa znaki podkreślenia (_).  
  
## <a name="syntax"></a>Składnia  
  
```  
auto_rename  
```  
  
## <a name="remarks"></a>Uwagi  
 Ten atrybut jest używany podczas importowania biblioteki typu, która korzysta z jednego lub więcej C++ słowa (słowa kluczowe lub makra) zarezerwowane jako nazwy zmiennych.  
  
 **KOŃCOWY określonego języka C++**
