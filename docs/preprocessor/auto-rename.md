# <a name="auto_rename-import-attribute"></a>auto_rename atrybut importowania

**Specyficzne dla języka C++**

Zmienia nazwy wyrazów zarezerwowanych w języku C++, dołączając dwa znaki podkreślenia (__) do nazwy zmiennej, aby rozwiązać potencjalne konflikty nazw.

## <a name="syntax"></a>Składnia

> **#import** **auto_rename** *biblioteki typów*

## <a name="remarks"></a>Uwagi

Ten atrybut jest używany podczas importowania biblioteki typów, która używa co najmniej jednego słowa zastrzeżonego C++ (słowa kluczowe lub makra) jako nazw zmiennych.
