# <a name="push_macro-pragma"></a>push_macro, pragma

Zapisuje wartość makra Macro *name* w górnej części stosu dla tego makra.

## <a name="syntax"></a>Składnia

> **#pragma push_macro ("** _Macro-Name_ **")**

## <a name="remarks"></a>Uwagi

Możesz pobrać wartość dla *nazwy makra* z `pop_macro`.

Zobacz [pop_macro](../preprocessor/pop-macro.md) , aby uzyskać przykład.
