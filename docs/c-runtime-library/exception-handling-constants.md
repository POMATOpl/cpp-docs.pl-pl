# <a name="exception-handling-constants"></a>Instrukcje obsługi wyjątków — Stałe

Stała `EXCEPTION_CONTINUE_SEARCH`, `EXCEPTION_CONTINUE_EXECUTION`lub `EXCEPTION_EXECUTE_HANDLER` jest zwracana, gdy wystąpi wyjątek podczas wykonywania chronionej sekcji instrukcji **try-except** . Wartość zwracana określa sposób obsługi wyjątku. Aby uzyskać więcej informacji, zobacz [Instrukcja try-except](../cpp/try-except-statement.md) w  *C++ dokumentacji języka*.
