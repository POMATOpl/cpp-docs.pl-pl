# <a name="environmental-constants"></a>Stałe środowiska

## <a name="syntax"></a>Składnia

```
#include <stdlib.h>
```

## <a name="remarks"></a>Uwagi

Ta stała określa długość środowiska dla ciągów.

|Stała|Znaczenie|
|--------------|-------------|
|`_MAX_ENV`|Maksymalny rozmiar ciągu dla ciągu środowiska.|
