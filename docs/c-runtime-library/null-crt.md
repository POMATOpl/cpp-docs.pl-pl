# <a name="null-crt"></a>NULL (CRT)

Wartość **null** jest wartością wskaźnika o wartości null używaną z wieloma operacjami i funkcjami wskaźnika. Jest to odpowiednik wartości 0. Zdefiniowano **wartość null** w następujących plikach nagłówkowych: CRTDBG. H, USTAWIENIA REGIONALNE. H, STDDEF. H, STDIO. H, STDLIB. H, CIĄG. H, UŻYWANIE TCHAR. H, CZAS. H i WCHAR. H.
