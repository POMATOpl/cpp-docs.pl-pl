# <a name="_debug"></a>_DEBUG

Kompilator definiuje `_DEBUG` w przypadku określenia opcji/MTD lub/MDD. Te opcje określają wersje debugowania biblioteki wykonawczej C.

Aby uzyskać więcej informacji, zobacz [techniki debugowania CRT](/visualstudio/debugger/crt-debugging-techniques).
