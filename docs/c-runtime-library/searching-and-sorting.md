|Funkcja|Wyszukaj lub Sortuj|
|--------------|--------------------|
|[bsearch](../c-runtime-library/reference/bsearch.md)|Wyszukiwanie binarne|
|[bsearch_s](../c-runtime-library/reference/bsearch-s.md)|Bezpieczniejsza wersja **bsearch**|
|[_lfind](../c-runtime-library/reference/lfind.md)|Wyszukiwanie liniowe dla danej wartości|
|[_lfind_s](../c-runtime-library/reference/lfind-s.md)|Bezpieczniejsza wersja **_lfind**|

|[qsort](../c-runtime-library/reference/qsort.md)|Szybkie sortowanie|
