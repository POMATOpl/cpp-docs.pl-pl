# <a name="cisqrt"></a>_CIsqrt
Oblicza pierwiastek kwadratowy wartości top w stosie.  
  
## <a name="syntax"></a>Składnia  
  
```  
void __cdecl _CIsqrt();  
```  
  
## <a name="remarks"></a>Uwagi  
 Ta wersja `sqrt` funkcja ma specjalne Konwencja wywoływania obsługującą przez kompilator. Przyspiesza wykonywanie, ponieważ uniemożliwia kopie generowany, a ułatwia alokacja rejestru.  
  
 Wartość wynikową spoczywa na wierzchu stosu.  
  
## <a name="requirements"></a>Wymagania  
 **Platforma:** x86
