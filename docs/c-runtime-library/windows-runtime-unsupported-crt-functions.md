# <a name="windows-runtime-unsupported-crt-functions"></a>Nieobsługiwane funkcje CRT środowiska uruchomieniowego systemu Windows

Nie można używać wielu C-run-time (CRT) interfejsów API w aplikacjach systemu Windows platformy Uniwersalnej, które są wykonywane w środowisku wykonawczym systemu Windows. Te aplikacje są tworzone za pomocą flagi kompilatora /ZW. Aby uzyskać listę nieobsługiwane funkcje CRT, zobacz [funkcje CRT, nie są obsługiwane w aplikacjach platformy uniwersalnej systemu Windows](../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).

Wszystkie interfejsy API CRT opisanym w [alfabetyczne odwołanie funkcji](../c-runtime-library/reference/crt-alphabetical-function-reference.md) sekcji dokumentacji.
