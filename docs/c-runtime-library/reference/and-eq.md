# <a name="andeq"></a>and_eq
Zamiast & = — operator.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
#define and_eq &=  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Makro daje operator & =.  
  
## <a name="example"></a>Przykład  
  
```  
// iso646_and_eq.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 3, b = 2, result;  
  
   result= a &= b;  
   cout << result << endl;  
  
   result= a and_eq b;  
   cout << result << endl;  
}  
```  
  
```Output  
2  
2  
```
