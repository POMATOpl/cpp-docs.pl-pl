# <a name="bitor"></a>bitor
Zamiast &#124; operator.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
#define bitor |  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Makro daje operatora &#124;.  
  
## <a name="example"></a>Przykład  
  
```  
// iso646_bitor.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 1, b = 2, result;  
  
   result = a | b;  
   cout << result << endl;  
  
   result= a bitor b;  
   cout << result << endl;  
}  
```  
  
```Output  
3  
3  
```
