# <a name="conj-conjf-conjl"></a>conj conjf, conjl
Pobiera zespolonej liczbą.  
  
## <a name="syntax"></a>Składnia  
  
```  
_Dcomplex conj(   
   _Dcomplex z   
);  
_Fcomplex conj(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex conj(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex conjf(   
   _Fcomplex z   
);  
_Lcomplex conjl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `z`  
 Liczba złożonych.  
  
## <a name="return-value"></a>Wartość zwracana  
 Zespolonej `z`.  Wynik zawiera tę samą część rzeczywistą i urojony jako `z`, ale z przeciwnym znakiem.  
  
## <a name="remarks"></a>Uwagi  
 Ponieważ C++ pozwala przeładowanie, można wywoływać przeciążenia `conj` który przyjmować i zwracać `_Fcomplex` i `_Lcomplex` wartości. W programie C `conj` zawsze przyjmuje i zwraca `_Dcomplex` wartość.
