# <a name="_set_fmode"></a>_set_fmode

Ustawia domyślny tryb tłumaczenia plików dla operacji we/wy na plikach.

## <a name="syntax"></a>Składnia

```C
errno_t _set_fmode(
   int mode
);
```

### <a name="parameters"></a>Parametry

*wyst*<br/>
Wymagany tryb tłumaczenia plików: **_O_TEXT** lub **_O_BINARY**.

## <a name="return-value"></a>Wartość zwracana

Zwraca zero, jeśli to się powiedzie, kod błędu w przypadku niepowodzenia. Jeśli *tryb* nie jest **_O_TEXT** lub **_O_BINARY** lub **_O_WTEXT**, zostanie wywołana procedura obsługi nieprawidłowego parametru, zgodnie z opisem w [walidacji parametru](../../c-runtime-library/parameter-validation.md). Jeśli wykonanie może być kontynuowane, ta funkcja ustawia **errno** na **EINVAL** i zwraca **EINVAL**.

## <a name="remarks"></a>Uwagi

Funkcja ustawia zmienną globalną [_fmode](../../c-runtime-library/fmode.md) . Ta zmienna określa domyślny tryb tłumaczenia plików dla operacji we/wy pliku **_open** i **_pipe**.

**_O_TEXT** i **_O_BINARY** są zdefiniowane w fcntl. h. **EINVAL** jest zdefiniowany w errno. h.

## <a name="example"></a>Przykład

```C
// crt_set_fmode.c
#include <stdlib.h>
#include <stdio.h>
#include <fcntl.h>     /* for _O_TEXT and _O_BINARY */
#include <errno.h>     /* for EINVAL */
#include <sys\stat.h>  /* for _S_IWRITE */
#include <share.h>     /* for _SH_DENYNO */

int main()
{
   int mode, fd, ret;
   errno_t err;
   int buf[12] = { 65, 66, 67, 68, 69, 70, 71, 72, 73, 74,
                   75, 76 };
   char * filename = "fmode.out";

   err = _get_fmode(&mode);
   if (err == EINVAL)
   {
      printf( "Invalid parameter: mode\n");
      return 1;
   }
   else
      printf( "Default Mode is %s\n", mode == _O_TEXT ? "text" :
              "binary");

   err = _set_fmode(_O_BINARY);
   if (err == EINVAL)
   {
      printf( "Invalid mode.\n");
      return 1;
   }

   if ( _sopen_s(&fd, filename, _O_RDWR | _O_CREAT, _SH_DENYNO, _S_IWRITE | _S_IREAD) != 0 )
   {
      printf( "Error opening the file %s\n", filename);
      return 1;
   }

   if (ret = _write(fd, buf, 12*sizeof(int)) < 12*sizeof(int))
   {
      printf( "Problem writing to the file %s.\n", filename);
      printf( "Number of bytes written: %d\n", ret);
   }

   if (_close(fd) != 0)
   {
      printf("Error closing the file %s. Error code %d.\n",
             filename, errno);
   }

   system("type fmode.out");
}
```

```Output
Default Mode is binary
A   B   C   D   E   F   G   H   I   J   K   L
```
