# <a name="carg-cargf-cargl"></a>carg cargf, cargl
Pobiera argument liczbą, oddział Wytnij rzeczywistych osi ujemna.  
  
## <a name="syntax"></a>Składnia  
  
```  
double carg(   
   _Dcomplex z   
);  
float carg(   
   _Fcomplex z   
);  // C++ only  
long double carg(   
   _Lcomplex z   
);  // C++ only  
float cargf(   
   _Fcomplex z   
);  
long double cargl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `z`  
 Liczba złożonych.  
  
## <a name="return-value"></a>Wartość zwracana  
 Argument (znanej także jako etap) `z`. Wynik jest w interwale [-π, + π].  
  
## <a name="remarks"></a>Uwagi  
 Ponieważ C++ pozwala przeładowanie, można wywoływać przeciążenia `carg` które trwają `_Fcomplex` lub `_Lcomplex` wartości i wróć `float` lub `long double` wartości. W programie C `carg` zawsze ma `_Dcomplex` wartości i zwraca `double` wartość.
