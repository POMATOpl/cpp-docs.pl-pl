# <a name="noteq"></a>not_eq

Alternatywa dla operatora !=.

## <a name="syntax"></a>Składnia

```C

#define not_eq !=
```

## <a name="remarks"></a>Uwagi

Makro daje operator !=.

## <a name="example"></a>Przykład

```cpp
// iso646_not_eq.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 0, b = 1;

   if (a != b)
      cout << "a is not equal to b" << endl;

   if (a not_eq b)
      cout << "a is not equal to b" << endl;
}
```

```Output
a is not equal to b
a is not equal to b
```
