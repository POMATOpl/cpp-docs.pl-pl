# <a name="not"></a>not

Zamiast! operator.

## <a name="syntax"></a>Składnia

```C

#define not !

```

## <a name="remarks"></a>Uwagi

Makro daje operator !.

## <a name="example"></a>Przykład

```cpp
// iso646_not.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 0;

   if (!a)
      cout << "a is zero" << endl;

   if (not(a))
      cout << "a is zero" << endl;
}
```

```Output
a is zero
a is zero
```
