# <a name="kbhit"></a>_kbhit
Sprawdza, czy konsoli dla klawiatury.  
  
> [!IMPORTANT]
>  Nie można używać tego interfejsu API w aplikacjach, które są wykonywane w środowisku wykonawczym systemu Windows. Aby uzyskać więcej informacji, zobacz [funkcje CRT, nie są obsługiwane w aplikacjach platformy uniwersalnej systemu Windows](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).  
  
## <a name="syntax"></a>Składnia  
  
```  
  
int _kbhit( void );  
```  
  
## <a name="return-value"></a>Wartość zwracana  
 `_kbhit` Zwraca wartość niezerową, jeśli klucz został naciśnięty. W przeciwnym razie zwraca wartość 0.  
  
## <a name="remarks"></a>Uwagi  
 `_kbhit` Funkcja sprawdza konsoli dla ostatnich naciśnięcie klawisza. Jeśli funkcja zwraca wartość niezerową, naciśnięcie klawisza oczekuje się w buforze. Program może wywoływać `_getch` lub `_getche` można pobrać naciśnięcie klawisza.

## <a name="example"></a>Przykład  
  
```  
// crt_kbhit.c  
// compile with: /c  
/* This program loops until the user  
 * presses a key. If _kbhit returns nonzero, a  
 * keystroke is waiting in the buffer. The program  
 * can call _getch or _getche to get the keystroke.  
 */  
  
#include <conio.h>  
#include <stdio.h>  
  
int main( void )  
{  
   /* Display message until key is pressed. */  
   while( !_kbhit() )  
      _cputs( "Hit me!! " );  
  
   /* Use _getch to throw key away. */  
   printf( "\nKey struck was '%c'\n", _getch() );  
}  
```

