# <a name="or_eq"></a>or_eq

Alternatywa dla operatora &#124;=.

## <a name="syntax"></a>Składnia

```C

#define or_eq |=
```

## <a name="remarks"></a>Uwagi

Makro daje operator &#124;=.

## <a name="example"></a>Przykład

```cpp
// iso646_oreq.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a |= b;
   cout << result << endl;

   result= a or_eq b;
   cout << result << endl;
}
```

```Output
3
3
```
