# <a name="and"></a>and

Alternatywa dla & & — operator.

## <a name="syntax"></a>Składnia

```C

#define and &&
```

## <a name="remarks"></a>Uwagi

Makro daje operator & &.

## <a name="example"></a>Przykład

```cpp
// iso646_and.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   bool a = true, b = false, result;

   boolalpha(cout);

   result= a && b;
   cout << result << endl;

   result= a and b;
   cout << result << endl;
}
```

```Output
false
false
```
