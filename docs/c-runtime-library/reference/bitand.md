# <a name="bitand"></a>bitand

Zamiast & operatora.

## <a name="syntax"></a>Składnia

```C

#define bitand &

```

## <a name="remarks"></a>Uwagi

Makro daje operator

## <a name="example"></a>Przykład

```cpp
// iso646_bitand.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 1, b = 2, result;

   result = a & b;
   cout << result << endl;

   result= a bitand b;
   cout << result << endl;
}
```

```Output
0
0
```
