# <a name="xor_eq"></a>xor_eq

Alternatywa dla operatora ^=.

## <a name="syntax"></a>Składnia

```C

#define xor_eq ^=
```

## <a name="remarks"></a>Uwagi

Makro daje operator ^=.

## <a name="example"></a>Przykład

```cpp
// iso646_xor_eq.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a ^= b;
   cout << result << endl;

   a = 3;
   b = 2;

   result= a xor_eq b;
   cout << result << endl;
}
```

```Output
1
1
```
