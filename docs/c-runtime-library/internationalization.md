# <a name="internationalization"></a>Internacjonalizacja

Biblioteka środowiska uruchomieniowego Microsoft udostępnia wiele procedur, które są przydatne do tworzenia różnych wersji programu na rynki międzynarodowe. Obejmuje to [procedury związane z ustawieniami regionalnymi](../c-runtime-library/locale.md), procedury o szerokim znaku, procedury znaków wielobajtowych i procedury tekstu ogólnego. Dla wygody, większość procedur związanych z ustawieniami regionalnymi również została ujęta wśród kategorii tego dokumentu odniesienia, zgodnie z operacjami, które wykonują. W tej sekcji, w kolejności alfabetycznej, opisano procedury obsługi znaków wielobajtowych i dwubajtowych, wraz z odpowiednikami dla znaków jednobajtowych, jeśli takie istnieją.

Omówiono też alternatywy operatora ISO646.
