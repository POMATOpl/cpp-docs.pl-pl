# <a name="exit_success-exit_failure"></a>EXIT_SUCCESS, EXIT_FAILURE

## <a name="required-header"></a>Wymagany nagłówek

```c
#include <stdlib.h>
```

## <a name="remarks"></a>Uwagi

Są to argumenty dla funkcji [Exit](reference/exit-exit-exit.md) i [_exit](reference/exit-exit-exit.md) oraz wartości zwracane dla funkcji [atexit —](reference/atexit.md) i [_onexit](reference/onexit-onexit-m.md) .

|Stała|Zdefiniowana wartość|
|-|-|
|EXIT_SUCCESS|0|
|EXIT_FAILURE|1|
