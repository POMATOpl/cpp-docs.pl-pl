# <a name="iob"></a>_iob
Tablica stdio — struktury sterujące.  
  
## <a name="syntax"></a>Składnia  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## <a name="remarks"></a>Uwagi  
 `IOB_ENTRIES` nie zdefiniowano jako 20 w stdio.h.
