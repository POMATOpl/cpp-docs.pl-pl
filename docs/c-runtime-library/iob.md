# <a name="iob"></a>_iob

Tablica stdio — struktur sterujących.

## <a name="syntax"></a>Składnia

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>Uwagi

`IOB_ENTRIES` jest zdefiniowany jako 20 w stdio.h.
