# <a name="mb_cur_max"></a>MB_CUR_MAX

Makro, które wskazuje maksymalną liczbę bajtów w znaku wielobajtowego dla bieżących ustawień regionalnych.

## <a name="syntax"></a>Składnia

```
#include <stdlib.h>
```

## <a name="remarks"></a>Uwagi

Kontekst: funkcje konwersji wielobajtowych ANSI i szerokich znaków

Wartość `MB_CUR_MAX` jest maksymalną liczbą bajtów w znaku wielobajtowym dla bieżących ustawień regionalnych.
