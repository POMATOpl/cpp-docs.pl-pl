# <a name="mbcurmax"></a>MB_CUR_MAX

Makra, która wskazuje maksymalną liczbę bajtów w znaków wielobajtowych dla bieżących ustawień regionalnych.

## <a name="syntax"></a>Składnia

`#include <stdlib.h>`

## <a name="remarks"></a>Uwagi

Kontekstu: Funkcje konwersji znaków wielobajtowych i całej ANSI

Wartość `MB_CUR_MAX` jest maksymalna liczba bajtów w znaków wielobajtowych dla bieżących ustawień regionalnych.
