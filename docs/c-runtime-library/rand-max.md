# <a name="randmax"></a>RAND_MAX
## <a name="syntax"></a>Składnia  
  
```  
  
#include <stdlib.h>  
  
```  
  
## <a name="remarks"></a>Uwagi  
 Stała `RAND_MAX` maksymalna wartość, która może być zwracany przez `rand` funkcji. `RAND_MAX` nie zdefiniowano jako wartość 0x7fff.
