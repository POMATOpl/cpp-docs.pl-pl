# <a name="randmax"></a>RAND_MAX

## <a name="syntax"></a>Składnia

```

#include <stdlib.h>
```

## <a name="remarks"></a>Uwagi

Stała `RAND_MAX` maksymalna wartość, które mogą być zwrócone przez `rand` funkcji. `RAND_MAX` jest zdefiniowany jako wartość 0x7fff.
