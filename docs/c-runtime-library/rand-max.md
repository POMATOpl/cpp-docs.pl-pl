# <a name="rand_max"></a>RAND_MAX

## <a name="syntax"></a>Składnia

```
#include <stdlib.h>
```

## <a name="remarks"></a>Uwagi

Stała `RAND_MAX` jest maksymalną wartością, którą można zwrócić przez `rand` funkcję. `RAND_MAX` jest definiowana jako wartość 0x7FFF.
