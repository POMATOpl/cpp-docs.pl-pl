# <a name="operator-newcrt"></a>operator new(CRT)

Począwszy od Visual Studio 2013, uniwersalne środowisko uruchomieniowe C (UCRT) nie obsługuje już operatora specyficznego dla języka C++ i operatora delete. Są one teraz częścią standardowej biblioteki języka C++. Aby uzyskać więcej informacji, zobacz temat [New and DELETE Operators](../cpp/new-and-delete-operators.md) and [New Operator](../cpp/new-operator-cpp.md) in C++ Language Reference.
