# <a name="operator-newcrt"></a>operator new(CRT)

Począwszy od Visual Studio 2013, uniwersalne środowisko uruchomieniowe języka C (UCRT) nie obsługuje C++już operatorów specyficznych dla operatora new i operator delete. Są one teraz częścią biblioteki C++ standardowej. Aby uzyskać więcej informacji, zobacz temat [New and DELETE Operators](../cpp/new-and-delete-operators.md) and [New Operator](../cpp/new-operator-cpp.md) in Reference C++ Language.
