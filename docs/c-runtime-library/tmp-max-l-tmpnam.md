```

```

## <a name="remarks"></a>Uwagi

`TMP_MAX` jest maksymalną liczbę unikatowych nazw plików, `tmpnam` może wygenerować funkcji. `L_tmpnam` to długość nazwy plików tymczasowych, generowane przez `tmpnam`.
