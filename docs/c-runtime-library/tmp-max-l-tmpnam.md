```

```

## <a name="remarks"></a>Uwagi

`TMP_MAX` to maksymalna liczba unikatowych nazw plików, które `tmpnam` może wygenerować funkcja. `L_tmpnam` jest długością tymczasowych nazw plików generowanych przez `tmpnam` .
