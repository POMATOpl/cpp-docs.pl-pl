# <a name="redistributing-database-support-files"></a>Redystrybuowanie plików obsługi baz danych
Przeprowadzać redystrybucję plików obsługi dla obiektów Data Access (DAO) oraz technologie bazy danych w programie Microsoft Data Access SDK.  
  
## <a name="installing-dao-support-files"></a>Instalowanie plików obsługi DAO  
 Aby uzyskać najnowszą wersję programu DAO, zobacz [artykuł 239114: jak uzyskać najnowszy dodatek service pack dla aparatu bazy danych Microsoft Jet 4.0](http://go.microsoft.com/fwlink/p/?linkid=198014) w witrynie Microsoft Support.  
  
## <a name="installing-microsoft-data-access-sdk-support-files"></a>Instalowanie plików pomocy technicznej zestaw SDK dostępu do danych firmy Microsoft  
 Aby zainstalować obsługę ODBC, OLE DB, ActiveX Data Objects (ADO) i usług danych zdalnych (RDS), należy użyć Mdac_typ.exe. Mdac_typ.exe znajduje się w.. Folder \WCU\MDAC28\ na nośniku instalacyjnym programu Visual Studio. Możesz również pobrać Mdac_typ.exe z [Microsoft Download Center](http://go.microsoft.com/fwlink/p/?linkid=198015) witryny sieci Web.  
  
 Aby uzyskać więcej informacji na [MSDN](http://go.microsoft.com/fwlink/p/?linkid=198016) witryny sieci Web, wyszukaj "Redystrybucja MDAC 2.8 SP1". Jeśli używasz projektów Instalatora Visual Studio do wdrażania aplikacji, zobacz [wdrożenia i zależności](https://msdn.microsoft.com/49e9b84d-bd6a-4388-b9ac-46ea79cf0733).
