# <a name="midl-property-pages-output"></a>Strony właściwości MIDL: Dane wyjściowe

**Dane wyjściowe** — strona właściwości w **MIDL** folderu określa następujące opcje kompilatora MIDL:

- Katalogu wyjściowego ([/out](https://msdn.microsoft.com/library/windows/desktop/aa367358))

- Plik nagłówkowy ([/h](https://msdn.microsoft.com/library/windows/desktop/aa367325))

- Plik danych biblioteki DLL ([/dlldata](https://msdn.microsoft.com/library/windows/desktop/aa367322))

- Plik IID ([/iid](https://msdn.microsoft.com/library/windows/desktop/aa367329))

- Plik serwera proxy ([/proxy](https://msdn.microsoft.com/library/windows/desktop/aa367362))

- Generuj bibliotekę typów ([notlb](https://msdn.microsoft.com/library/windows/desktop/aa367343))

- Biblioteka typów ([/TLB](https://msdn.microsoft.com/library/windows/desktop/aa367372))

Aby uzyskać informacje dotyczące uzyskiwania dostępu do **dane wyjściowe** — strona właściwości w **MIDL** folderów, zobacz [Praca z właściwościami projektu](../ide/working-with-project-properties.md).

Aby uzyskać informacje o tym, jak programowo uzyskać dostęp do MIDL opcji dla projektów języka C++, zobacz <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool>.
