# <a name="midl-property-pages-output"></a>Strony właściwości MIDL: wyjściowe
**Dane wyjściowe** stronę właściwości w **MIDL** folderu określa następujące opcje kompilatora MIDL:  
  
-   Katalog wyjściowy ([/out](http://msdn.microsoft.com/library/windows/desktop/aa367358))  
  
-   Plik nagłówka ([/h](http://msdn.microsoft.com/library/windows/desktop/aa367325))  
  
-   Plik danych biblioteki DLL ([/dlldata](http://msdn.microsoft.com/library/windows/desktop/aa367322))  
  
-   Plik IID ([/iid](http://msdn.microsoft.com/library/windows/desktop/aa367329))  
  
-   Plik serwera proxy ([/proxy](http://msdn.microsoft.com/library/windows/desktop/aa367362))  
  
-   Generuj bibliotekę typów ([notlb](http://msdn.microsoft.com/library/windows/desktop/aa367343))  
  
-   Biblioteki typów ([/TLB](http://msdn.microsoft.com/library/windows/desktop/aa367372))  
  
 Aby uzyskać informacje dotyczące dostępu do **dane wyjściowe** stronę właściwości w **MIDL** folderu, zobacz [Praca z właściwościami projektu](../ide/working-with-project-properties.md).  
  
 Aby uzyskać informacje dotyczące sposobu uzyskania programowego dostępu do MIDL opcje dla projektów C++, zobacz <xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool>.
