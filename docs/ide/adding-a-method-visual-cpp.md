# <a name="adding-a-method--visual-c"></a>Dodawanie metody (Visual C++)
Można użyć [Kreator dodawania metody](../ide/add-method-wizard.md) dodanie metody interfejsu w projekcie. Jeśli projekt zawiera klasę skojarzoną z interfejsem, Kreator modyfikuje klasy, zbyt.  
  
### <a name="to-add-a-method-to-your-object"></a>Aby dodać metody do obiektu  
  
1.  W widoku klas rozwiń węzeł projektu, aby wyświetlić interfejsu, do której chcesz dodać metody.  
  
    > [!NOTE]
    >  Można również dodać metody do dispinterfaces, który, chyba że projekt jest przypisane, znajdują się w węźle biblioteki.  
  
2.  Kliknij prawym przyciskiem myszy nazwę interfejsu.  
  
3.  W menu skrótów kliknij **Dodaj**, a następnie kliknij przycisk **Dodaj metodę**.  
  
4.  W Kreatorze dodawania metody Podaj informacje, aby utworzyć metody.  
  
5.  Określ jakiekolwiek ustawienia języka definicji interfejsu dla metody w [atrybuty IDL](../ide/idl-attributes-add-method-wizard.md) stronie kreatora.  
  
6.  Kliknij przycisk **Zakończ** można dodać metody.
