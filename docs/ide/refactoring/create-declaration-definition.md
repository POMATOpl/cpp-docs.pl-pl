# <a name="create-declaration--definition"></a>Utwórz deklarację/definicję

**Co:** Umożliwia natychmiastowe wygenerowanie deklaracji lub definicji funkcji.

**Kiedy:** Masz funkcję, która wymaga deklaracji lub na odwrót.

**Dlaczego:** Można ręcznie utworzyć deklarację/definicję, ale spowoduje to automatyczne utworzenie pliku nagłówka/kodu, jeśli jest to wymagane.

**Jaka**

1. Umieść kursor tekstowy lub wskaźnik myszy nad funkcją, dla której chcesz utworzyć deklarację lub definicję.

   ![Wyróżniony kod](images/createdefinition_highlight.png)

1. Następnie wykonaj jedną z następujących czynności:
   * **Klawiatura**
     * Naciśnij **klawisze CTRL +.** Aby wyzwolić menu **szybkie akcje i refaktoryzacje** , a następnie wybierz pozycję **Utwórz deklarację/definicję** z menu kontekstowego.
   * **Mysz**
     * Kliknij prawym przyciskiem myszy i wybierz menu **szybkie akcje i refaktoryzacje** i wybierz polecenie **Utwórz deklarację/definicję** z menu kontekstowego.

1. Deklaracja/definicja funkcji zostanie utworzona w pliku źródłowym lub nagłówkowym, który będzie widoczny w oknie podglądu okienka podręcznego.  Jeśli plik źródłowy lub nagłówek nie istnieje, również zostanie utworzony i umieszczony w projekcie.

   ![Utwórz wynik deklaracji/definicji](images/createdefinition_result.png)
