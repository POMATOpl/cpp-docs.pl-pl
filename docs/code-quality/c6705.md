# <a name="c6705"></a>C6705

> Ostrzeżenie C6705: Oczekiwano błędu adnotacji <expected_number> argumentów dla adnotacji \<parameter> znalezionych <actual_number>.

To ostrzeżenie jest zgłaszane w przypadku błędu w adnotacjach.
