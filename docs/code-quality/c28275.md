# <a name="c28275"></a>C28275

> Ostrzeżenie C28275: parametr do \_ wartości makra \_ \_ ma wartość null

To ostrzeżenie wskazuje, że w pliku modelu występuje błąd wewnętrzny, a nie w analizowanym kodzie. Funkcja *macroValue* została wywołana bez parametru.
