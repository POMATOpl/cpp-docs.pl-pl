# <a name="c28214"></a>C28214

> Ostrzeżenie C28214: nazwy parametrów atrybutu muszą być P1... P9

To ostrzeżenie wskazuje, że podczas konstruowania deklaracji adnotacji nazwy parametrów są ograniczone do P1... P9. To ostrzeżenie nigdy nie powinno wystąpić w normalnych warunkach użytkowania.
