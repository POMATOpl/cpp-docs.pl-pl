# <a name="c1252"></a>C1252

> Ostrzeżenie C1252: cykliczna lub Brakująca zależność między wtyczkami: wymaga identyfikatora GUID

Narzędzie do analizy kodu zgłasza to ostrzeżenie w przypadku błędu wewnętrznego z zależnościami wtyczki, a nie w analizowanym kodzie.
