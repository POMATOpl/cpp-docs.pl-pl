# <a name="c28350"></a>C28350

> Ostrzeżenie C28350: adnotacja \<annotation> opisuje sytuację, która nie jest warunkowo stosowana.

Zwykle to ostrzeżenie jest generowane, gdy jest stosowana Adnotacja, w której jest przeprowadzana inspekcja typu C/C++.
