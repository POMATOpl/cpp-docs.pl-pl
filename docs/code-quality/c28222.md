# <a name="c28222"></a>C28222

> Ostrzeżenie C28222: \_Yes\_, \_nie\_lub \_mogą\_ oczekiwane dla adnotacji

To ostrzeżenie wskazuje, że parametr do adnotacji powinien być jednym z symboli `_Yes_`, `_No_`lub `_Maybe_`i został napotkany inny symbol. Zwykle oznacza to nieprawidłowe kodowane makro adnotacji.
