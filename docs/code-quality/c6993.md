# <a name="c6993"></a>C6993

> Ostrzeżenie C6993: Analiza kodu ignoruje Konstrukcje OpenMP; Analizowanie kodu jednowątkowego

To ostrzeżenie wskazuje, że analizator kodu napotkał pragmy Open MP, których nie może przeanalizować.
