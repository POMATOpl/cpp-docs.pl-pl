# <a name="c28196"></a>C28196

> Ostrzeżenie C28196: wymagania nie są spełnione. (Wyrażenie nie jest szacowane na wartość true).

To ostrzeżenie wskazuje, że prototyp funkcji dla analizowanej funkcji ma `__notnull` `__null` parametr, lub `__drv_valueIs` na `_Out_` wartość zwracaną, ale zwrócona wartość jest niespójna z tą adnotacją.
