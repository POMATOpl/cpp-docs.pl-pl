# <a name="c28283"></a>C28283

> Ostrzeżenie C28283: dla symbolu, określona specyfikacja rozmiaru nie jest jeszcze obsługiwana

Ostrzeżenie wskazuje, że adnotacja o rozmiarze "wskaźnik" otrzymała wartość inną niż zero. Zasadniczo obiekt wywołujący próbował powiedzieć, że ciąg jest zakończony znakiem innym niż Binary zero.
