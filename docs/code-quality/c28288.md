# <a name="c28288"></a>C28288

> Ostrzeżenie C28288: dla funkcji, błąd składniowy w \_pod\_() adnotacji (Nieprawidłowa nazwa parametru)

Narzędzie analizy kodu zgłasza to ostrzeżenie, gdy zostanie użyta adnotacja `SAL_at` (`__drv_at`) i wyrażenie parametru nie może być interpretowane w bieżącym kontekście. Może to obejmować użycie błędnej pisowni parametru lub nazwy elementu członkowskiego, lub błędnego pisowni słów kluczowych "return" lub "This".
