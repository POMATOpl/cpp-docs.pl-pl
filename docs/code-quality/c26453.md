# <a name="warning-c26453-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>Ostrzeżenie C26453: Przepełnienie arytmetyczne: przesunięcie w lewo ujemnej podpisanej liczby jest zachowaniem nieokreślonym

To ostrzeżenie wskazuje, że jesteśmy w lewo przesunięcie ujemnej podpisanej wartości integralnej, co jest złym pomysłem i wyzwala zachowanie zdefiniowane w implementacji.

## <a name="example"></a>Przykład

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

Aby poprawić to ostrzeżenie, użyj następującego kodu:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
