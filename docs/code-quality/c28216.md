# <a name="c28216"></a>C28216

> Ostrzeżenie C28216: \_ klauzula Check \_ Return \_ dotyczy tylko warunków post dla określonego parametru funkcji.

`_Check_return_`Adnotacja została zastosowana w kontekście innym niż `__post` ; może być potrzebny `__post` modyfikator (or `__drv_out` ) lub może być nieprawidłowo umieszczona.
