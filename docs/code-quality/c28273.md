# <a name="c28273"></a>C28273

> Ostrzeżenie C28273: dla funkcji, wskazówki są niespójne z deklaracją funkcji

To ostrzeżenie wskazuje błąd w adnotacjach, a nie w analizowanym kodzie. Adnotacje wyświetlane w definicji funkcji są niespójne z tymi, które występują w deklaracji. Dwie adnotacje powinny zostać rozpoznane jako zgodne.
