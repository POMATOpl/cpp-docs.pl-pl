# <a name="c28286"></a>C28286

> Ostrzeżenie C28286: dla funkcji, błąd składniowy blisko końca

Narzędzie do analizy kodu zgłasza to ostrzeżenie, jeśli w pliku modelu wystąpi prawdopodobny błąd. Niektóre błędy plików źródłowych mogą również spowodować takie błędy.
