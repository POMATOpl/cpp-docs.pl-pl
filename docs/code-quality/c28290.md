# <a name="c28290"></a>C28290

> Ostrzeżenie C28290: adnotacja dla funkcji zawiera więcej elementów zewnętrznych niż rzeczywista liczba parametrów

Narzędzie analizy kodu zgłasza to ostrzeżenie, gdy adnotacja dla funkcji zawiera więcej elementów zewnętrznych niż rzeczywista liczba parametrów.
