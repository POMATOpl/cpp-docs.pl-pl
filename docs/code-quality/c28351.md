# <a name="c28351"></a>C28351

> Ostrzeżenie C28351: Adnotacja, \<Adnotacja > opisuje, gdzie w warunku nie można użyć wartości dynamicznej (zmiennej).

To ostrzeżenie jest zgłaszane, gdy jest stosowana Adnotacja, w którejC++ jest przeprowadzana inspekcja C/typ.

Wyrażenie w `_When_` powinno być oceniane jako stała. `_When_` jest ignorowana.
