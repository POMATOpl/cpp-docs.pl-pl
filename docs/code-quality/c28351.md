# <a name="c28351"></a>C28351

> Ostrzeżenie C28351: adnotacja \<annotation> opisuje, gdzie wartość dynamiczna (zmienna) nie może być użyta w warunku.

To ostrzeżenie jest zgłaszane, gdy jest stosowana Adnotacja, w której jest przeprowadzana inspekcja typu C/C++.

Wyrażenie, `_When_` które ma zostać obliczone, jest stałe. Wartość `_When_` jest ignorowana.
