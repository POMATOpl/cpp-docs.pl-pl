# <a name="c28301"></a>C28301

> Ostrzeżenie C28301: brak adnotacji dla pierwszej deklaracji \<function> .\<note1> Zobacz \<filename> ( \<line> ). \<note2>

To ostrzeżenie jest zgłaszane, gdy nie znaleziono adnotacji w pierwszej deklaracji danej funkcji.
