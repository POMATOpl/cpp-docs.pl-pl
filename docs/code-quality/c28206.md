# <a name="c28206"></a>C28206

> Ostrzeżenie C28206: \<expression> : Lewy argument operacji wskazuje na strukturę, użyj `->`

To ostrzeżenie jest zgłaszane, gdy oczekiwano notacji odwołania wskaźnika struktury `->` .
