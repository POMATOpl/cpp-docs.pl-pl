# <a name="c28211"></a>C28211

> Ostrzeżenie C28211: Oczekiwano nazwy kontekstu statycznego dla SAL_context

To ostrzeżenie wskazuje, że operand do `_Static_context_` adnotacji musi być nazwą kontekstu zdefiniowanego przez narzędzie. To ostrzeżenie nigdy nie powinno wystąpić w normalnych warunkach użytkowania.
