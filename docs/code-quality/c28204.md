# <a name="c28204"></a>C28204

> Ostrzeżenie C28204: \<function> : tylko jedno z tych przeciążeń i jeden w \<filename> ( \<line> ) jest adnotacją dla \<paramname> : oba lub nie muszą mieć adnotacji.

To ostrzeżenie jest zgłaszane w przypadku błędu w adnotacjach.
