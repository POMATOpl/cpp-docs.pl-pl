# <a name="c28232"></a>C28232

> Ostrzeżenie C28232: \_ pre \_ , \_ post \_ lub \_ DEREF \_ nie została zastosowana do żadnej adnotacji

To ostrzeżenie wskazuje, że `_Pre_` `_Post_` operator, lub `_Deref_` jest wyświetlany w wyrażeniu adnotacji bez późniejszej adnotacji funkcjonalnej; modyfikator został zignorowany, ale wskazuje nieprawidłowo zakodowaną adnotację.
