# <a name="c28232"></a>C28232

> Ostrzeżenie C28232: \_pre\_, \_post\_lub \_DEREF\_ nie zostały zastosowane do żadnej adnotacji

To ostrzeżenie wskazuje, że operator `_Pre_`, `_Post_`lub `_Deref_` pojawia się w wyrażeniu adnotacji bez późniejszej adnotacji funkcjonalnej; modyfikator został zignorowany, ale wskazuje nieprawidłowo zakodowaną adnotację.
