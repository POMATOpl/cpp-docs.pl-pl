# <a name="c28245"></a>C28245

> Ostrzeżenie C28245: adnotacja dla funkcji zawiera adnotacje "This" dla funkcji nie będącej składową

Wystąpił błąd wewnętrzny w pliku modelu PREfast. To ostrzeżenie nie powinno występować w typowym użyciu.
