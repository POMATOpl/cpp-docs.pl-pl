# <a name="c28219"></a>C28219

> Ostrzeżenie C28219: oczekiwano elementu członkowskiego wyliczenia dla adnotacji, parametr w adnotacji

Parametr do adnotacji powinien być elementem członkowskim nazwanego **`enum`** typu i został napotkany inny symbol; Użyj elementu członkowskiego tego **`enum`** typu. Zwykle oznacza to nieprawidłowe kodowane makro adnotacji.
