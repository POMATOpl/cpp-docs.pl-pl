# <a name="c28160"></a>C28160

> Ostrzeżenie C28160: błąd adnotacji

To ostrzeżenie jest zgłaszane, gdy zostanie `__drv_error` napotkana adnotacja. Ta adnotacja służy do flagi, które powinny być naprawione, i może być używana z `__drv_when` adnotacją, aby wskazać konkretne kombinacje parametrów.
