# <a name="c28209"></a>C28209

> Ostrzeżenie C28209: Deklaracja symbolu ma sprzeczną deklarację

To ostrzeżenie wskazuje nieprawidłowo skonstruowaną deklarację adnotacji. To ostrzeżenie nigdy nie powinno wystąpić w normalnych warunkach użytkowania.
