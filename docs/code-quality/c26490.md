# <a name="c26490-no_reinterpret_cast"></a>C26490 NO_REINTERPRET_CAST

Nie używaj `reinterpret_cast` . 
## <a name="see-also"></a>Zobacz też 
[Typ podstawowe wytyczne dotyczące języka C++. 1](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).

## <a name="example"></a>Przykład
```cpp
void function(void* ptr)
{
    std::size_t val = reinterpret_cast<std::size_t>(ptr); // C26490, Don't use reinterpret_cast
}
```
