# <a name="c28221"></a>C28221

> Ostrzeżenie C28221: oczekiwano wyrażenia ciągu dla parametru w adnotacji

To ostrzeżenie wskazuje, że parametr do adnotacji powinien być ciągiem, a został napotkany inny typ. Zwykle wskazuje to niepoprawnie zakodowane makro adnotacji i nie oczekuje się, że jest ono widoczne w normalnych warunkach użytkowania.
