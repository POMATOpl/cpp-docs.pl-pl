# <a name="c28021"></a>C28021

> Ostrzeżenie C28021: parametr \<param> z adnotacją \<anno> musi być wskaźnikiem

To ostrzeżenie jest zgłaszane, gdy obiekt z adnotacjami nie jest typem wskaźnika. Tej adnotacji nie można używać z **`void`** typami całkowitymi ani.
