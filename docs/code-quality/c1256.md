# <a name="c1256"></a>C1256

> Ostrzeżenie C1256: PCH musi mieć zarówno zapis, jak i odczyt.

Narzędzie analizy kodu zgłasza to ostrzeżenie, gdy w narzędziu wystąpi błąd wewnętrzny, a nie w analizowanym kodzie.
