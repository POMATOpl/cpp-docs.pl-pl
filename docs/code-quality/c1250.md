# <a name="c1250"></a>C1250

> Ostrzeżenie C1250: nie można załadować wtyczki.

Narzędzie analizy kodu raportuje to ostrzeżenie, gdy w wtyczki wystąpi błąd wewnętrzny, a nie w analizowanym kodzie.
