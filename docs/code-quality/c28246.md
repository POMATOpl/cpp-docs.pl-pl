# <a name="c28246"></a>C28246

> Ostrzeżenie C28246: adnotacja dla funkcji " \<name> " — parametr " \<parameter> " nie jest zgodna z typem parametru

Operator __deref został zastosowany do typu niebędącego wskaźnikiem podczas tworzenia adnotacji.
