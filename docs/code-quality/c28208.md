# <a name="c28208"></a>C28208

> Ostrzeżenie C28208: funkcja \<funkcji > została wcześniej zdefiniowana z inną listą parametrów w \<pliku > (\<wiersz >). Niektóre narzędzia analityczne spowodują nieprawidłowe wyniki

To ostrzeżenie jest zgłaszane, gdy znana definicja funkcji nie jest zgodna z innym wystąpieniem.
