# <a name="c28310"></a>C28310

> Ostrzeżenie C28310: < annotation_name > Adnotacja w funkcji \<> \<parametru > nie ma wersji SAL.

Wszystkie adnotacje SAL używane w kodzie źródłowym powinny mieć wersję adnotacji stosowaną przy użyciu SAL_name. Należy poprawić to w definicji makra.

To ostrzeżenie jest zgłaszane tylko raz na deklarację. Sprawdź pozostałą część deklaracji pod kątem przestarzałego SAL.
