# <a name="game-development-with-c"></a>Programowanie gier w języku C++

Tworząc grę dla systemu Windows 10, masz możliwość dotarcia do milionów graczy na całym świecie na całym telefonie, komputerze i konsoli Xbox One. Dzięki konsoli Xbox w systemie Windows, usłudze Xbox Live, wieloosobowej grze wieloosobowej na różnych urządzeniach, niesamowitej społeczności graczy i nowym, potężnym funkcjom, takim jak uniwersalna platforma systemu Windows (UWP) i DirectX 12, gry z systemem Windows 10 ekscytują graczy w każdym wieku i gatunku. Nowa platforma uniwersalna systemu Windows (UWP) zapewnia zgodność gry na różnych urządzeniach z systemem Windows 10 ze wspólnym interfejsem API dla telefonów, komputerów i konsoli Xbox One, a także narzędziami i opcjami dostosowanymi do każdego urządzenia.

Tworzenie gier jest udokumentowane w [Centrum deweloperów systemu Windows](https://docs.microsoft.com/windows/uwp/gaming/getting-started).
