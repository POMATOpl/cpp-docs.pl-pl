# <a name="support-for-ansi"></a>Obsługa standardu ANSI
Większość metod i klas MFC obsługuje zestawu znaków ANSI, mimo że struktura MFC jako całość rozwija się stopniowo do obsługi tylko Unicode zestaw znaków. Trwającą ulepszeniom Windows Vista i Windows wspólnych formantów w wersji 6.1 obsługę kilku metod i klas ANSI jest przestarzały.  Aby uzyskać więcej informacji, zobacz [przestarzałe interfejsy API ANSI](../mfc/deprecated-ansi-apis.md) i [Obsługa formatu Unicode](../text/support-for-unicode.md).
