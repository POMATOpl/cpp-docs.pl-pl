# <a name="support-for-ansi"></a>Obsługa standardu ANSI

Większość klas MFC i metod obsługuje zestaw znaków ANSI, chociaż platforma MFC jako całość jest stopniowo rozwijana w kierunku obsługi tylko zestawu znaków Unicode. Ze względu na bieżące ulepszenia systemu Windows Vista i wspólnych kontrolek systemu Windows w wersji 6,1, Obsługa kilku klas i metod ANSI jest przestarzała.  Aby uzyskać więcej informacji, zobacz [przestarzałe interfejsy API ANSI](../mfc/deprecated-ansi-apis.md) i [Obsługa standardu Unicode](../text/support-for-unicode.md).
