# <a name="atl-registry-component-registrar"></a>Składnik rejestru ALT (Rejestrator)

Rejestrator ALT udostępnia zoptymalizowane do rejestru systemowego za pomocą niestandardowego interfejsu. Rejestrator jest bezwątkowy i umożliwia łączenie statyczne kodu dla klientów w języku C++.

> [!NOTE]
>  Kod źródłowy Rejestrator ALT znajdują się w atlmfc\include\atliface.h.
