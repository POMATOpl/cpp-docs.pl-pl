# <a name="data-transfer-classes"></a>Klasy Transfer danych

Poniższe klasy obsługują różne typy transferów danych:

- [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) Program obsługuje Uniform Data Transfer przy użyciu standardowych formatów pobierania i ustawiania danych. Obsługuje powiadomienia o zmianach danych przez Zarządzanie połączeniami w celu doradzania ujściami.

- [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) Zezwala asynchronicznej monikerowi na wysyłanie i odbieranie informacji o asynchronicznym przekazywaniu danych do i z obiektu.
