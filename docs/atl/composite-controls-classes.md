# <a name="composite-controls-classes"></a>Klasy formantów złożonych

Następująca Klasa zapewnia obsługę tworzenia kontrolek złożonych

- [CComCompositeControl](../atl/reference/ccomcompositecontrol-class.md) Formanty ActiveX pochodne z `CComCompositeControl` są hostowane przez standardowe okno dialogowe. Te typy kontrolek są nazywane kontrolkami złożonymi, ponieważ mogą hostować inne kontrolki (natywne formanty systemu Windows i kontrolki ActiveX).
