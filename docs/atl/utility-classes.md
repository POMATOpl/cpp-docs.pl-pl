# <a name="utility-classes"></a>Klasy narzędzi
Dostępne są następujące klasy MFC niezależny narzędzie:  
  
-   [CImage](../atl-mfc-shared/reference/cimage-class.md) zapewnia Rozbudowana obsługa mapy bitowej, łącznie z możliwością przy ładowaniu i zapisywaniu obrazy JPEG, GIF, BMP, i formatuje Portable Network Graphics (PNG).  
  
-   [CPoint](../atl-mfc-shared/reference/cpoint-class.md) udostępnia implementację na potrzeby przechowywania par współrzędne (x, y).  
  
-   [CRect](../atl-mfc-shared/reference/crect-class.md) udostępnia implementację na potrzeby przechowywania współrzędne obszary prostokątne.  
  
-   [CSize](../atl-mfc-shared/reference/csize-class.md) udostępnia implementację na potrzeby przechowywania odległość, położenia lub par wartości.  
  
-   [Cstring —](../atl-mfc-shared/reference/cstringt-class.md) udostępnia implementację na potrzeby przechowywania ciągów znaków.  
  
-   [CAdapt](../atl/reference/cadapt-class.md) prostego szablonu stosowanego do opakowywania klasy, które ponownie zdefiniować operator address-of.  
  
-   [_U_RECT](../atl/reference/u-rect-class.md) klasę karty argumentu, która umożliwia albo `RECT` wskaźniki lub odwołania do przekazania do funkcji, która jest zaimplementowana w postaci wskaźników.
