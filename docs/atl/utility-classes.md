# <a name="utility-classes"></a>Klasy narzędzi

Dostępne są następujące klasy narzędzi niezależnych od MFC:

- [Funkcji CImage](../atl-mfc-shared/reference/cimage-class.md) Zapewnia obsługę ulepszonej mapy bitowej, w tym możliwość ładowania i zapisywania obrazów w formatach JPEG, GIF, BMP i Portable Network Graphics (PNG).

- [CPoint](../atl-mfc-shared/reference/cpoint-class.md) Zawiera implementację do przechowywania par współrzędnych (x, y).

- [CRect](../atl-mfc-shared/reference/crect-class.md) Zapewnia implementację do przechowywania współrzędnych prostokątnych obszarów.

- [CSize](../atl-mfc-shared/reference/csize-class.md) Zawiera implementację do przechowywania odległości, względne położenia lub par wartości.

- [CString](../atl-mfc-shared/reference/cstringt-class.md) Zawiera implementację do przechowywania ciągów znaków.

- [CAdapt](../atl/reference/cadapt-class.md) Prosty szablon służący do zawijania klas, które definiują operator address-of.

- [_U_RECT](../atl/reference/u-rect-class.md) Klasa adaptera argumentów, która umożliwia `RECT` przekazywanie wskaźników lub odwołań do funkcji, która jest zaimplementowana w wskaźnikach.
