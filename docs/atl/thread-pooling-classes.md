# <a name="thread-pooling-classes"></a>Klasy Pula wątków
Następujące klasy obsługuje korzystanie z puli wątków:  
  
-   [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) implementuje moduł EXE z obsługą wielu wątków w puli apartamentach.  
  
-   [CComApartment](../atl/reference/ccomapartment-class.md) zarządza typu apartment w module EXE puli wątków.  
  
-   [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) zarządza wyboru wątku dla modułu EXE.
