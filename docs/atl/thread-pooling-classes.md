# <a name="thread-pooling-classes"></a>Klasy puli wątków

Następujące klasy obsługują pule wątków:

- [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) Implementuje moduł EXE z obsługą wielu apartamentach z pulą wątków.

- [CComApartment](../atl/reference/ccomapartment-class.md) Zarządza komórką w module programu EXE w puli wątków.

- [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) Zarządza wyborem wątku dla modułu EXE.
