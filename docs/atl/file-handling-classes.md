# <a name="file-handling-classes"></a>Klasy obsługi plików

Klasy te zapewniają metody obsługi plików, plików tymczasowych i plików mapowanych na pamięć.

- [CAtlFile](../atl/reference/catlfile-class.md) Ta klasa udostępnia cienkią otokę wokół interfejsu API obsługi plików systemu Windows.

- [CAtlFileMapping](../atl/reference/catlfilemapping-class.md) Ta klasa reprezentuje plik mapowany w pamięci, dodając operator rzutowania do metod [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md).

- [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md) Ta klasa reprezentuje plik mapowany w pamięci.

- [CAtlTemporaryFile](../atl/reference/catltemporaryfile-class.md) Ta klasa zawiera metody tworzenia i używania pliku tymczasowego.
