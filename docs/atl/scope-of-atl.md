# <a name="scope-of-atl"></a>Zakres ATL

ATL pozwala łatwo tworzyć obiekty COM, serwery automatyzacji i kontrolki ActiveX. ATL oferuje wbudowaną obsługę wielu podstawowych interfejsów COM.

ATL jest dostarczany jako kod źródłowy, który jest dołączany do aplikacji. Ponadto ATL udostępnia bibliotekę DLL (atl90.dll), która zawiera kod, który może być współużytkowany przez składniki programu. Jednak ta biblioteka DLL nie jest konieczna.
