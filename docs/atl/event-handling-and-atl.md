# <a name="event-handling-and-atl"></a>Obsługa zdarzeń i ATL

Aby uzyskać informacje na temat sposobu uruchamiania zdarzeń i implementowania punktów połączenia, Odczytaj [punkty połączenia ATL](../atl/atl-connection-points.md).
