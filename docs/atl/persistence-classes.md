# <a name="persistence-classes"></a>Klasy trwałości

Następujące klasy implementują trwałość obiektu:

- [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) Umożliwia klientowi załadowanie i zapisanie właściwości obiektu w zbiorze właściwości.

- [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) Umożliwia klientowi załadowanie i zapisanie trwałych danych obiektu w strumieniu.

- [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) Umożliwia klientowi załadowanie i zapisanie trwałych danych obiektu w magazynie.
