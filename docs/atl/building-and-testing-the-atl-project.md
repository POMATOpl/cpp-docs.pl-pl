# <a name="building-and-testing-the-atl-project"></a>Kompilowanie i testowanie projektu ATL

Jak wspomniano w [wstawieniu kontrolki złożonej](../atl/inserting-a-composite-control.md), jeden z początkowych składników projektu jest domyślną stroną HTML, która hostuje nowy formant złożony. Po zakończeniu modyfikowania kontrolki złożonej kliknij pozycję **Kompiluj rozwiązanie** lub **Skompiluj ponownie rozwiązanie** w menu **kompilacja** . Po pomyślnym skompilowaniu projektu Załaduj stronę HTML, która znajduje się w katalogu głównym projektu kontrolki złożonej, w programie Internet Explorer lub w innej przeglądarce i przetestuj funkcje formantu.

Możesz również testować formant złożony przy użyciu narzędzia kontenera testowego lub innej aplikacji, która może hostować formant ActiveX. Zobacz [testowanie właściwości i zdarzeń za pomocą kontenera testów,](../mfc/testing-properties-and-events-with-test-container.md) Aby uzyskać informacje na temat uzyskiwania dostępu do kontenera testowego.
