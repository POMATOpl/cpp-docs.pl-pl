# <a name="atl-composite-control-fundamentals"></a>Podstawy złożonych kontrolek ALT

Formant złożony jest typem kontrolki ActiveX, która może zawierać (podobnie jak okno dialogowe) inne kontrolki ActiveX lub formanty systemu Windows. Po skompilowaniu złożonego formantu można go wstawić wszędzie tam, gdzie Kontrolka ActiveX może być hostowana.

Kreator projektu ATL i okno dialogowe **Dodawanie klasy** automatyzują proces tworzenia i wdrażania projektu kontrolki złożonej, podobnie jak w wyniku uruchomienia Kreatora aplikacji w celu utworzenia struktury aplikacji MFC. Proces opracowywania składa się z pięciu kroków:

- [Tworzenie projektu ATL](../atl/reference/creating-an-atl-project.md)

- [Wstawianie kontrolki złożonej](../atl/inserting-a-composite-control.md)

- [Modyfikowanie projektu ATL](../atl/modifying-the-atl-project.md)

- [Dodawanie funkcji do kontrolek złożonych](../atl/adding-functionality-to-the-composite-control.md)

- [Tworzenie i testowanie projektu ATL](../atl/building-and-testing-the-atl-project.md)
