# <a name="using-task-manager"></a>Korzystanie z Menedżera zadań

Jednym z najprostszych metod debugowania usługi jest użycie Menedżera zadań. Gdy usługa jest uruchomiona, uruchom Menedżera zadań, a następnie kliknij kartę **procesy** . Kliknij prawym przyciskiem myszy nazwę pliku EXE, a następnie kliknij polecenie **Debuguj**. Spowoduje to uruchomienie Visual C++ dołączone do tego uruchomionego procesu. Teraz kliknij przycisk **Przerwij** w menu **debugowanie** , aby umożliwić ustawianie punktów przerwania w kodzie. Kliknij przycisk **Uruchom** , aby uruchomić do wybranych punktów przerwania.
