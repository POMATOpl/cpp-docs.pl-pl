# <a name="stock-properties-atl-control-wizard"></a>Właściwości podstawowe, Kreator kontrolki ATL

Ta strona kreatora służy do identyfikowania właściwości podstawowych obsługiwanych przez formant. Domyślnie nie są identyfikowane żadne właściwości.

- **Nieobsługiwany**

   Wskazuje dostępne właściwości, które nie są obecnie obsługiwane dla formantu.

- **Obsługiwane**

   Wskazuje właściwości, które są obecnie obsługiwane dla formantu.

|Przycisk transferu|Opis|
|---------------------|-----------------|
|**>**|Dodaje do **obsługiwanej** listy nazwę właściwości aktualnie wybraną z listy **nieobsługiwane** .|
|**>>**|Dodaje do **obsługiwanej** listy wszystkie nazwy właściwości dostępne na liście **nieobsługiwane** .|
|**\<**|Usuwa nazwę właściwości aktualnie wybraną z listy **obsługiwanych** .|
|**\<\<**|Usuwa wszystkie nazwy właściwości aktualnie wymienione na liście **obsługiwanych** .|
