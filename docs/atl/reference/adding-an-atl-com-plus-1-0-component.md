# <a name="adding-an-atl-com-10-component"></a>Dodawanie składnika ATL COM+ 1,0

::: moniker range="vs-2019"

Kreator składnika ATL COM+ 1,0 nie jest dostępny w programie Visual Studio 2019 lub nowszym.

::: moniker-end

::: moniker range="<=vs-2017"

Ten Kreator dodaje do projektu obiekt, który obsługuje usługi modelu COM+ 1,0, w tym transakcje.

## <a name="to-add-an-atl-com-10-component-to-your-project"></a>Aby dodać składnik ATL COM+ 1,0 do projektu

1. W **Eksplorator rozwiązań** lub [Widok klasy](/visualstudio/ide/viewing-the-structure-of-code), kliknij prawym przyciskiem myszy nazwę projektu, do którego chcesz dodać składnik ATL COM+ 1,0.

1. W menu skrótów kliknij polecenie **Dodaj**, a następnie kliknij przycisk **Dodaj klasę**.

1. W oknie dialogowym [Dodawanie klasy](../../ide/add-class-dialog-box.md) w okienku **Szablony** kliknij węzeł **ATL COM+ 1,0 składnika**, a następnie kliknij przycisk **Otwórz** , aby wyświetlić [Kreatora składnika ATL COM+ 1,0](../../atl/reference/atl-com-plus-1-0-component-wizard.md).
