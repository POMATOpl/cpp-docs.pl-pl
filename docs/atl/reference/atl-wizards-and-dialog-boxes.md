# <a name="atl-wizards-and-dialog-boxes"></a>Kreatorzy ATL i okna dialogowe

Kreatory Active Template Library (ATL) generują kod standardowy dla różnych rodzajów obiektów COM. Kreatora można uruchomić, otwierając menu skrótów dla projektu w **Eksplorator rozwiązań** i wybierając polecenie **Dodaj** **klasy**.
