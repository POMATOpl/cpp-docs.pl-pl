# <a name="catldebuginterfacesmodule-class"></a>Klasa CAtlDebugInterfacesModule

Ta klasa zapewnia obsługę interfejsów debugowania.

## <a name="syntax"></a>Składnia

```cpp
class CAtlDebugInterfacesModule
```

## <a name="remarks"></a>Uwagi

`CAtlDebugInterfacesModule` zapewnia obsługę wymaganą do debugowania interfejsów. Jest ona uwzględniona w każdym projekcie, który definiuje [_ATL_DEBUG_QI](debugging-and-error-reporting-macros.md#_atl_debug_qi)symbolu.

## <a name="requirements"></a>Wymagania

**Nagłówek:** atlbase. h
