# <a name="atl-structures"></a>Struktury ATL


Biblioteki Active Template Library obejmuje następujące struktury.  
  
## <a name="in-this-section"></a>W tej sekcji  
 [ATL_DRAWINFO](../../atl/reference/atl-drawinfo-structure.md)  
 Zawiera informacje używane do renderowania do różnych celów, takich jak drukarki, metaplik lub formantu ActiveX.  
  
 [_AtlCreateWndData](../../atl/reference/atlcreatewnddata-structure.md)  
 Zawiera dane wystąpienia klasy w kodzie okien w ATL.  
  
 [_ATL_BASE_MODULE70](../../atl/reference/atl-base-module70-structure.md)  
 Używane przez dowolny projekt, który używa ATL.  
  
 [_ATL_COM_MODULE70](../../atl/reference/atl-com-module70-structure.md)  
 Używany przez kod związanych z modelu COM w ATL.  
  
 [_ATL_FUNC_INFO](../../atl/reference/atl-func-info-structure.md)  
 Zawiera informacje o typie opisuje metody lub właściwości w elemencie dispinterface.  
  
 [_ATL_MODULE70](../../atl/reference/atl-module70-structure.md)  
 Zawiera dane używane przez co moduł ATL.  
  
 [_ATL_WIN_MODULE70](../../atl/reference/atl-win-module70-structure.md)  
 Używany przez kod okien w ATL.
