# <a name="default-atl-project-configurations"></a>Domyślne konfiguracje projektu ATL

Kreator projektu ATL domyślnie tworzy dwie konfiguracje projektu:

|Konfiguracja|Zestaw znaków|Użycie ATL|
|-------------------|-------------------|----------------|
|Release|MBCS|DLL|
|Debugowanie|MBCS|DLL|

**Zestaw znaków**, **użycie ATL** i wszystko można zmienić w oknie dialogowym **Ustawienia projektu** na karcie **Ogólne** . Możesz również dodać własne konfiguracje przy użyciu Configuration Manager. Aby uzyskać szczegółowe informacje, zobacz [Konfiguracje kompilacji](/visualstudio/ide/understanding-build-configurations).
