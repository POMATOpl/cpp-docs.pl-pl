# <a name="default-atl-project-configurations"></a>Domyślne konfiguracje projektu ATL

Domyślnie Kreator projektów ATL tworzy dwie konfiguracje projektu:

|Konfiguracja|Zestaw znaków|Użycie ATL|
|-------------------|-------------------|----------------|
|Wydanie|MBCS|DLL|
|Debugowanie|MBCS|DLL|

**Zestaw znaków**, **użycie ATL** i można zmienić w **ustawienia projektu** , okno dialogowe **ogólne** kartę. Można również dodać własne konfiguracje za pomocą programu Configuration Manager. Aby uzyskać więcej informacji, zobacz [konfiguracji kompilacji](/visualstudio/ide/understanding-build-configurations).
