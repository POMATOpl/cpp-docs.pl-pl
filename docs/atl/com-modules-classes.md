# <a name="com-modules-classes"></a>Klasy modułów COM

Następujące klasy zapewniają obsługę modułu COM:

- [CAtlBaseModule](../atl/reference/catlbasemodule-class.md) Ta klasa jest tworzona w każdym projekcie ATL.

- [CAtlComModule](../atl/reference/catlcommodule-class.md) Ta klasa implementuje moduł serwera COM.

- [CAtlModule](../atl/reference/catlmodule-class.md) Ta klasa udostępnia metody używane przez kilka klas modułów ATL.

- [CAtlModuleT](../atl/reference/catlmodulet-class.md) Ta klasa implementuje moduł ATL.

- [CAtlExeModuleT](../atl/reference/catlexemodulet-class.md) Ta klasa reprezentuje moduł aplikacji.

- [Funkcja CAtlServiceModuleT](../atl/reference/catlservicemodulet-class.md) Ta klasa implementuje usługę.

- [CAtlWinModule](../atl/reference/catlwinmodule-class.md) Ta klasa zapewnia obsługę składników okienek ATL.

- [CComModule](../atl/reference/ccommodule-class.md) Ta klasa implementuje moduł DLL lub EXE. Przestarzałe w ATL 7,0.

- [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) Ta klasa implementuje moduł EXE z obsługą wielu apartamentach z pulą wątków. Przestarzałe w ATL 7,0.
