# <a name="mmc-snap-in-classes"></a>Klasy Snap-In MMC

Następujące klasy zapewniają obsługę tworzenia składników przystawki programu Microsoft Management Console (MMC):

- [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) Implementuje obiekt węzła przystawki, taki jak dodawanie elementów menu i pasków narzędzi, i przekazywanie poleceń dla węzła przystawki do odpowiedniej funkcji obsługi.

- [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) Implementuje obiekt strony właściwości przystawki.
