# <a name="mmc-snap-in-classes"></a>Klasy przystawki programu MMC
Następujące klasy umożliwiają tworzenie programu Microsoft Management Console (MMC) przystawki składników:  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) implementuje obiektu przystawki węzła, takie jak dodawanie elementów menu i paski narzędzi i przesyłania poleceń dla węzła przystawki do funkcji odpowiedni program obsługi.  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) implementuje obiekt strony właściwości przystawki.
