# <a name="mmc-snap-in-classes"></a>Klasy przystawki programu MMC
Następujące klasy zapewniają obsługę dla opracowywania programu Microsoft Management Console (MMC) przystawki składników:  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) implementuje obiektu przystawki węzeł, na przykład dodawanie elementów menu i pasków narzędzi oraz przekazywania poleceń dla funkcji obsługi odpowiedniego węzła przystawki.  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) implementuje obiekt strony właściwości przystawki.
