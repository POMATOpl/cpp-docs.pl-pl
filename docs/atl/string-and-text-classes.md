# <a name="string-and-text-classes"></a>Klasy String i text

Te klasy zapewniają obsługę ciągów i konwersji ciągów tekstowych.

- [CA2AEX](../atl/reference/ca2aex-class.md) Ta klasa jest używana przez makra konwersji ciągów CA2TEX i CT2AEX oraz typedef CA2A.

- [CA2CAEX](../atl/reference/ca2caex-class.md) Ta klasa jest używana przez makra konwersji ciągów CA2CTEX i CT2CAEX oraz typedef CA2CA.

- [CA2WEX](../atl/reference/ca2wex-class.md) Ta klasa jest używana przez makra konwersji ciągów CA2TEX, CA2CTEX, CT2WEX i CT2CWEX oraz typedef CA2W.

- [CW2AEX](../atl/reference/cw2aex-class.md) Ta klasa jest używana przez makra konwersji ciągów CT2AEX, CW2TEX, CW2CTEX i CT2CAEX oraz typedef CW2A.

- [CW2CWEX](../atl/reference/cw2cwex-class.md) Ta klasa jest używana przez makra konwersji ciągów CW2CTEX i CT2CWEX oraz typedef CW2CW.

- [CW2WEX](../atl/reference/cw2wex-class.md) Ta klasa jest używana przez makra konwersji ciągów CW2TEX i CT2WEX oraz typedef CW2W.

- [CComBSTR](../atl/reference/ccombstr-class.md) Ta klasa jest otoką dla BSTRs.

- [_U_STRINGorID](../atl/reference/u-stringorid-class.md) Ta klasa adaptera argumentów umożliwia przekazywanie nazw zasobów `LPCTSTR` lub identyfikatorów zasobów (UINTs) do funkcji bez konieczności, aby obiekt wywołujący PRZEKONWERTOWAŁ identyfikator na ciąg za pomocą makra MAKEINTRESOURCE.
