# <a name="atl-com-desktop-components"></a>Składniki ATL COM pulpitu

Dokumentacja ATL dokumentuje Active Template Library (ATL), zestaw klas C++ opartych na szablonach, które upraszczają programowanie obiektów Component Object Model (COM). COM to specyfikacja binarna służąca do tworzenia i używania składników oprogramowania w systemie Windows. Aby w pełni wykorzystać możliwości biblioteki ATL, zdecydowanie zalecane jest korzystanie z programu z modelem COM. Aby uzyskać więcej informacji na temat modelu COM, zobacz [Component Object Model (com)](/windows/win32/com/component-object-model--com--portal).
