# <a name="enumerators-and-collections-classes"></a>Moduły wyliczające i klasy kolekcji
Następujące klasy zapewniają obsługę dla modelu COM kolekcje i wyliczenia:  
  
-   [CComEnum](../atl/reference/ccomenum-class.md) definiuje COM enumerator — obiekt na podstawie tablicy.  
  
-   [CComEnumImpl](../atl/reference/ccomenumimpl-class.md) zawiera implementację interfejsu COM modułu wyliczającego przechowywania wyliczany elementów w tablicy.  
  
-   [CComEnumOnSTL](../atl/reference/ccomenumonstl-class.md) obiekt modułu wyliczającego definiuje COM oparte na kolekcji standardowa biblioteka C++.  
  
-   [IEnumOnSTLImpl](../atl/reference/ienumonstlimpl-class.md) zawiera implementację interfejsu COM modułu wyliczającego przechowywania wyliczany elementów w kontenerze zgodnej biblioteki C++ Standard.  
  
-   [ICollectionOnSTLImpl](../atl/reference/icollectiononstlimpl-class.md) udostępnia implementację dla **liczba**, **elementu**, i `_NewEnum` właściwości interfejsu kolekcji.
