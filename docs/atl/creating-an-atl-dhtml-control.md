# <a name="creating-an-atl-dhtml-control"></a>Tworzenie kontrolki DHTML ATL

Kreator kontrolki ATL automatyzuje proces tworzenia kontrolki DHTML. Generuje on niezbędne pliki zasobów, w tym plik HTML zawierający przykładowy kod.

## <a name="to-create-an-atl-dhtml-control"></a>Aby utworzyć formant ATL DHTML

1. Wykonaj kroki opisane w temacie [Tworzenie projektu ATL](../atl/reference/creating-an-atl-project.md).

1. W **Widok klasy** kliknij prawym przyciskiem myszy węzeł projektu, wskaż polecenie **Dodaj**, a następnie kliknij polecenie **Dodaj klasę** z menu skrótów. W oknie dialogowym **Dodawanie klasy** kliknij dwukrotnie [Kreatora kontrolki ATL](../atl/reference/atl-control-wizard.md). W **Kreatorze formantu ATL** kliknij kartę **Opcje** , a następnie wybierz pozycję **formant DHTML**.

Teraz można [testować domyślny formant](../atl/testing-the-atl-dhtml-control.md).
