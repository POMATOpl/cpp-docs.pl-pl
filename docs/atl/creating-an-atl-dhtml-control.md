# <a name="creating-an-atl-dhtml-control"></a>Tworzenie formantu ATL DHTML
Kreator formantu ATL automatyzuje proces tworzenia kontrolek DHTML. Generuje pliki niezbędne zasobów, w tym pliku HTML zawierającego przykładowy kod.  
  
#### <a name="to-create-an-atl-dhtml-control"></a>Można utworzyć formantu ATL DHTML  
  
1.  Postępuj zgodnie z instrukcjami [tworzenie Projekt ATL](../atl/reference/creating-an-atl-project.md).  
  
2.  W widoku klas, kliknij prawym przyciskiem myszy węzeł projektu, wskaż pozycję **Dodaj**i kliknij przycisk **Dodaj klasę** z menu skrótów. W **Dodaj klasę** okno dialogowe, kliknij dwukrotnie [Kreator formantu ATL](../atl/reference/atl-control-wizard.md). W Kreatorze formantu ATL kliknij **opcje** i wybierz **kontroli DHTML**.  
  
 Możesz teraz [sprawdzić formant domyślne](../atl/testing-the-atl-dhtml-control.md).
