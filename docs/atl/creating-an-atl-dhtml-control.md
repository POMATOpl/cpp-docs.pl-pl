# <a name="creating-an-atl-dhtml-control"></a>Tworzenie kontrolki ATL DHTML

Kreator kontrolki ATL automatyzuje proces tworzenia kontrolki DHTML. Generuje pliki potrzebnych zasobów, w tym pliku HTML zawierającego przykładowy kod.

## <a name="to-create-an-atl-dhtml-control"></a>Aby utworzyć kontrolki ATL DHTML

1. Postępuj zgodnie z instrukcjami w [Tworzenie projektu ATL](../atl/reference/creating-an-atl-project.md).

1. W **Widok klas**, kliknij prawym przyciskiem myszy węzeł projektu, wskaż **Dodaj**i kliknij przycisk **Dodaj klasę** z menu skrótów. W **Dodaj klasę** okno dialogowe, kliknij dwukrotnie [Kreator kontrolki ATL](../atl/reference/atl-control-wizard.md). W **Kreator kontrolki ATL**, kliknij przycisk **opcje** kartę, a następnie wybierz pozycję **kontrolki DHTML**.

Możesz teraz [test domyślny formant](../atl/testing-the-atl-dhtml-control.md).
