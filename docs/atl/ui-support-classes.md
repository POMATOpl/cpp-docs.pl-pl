# <a name="ui-support-classes"></a>Klasy obsługi interfejsu użytkownika

Następujące klasy zapewniają obsługę ogólnych interfejsów użytkownika:

- [IDocHostUIHandlerDispatch](../atl/reference/idochostuihandlerdispatch-interface.md) Interfejs do aparatu analizy i renderowania HTML firmy Microsoft.

- [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) Zapewnia podstawowe metody, za pomocą których kontener komunikuje się z kontrolką. Zarządza aktywacją i dezaktywacją formantów w miejscu.

- [IOleInPlaceObjectWindowlessImpl](../atl/reference/ioleinplaceobjectwindowlessimpl-class.md) Zarządza ponowną aktywacją kontrolek miejscowych. Umożliwia kontrolowanie bezokienkowe komunikatów, a także uczestnictwo w operacjach przeciągania i upuszczania.

- [IOleInPlaceActiveObjectImpl](../atl/reference/ioleinplaceactiveobjectimpl-class.md) Pomaga komunikować się między kontrolką miejscową i jej kontenerem.

- [IViewObjectExImpl](../atl/reference/iviewobjecteximpl-class.md) Umożliwia kontrolce wyświetlanie samej siebie bezpośrednio i powiadamianie kontenera o zmianach w jego wyświetlaczu. Zapewnia obsługę rysowania bez migotania, formantów nieprostokątnych i przezroczystych oraz testowania trafień.
