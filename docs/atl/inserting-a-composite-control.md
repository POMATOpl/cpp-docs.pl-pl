# <a name="inserting-a-composite-control"></a>Wstawianie kontrolki złożonej

**Dodaj klasę** okno dialogowe umożliwia wstawianie obiektu ATL do projektu. Dostęp do tego okna dialogowego, klikając prawym przyciskiem myszy nazwę projektu w Eksploratorze rozwiązań, wskazując **Dodaj**, a następnie klikając polecenie **Dodaj klasę**.

W **Dodaj klasę** okna dialogowego wybierz **kontrolka ATL**. Spowoduje to uruchomienie [Kreator kontrolki ATL](../atl/reference/atl-control-wizard.md). Aby utworzyć kontrolki złożonej, wybierz **opcje** kartę, a następnie kliknij przycisk **złożonej kontrolki** pole wyboru.

Domyślną stronę HTML, zostanie utworzony w celu wyświetlania kontrolki.
