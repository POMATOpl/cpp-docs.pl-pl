# <a name="inserting-a-composite-control"></a>Wstawianie kontrolki złożonej

Okno dialogowe **Dodawanie klasy** umożliwia wstawianie obiektu ATL do projektu. Aby uzyskać dostęp do tego okna dialogowego, kliknij prawym przyciskiem myszy nazwę projektu w Eksplorator rozwiązań, wskaż polecenie **Dodaj**, a następnie kliknij polecenie **Dodaj klasę**.

W oknie dialogowym **Dodaj klasę** wybierz opcję **formant ATL**. Spowoduje to uruchomienie [Kreatora kontrolki ATL](../atl/reference/atl-control-wizard.md). Aby utworzyć formant złożony, wybierz kartę **Opcje** , a następnie kliknij pole wyboru **formant złożony** .

Zostanie utworzona domyślna strona HTML do wyświetlania formantu.
