# <a name="deleting-a-device-image-image-editor-for-icons"></a>Usuwanie obrazu urządzenia (Edytor obrazów dla ikon)
### <a name="to-delete-a-device-image"></a>Aby usunąć obrazu urządzenia  
  
1.  Podczas wyświetlania obrazu ikony w **obrazu** edytora, wybierz opcję **usuwanie obrazu urządzenia** z **obrazu** menu. Po usunięciu ostatniego obrazu ikony w zasobie zasobu są także usuwane.  
  
    > [!NOTE]
    >  Po naciśnięciu klawisza DEL, obrazy i kolorów, które mają być rysowane ikonę zostaną usunięte, ale pozostaje ikony; Teraz można zmodyfikowanie. Jeśli naciśniesz DEL przez pomyłkę, możesz nacisnąć klawisze CTRL + Z, aby cofnąć.  
  
 Aby uzyskać informacje o dodawaniu zasobów do projektów zarządzanych, zobacz [zasobów w aplikacjach pulpitu](/dotnet/framework/resources/index) w *Przewodnik programistów platformy .NET Framework.* Aby uzyskać informacje na ręczne dodanie do projektów zarządzanych plików zasobów, uzyskiwanie dostępu do zasobów, wyświetlanie zasoby statyczne i przypisanie do właściwości ciągów zasobów, zobacz [tworzenie plików zasobów dla aplikacji pulpitu](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Informacje dotyczące globalizacji i lokalizacji zasobów w zarządzanych aplikacjach, zobacz [Globalizing i lokalizacja aplikacji .NET Framework](/dotnet/standard/globalization-localization/index).
