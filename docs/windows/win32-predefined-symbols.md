# <a name="win32-predefined-symbols"></a>Wstępnie zdefiniowane symbole Win32
Te symbole są definiowane w pliki nagłówkowe Win32 i obsługują standardowe funkcje aplikacji systemu Windows i działań. Te symbole są używane głównie z typowymi elementami interfejsu użytkownika. Podczas pracy z kontrolkami w edytory zasobów symbole te będą wyświetlane w [okna właściwości](/visualstudio/ide/reference/properties-window) skojarzone z formantów standardowych. Na przykład jeśli ikona aplikacji powinien być wyświetlany pasek narzędzi, ikony zostanie skojarzona z symbol IDI_SMALL w oknie właściwości.  
  
|||  
|-|-|  
|IDABORT|Formantu: Przycisk Przerwij pola okna dialogowego|  
|IDC_STATIC|Kontrola: Statyczny tekst w oknie dialogowym|  
|IDCANCEL|Sterowania: Przycisk Anuluj okno dialogowe|  
|IDD_ABOUTBOX|Okno dialogowe: Produktu okno dialogowe|  
|IDI_PROJECTNAME|Ikona: Bieżąca ikona projektu|  
|IDI_SMALL|Ikona: Bieżący projekt małe ikony|  
|IDIGNORE|Kontrola: Używany przycisk Ignoruj, w oknach dialogowych|  
|IDM_ABOUT|Element menu: Używane za pomocą... — Informacje...|  
|IDM_EXIT|Element menu: Używane z pliku... Zakończ...|  
|IDNO|Kontrola: Nie przycisk okna dialogowego|  
|IDOK|Formantu: Przycisk pola OK okna dialogowego|  
|IDRETRY|Sterowania: Przycisk Ponów — okno dialogowe|  
|IDS_APP_TITLE|Ciągu: Nazwa bieżącej aplikacji|  
|IDYES|Kontrola: Okno dialogowe tak przycisku.|
