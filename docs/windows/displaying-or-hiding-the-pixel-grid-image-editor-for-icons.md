# <a name="displaying-or-hiding-the-pixel-grid-image-editor-for-icons"></a>Wyświetlanie i ukrywanie siatki pikseli (Edytor obrazów dla ikon)
Dla wszystkich okienek edytor obrazów z współczynnika powiększenia 4 lub nowszym można wyświetlić siatkę ograniczającego piksele obrazu.  
  
> [!TIP]
>  Etykietki narzędzi są wyświetlane po umieszczeniu kursora na przycisku paska narzędzi. Te porady mogą pomóc w identyfikacji funkcji każdego przycisku.  
  
### <a name="to-display-or-hide-the-pixel-grid"></a>Umożliwia wyświetlanie lub ukrywanie siatki pikseli  
  
1.  Na **obrazu** menu, kliknij przycisk **ustawienia siatki**.  
  
2.  Wybierz **siatki pikseli** pole wyboru, aby wyświetlić siatki, lub wyczyść pole, aby ukryć siatki.
