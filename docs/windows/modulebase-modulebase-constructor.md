# <a name="modulebasemodulebase-constructor"></a>ModuleBase::ModuleBase — Konstruktor

## <a name="syntax"></a>Składnia  
  
```  
ModuleBase();  
```  
  
## <a name="remarks"></a>Uwagi  
 Inicjuje wystąpienie `Module` klasy.
