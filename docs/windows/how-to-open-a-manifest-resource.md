# <a name="how-to-open-a-manifest-resource"></a>Porady: otwarcie zasobu manifestu
Jeśli otwarcie zasobu manifestu z [widok zasobów](../windows/resource-view-window.md), zasób zostanie otwarty w trybie binarnym. Aby wyświetlić zawartość zasobu manifestu w formacie bardziej widoczne, należy otworzyć zasobu w Eksploratorze rozwiązań.  
  
### <a name="to-open-a-manifest-resource-in-the-text-editor"></a>Aby otworzyć zasobu manifestu w edytorze tekstu  
  
1.  Przy otwarciu projektu w `Solution Explorer`, rozwiń węzeł **pliki zasobów** folderu.  
  
2.  Kliknij dwukrotnie plik .manifest.  
  
     Zasób manifestu zostanie otwarty w edytorze tekstu.  
  
### <a name="to-open-a-manifest-resource-in-another-editor"></a>Aby otworzyć zasobu manifestu w innym edytorze  
  
1.  W Eksploratorze rozwiązań kliknij prawym przyciskiem myszy plik .manifest, a następnie wybierz pozycję **Otwórz za pomocą...**  z menu skrótów.  
  
2.  W **Otwórz za pomocą** okna dialogowego, określ Edytor chcesz użyć, a następnie kliknij przycisk **Otwórz**.  
  
 Aby uzyskać informacje o dodawaniu zasobów do projektów zarządzanych, zobacz [zasobów w aplikacjach pulpitu](/dotnet/framework/resources/index) w *Przewodnik programistów platformy .NET Framework.*
