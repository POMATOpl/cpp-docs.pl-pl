# <a name="ole-db-consumer-attributes"></a>Atrybuty konsumentów OLE DB

Atrybuty konsumenta OLE DB wstrzyknąć kod, na podstawie [szablonów konsumenta OLE DB](../../data/oledb/ole-db-consumer-templates-reference.md), aby utworzyć działającego konsumenta OLE DB, który wykonuje zadania, takie jak otwieranie tabel, wykonywanie poleceń i uzyskiwanie dostępu do danych.

|Atrybut|Opis|
|---------------|-----------------|
|[db_accessor](db-accessor.md)|Wiąże kolumny w zestawie wierszy i wiąże je z odpowiednimi mapami akcesora.|
|[db_column](db-column.md)|Wiąże określoną kolumnę z zestawem wierszy.|
|[db_command](db-command.md)|Wykonuje polecenie OLE DB.|
|[db_param](db-param.md)|Kojarzy określoną zmienną element członkowną z parametrem wejściowym lub wyjściowym.|
|[db_source](db-source.md)|Tworzy i hermetyzuje połączenie, za pośrednictwem dostawcy, do źródła danych.|
|[db_table](db-table.md)|Otwiera tabelę OLE DB.|
