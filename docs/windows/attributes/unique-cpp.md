# <a name="unique-c"></a>unique (C++)

Określa unikatowy wskaźnik.

## <a name="syntax"></a>Składnia

```cpp
[unique]
```

## <a name="remarks"></a>Uwagi

**Unikatowy** C++ atrybut ma taką samą funkcjonalność jak [unikatowy](/windows/win32/Midl/unique) atrybut MIDL.
