# <a name="oleautomation"></a>oleautomation

Wskazuje, że interfejs jest zgodny z automatyzacją.

## <a name="syntax"></a>Składnia

```cpp
[oleautomation]
```

## <a name="remarks"></a>Uwagi

Atrybut **OleAutomation** C++ ma takie same funkcje jak atrybut [OleAutomation](/windows/win32/Midl/oleautomation) MIDL.
