# <a name="oleautomation"></a>oleautomation

Wskazuje, że interfejs jest zgodna z usługą Automation.

## <a name="syntax"></a>Składnia

```cpp
[oleautomation]
```

## <a name="remarks"></a>Uwagi

**Oleautomation —** atrybut C++ ma taką samą funkcjonalność jak [oleautomation —](/windows/desktop/Midl/oleautomation) atrybutów w MIDL.

## <a name="example"></a>Przykład

Zobacz przykłady dla [defaultvalue](defaultvalue.md) i [nonextensible](nonextensible.md) do użytku przykładowe **oleautomation —**.
