# <a name="attributes-by-group"></a>Atrybuty według grup

C++ Atrybuty są zorganizowane w następujące grupy funkcjonalne.

|Atrybut|Opis|
|---------------|-----------------|
|[Atrybuty COM](com-attributes.md)|Wsuń kod obsługujący wiele obszarów programowania modelu COM i .NET Framework tworzenia środowiska uruchomieniowego języka wspólnego.|
|[Atrybuty IDL](idl-attributes.md)|Włączenie modyfikacji pliku. idl z pliku kodu źródłowego bez użycia kreatora i bez znajomości struktury i składni tego pliku.|
|[Atrybuty konsumentów OLE DB](ole-db-consumer-attributes.md)|Wsuń kod — na podstawie [OLE DB szablonów konsumentów](../../data/oledb/ole-db-consumer-templates-reference.md), aby utworzyć działającego klienta OLE DB, który wykonuje zadania, takie jak otwieranie tabel, wykonywanie poleceń i uzyskiwanie dostępu do danych.|
|[Atrybuty kompilatora](compiler-attributes.md)|Dostarczone przez kompilator firmy C++ Microsoft.|

Aby uzyskać informacje na temat sposobu definiowania atrybutów niestandardowych dla aplikacji .NET Framework, zobacz [atrybuty zdefiniowane przez użytkownika](../../extensions/user-defined-attributes-cpp-component-extensions.md).
