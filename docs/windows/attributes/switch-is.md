# <a name="switchis"></a>switch_is

Określa wyrażenie lub identyfikator działający jako discriminant Unii, który wybiera składowa typu Unii.

## <a name="syntax"></a>Składnia

```cpp
[switch_is]
```

## <a name="remarks"></a>Uwagi

**Switch_is —** atrybut C++ ma taką samą funkcjonalność jak [switch_is —](/windows/desktop/Midl/switch-is) atrybutów w MIDL.
