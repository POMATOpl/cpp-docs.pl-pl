# <a name="array-attributes"></a>Atrybuty tablicy

Następujące atrybuty dotyczą tablic lub `SAFEARRAY`s.

|Atrybut|Opis|
|---------------|-----------------|
|[library_block](library-block.md)|Umieszcza konstrukcję wewnątrz bloku biblioteki pliku .idl.|
|[satype](satype.md)|Określa typ danych `SAFEARRAY` struktury.|
|[string](string-cpp.md)|Oznacza to, że jednowymiarowy **char**, **wchar_t**, `byte` (lub równoważnego) tablicy lub wskaźnika do tablicy takie, które muszą być traktowane jako ciąg.|
