|---------------|-----------------|
|[library_block](library-block.md)|Umieszcza konstrukcję w bloku biblioteki pliku IDL.|
|[satype](satype.md)|Określa typ danych struktury `SAFEARRAY`.|
|[string](string-cpp.md)|Wskazuje **, że tablica Jednowymiarowa**, **wchar_t**, `byte` (lub równoważna) lub wskaźnik do takiej tablicy musi być traktowana jako ciąg.|
