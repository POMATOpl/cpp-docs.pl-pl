# <a name="selecting-controls"></a>Zaznaczanie kontrolek
Wybierz formanty do rozmiaru, Dopasuj, przenieść, skopiować, lub usuń je, a następnie wykonaj operację, którą chcesz. W większości przypadków musisz wybrać więcej niż jeden formant za pomocą narzędzi rozmiaru i wyrównanie na [paska narzędzi edytora okien dialogowych](../windows/showing-or-hiding-the-dialog-editor-toolbar.md).  
  
 Po wybraniu formantu, ma przyciemnione obramowanie z stałe (aktywny) lub pusty (nieaktywny) małe "uchwytów," kwadraty, które pojawiają się w zaznaczenia. Po wybraniu wielu formantów, formantu dominującego ma uchwyty rozmiaru stałe; wszystkie inne wybrane formanty mają uchwyty pusty zmiany rozmiaru.  
  
 Gdy są zmiany rozmiaru lub wyrównywanie wiele formantów, Edytor okien dialogowych używa "formantu dominującego" ustalenie sposobu o rozmiarze lub wyrównane inne formanty. Domyślnie formantu dominującego jest pierwszą kontrolkę wybrane.  
  
-   [Wybieranie wielu formantów](../windows/selecting-multiple-controls.md)  
  
-   [Określanie formantu dominującego](../windows/specifying-the-dominant-control.md)  
  
 Aby uzyskać informacje o dodawaniu zasobów do projektów zarządzanych, zobacz [zasobów w aplikacjach pulpitu](https://msdn.microsoft.com/library/f45fce5x.aspx) w *Przewodnik programistów platformy .NET Framework.* Aby uzyskać informacje na ręczne dodanie do projektów zarządzanych plików zasobów, uzyskiwanie dostępu do zasobów, wyświetlanie zasoby statyczne i przypisanie do właściwości ciągów zasobów, zobacz [tworzenie plików zasobów dla aplikacji pulpitu](https://msdn.microsoft.com/library/xbx3z216.aspx). Informacje dotyczące globalizacji i lokalizacji zasobów w zarządzanych aplikacjach, zobacz [Globalizing i lokalizacja aplikacji .NET Framework](https://msdn.microsoft.com/library/h6270d0z.aspx).
