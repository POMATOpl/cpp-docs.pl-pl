# <a name="propput"></a>propput
Określa funkcja ustawienie właściwości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
[propput]  
  
```  
  
## <a name="remarks"></a>Uwagi  
 **Propput** atrybut C++ ma te same funkcje co [propput](http://msdn.microsoft.com/library/windows/desktop/aa367146) MIDL atrybutu.
