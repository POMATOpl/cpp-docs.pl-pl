# <a name="selecting-multiple-menus-or-menu-commands"></a>Zaznaczanie kilku menu lub poleceń menu
Możesz wybrać wiele nazw menu lub poleceń menu, aby wykonywać operacje zbiorcze, takich jak usuwanie lub zmiana właściwości.  
  
### <a name="to-select-multiple-menu-commands"></a>Aby wybrać wiele poleceń menu  
  
1.  Trzymając wciśnięty klawisz CTRL, kliknij menu lub poleceń podmenu, które mają.  
  
 Aby uzyskać informacje o dodawaniu zasobów do projektów zarządzanych, zobacz [zasobów w aplikacjach pulpitu](/dotnet/framework/resources/index) w *Przewodnik programistów platformy .NET Framework.*  
  
 **Wymagania**  
  
 Win32
