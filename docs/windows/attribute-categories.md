# <a name="attribute-categories"></a>Kategorie atrybutów
Odwołanie do atrybutów opisano różne typy atrybutów, których można użyć w programie Visual C++.  
  
## <a name="related-sections"></a>Sekcje pokrewne  
 [Atrybuty COM](../windows/com-attributes.md)  
 Injects kod obsługujący wiele obszarów programowanie COM.  
  
 [Atrybuty IDL](../windows/idl-attributes.md)  
 Injects kod, który tworzy lub modyfikuje pliku .idl skojarzone z w pliku kodu źródłowego bez użycia kreatora i bez konieczności znajomości struktury i Składnia tego pliku.  
  
 [Atrybuty konsumentów OLE DB](../windows/ole-db-consumer-attributes.md)  
 Injects kodu na podstawie [szablony OLE DB konsumenta](../data/oledb/ole-db-consumer-templates-reference.md), aby utworzyć pracy OLE DB konsumenta wykonujące zadania, takie jak tabele otwierania wykonywania poleceń i uzyskiwanie dostępu do danych.  
  
 [Atrybuty kompilatora](../windows/compiler-attributes.md)  
 Injects kod, który wykonuje szereg typowych zadań programistycznych.  
  
 [Programowanie oparte na atrybutach](../windows/attributed-programming-concepts.md)  
 Zawiera łącza do pojęć opisujące Programowanie oparte na atrybutach.
