# <a name="attribute-categories"></a>Kategorie atrybutów

Dokumentacja atrybutów w tym artykule opisano różne typy atrybutów, których można użyć w programie Visual C++.
  
## <a name="related-sections"></a>Sekcje pokrewne

[Atrybuty COM](../windows/com-attributes.md)<br/>
Wprowadza kod obsługujący wiele obszarów projektowania modelu COM.
  
[Atrybuty IDL](../windows/idl-attributes.md)<br/>
Wprowadza kod, który tworzy lub modyfikuje pliku .idl skojarzone z w pliku kodu źródłowego bez użycia kreatora i bez struktury i składni tego pliku.
  
[Atrybuty konsumentów OLE DB](../windows/ole-db-consumer-attributes.md)<br/>
Wprowadza kodu, w oparciu o [szablony OLE DB konsumenta](../data/oledb/ole-db-consumer-templates-reference.md), aby utworzyć pracy OLE DB użytkownikowi, który wykonuje zadania, takie jak otwieranie tabel, wykonując polecenia i uzyskiwanie dostępu do danych.
  
[Atrybuty kompilatora](../windows/compiler-attributes.md)<br/>
Wprowadza kod, który wykonuje szereg typowych zadań programistycznych.
  
[Programowanie oparte na atrybutach](../windows/attributed-programming-concepts.md)<br/>
Zawiera łącza do tematów pojęciowych opisujące Programowanie oparte na atrybutach.
