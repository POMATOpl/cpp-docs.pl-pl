# <a name="specifying-the-dominant-control"></a>Określanie formantu dominującego
Wybrany formant jest najpierw formantu dominującego.  
  
### <a name="to-specify-the-dominant-control"></a>Aby określić formantu dominującego  
  
1.  Naciśnij i przytrzymaj **CTRL** klucza, a następnie kliknij przycisk kontroli chcesz użyć, aby mieć wpływ na rozmiar lub lokalizacji innych kontrolek *pierwszy*.  
  
     **Uwaga** uchwyty zmiany rozmiaru formantu dominującego są wypełnione podczas uchwyty kontrolek podrzędnych są puste. Wszystkie dodatkowe zmiany rozmiaru lub wyrównania jest oparty na formantu dominującego.  
  
### <a name="to-change-the-dominant-control"></a>Aby zmienić formantu dominującego  
  
1.  Wyczyść bieżącego zaznaczenia, klikając poza wszystkie obecnie wybrane kontrolki.  
  
2.  Powtórzyć poprzednią procedurę, wybierając inny formant najpierw.  
  
 Aby uzyskać informacje o dodawaniu zasobów do projektów zarządzanych, zobacz [zasobów w aplikacjach pulpitu](/dotnet/framework/resources/index) w *Przewodnik programistów platformy .NET Framework.* Aby uzyskać informacje na ręczne dodanie do projektów zarządzanych plików zasobów, uzyskiwanie dostępu do zasobów, wyświetlanie zasoby statyczne i przypisanie do właściwości ciągów zasobów, zobacz [tworzenie plików zasobów dla aplikacji pulpitu](/dotnet/framework/resources/creating-resource-files-for-desktop-apps).
