# <a name="propget"></a>propget
Określa funkcja dostępu właściwości.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
[propget]  
  
```  
  
## <a name="remarks"></a>Uwagi  
 **Propget** atrybut C++ ma te same funkcje co [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL atrybutu.
