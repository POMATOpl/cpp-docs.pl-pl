# <a name="predefined-symbol-ids"></a>Wstępnie zdefiniowane symbole identyfikatorów

Po rozpoczęciu nowego projektu, w zależności od typu projektu, niektóre identyfikatory symboli są wstępnie zdefiniowane do użycia. Te identyfikatory symboli obsługują różne biblioteki i typy projektów, takie jak MFC. Reprezentują one typowe zadania, które zwykle są uwzględniane w dowolnej aplikacji lub akcji elementów sprzętowych, takich jak mysz lub drukarka.

Te identyfikatory symboli stają się ważne podczas pracy z zasobami. Są one dostępne podczas edytowania tabel akceleratorów, a niektóre z nich są już skojarzone z kluczami wirtualnymi. Są one również dostępne przez [okno właściwości](/visualstudio/ide/reference/properties-window). Do nowych zasobów można przypisać dowolne wstępnie zdefiniowane identyfikatory symboli lub przypisać do nich klawisze skrótów, a funkcje skojarzone z IDENTYFIKATORem symbol są automatycznie kojarzone z tą kombinacją klawiszy.

Biblioteki mają wstępnie zdefiniowane symbole, które będą wyświetlane jako część projektu:

- [Wstępnie zdefiniowane symbole ATL](../windows/atl-predefined-symbols.md)

- [Wstępnie zdefiniowane symbole MFC](../windows/mfc-predefined-symbols.md)

- [Wstępnie zdefiniowane symbole Win32](../windows/win32-predefined-symbols.md)

> [!NOTE]
> Wstępnie zdefiniowane symbole są zawsze tylko do odczytu.

## <a name="requirements"></a>Wymagania

Win32, MFC lub ATL
