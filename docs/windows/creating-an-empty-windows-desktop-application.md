# <a name="creating-an-empty-windows-desktop-application"></a>Tworzenie aplikacji klasycznych pusty systemu Windows
### <a name="to-create-an-empty-windows-desktop-application"></a>Aby utworzyć pustą aplikację pulpitu systemu Windows  
  
1.  Z **pliku** menu, wybierz opcję **nowy, projekt**.  
  
2.  W lewym okienku **nowy projekt** okna dialogowego, kliknij przycisk **Win32** i w środkowym okienku wybierz **aplikacji konsoli Win32**.  
  
3.  Wprowadź nazwę dla nowego projektu, ścieżka do katalogu projektu, a następnie kliknij przycisk **OK**.  
  
4.  W [Kreator aplikacji Win32](../windows/win32-application-wizard.md), kliknij przycisk **ustawienia aplikacji** strony. Wybierz **typu aplikacji** chcesz utworzyć plik kodu źródłowego, a następnie wybierz **pusty projekt** pole wyboru w obszarze **dodatkowe opcje**.  
  
5.  Kliknij przycisk **OK**.  
  
 Projekt zostanie wyświetlony w **Eksploratora rozwiązań** z trzech katalogów zawiera pliki źródłowe, pliki nagłówkowe i pliki zasobów.  
  
 Następnie można [Dodaj pliki do pustego projektu Visual C++](../windows/adding-files-to-an-empty-win32-applications.md).
