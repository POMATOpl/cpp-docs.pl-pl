# <a name="new-custom-resource-dialog-box"></a>Okno dialogowe Nowy zasób niestandardowy
Pozwala utworzyć nowy zasób niestandardowy.  
  
### <a name="resource-type"></a>Typ zasobu  
 Zawiera pole tekstowe do wprowadzenia nazwy typu zasobów niestandardowych. Visual C++ automatycznie powoduje rozpoczynanie nazwę podczas zamykania **nowy zasób niestandardowy** okno dialogowe.  
  
## <a name="remarks"></a>Uwagi  
 Nie określaj nazwy zasobu lub identyfikator, który jest zarezerwowana przez [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]. Zarezerwowane nazwy to DESIGNINFO, HWB i TEXTINCLUDE, a zarezerwowany identyfikator to 255.
