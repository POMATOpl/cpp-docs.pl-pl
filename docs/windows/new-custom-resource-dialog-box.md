# <a name="new-custom-resource-dialog-box-c"></a>Okno dialogowe Nowy zasób niestandardowy (C++)

Pozwala utworzyć nowy niestandardowy zasób w projektach C++.

### <a name="resource-type"></a>Typ zasobu

Zawiera pole tekstowe do wprowadzenia nazwy typu zasobów niestandardowych. Visual C++ automatycznie powoduje rozpoczynanie nazwę podczas zamykania **nowy zasób niestandardowy** okno dialogowe.

## <a name="remarks"></a>Uwagi

Nie określaj nazwy zasobu lub identyfikator, który jest zarezerwowany przez program Visual Studio. Zarezerwowane nazwy to DESIGNINFO, HWB i TEXTINCLUDE, a zarezerwowany identyfikator to 255.
