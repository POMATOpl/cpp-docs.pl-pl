# <a name="ole-db-consumer-attributes"></a>Atrybuty konsumentów OLE DB
Atrybuty konsumentów OLE DB wstrzyknięcie kodu, w oparciu o [szablony OLE DB konsumenta](../data/oledb/ole-db-consumer-templates-reference.md), aby utworzyć pracy OLE DB użytkownikowi, który wykonuje zadania, takie jak otwieranie tabel, wykonując polecenia i uzyskiwanie dostępu do danych.
  
|Atrybut|Opis|
|---------------|-----------------|
|[db_accessor](../windows/db-accessor.md)|Tworzy powiązanie kolumn w zestawie wierszy i wiąże ich odpowiednich mapowania dostępu.|
|[db_column](../windows/db-column.md)|Wiąże określonej kolumny zestawu wierszy.|
|[db_command](../windows/db-command.md)|Wykonuje polecenie OLE DB.|
|[db_param](../windows/db-param.md)|Kojarzy zmiennej określonego elementu członkowskiego z parametrów wejściowych lub wyjściowych.|
|[db_source](../windows/db-source.md)|Tworzy i hermetyzuje połączenie za pośrednictwem dostawcy, ze źródłem danych.|
|[db_table](../windows/db-table.md)|Zostanie otwarty tabeli OLE DB.|
