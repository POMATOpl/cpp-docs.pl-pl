# <a name="out-c"></a>out (C++)
Określa parametry wskaźnika, które zostaną zwrócone z procedury wywoływanej do procedury wywołującej (z serwera do klienta).  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
[out]  
```  
  
## <a name="remarks"></a>Uwagi  
 **Się** atrybut C++ ma taką samą funkcjonalność jak [się](http://msdn.microsoft.com/library/windows/desktop/aa367136) atrybutów w MIDL.  
  
## <a name="example"></a>Przykład  
 Zobacz przykład [możliwej do wiązania](../windows/bindable.md) do użytku przykładowe **się**.
