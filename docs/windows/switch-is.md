# <a name="switchis"></a>switch_is
Określa wyrażenie lub identyfikator działający jako discriminant Unii, który wybiera element członkowski typu union.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
[switch_is]  
  
```  
  
## <a name="remarks"></a>Uwagi  
 **Switch_is —** atrybut C++ ma te same funkcje co [switch_is —](http://msdn.microsoft.com/library/windows/desktop/aa367275) MIDL atrybutu.
