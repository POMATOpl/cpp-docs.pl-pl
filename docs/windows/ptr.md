# <a name="ptr"></a>ptr
Określa wskaźnik jako pełna wskaźnika.  
  
## <a name="syntax"></a>Składnia  
  
```  
[ptr]  
```  
  
## <a name="remarks"></a>Uwagi  
 **Ptr** atrybut C++ ma taką samą funkcjonalność jak [ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) atrybutów w MIDL.  
  
## <a name="example"></a>Przykład  
 Zobacz przykład [defaultvalue](../windows/defaultvalue.md) do użytku przykładowe **ptr**.
