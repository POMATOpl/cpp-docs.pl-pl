# <a name="ptr"></a>ptr
Określa wskaźnik jako pełne wskaźnika.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
[ptr]  
  
```  
  
## <a name="remarks"></a>Uwagi  
 **Ptr** atrybut C++ ma te same funkcje co [ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL atrybutu.  
  
## <a name="example"></a>Przykład  
 Zobacz przykład [defaultvalue](../windows/defaultvalue.md) użytku próbki **ptr**.
