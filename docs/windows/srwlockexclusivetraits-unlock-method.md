# <a name="srwlockexclusivetraitsunlock-method"></a>SRWLockExclusiveTraits::Unlock — Metoda

```  
inline static void Unlock(  
   _In_ Type srwlock  
);  
```  
  
#### <a name="parameters"></a>Parametry

 Dojście do obiektu srwlock —.
