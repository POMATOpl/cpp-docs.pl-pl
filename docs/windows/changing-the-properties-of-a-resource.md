# <a name="changing-the-properties-of-a-resource"></a>Zmiana właściwości zasobu
### <a name="to-edit-the-properties-of-a-resource"></a>Aby edytować właściwości zasobu  
  
1.  W [widok zasobów](../windows/resource-view-window.md), kliknij prawym przyciskiem myszy zasób chcesz edytować, a następnie wybierz pozycję **właściwości** z menu skrótów.  
  
     **Uwaga** Jeśli projekt nie zawiera już plik .rc, zobacz [Tworzenie nowego pliku skryptu zasobu](../windows/how-to-create-a-resource-script-file.md).  
  
2.  W [okna właściwości](/visualstudio/ide/reference/properties-window), zmień właściwości zasobu.  
  
### <a name="to-undo-a-change-youve-made-to-the-properties-of-a-resource"></a>Aby cofnąć zmiany wprowadzone do właściwości zasobu  
  
1.  Upewnij się, że zasób ma fokus w widoku zasobów.  
  
2.  Wybierz **Cofnij** z **Edytuj** menu.
