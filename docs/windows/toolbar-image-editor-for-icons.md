# <a name="toolbar-image-editor-for-icons"></a>Pasek narzędzi (Edytor obrazów dla ikon)
**Edytor obrazów** narzędzi zawiera narzędzia do rysowania, malowania wprowadzania tekstu, wymazywanie i manipulowanie widoków. Zawiera ona także Selektor opcji, z którym można wybrać opcje dotyczące korzystania z narzędzia. Na przykład można wybrać różne szerokości pędzla, czynniki powiększenia i style linii.  
  
> [!NOTE]
>  Wszystkie narzędzia dostępne w **edytor obrazów** narzędzi są także dostępne w **obrazu** menu (w obszarze **narzędzia** polecenia).  
  
 ![Pasek narzędzi edytora obrazów](../mfc/media/vcimageeditortoolbar.gif "vcImageEditorToolbar")  
Pasek narzędzi edytora obrazów  
  
 Aby użyć **edytor obrazów** narzędzi i **opcji** selektora, kliknij narzędzie lub opcji, które mają.  
  
> [!TIP]
>  Etykietki narzędzi są wyświetlane po umieszczeniu kursora na przycisku paska narzędzi. Te porady mogą pomóc w identyfikacji funkcji każdego przycisku.  
  
 Z **opcji** selektora można określić szerokości linii, pędzla itp. Ikona na **opcji** selektora przycisk zmiany w zależności od narzędzie, które zostały wybrane.  
  
 ![Rysowanie&#45;selektora kształtu na pasku narzędzi edytora obrazów](../mfc/media/vcimageeditortoolbaroptionselector.gif "vcImageEditorToolbarOptionSelector")  
Selektor opcji na pasku narzędzi edytora obrazów

Informacje dotyczące globalizacji i lokalizacji zasobów w zarządzanych aplikacjach, zobacz [Globalizing i lokalizacja aplikacji .NET Framework](/dotnet/standard/globalization-localization/index).
