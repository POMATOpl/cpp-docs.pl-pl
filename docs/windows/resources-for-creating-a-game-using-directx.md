# <a name="resources-for-creating-a-c-game-using-directx"></a>Zasoby służące do tworzenia gier C++ za pomocą programu DirectX

Aby tworzyć najlepsze gry, używaj Visual C++ w połączeniu z DirectX.

## <a name="resources"></a>Resources

Visual C++ jest zawarty we wszystkich wersjach programu Visual Studio, ale nie jest instalowany domyślnie. Aby uzyskać informacje o sposobie instalowania programu Visual Studio i narzędzia programistyczne C++, zobacz [Instalowanie obsługi języka C++ w programie Visual Studio](../build/vscpp-step-0-installation.md).

Windows 8.1 i nowszych zestawu SDK programu DirectX znajduje się w zestawie Windows SDK. Nie trzeba dodatkowo niczego pobierać. Aby uzyskać więcej informacji, zobacz [gdzie jest DirectX SDK?](https://msdn.microsoft.com/library/windows/desktop/ee663275.aspx).

Aby utworzyć aplikację DirectX dla Windows 7, Windows Vista lub Windows XP, możesz pobrać autonomiczny [zestawu SDK programu DirectX](http://www.microsoft.com/download/details.aspx?displaylang=en&id=6812).

[Grafika i gry DirectX](https://msdn.microsoft.com/library/windows/desktop/ee663274) zawiera informacje i przykłady ułatwiające rozpoczęcie pracy przy użyciu technologii DirectX programowania.

[Aplikacje Windows Win32](../windows/windows-desktop-applications-cpp.md) zawiera artykuły opisujące, jak utworzyć podstawową aplikację pulpitu Windows.

[Tworzenie gier](https://msdn.microsoft.com/library/windows/apps/hh452744.aspx) omawia tworzenie gier Universal Windows Platform, ale wiele koncepcji dotyczy także pulpitów gier.
