# <a name="resources-for-creating-a-game-using-directx"></a>Zasoby służące do tworzenia gier za pomocą programu DirectX
Aby tworzyć najlepsze gry, używaj Visual C++ w połączeniu z DirectX.  
  
## <a name="resources"></a>Zasoby  
 Visual C++ jest zawarty we wszystkich edycjach Visual Studio. Aby uzyskać informacje dotyczące programu Visual Studio, zobacz [programu Visual Studio](http://www.visualstudio.com/) witryny sieci Web.  
  
 Aby uzyskać [!INCLUDE[win81](../misc/includes/win81_md.md)], DirectX znajduje się w zestawie Windows SDK. Nie trzeba dodatkowo niczego pobierać. Aby uzyskać więcej informacji, zobacz [w przypadku zestawu SDK programu DirectX?](http://msdn.microsoft.com/library/windows/desktop/ee663275.aspx).  
  
 Aby utworzyć DirectX aplikacji dla systemu Windows 7, Windows Vista lub Windows XP, możesz pobrać autonomiczną [zestawu SDK programu DirectX](http://www.microsoft.com/download/details.aspx?displaylang=en&id=6812).  
  
 [Grafika DirectX i gier](http://msdn.microsoft.com/library/windows/desktop/ee663274\(v=vs.85\).aspx) zawiera informacje i przykłady ułatwiające rozpoczęcie pracy z DirectX programowania.  
  
 [Aplikacje Win32 systemu Windows](../windows/windows-desktop-applications-cpp.md) zawiera artykuły, które zawierają opis sposobu tworzenia podstawowej aplikacji klasycznej systemu Windows.  
  
 [Projektowanie gier](http://msdn.microsoft.com/library/windows/apps/hh452744.aspx) omówiono tworzenie gier dla platformy uniwersalnej systemu Windows, ale wiele pojęć dotyczą także gry pulpitu.
