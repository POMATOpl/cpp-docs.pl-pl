# <a name="redistributing-web-client-applications"></a>Ponowne dystrybuowanie Aplikacji klienta sieci Web

Jeśli aplikacja używa klas MFC implementujących formant WebBrowser (na przykład `CHtmlView` lub `CHtmlEditView` ), program Microsoft Internet Explorer 4,0 lub nowszy musi być co najmniej zainstalowany na komputerze docelowym.

Zainstalowanie najnowszej wersji programu Internet Explorer gwarantuje również, że na komputerze docelowym znajdują się najnowsze pliki formantów wspólnych. Aby uzyskać więcej informacji, zobacz [Instalowanie i wdrażanie programu Internet Explorer 11](/internet-explorer/ie11-deploy-guide/install-and-deploy-ie11).
