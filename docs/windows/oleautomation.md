# <a name="oleautomation"></a>oleautomation
Wskazuje, że interfejs jest zgodna z usługą Automation.  
  
## <a name="syntax"></a>Składnia  
  
```cpp  
[oleautomation]  
```  
  
## <a name="remarks"></a>Uwagi  
 **Oleautomation —** atrybut C++ ma taką samą funkcjonalność jak [oleautomation —](http://msdn.microsoft.com/library/windows/desktop/aa367129) atrybutów w MIDL.  
  
## <a name="example"></a>Przykład  
 Zobacz przykłady dla [defaultvalue](../windows/defaultvalue.md) i [nonextensible](../windows/nonextensible.md) do użytku przykładowe **oleautomation —**.
