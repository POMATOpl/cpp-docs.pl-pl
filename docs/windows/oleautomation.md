# <a name="oleautomation"></a>oleautomation
Wskazuje, że interfejs jest zgodna z automatyzacji.  
  
## <a name="syntax"></a>Składnia  
  
```  
  
[oleautomation]  
  
```  
  
## <a name="remarks"></a>Uwagi  
 **Oleautomation** atrybut C++ ma te same funkcje co [oleautomation](http://msdn.microsoft.com/library/windows/desktop/aa367129) MIDL atrybutu.  
  
## <a name="example"></a>Przykład  
 Przykłady dla [defaultvalue](../windows/defaultvalue.md) i [nonextensible —](../windows/nonextensible.md) użytku próbki **oleautomation**.
