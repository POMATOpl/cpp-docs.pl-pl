## <a name="syntax"></a>Składnia  
  
```  
FactoryCache* factoryCache;  
```  
  
## <a name="remarks"></a>Uwagi  
 Przechowuje wskaźnik do pamięci podręcznej fabryki dla **creatormap —**.
