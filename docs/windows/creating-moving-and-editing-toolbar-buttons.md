# <a name="creating-moving-and-editing-toolbar-buttons"></a>Tworzenie, przenoszenie i edytowanie przycisków paska narzędzi
Można łatwo utworzyć, przenieść, kopiowanie i edytowanie przycisków paska narzędzi.  
  
 Domyślnie nowy lub pusty przycisk jest wyświetlany na prawym końcu paska narzędzi. Ten przycisk, można przenieść przed rozpoczęciem edycji. Podczas tworzenia nowego przycisku innego pusty przycisk pojawia się po prawej stronie przycisku edytowany. Po zapisaniu paska narzędzi przycisk puste nie są zapisywane.
