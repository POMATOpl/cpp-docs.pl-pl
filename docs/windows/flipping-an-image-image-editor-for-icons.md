# <a name="flipping-an-image-image-editor-for-icons"></a>Przerzucanie obrazu (Edytor obrazów dla ikon)
Można obracać lub obraz do:  
  
-   Utwórz odbicie lustrzane oryginalnej.  
  
-   Włącz odwrócony obraz.  
  
-   Obróć obraz po prawej stronie 90 stopni w czasie.  
  
### <a name="to-flip-the-image-horizontally-mirror-image"></a>Aby przerzucić obraz w poziomie (lustrzanego)  
  
1.  Z **obrazu** menu, wybierz **Przerzuć w poziomie**.  
  
### <a name="to-flip-the-image-vertically-turn-upside-down"></a>Aby przerzucić obraz w pionie (Włącz odwrotnie)  
  
1.  Z **obrazu** menu, wybierz **Przerzuć w pionie**.  
  
### <a name="to-rotate-the-image-90-degrees"></a>Do obrazu Obrót o 90 stopni  
  
1.  Z **obrazu** menu, wybierz **Obrót o 90 stopni**.  
  
    > [!NOTE]
    >  Można również użyć [klawisze skrótów (skrót)](../windows/accelerator-keys-image-editor-for-icons.md) dla tych poleceń lub dostępu do poleceń menu skrótów (kliknij poza obrazu znajduje się w edytorze obrazów).
