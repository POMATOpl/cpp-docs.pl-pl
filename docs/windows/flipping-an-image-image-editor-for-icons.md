# <a name="flipping-an-image-image-editor-for-icons"></a>Przerzucanie obrazu (Edytor obrazów dla ikon)
Można przerzucić lub obrócić obraz:  
  
-   Utwórz obraz lustrzany oryginalny.  
  
-   Nogami obrazu.  
  
-   Obróć obraz w prawo o 90 stopni w danym momencie.  
  
### <a name="to-flip-the-image-horizontally-mirror-image"></a>Aby przerzucić obraz w poziomie (obraz)  
  
1.  Z **obraz** menu, wybierz **Przerzuć w poziomie**.  
  
### <a name="to-flip-the-image-vertically-turn-upside-down"></a>Aby przerzucić obraz w pionie (nogami)  
  
1.  Z **obraz** menu, wybierz **Przerzuć w pionie**.  
  
### <a name="to-rotate-the-image-90-degrees"></a>Aby obrócić obraz o 90 stopni  
  
1.  Z **obraz** menu, wybierz **Obrót o 90 stopni**.  
  
    > [!NOTE]
    >  Można również użyć [klawisze skrótów (skrót)](../windows/accelerator-keys-image-editor-for-icons.md) dla tych poleceń lub uzyskać dostęp do poleceń menu skrótów (kliknij poza obrazu, który znajduje się w edytorze obrazów).
