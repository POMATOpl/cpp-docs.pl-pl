# <a name="visual-studio-documentation-for-microsoft-c"></a>Dokumentacja programu Visual Studio dla platformy Microsoft C++

Witaj! To repozytorium zawiera pliki źródłowe do dokumentacji technicznej dotyczącej pracy w toku Microsoft C++ (MSVC lub Visual C++). Artykuły są publikowane na platformie [C++ w witrynie dokumentacji programu Visual Studio](https://docs.microsoft.com/cpp).

Dokumentacja dla Visual Basic i Visual C# znajduje się w oddzielnym repozytorium w [https://github.com/dotnet/core-docs](https://github.com/dotnet/core-docs)programie, a dokumentacja programu Visual Studio znajduje się w repozytorium znajdującym [https://github.com/Microsoft/visualstudio-docs](https://github.com/Microsoft/visualstudio-docs)się w lokalizacji.

## <a name="contributing-to-the-documentation"></a>Współtworzenie dokumentacji

Witamy w Twoich wkładach, aby pomóc nam w ulepszaniu MSVC dokumentów. Aby uzyskać kompleksowy przewodnik dotyczący współtworzenia docs.microsoft.com, zobacz [Omówienie przewodnika współautora Microsoft docs](https://docs.microsoft.com/contribute). Aby uzyskać szczegółowe informacje na temat sposobu udostępniania dokumentacji MSVC, zobacz [Przewodnik dotyczący](CONTRIBUTING.md)współtworzenia.

Niektóre obszary funkcji MSVC mają swoje własne foldery w tym repozytorium, na przykład w `standard-library` przypadku artykułów w standardowej bibliotece języka c++, `ide` w przypadku artykułów specyficznych dla języka C++ w programie Visual Studio Interactive Development Environment (IDE) i tak dalej. `/media` Podfolder w każdym folderze zawiera pliki sztuki dla artykułów. [Przewodnik współautora](CONTRIBUTING.md) zawiera więcej informacji.
